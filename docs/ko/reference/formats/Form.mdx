<div id="description">
  ## 설명
</div>

`Form` 포맷은 데이터가 `key1=value1&key2=value2`와 같이 구성되는 application/x-www-form-urlencoded 포맷의 단일 레코드를 읽는 데 사용할 수 있습니다.

<div id="example-usage">
  ## 사용 예시
</div>

일부 URL 인코딩된 데이터가 포함된 파일 `data.tmp`가 `user_files` 경로에 있다고 가정합니다:

```text title="data.tmp"
t_page=116&c.e=ls7xfkpm&c.tti.m=raf&rt.start=navigation&rt.bmr=390%2C11%2C10
```

```sql title="Query"
SELECT * FROM file(data.tmp, Form) FORMAT vertical;
```

```response title="Response"
Row 1:
──────
t_page:   116
c.e:      ls7xfkpm
c.tti.m:  raf
rt.start: navigation
rt.bmr:   390,11,10
```
