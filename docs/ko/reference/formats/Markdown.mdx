<div id="description">
  ## 설명
</div>

[Markdown](https://en.wikipedia.org/wiki/Markdown) 포맷을 사용하면 `.md` 파일에 바로 붙여넣을 수 있는 형태로 결과를 내보낼 수 있습니다:

Markdown 표는 자동으로 생성되며 GitHub와 같은 Markdown 지원 플랫폼에서 사용할 수 있습니다. 이 포맷은 출력에만 사용됩니다.

<div id="example-usage">
  ## 사용 예시
</div>

```sql
SELECT
    number,
    number * 2
FROM numbers(5)
FORMAT Markdown
```

```results
| number | multiply(number, 2) |
|-:|-:|
| 0 | 0 |
| 1 | 2 |
| 2 | 4 |
| 3 | 6 |
| 4 | 8 |
```
