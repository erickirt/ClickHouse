이 설정은 [system.merge&#95;tree&#95;settings](/ko/reference/system-tables/merge_tree_settings)에서 확인할 수 있으며, ClickHouse 소스에서 자동으로 생성됩니다.

<div id="min_rows_for_compact_part">
  ## min_rows_for_compact_part
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

더 이상 사용되지 않는 설정으로, 아무런 동작도 하지 않습니다.

<div id="min_rows_for_full_part_storage">
  ## min_rows_for_full_part_storage
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

ClickHouse Cloud에서만 사용할 수 있습니다. 데이터 파트에 패킹 대신 Full 스토리지 유형을 사용할 때 필요한 최소 행 수입니다.

<div id="min_rows_for_wide_part">
  ## min_rows_for_wide_part
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

데이터 파트를 `Compact` 대신 `Wide` 포맷으로 생성하는 데 필요한 최소 행 수입니다.

<div id="min_rows_to_fsync_after_merge">
  ## min_rows_to_fsync_after_merge
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

머지 후 파트에 대해 fsync를 수행하는 최소 행 수입니다(0이면 비활성화됨)
