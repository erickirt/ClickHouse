<div id="merge_workload">
  ## merge_workload
</div>

머지와
다른 워크로드 간의 리소스 사용 및 공유 방식을 제어하는 데 사용됩니다. 지정된 값은
이 테이블의 백그라운드 머지에 대한 `workload` 설정값으로 사용됩니다. 지정하지 않은 경우(빈 문자열),
대신 서버 설정 `merge_workload`가 사용됩니다.
