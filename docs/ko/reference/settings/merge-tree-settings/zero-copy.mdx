<div id="zero_copy_concurrent_part_removal_max_postpone_ratio">
  ## zero_copy_concurrent_part_removal_max_postpone_ratio
</div>

<SettingsInfoBlock type="Float" default_value="0.05" />

더 작은 독립 범위를 확보하기 위해 제거를 연기할 최상위 수준 파트의 최대 비율입니다.
변경하지 않는 것이 좋습니다.

<div id="zero_copy_concurrent_part_removal_max_split_times">
  ## zero_copy_concurrent_part_removal_max_split_times
</div>

<SettingsInfoBlock type="UInt64" default_value="5" />

독립적인 오래된 파트 범위를 더 작은 하위 범위로 분할할 때의 최대 재귀 깊이입니다.
변경하지 않는 것을 권장합니다.
