이 설정은 [system.merge&#95;tree&#95;settings](/ko/reference/system-tables/merge_tree_settings)에서 확인할 수 있으며, ClickHouse 소스에서 자동 생성됩니다.

<div id="min_delay_to_insert_ms">
  ## min_delay_to_insert_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="10" />

단일 파티션에 머지되지 않은 파트가 많을 때 MergeTree 테이블에 데이터를 삽입하는 최소 지연 시간(밀리초)입니다.

<div id="min_delay_to_mutate_ms">
  ## min_delay_to_mutate_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="10" />

완료되지 않은 뮤테이션이 많은 경우, MergeTree 테이블에서 뮤테이션을 수행하기 전의 최소 지연 시간(밀리초)입니다.
