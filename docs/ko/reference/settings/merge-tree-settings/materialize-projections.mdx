활성화되면 INSERT 시 프로젝션이 포함된 새 파트가 생성됩니다.
그렇지 않으면 명시적인 [MATERIALIZE PROJECTION](/ko/reference/statements/alter/projection#materialize-projection)을 통해 생성하거나,
[materialize&#95;projections&#95;on&#95;merge](/ko/reference/settings/merge-tree-settings/materialize-projections#materialize_projections_on_merge)를 사용하는 머지 과정에서 생성할 수 있습니다.

<div id="materialize_projections_on_merge">
  ## materialize_projections_on_merge
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.6"},{"label": "0"},{"label": "새로운 설정"}]}]} />

활성화하면 머지 시 모든 소스 파트에 없는 프로젝션(예를 들어 `materialize_projections_on_insert = 0`으로
삽입된 경우)을 다시 생성하므로, 병합된 파트에 해당 프로젝션이 포함됩니다.

머지는 여전히 동일한 프로젝션 집합을 공유하는 파트만 결합합니다. 기존의 모든 파트에 프로젝션을 백필하려면
명시적으로 [MATERIALIZE PROJECTION](/ko/reference/statements/alter/projection#materialize-projection)을 사용하십시오. 프로젝션은
[materialize&#95;projections&#95;on&#95;insert](/ko/reference/settings/merge-tree-settings/materialize-projections#materialize_projections_on_insert)를 사용하는 INSERT 중에도 생성됩니다.
