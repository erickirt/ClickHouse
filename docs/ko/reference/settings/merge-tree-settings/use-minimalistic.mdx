<div id="use_minimalistic_checksums_in_zookeeper">
  ## use_minimalistic_checksums_in_zookeeper
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

일반적인 포맷(수십 KB) 대신 ZooKeeper의 파트 checksums에 작은 포맷(수십 바이트)을 사용합니다. 활성화하기 전에 모든 레플리카가 새 포맷을 지원하는지 확인하십시오.

<div id="use_minimalistic_part_header_in_zookeeper">
  ## use_minimalistic_part_header_in_zookeeper
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

ZooKeeper에서 데이터 파트 헤더를 저장하는 방식입니다. 활성화하면 ZooKeeper에
저장되는 데이터 양이 줄어듭니다. 자세한 내용은 [여기](/ko/reference/settings/server-settings/settings#use_minimalistic_part_header_in_zookeeper)를 참조하십시오.
