Compact 파트의 JSON 공유 데이터 직렬화에서 사용하는 버킷 수입니다. `map_with_buckets` 및 `advanced` 공유 데이터 직렬화와 함께 작동합니다.
허용되는 최대값은 256입니다.

허용되는 최대값은 256입니다.

  ## object_shared_data_serialization_version

<SettingsInfoBlock type="MergeTreeObjectSharedDataSerializationVersion" default_value="advanced" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.8"},{"label": "map"},{"label": "JSON serialization version을 제어하는 설정 추가"}]}, {"id": "row-2","items": [{"label": "25.12"},{"label": "advanced"},{"label": "기본적으로 고급 공유 데이터 직렬화 버전 활성화"}]}]} />

JSON 데이터 타입 내부 공유 데이터의 직렬화 버전입니다.

Possible values:

* `map` - 공유 데이터를 `Map(String, String)`으로 저장합니다
* `map_with_buckets` - 공유 데이터를 여러 개의 개별 `Map(String, String)` 컬럼으로 저장합니다. 버킷을 사용하면 공유 데이터에서 개별 경로를 읽는 성능이 향상됩니다.
* `advanced` - 공유 데이터에서 개별 경로 읽기 성능을 크게 향상하도록 설계된 공유 데이터용 특수 serialization입니다.
  이 serialization은 많은 추가 정보를 저장하므로 디스크에서 공유 데이터의 저장 크기가 증가합니다.

`map_with_buckets` 및 `advanced` serialization의 버킷 수는 다음 설정에 따라 결정됩니다.
[object&#95;shared&#95;data&#95;buckets&#95;for&#95;compact&#95;part](/ko/reference/settings/merge-tree-settings/object-shared#object_shared_data_buckets_for_compact_part)/[object&#95;shared&#95;data&#95;buckets&#95;for&#95;wide&#95;part](/ko/reference/settings/merge-tree-settings/object-shared#object_shared_data_buckets_for_wide_part).

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.8"},{"label": "map"},{"label": "0레벨 파트의 JSON 직렬화 버전을 제어하는 설정 추가"}]}, {"id": "row-2","items": [{"label": "25.12"},{"label": "map_with_buckets"},{"label": "기본적으로 0레벨 파트에 map_with_buckets 공유 데이터 직렬화 버전 활성화"}]}]} />

이 설정을 사용하면 삽입 중 생성되는 0레벨 파트의 JSON 타입 내부 공유 데이터에 대해
서로 다른 직렬화 버전을 지정할 수 있습니다.
`advanced` 공유 데이터 직렬화는 삽입 시간을 크게 늘릴 수 있으므로 0레벨 파트에는 사용하지 않는 것이 좋습니다.
