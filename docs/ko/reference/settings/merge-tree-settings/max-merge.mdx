<div id="max_merge_selecting_sleep_ms">
  ## max_merge_selecting_sleep_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="60000" />

선택된 파트가 없을 때, 다시 머지할 파트를 선택하기 전에 대기하는 최대 시간입니다. 이 설정값이 낮을수록
background&#95;schedule&#95;pool에서 파트를 선택하는 작업이 더 자주 트리거되며,
대규모 클러스터에서는 ZooKeeper로 보내는 요청 수가 크게 늘어날 수 있습니다.
