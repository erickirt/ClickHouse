<div id="always_fetch_merged_part">
  ## always_fetch_merged_part
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

true로 설정하면 이 레플리카는 파트를 머지하지 않고, 항상 다른 레플리카에서 병합된 파트를 다운로드합니다.

가능한 값:

* true, false

<div id="always_use_copy_instead_of_hardlinks">
  ## always_use_copy_instead_of_hardlinks
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

뮤테이션, 대체, detach 등의 작업 중에는 하드링크를 사용하는 대신 항상 데이터를 복사합니다.
