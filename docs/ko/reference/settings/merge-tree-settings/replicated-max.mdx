<div id="replicated_max_mutations_in_one_entry">
  ## replicated_max_mutations_in_one_entry
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" />

함께 머지되어 하나의 MUTATE&#95;PART entry에서 실행될 수 있는 뮤테이션 명령의 최대 개수입니다.
(0은 무제한을 의미합니다)
