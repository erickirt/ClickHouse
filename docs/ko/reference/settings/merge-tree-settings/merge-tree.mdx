  ## merge_tree_clear_old_parts_interval_seconds
</div>

<SettingsInfoBlock type="UInt64" default_value="1" />

파트, WAL, 뮤테이션을 정리하는 작업을 실행하는 인터벌을 초 단위로 설정합니다.

가능한 값:

* 양의 정수.
