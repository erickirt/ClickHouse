이 섹션에는 서버 설정에 대한 설명이 포함되어 있습니다. 이러한 설정은
세션이나 쿼리 수준에서는 변경할 수 없습니다.

ClickHouse의 설정 파일에 대한 자세한 내용은 [&quot;&quot;설정 파일&quot;&quot;](/ko/concepts/features/configuration/server-config/configuration-files)을 참조하십시오.

다른 설정은 &quot;&quot;[설정](/ko/concepts/features/configuration/settings/overview)&quot;&quot; 섹션에 설명되어 있습니다.
설정을 살펴보기 전에 [설정 파일](/ko/concepts/features/configuration/server-config/configuration-files)
섹션을 먼저 읽고 치환(substitutions)의 사용 방식(`incl` 및 `optional` 속성)에 유의할 것을 권장합니다.
