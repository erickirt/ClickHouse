이러한 설정은 ClickHouse 서버를 구성하며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="encryption_header_cache_policy">
  ## encryption_header_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />암호화 헤더 캐시 정책의 이름입니다.

<div id="encryption_header_cache_size">
  ## encryption_header_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="52428800" changeable_without_restart="Yes" />

암호화된 파일에서 읽은 암호화 헤더 캐시의 최대 크기입니다. 실험적 ReaderExecutor 읽기 경로에서만 사용됩니다.

<Note>
  이 설정은 런타임에 수정할 수 있으며 즉시 적용됩니다.
</Note>

<div id="encryption_header_cache_size_ratio">
  ## encryption_header_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />암호화 헤더 캐시에서 보호 큐(SLRU 정책인 경우)가 전체 캐시 크기에서 차지하는 비율입니다.
