이 설정은 ClickHouse 서버를 구성하며, ClickHouse 소스에서 자동 생성됩니다.

<div id="global_profiler_cpu_time_period_ns">
  ## global_profiler_cpu_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />글로벌 프로파일러의 CPU 클록 타이머 주기(나노초 단위)입니다. CPU 클록 글로벌 프로파일러를 끄려면 값을 0으로 설정하십시오. 권장 값은 단일 쿼리의 경우 최소 10000000(초당 100회)이며, 클러스터 전체 프로파일링의 경우 1000000000(초당 1회)입니다.

<div id="global_profiler_real_time_period_ns">
  ## global_profiler_real_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />글로벌 프로파일러의 실시간 클록 타이머 주기입니다(나노초 단위). 값을 0으로 설정하면 실시간 클록 글로벌 프로파일러가 비활성화됩니다. 권장값은 단일 쿼리의 경우 최소 10000000(초당 100회), 클러스터 전체 프로파일링의 경우 1000000000(초당 1회)입니다.
