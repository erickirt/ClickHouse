이러한 설정은 ClickHouse 서버를 구성하며 ClickHouse 소스 코드에서 자동으로 생성됩니다.

<div id="unique_key_bitmap_cache_policy">
  ## unique_key_bitmap_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />UNIQUE KEY delete-bitmap 캐시 정책의 이름입니다(SLRU 또는 LRU).

<div id="unique_key_bitmap_cache_size_bytes">
  ## unique_key_bitmap_cache_size_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" changeable_without_restart="Yes" />UNIQUE KEY의 파트별 삭제 비트맵용 in-process 캐시의 최대 크기(바이트)입니다. 캐시를 비활성화하려면 0으로 설정하십시오.

<div id="unique_key_bitmap_cache_size_ratio">
  ## unique_key_bitmap_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />UNIQUE KEY delete-bitmap cache에서 보호 큐(SLRU 정책 사용 시)가 캐시 전체 크기에서 차지하는 비율입니다.

<div id="unique_key_index_cache_policy">
  ## unique_key_index_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />UNIQUE KEY 인덱스 캐시 정책의 이름입니다(SLRU 또는 LRU).

<div id="unique_key_index_cache_size_bytes">
  ## unique_key_index_cache_size_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" changeable_without_restart="No" />UNIQUE KEY 인덱스(SST) blocks용 in-process 캐시의 최대 크기(바이트)입니다. 캐시를 비활성화하려면 0으로 설정하십시오.

<div id="unique_key_index_cache_size_ratio">
  ## unique_key_index_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />UNIQUE KEY 인덱스 캐시에서 보호 큐(정책 사용 시)의 크기가 캐시 전체 크기에서 차지하는 비율입니다.
