이 설정은 ClickHouse 서버를 구성하며 ClickHouse 소스 코드에서 자동 생성됩니다.

<div id="custom_cached_disks_base_directory">
  ## custom_cached_disks_base_directory
</div>

이 설정은 사용자 지정(SQL로 생성된) 캐시된 디스크의 캐시 경로를 지정합니다.
`custom_cached_disks_base_directory`는 사용자 지정 디스크에서 `filesystem_caches_path`(`filesystem_caches_path.xml`에 있음)보다 우선순위가 높으며,
전자가 없을 때는 후자가 사용됩니다.
파일 시스템 캐시 설정 경로는 해당 디렉터리 내부에 있어야 하며,
그렇지 않으면 디스크가 생성되지 않도록 예외가 발생합니다.

<Note>
  이 설정은 이전 버전에서 생성된 디스크를 사용하다가 server를 업그레이드한 경우에는 영향을 주지 않습니다.
  이 경우에는 server가 정상적으로 시작될 수 있도록 예외가 발생하지 않습니다.
</Note>

예시:

```xml
<custom_cached_disks_base_directory>/var/lib/clickhouse/caches/</custom_cached_disks_base_directory>
```
