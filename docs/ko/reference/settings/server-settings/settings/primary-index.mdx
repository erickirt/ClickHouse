이 설정은 ClickHouse 서버 구성에 사용되며 ClickHouse 소스에서 자동 생성됩니다.

<div id="primary_index_cache_policy">
  ## primary_index_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />프라이머리 인덱스 캐시 정책의 이름입니다.

<div id="primary_index_cache_prewarm_ratio">
  ## primary_index_cache_prewarm_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.95" changeable_without_restart="No" />예열 시 채울 마크 캐시 전체 크기의 비율입니다.

<div id="primary_index_cache_size">
  ## primary_index_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="5368709120" changeable_without_restart="Yes" />프라이머리 인덱스(MergeTree 엔진 계열 테이블의 인덱스)용 캐시의 최대 크기입니다.

<div id="primary_index_cache_size_ratio">
  ## primary_index_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />프라이머리 인덱스 캐시(primary index cache)에서 보호 큐의 크기가 전체 캐시 크기에서 차지하는 비율입니다(SLRU 정책을 사용하는 경우).
