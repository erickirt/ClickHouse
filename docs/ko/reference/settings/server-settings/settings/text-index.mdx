이 설정은 ClickHouse 서버 구성을 위한 것이며 ClickHouse 소스에서 자동 생성됩니다.

<div id="text_index_header_cache_max_entries">
  ## text_index_header_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="100000" changeable_without_restart="No" />text index header 캐시의 크기(엔트리 수)입니다. 0이면 비활성화됩니다.

<div id="text_index_header_cache_policy">
  ## text_index_header_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />텍스트 인덱스 헤더 캐시 정책의 이름입니다.

<div id="text_index_header_cache_size">
  ## text_index_header_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" changeable_without_restart="Yes" />text index 헤더의 캐시 크기입니다. 0은 비활성화를 의미합니다.

<Note>
  이 설정은 런타임에 변경할 수 있으며 즉시 적용됩니다.
</Note>

<div id="text_index_header_cache_size_ratio">
  ## text_index_header_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />텍스트 인덱스 헤더 캐시에서 보호 큐(SLRU 정책 사용 시)가 전체 캐시 크기에서 차지하는 비율입니다.

<div id="text_index_postings_cache_max_entries">
  ## text_index_postings_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000" changeable_without_restart="No" />text index 포스팅 리스트 캐시의 크기입니다. 단위는 항목 수이며, 0이면 비활성화됩니다.

<div id="text_index_postings_cache_policy">
  ## text_index_postings_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />텍스트 인덱스 포스팅 리스트 캐시 정책의 이름입니다.

<SettingsInfoBlock type="UInt64" default_value="2147483648" changeable_without_restart="Yes" />text index 포스팅 리스트 캐시의 크기입니다. 0은 비활성화를 의미합니다.

<Note>
