이 설정은 ClickHouse 서버 구성에 사용되며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="mlock_executable">
  ## mlock_executable
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />

시작 후 `<mlockall>`을 수행하여 첫 쿼리의 지연 시간을 줄이고, 높은 IO 부하 시 ClickHouse 실행 파일이 메모리에서 페이지 아웃되지 않도록 합니다.

<Note>
  이 옵션을 활성화하는 것을 권장하지만, 시작 시간이 몇 초까지 늘어날 수 있습니다. 또한 이 설정은 &quot;CAP&#95;IPC&#95;LOCK&quot; capability가 없으면 작동하지 않습니다.
</Note>

**예시**

```xml
<mlock_executable>false</mlock_executable>
```
