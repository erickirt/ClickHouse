이 설정은 ClickHouse 서버를 구성하는 데 사용되며, ClickHouse 소스에서 자동으로 생성됩니다.

<div id="merge_tree">
  ## merge_tree
</div>

[MergeTree](/ko/reference/engines/table-engines/mergetree-family/mergetree) 테이블의 세부 설정입니다.

자세한 내용은 MergeTreeSettings.h 헤더 파일을 참조하십시오.

**예시**

```xml
<merge_tree>
    <max_suspicious_broken_parts>5</max_suspicious_broken_parts>
</merge_tree>
```

<div id="merge_workload">
  ## merge_workload
</div>

<SettingsInfoBlock type="String" default_value="default" changeable_without_restart="Yes" />

머지와 다른 워크로드 사이에서 리소스를 사용하고 공유하는 방식을 조정하는 데 사용됩니다. 지정한 값은 모든 백그라운드 머지의 `workload` 설정 값으로 사용됩니다. 이 값은 MergeTree 설정으로 재정의할 수 있습니다.
