이 설정은 ClickHouse 서버 구성에 사용되며 ClickHouse 소스에서 자동 생성됩니다.

<div id="uncompressed_cache_policy">
  ## uncompressed_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />비압축 캐시 정책 이름입니다.

<div id="uncompressed_cache_size">
  ## uncompressed_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

MergeTree 엔진 계열의 테이블 엔진에서 사용하는 비압축 데이터의 최대 크기(바이트)를 지정합니다.

서버에는 하나의 공유 캐시가 있습니다. 메모리는 필요할 때 할당됩니다. `use_uncompressed_cache` 옵션이 활성화되어 있으면 이 캐시가 사용됩니다.

비압축 캐시는 특정 상황에서 실행 시간이 매우 짧은 쿼리에 유리합니다.

<Note>
  값이 `0`이면 비활성화됨을 의미합니다.

  이 설정은 런타임에 수정할 수 있으며 즉시 적용됩니다.
</Note>

<div id="uncompressed_cache_size_ratio">
  ## uncompressed_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />비압축 캐시에서 보호 큐(정책이 SLRU인 경우)가 차지하는 크기의 비율로, 캐시 전체 크기를 기준으로 합니다.
