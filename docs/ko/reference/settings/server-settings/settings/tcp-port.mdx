이 설정은 ClickHouse 서버 구성에 사용되며 ClickHouse 소스에서 자동 생성됩니다.

<div id="tcp_port">
  ## tcp_port
</div>

TCP protocol을 통해 클라이언트와 통신하기 위한 포트입니다.

**예시**

```xml
<tcp_port>9000</tcp_port>
```

<div id="tcp_port_secure">
  ## tcp_port_secure
</div>

클라이언트와 보안 통신을 수행하는 TCP 포트입니다. [OpenSSL](/ko/reference/settings/server-settings/settings/other#openssl) 설정과 함께 사용합니다.

**기본값**

```xml
<tcp_port_secure>9440</tcp_port_secure>
```
