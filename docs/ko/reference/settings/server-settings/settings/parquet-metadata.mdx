이 설정은 ClickHouse 서버를 구성하며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="parquet_metadata_cache_max_entries">
  ## parquet_metadata_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="5000" changeable_without_restart="No" />Parquet 메타데이터 파일 캐시의 최대 항목 수입니다. 0은 비활성화를 의미합니다.

<div id="parquet_metadata_cache_policy">
  ## parquet_metadata_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Parquet 메타데이터 캐시 정책 이름입니다.

<div id="parquet_metadata_cache_size">
  ## parquet_metadata_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="536870912" changeable_without_restart="Yes" />Parquet 메타데이터 캐시의 최대 크기(바이트)입니다. 0은 비활성화를 의미합니다.

<div id="parquet_metadata_cache_size_ratio">
  ## parquet_metadata_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />Parquet 메타데이터 캐시에서 보호 큐(SLRU 정책 사용 시)가 전체 캐시 크기에서 차지하는 비율입니다.
