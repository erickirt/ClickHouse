이 설정은 ClickHouse 서버 구성에 사용되며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="max_prefixes_deserialization_thread_pool_free_size">
  ## max_prefixes_deserialization_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

prefixes 역직렬화용 스레드 풀의 **유휴** 스레드 수가 `max_prefixes_deserialization_thread_pool_free_size`를 초과하면, ClickHouse는 유휴 스레드가 점유한 리소스를 해제하고 풀 크기를 줄입니다. 필요하면 스레드를 다시 생성할 수 있습니다.

<div id="max_prefixes_deserialization_thread_pool_size">
  ## max_prefixes_deserialization_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="100" changeable_without_restart="Yes" />

ClickHouse는 MergeTree의 Wide 파트에서 파일 프리픽스의 컬럼 및 서브컬럼 메타데이터를 병렬로 읽기 위해 prefixes 역직렬화용 스레드 풀의 스레드를 사용합니다. `max_prefixes_deserialization_thread_pool_size`는 이 풀의 최대 스레드 수를 제한합니다.
