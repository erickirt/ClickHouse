이 설정은 ClickHouse 서버를 구성하며, ClickHouse 소스에서 자동으로 생성됩니다.

<div id="logger.console">
  ## logger.console
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />콘솔 로깅을 활성화합니다. 활성화하려면 `<1>` 또는 `<true>`로 설정하십시오. ClickHouse가 데몬 모드로 실행되지 않는 경우 기본값은 `<1>`이고, 그 외에는 `<0>`입니다.

<div id="logger.console_log_level">
  ## logger.console_log_level
</div>

<SettingsInfoBlock type="String" default_value="trace" changeable_without_restart="No" />콘솔 출력용 로그 레벨입니다. 기본값은 `<level>`입니다.
