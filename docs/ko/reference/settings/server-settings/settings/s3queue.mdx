이 설정은 ClickHouse 서버를 구성하는 데 사용되며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="s3queue_disable_streaming">
  ## s3queue_disable_streaming
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Yes" />테이블이 생성되어 있고 attached 상태인 구체화된 뷰(Materialized View)가 있더라도 S3Queue에서 streaming을 비활성화합니다

<div id="s3queue_log">
  ## s3queue_log
</div>

`s3queue_log` system table의 설정입니다.

<SystemLogParameters />

기본 설정은 다음과 같습니다.

```xml
<s3queue_log>
    <database>system</database>
    <table>s3queue_log</table>
    <partition_by>toYYYYMM(event_date)</partition_by>
    <flush_interval_milliseconds>7500</flush_interval_milliseconds>
</s3queue_log>
```
