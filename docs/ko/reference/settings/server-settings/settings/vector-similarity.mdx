이 설정은 ClickHouse 서버를 구성하며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="vector_similarity_index_cache_max_entries">
  ## vector_similarity_index_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000" changeable_without_restart="No" />벡터 유사성 인덱스 캐시의 크기를 항목 수로 지정합니다. 0이면 비활성화됩니다.

<div id="vector_similarity_index_cache_policy">
  ## vector_similarity_index_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />벡터 유사성 인덱스 캐시 정책의 이름입니다.

<div id="vector_similarity_index_cache_size">
  ## vector_similarity_index_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="5368709120" changeable_without_restart="Yes" />벡터 유사성 인덱스를 위한 캐시 크기입니다. 0은 비활성화를 의미합니다.

<Note>
  이 설정은 런타임에 변경할 수 있으며 즉시 적용됩니다.
</Note>

<div id="vector_similarity_index_cache_size_ratio">
  ## vector_similarity_index_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />벡터 유사성 인덱스 캐시에서 보호 큐(SLRU policy를 사용하는 경우)의 크기가 캐시 전체 크기에서 차지하는 비율입니다.
