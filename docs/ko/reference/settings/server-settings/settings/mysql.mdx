이 설정은 ClickHouse 서버 구성에 사용되며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="mysql_port">
  ## mysql_port
</div>

MySQL protocol을 통해 클라이언트와 통신하는 데 사용하는 포트입니다.

<Note>
  * 양의 정수는 수신 대기할 포트 번호를 지정합니다.
  * 빈 값은 MySQL protocol을 통한 클라이언트 통신을 비활성화하는 데 사용됩니다.
</Note>

**예시**

```xml
<mysql_port>9004</mysql_port>
```

<div id="mysql_require_secure_transport">
  ## mysql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />`true`로 설정하면 [mysql&#95;port](/ko/reference/settings/server-settings/settings/mysql#mysql_port)를 통해 클라이언트와 보안 통신해야 합니다. `<--ssl-mode=none>` 옵션을 사용하는 connection은 거부됩니다. [OpenSSL](/ko/reference/settings/server-settings/settings/other#openssl) 설정과 함께 사용하십시오.
