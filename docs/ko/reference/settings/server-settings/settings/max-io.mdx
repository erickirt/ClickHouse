이 설정은 ClickHouse 서버를 구성하며 ClickHouse 소스에서 자동 생성됩니다.

<div id="max_io_thread_pool_free_size">
  ## max_io_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

IO 스레드 풀의 **idle** 스레드 수가 `max_io_thread_pool_free_size`를 초과하면 ClickHouse는 유휴 상태인 스레드가 사용하던 리소스를 해제하고 풀 크기를 줄입니다. 필요하면 스레드를 다시 생성할 수 있습니다.

<div id="max_io_thread_pool_size">
  ## max_io_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="100" changeable_without_restart="Yes" />

ClickHouse는 일부 IO 작업(예: S3와 상호작용할 때)에 IO 스레드 풀의 스레드를 사용합니다. `max_io_thread_pool_size`는 이 풀의 최대 스레드 수를 제한합니다.
