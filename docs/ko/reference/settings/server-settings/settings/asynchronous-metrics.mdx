이 설정은 ClickHouse 서버를 구성하기 위한 것이며 ClickHouse 소스에서 자동 생성됩니다.

<div id="asynchronous_metrics_enable_heavy_metrics">
  ## asynchronous_metrics_enable_heavy_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />부하가 큰 비동기 메트릭의 계산을 활성화합니다.

<div id="asynchronous_metrics_keeper_metrics_only">
  ## asynchronous_metrics_keeper_metrics_only
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />비동기 메트릭이 Keeper 관련 메트릭만 계산하게 합니다.

<div id="asynchronous_metrics_update_period_s">
  ## asynchronous_metrics_update_period_s
</div>

<SettingsInfoBlock type="UInt32" default_value="1" changeable_without_restart="No" />비동기 메트릭 업데이트 주기(초)입니다.
