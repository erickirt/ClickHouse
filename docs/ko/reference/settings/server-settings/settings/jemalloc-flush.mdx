이 설정은 ClickHouse 서버를 구성하며, ClickHouse 소스에서 자동으로 생성됩니다.

<div id="jemalloc_flush_profile_interval_bytes">
  ## jemalloc_flush_profile_interval_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />전역 피크 메모리 사용량이 jemalloc&#95;flush&#95;profile&#95;interval&#95;bytes만큼 증가한 후 jemalloc 프로파일이 플러시됩니다

<div id="jemalloc_flush_profile_on_memory_exceeded">
  ## jemalloc_flush_profile_on_memory_exceeded
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />총 메모리 한도 초과 오류가 발생하면 jemalloc 프로파일이 플러시됩니다

<div id="jemalloc_flush_profile_on_memory_exceeded_interval">
  ## jemalloc_flush_profile_on_memory_exceeded_interval
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />0이 아니면 총 메모리 한도 초과 오류 발생 시 jemalloc 프로파일을 플러시하는 최소 간격(초)을 설정합니다. 예를 들어 5이면 5초마다 profile 플러시는 최대 1번만 수행됩니다. 이 설정은 `jemalloc_flush_profile_on_memory_exceeded`보다 우선합니다.
