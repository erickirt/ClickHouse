이 설정은 ClickHouse 서버를 구성하는 항목이며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="keeper_hosts">
  ## keeper_hosts
</div>

동적 설정입니다. ClickHouse가 연결할 수 있는 [Zoo]Keeper 호스트 집합이 포함됩니다. `<auxiliary_zookeepers>`의 정보는 노출되지 않습니다.

<div id="keeper_multiread_batch_size">
  ## keeper_multiread_batch_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />

배칭을 지원하는 [Zoo]Keeper에 보내는 MultiRead request 배치의 최대 크기입니다. 0으로 설정하면 배칭이 비활성화됩니다. ClickHouse Cloud에서만 사용할 수 있습니다.
