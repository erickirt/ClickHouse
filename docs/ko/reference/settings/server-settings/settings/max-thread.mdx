이 설정은 ClickHouse 서버를 구성하며 ClickHouse 소스 코드로부터 자동 생성됩니다.

<div id="max_thread_pool_free_size">
  ## max_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />

Global Thread 풀의 **대기 중인** 스레드 수가 [`max_thread_pool_free_size`](/ko/reference/settings/server-settings/settings/max-thread#max_thread_pool_free_size)보다 많으면, ClickHouse는 일부 스레드가 점유하고 있는 리소스를 해제하여 풀 크기를 줄입니다. 필요하면 스레드를 다시 생성할 수 있습니다.

**예시**

```xml
<max_thread_pool_free_size>1200</max_thread_pool_free_size>
```

<div id="max_thread_pool_size">
  ## max_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />

ClickHouse는 쿼리를 처리할 때 Global Thread 풀의 스레드를 사용합니다. 쿼리를 처리할 idle 스레드가 없으면 풀에 새 스레드가 생성됩니다. `max_thread_pool_size`는 풀에서 사용할 수 있는 스레드의 최대 개수를 제한합니다.

**예시**

```xml
<max_thread_pool_size>12000</max_thread_pool_size>
```
