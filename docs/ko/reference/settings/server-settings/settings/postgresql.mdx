이 설정은 ClickHouse 서버 구성을 위한 것이며 ClickHouse 소스에서 자동 생성됩니다.

<div id="postgresql_port">
  ## postgresql_port
</div>

PostgreSQL protocol을 통해 클라이언트와 통신하는 데 사용하는 포트입니다.

<Note>
  * 양의 정수는 수신 대기할 포트 번호를 지정합니다.
  * 빈 값은 PostgreSQL protocol을 통한 클라이언트와의 통신을 비활성화하는 데 사용됩니다.
</Note>

**예시**

```xml
<postgresql_port>9005</postgresql_port>
```

<div id="postgresql_require_secure_transport">
  ## postgresql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />true로 설정하면 [postgresql&#95;port](/ko/reference/settings/server-settings/settings/postgresql#postgresql_port)를 통해 클라이언트와의 보안 통신이 필요합니다. `<sslmode=disable>` 옵션을 사용하는 연결은 거부됩니다. [OpenSSL](/ko/reference/settings/server-settings/settings/other#openssl) 설정과 함께 사용하십시오.
