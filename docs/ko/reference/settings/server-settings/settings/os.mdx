<div id="os_collect_psi_metrics">
  ## os_collect_psi_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />/proc/pressure/ 파일의 PSI 메트릭 수집을 활성화합니다.

<div id="os_cpu_busy_time_threshold">
  ## os_cpu_busy_time_threshold
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000" changeable_without_restart="No" />CPU가 실제로 유용한 작업을 수행하고 있다고 판단하는 기준이 되는 OS CPU busy time의 임계값(마이크로초 단위)입니다(OSCPUVirtualTimeMicroseconds 메트릭). busy time이 이 값보다 낮으면 CPU 과부하는 없는 것으로 간주됩니다.
