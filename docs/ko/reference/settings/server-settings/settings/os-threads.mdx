이 설정은 ClickHouse 서버 구성을 위한 것이며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="os_threads_nice_value_distributed_cache_tcp_handler">
  ## os_threads_nice_value_distributed_cache_tcp_handler
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="No" />

Distributed Cache TCP handler 스레드의 Linux nice value입니다. 값이 낮을수록 CPU 우선순위가 높습니다.

CAP&#95;SYS&#95;NICE capability가 필요하며, 그렇지 않으면 no-op입니다.

Possible values: -20~19.

<div id="os_threads_nice_value_merge_mutate">
  ## os_threads_nice_value_merge_mutate
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="No" />

머지 및 mutation 스레드의 Linux nice value입니다. 값이 낮을수록 CPU 우선순위가 높아집니다.

CAP&#95;SYS&#95;NICE capability가 필요하며, 없으면 아무 작업도 수행되지 않습니다.

가능한 값: -20~19.

<div id="os_threads_nice_value_zookeeper_client_send_receive">
  ## os_threads_nice_value_zookeeper_client_send_receive
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="No" />

ZooKeeper 클라이언트의 송신 및 수신 스레드에 대한 Linux nice value입니다. 값이 낮을수록 CPU 우선순위가 높습니다.

CAP&#95;SYS&#95;NICE capability가 필요하며, 그렇지 않으면 no-op입니다.

Possible values: -20 to 19.
