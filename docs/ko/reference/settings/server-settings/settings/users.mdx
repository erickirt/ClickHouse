이 설정은 ClickHouse 서버를 구성하는 항목이며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="users_config">
  ## users_config
</div>

다음 내용이 포함된 파일 경로:

* 사용자 구성 정보
* 접근 권한
* 설정 프로필
* QUOTA 설정

**예시**

```xml
<users_config>users.xml</users_config>
```

<div id="users_to_ignore_early_memory_limit_check">
  ## users_to_ignore_early_memory_limit_check
</div>

초기 메모리 한도 검사(early memory limit check)에서 제외할 사용자의 쉼표로 구분된 목록입니다. 사용자가 이 목록에 없으면 전체 메모리 사용량(memory usage)이 한도를 초과할 경우 쿼리가 거부됩니다.
