이 설정은 ClickHouse 서버 구성을 위한 것으로, ClickHouse 소스에서 자동 생성됩니다.

<div id="index_uncompressed_cache_policy">
  ## index_uncompressed_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />보조 인덱스 비압축 캐시 정책의 이름입니다.

<div id="index_uncompressed_cache_size">
  ## index_uncompressed_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

`MergeTree` 인덱스의 비압축 블록 캐시 최대 크기입니다.

<Note>
  값이 `0`이면 비활성화된 상태를 의미합니다.

  이 설정은 런타임에 변경할 수 있으며 즉시 적용됩니다.
</Note>

<div id="index_uncompressed_cache_size_ratio">
  ## index_uncompressed_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />보조 인덱스 비압축 캐시에서 보호 큐(SLRU 정책 사용 시)가 캐시 전체 크기에서 차지하는 비율입니다.
