이 설정은 ClickHouse 서버 구성에 사용되며 ClickHouse 소스에서 자동 생성됩니다.

<div id="enable_azure_sdk_logging">
  ## enable_azure_sdk_logging
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Azure sdk의 로깅을 활성화합니다.

<div id="enable_webterminal">
  ## enable_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />`/webterminal` HTTP 엔드포인트에서 웹 터미널 인터페이스를 활성화합니다. WebSocket을 통해 브라우저에서 대화형 `clickhouse-client` 세션을 제공합니다. `false`로 설정하면 `/webterminal` 요청에 대해 HTTP 상태 `403 Forbidden`을 반환합니다.
