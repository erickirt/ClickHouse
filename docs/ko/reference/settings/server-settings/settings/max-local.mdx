이 설정은 ClickHouse 서버를 구성하기 위한 것이며 ClickHouse 소스에서 자동 생성됩니다.

<div id="max_local_read_bandwidth_for_server">
  ## max_local_read_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

로컬 읽기의 초당 최대 속도(바이트)입니다.

<Note>
  값이 `0`이면 무제한을 의미합니다.
</Note>

<div id="max_local_write_bandwidth_for_server">
  ## max_local_write_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

로컬 쓰기의 최대 속도입니다. 단위는 초당 바이트입니다.

<Note>
  값이 `0`이면 무제한입니다.
</Note>
