이 설정은 ClickHouse 서버 구성에 사용되며 ClickHouse 소스에서 자동 생성됩니다.

<div id="max_remote_read_connections">
  ## max_remote_read_connections
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />순차 읽기 최적화를 위해 `ReaderExecutor`가 유지하는 열린 원격 읽기 연결의 최대 개수입니다. 0으로 설정하면 연결 재사용이 비활성화됩니다.

<div id="max_remote_read_network_bandwidth_for_server">
  ## max_remote_read_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

읽기 작업의 네트워크 데이터 전송 속도 최대값으로, 단위는 초당 바이트 수입니다.

<Note>
  값이 `0`(기본값)이면 제한이 없음을 의미합니다.
</Note>

<div id="max_remote_write_network_bandwidth_for_server">
  ## max_remote_write_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

쓰기 시 네트워크를 통한 데이터 교환의 최대 속도를 초당 바이트 수로 지정합니다.

<Note>
  값이 `0`(기본값)이면 무제한을 의미합니다.
</Note>
