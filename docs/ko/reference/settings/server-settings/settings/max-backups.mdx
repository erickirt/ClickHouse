이 설정은 ClickHouse 서버를 구성하는 데 사용되며, ClickHouse 소스에서 자동 생성됩니다.

<div id="max_backups_io_thread_pool_free_size">
  ## max_backups_io_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="예" />Backups IO Thread pool의 **유휴** 스레드 수가 `max_backup_io_thread_pool_free_size`를 초과하면 ClickHouse는 유휴 상태의 스레드가 점유한 리소스를 해제하고 풀 크기를 줄입니다. 필요하면 스레드를 다시 생성할 수 있습니다.

<div id="max_backups_io_thread_pool_size">
  ## max_backups_io_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="Yes" />ClickHouse는 S3 백업 IO 작업을 수행할 때 Backups IO Thread pool의 스레드를 사용합니다. `max_backups_io_thread_pool_size`는 이 풀에서 사용할 수 있는 최대 스레드 수를 제한합니다.
