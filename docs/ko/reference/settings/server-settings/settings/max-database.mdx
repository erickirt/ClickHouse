이 설정은 ClickHouse 서버 구성을 위한 항목이며 ClickHouse 소스에서 자동 생성됩니다.

<div id="max_database_num_to_throw">
  ## max_database_num_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />데이터베이스 수가 이 값보다 크면 서버에서 예외를 발생시킵니다. 0은 제한이 없음을 의미합니다.

<div id="max_database_num_to_warn">
  ## max_database_num_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="Yes" />

attached 상태인 데이터베이스의 수가 지정된 값을 초과하면, ClickHouse 서버는 `system.warnings` 테이블에 경고 메시지를 추가합니다.

**예시**

```xml
<max_database_num_to_warn>50</max_database_num_to_warn>
```

<div id="max_database_replicated_create_table_thread_pool_size">
  ## max_database_replicated_create_table_thread_pool_size
</div>

<SettingsInfoBlock type="UInt32" default_value="1" changeable_without_restart="No" />DatabaseReplicated에서 레플리카 복구 중 테이블 생성에 사용하는 스레드 수입니다. 0은 스레드 수를 코어 수와 같게 설정함을 의미합니다.
