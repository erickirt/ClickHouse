이 설정은 ClickHouse 서버를 구성하기 위한 것이며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="logger.async">
  ## logger.async
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />`<true>`(기본값)인 경우 로깅은 비동기로 수행됩니다(출력 채널당 하나의 백그라운드 스레드 사용). 그렇지 않으면 LOG를 호출한 스레드 내부에서 로깅이 수행됩니다.

<div id="logger.async_queye_max_size">
  ## logger.async_queye_max_size
</div>

<SettingsInfoBlock type="UInt64" default_value="65536" changeable_without_restart="No" />async 로깅을 사용할 때 플러시를 기다리며 큐에 보관되는 최대 메시지 수입니다. 이를 초과하는 메시지는 버려집니다.
