이 설정은 ClickHouse 서버를 구성하며 ClickHouse 소스에서 자동 생성됩니다.

<div id="backups">
  ## 백업
</div>

[`BACKUP` 및 `RESTORE`](/ko/concepts/features/backup-restore/overview) SQL 문을 실행할 때 사용하는 백업 설정입니다.

다음 설정은 하위 태그를 사용해 구성할 수 있습니다:

<div id="backups_io_thread_pool_queue_size">
  ## backups_io_thread_pool_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Backups IO Thread pool에 스케줄할 수 있는 최대 작업 수입니다. 현재 S3 백업 로직상 이 큐는 무제한으로 유지하는 것이 좋습니다.

<Note>
  `0` 값(기본값)은 무제한을 의미합니다.
</Note>
