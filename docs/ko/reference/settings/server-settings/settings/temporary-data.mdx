이 설정은 ClickHouse 서버를 구성하는 데 사용되며 ClickHouse 소스에서 자동 생성됩니다.

<div id="temporary_data_in_cache">
  ## temporary_data_in_cache
</div>

이 옵션을 사용하면 임시 데이터가 해당 디스크의 캐시에 저장됩니다.
이 섹션에서는 유형이 `cache`인 디스크 이름을 지정해야 합니다.
이 경우 캐시와 임시 데이터가 동일한 공간을 공유하며, 임시 데이터를 생성하기 위해 디스크 캐시 항목이 제거될 수 있습니다.

<Note>
  임시 데이터 저장소를 구성하는 옵션으로는 `tmp_path`, `tmp_policy`, `temporary_data_in_cache` 중 하나만 사용할 수 있습니다.
</Note>

**예시**

`local_disk`의 캐시와 임시 데이터는 모두 `tiny_local_cache`가 관리하는 파일 시스템의 `/tiny_local_cache`에 저장됩니다.

```xml highlight={9-16,20}
<clickhouse>
<storage_configuration>
<disks>
<local_disk>
<type>local</type>
<path>/local_disk/</path>
</local_disk>

<tiny_local_cache>
<type>cache</type>
<disk>local_disk</disk>
<path>/tiny_local_cache/</path>
<max_size_rows>10M</max_size_rows>
<max_file_segment_size>1M</max_file_segment_size>
<cache_on_write_operations>1</cache_on_write_operations>
</tiny_local_cache>
</disks>
</storage_configuration>

<temporary_data_in_cache>tiny_local_cache</temporary_data_in_cache>
</clickhouse>
```
