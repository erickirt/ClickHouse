이 설정은 ClickHouse 서버 구성에 사용되며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="index_mark_cache_policy">
  ## index_mark_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />보조 인덱스 마크 캐시 정책의 이름입니다.

<div id="index_mark_cache_prewarm_ratio">
  ## index_mark_cache_prewarm_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.95" changeable_without_restart="No" />예열 중 인덱스 마크 캐시를 채울 때 사용할 전체 크기 대비 비율입니다.

<div id="index_mark_cache_size">
  ## index_mark_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="5368709120" changeable_without_restart="Yes" />

인덱스 마크 캐시의 최대 크기입니다.

<Note>
  값이 `0`이면 비활성화됩니다.

  이 설정은 런타임에 변경할 수 있으며 즉시 적용됩니다.
</Note>

<div id="index_mark_cache_size_ratio">
  ## index_mark_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.3" changeable_without_restart="No" />보조 인덱스 마크 캐시에서 보호 큐(SLRU 정책 사용 시)가 캐시 전체 크기에서 차지하는 비율입니다.
