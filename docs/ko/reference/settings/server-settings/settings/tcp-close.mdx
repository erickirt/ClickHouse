이 설정은 ClickHouse 서버를 구성하며, ClickHouse 소스에서 자동으로 생성됩니다.

<div id="tcp_close_connection_after_queries_num">
  ## tcp_close_connection_after_queries_num
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />연결이 종료되기 전에 TCP 연결당 허용되는 최대 쿼리 수입니다. 쿼리 수를 무제한으로 하려면 0으로 설정합니다.

<div id="tcp_close_connection_after_queries_seconds">
  ## tcp_close_connection_after_queries_seconds
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />닫히기 전까지 TCP 연결이 유지될 수 있는 최대 시간(초)입니다. 연결 수명을 무제한으로 하려면 0으로 설정하십시오.
