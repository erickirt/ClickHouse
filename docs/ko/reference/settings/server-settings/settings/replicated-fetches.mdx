이 설정은 ClickHouse 서버를 구성하며 ClickHouse 소스에서 자동으로 생성됩니다.

<div id="replicated_fetches_http_connection_timeout">
  ## replicated_fetches_http_connection_timeout
</div>

<SettingsInfoBlock type="Seconds" default_value="0" changeable_without_restart="No" />part fetch 요청의 HTTP 연결 타임아웃입니다. 명시적으로 설정하지 않은 경우 기본 프로필 `http_connection_timeout`에서 상속됩니다.

<div id="replicated_fetches_http_receive_timeout">
  ## replicated_fetches_http_receive_timeout
</div>

<SettingsInfoBlock type="Seconds" default_value="0" changeable_without_restart="No" />part를 가져오는 요청에 대한 HTTP 수신 타임아웃입니다. 명시적으로 설정하지 않으면 기본 프로필 `http_receive_timeout`에서 상속됩니다.

<div id="replicated_fetches_http_send_timeout">
  ## replicated_fetches_http_send_timeout
</div>

<SettingsInfoBlock type="Seconds" default_value="0" changeable_without_restart="No" />part fetch 요청의 HTTP 전송 타임아웃입니다. 명시적으로 설정하지 않으면 기본 프로필 `http_send_timeout`에서 상속됩니다.
