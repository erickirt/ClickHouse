<div id="log_query_threads">
  ## log_query_threads
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

쿼리 스레드 로깅을 설정합니다.

쿼리 스레드의 로그는 [system.query&#95;thread&#95;log](/ko/reference/system-tables/query_thread_log) 테이블에 기록됩니다. 이 설정은 [log&#95;queries](/ko/reference/settings/session-settings/log-queries#log_queries)가 true일 때만 적용됩니다. 이 설정이 구성된 경우, ClickHouse에서 실행되는 쿼리의 스레드는 [query&#95;thread&#95;log](/ko/reference/settings/server-settings/settings#query_thread_log) 서버 구성 매개변수의 규칙에 따라 기록됩니다.

가능한 값:

* 0 — 비활성화됨.
* 1 — 활성화됨.

**예시**

```text
log_query_threads=1
```

<div id="log_query_views">
  ## log_query_views
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

쿼리 view 로깅을 설정합니다.

이 설정이 활성화된 상태에서 ClickHouse가 실행한 쿼리에 관련된 view(구체화된 view 또는 라이브 view)가 있으면, 해당 view는 서버 구성 매개변수인 [query&#95;views&#95;log](/ko/reference/settings/server-settings/settings#query_views_log)에 기록됩니다.

예시:

```text
log_query_views=1
```
