<div id="cloud_mode_engine">
  ## cloud_mode_engine
</div>

<SettingsInfoBlock type="UInt64" default_value="1" />

Cloud에서 허용되는 엔진 계열입니다.

* 0 - 모든 것을 허용합니다
* 1 - DDLs를 *ReplicatedMergeTree를 사용하도록 재작성합니다
* 2 - DDLs를 SharedMergeTree를 사용하도록 재작성합니다
* 3 - 원격 디스크를 명시적으로 지정한 경우를 제외하고 DDLs를 SharedMergeTree를 사용하도록 재작성합니다
* 4 - 3과 동일하며, 추가로 Distributed 대신 Alias를 사용합니다(Alias 테이블은 Distributed 테이블의 대상 테이블을 가리키므로 해당 로컬 테이블을 사용합니다)

public part를 최소화하기 위해 UInt64를 사용합니다.

Cloud 기본값: `2`.
