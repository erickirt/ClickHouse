<div id="stream_flush_interval_ms">
  ## stream_flush_interval_ms
</div>

<SettingsInfoBlock type="밀리초" default_value="7500" />

이 설정은 스트리밍을 사용하는 테이블에서 타임아웃이 발생하거나 스레드가 [max&#95;insert&#95;block&#95;size](/ko/reference/settings/session-settings/max-insert#max_insert_block_size)개의 행을 생성할 때 적용됩니다.

기본값은 7500입니다.

값이 작을수록 데이터가 테이블에 더 자주 플러시됩니다. 값을 너무 낮게 설정하면 성능이 저하됩니다.

<div id="stream_poll_timeout_ms">
  ## stream_poll_timeout_ms
</div>

<SettingsInfoBlock type="Milliseconds" default_value="500" />

스트리밍 스토리지에서/로 데이터를 폴링할 때의 timeout입니다.
