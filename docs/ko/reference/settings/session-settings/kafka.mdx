<div id="kafka_disable_num_consumers_limit">
  ## kafka_disable_num_consumers_limit
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

사용 가능한 CPU 코어 수에 따라 결정되는 `kafka_num_consumers` 제한을 비활성화합니다.

<div id="kafka_max_wait_ms">
  ## kafka_max_wait_ms
</div>

<SettingsInfoBlock type="밀리초" default_value="5000" />

재시도하기 전에 [Kafka](/ko/reference/engines/table-engines/integrations/kafka)에서 메시지를 읽을 때 대기하는 시간(밀리초)입니다.

가능한 값:

* 양의 정수입니다.
* 0 — 무한 대기 시간입니다.

관련 항목:

* [Apache Kafka](https://kafka.apache.org/)
