BACKUP ON CLUSTER 또는 RESTORE ON CLUSTER 작업 중 호스트가 이 시간 동안 ZooKeeper에서 자신의 임시 &#39;alive&#39; 노드를 다시 생성하지 못하면 전체 백업 또는 복원 작업이 실패한 것으로 간주됩니다.
이 값은 장애 발생 후 호스트가 ZooKeeper에 다시 연결하는 데 걸릴 수 있는 합리적인 시간보다 커야 합니다.
0은 무제한을 의미합니다.
