  ## cluster_function_process_archive_on_multiple_nodes

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.7"},{"label": "1"},{"label": "새로운 설정"}]}]} />

`true`로 설정하면 cluster function에서 아카이브를 처리할 때 성능이 향상됩니다. 이전 버전에서 아카이브와 함께 cluster function을 사용하는 경우, 25.7+로 업그레이드하는 동안 compatibility를 유지하고 오류를 방지하려면 `false`로 설정해야 합니다.
