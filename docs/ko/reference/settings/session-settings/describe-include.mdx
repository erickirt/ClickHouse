<div id="describe_include_subcolumns">
  ## describe_include_subcolumns
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

[DESCRIBE](/ko/reference/statements/describe-table) 쿼리에서 서브컬럼을 설명에 포함할지 설정합니다. 예를 들어 [Tuple](/ko/reference/data-types/tuple)의 멤버나 [Map](/ko/reference/data-types/map#reading-subcolumns-of-map), [Nullable](/ko/reference/data-types/nullable#finding-null), [Array](/ko/reference/data-types/array#array-size) 데이터 타입의 서브컬럼이 포함됩니다.

가능한 값:

* 0 — `DESCRIBE` 쿼리에 서브컬럼이 포함되지 않습니다.
* 1 — `DESCRIBE` 쿼리에 서브컬럼이 포함됩니다.

**예시**

[DESCRIBE](/ko/reference/statements/describe-table) SQL 문의 예시를 참조하십시오.

<div id="describe_include_virtual_columns">
  ## describe_include_virtual_columns
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

true이면 테이블의 가상 컬럼이 DESCRIBE 쿼리 결과에 포함됩니다.
