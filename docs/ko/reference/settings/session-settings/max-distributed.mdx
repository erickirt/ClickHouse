이 설정은 [system.settings](/ko/reference/system-tables/settings)에서 확인할 수 있으며, [소스 코드](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp)에서 자동 생성됩니다.

<div id="max_distributed_connections">
  ## max_distributed_connections
</div>

<SettingsInfoBlock type="UInt64" default_value="1024" />

하나의 분산 테이블에 대해 단일 쿼리를 분산 처리할 때 원격 서버와 동시에 맺을 수 있는 최대 연결 수입니다. 이 값은 클러스터의 서버 수 이상으로 설정하는 것이 좋습니다.

다음 매개변수는 분산 테이블을 생성할 때(그리고 서버를 시작할 때)만 사용되므로 런타임에 변경할 이유가 없습니다.

<div id="max_distributed_depth">
  ## max_distributed_depth
</div>

<SettingsInfoBlock type="UInt64" default_value="5" />

[분산](/ko/reference/engines/table-engines/special/distributed) 테이블의 재귀 쿼리 최대 깊이를 제한합니다.

이 값을 초과하면 server가 예외를 발생시킵니다.

가능한 값:

* 양의 정수
* 0 — 깊이 제한 없음
