기존 조인 조건에서 전이적 동등 조인 프레디케이트를 추론합니다.
예를 들어 `A.x = B.x` 및 `B.x = C.x`가 주어지면, 조인 순서 최적화기가 직접 `A JOIN C` 계획을
고려할 수 있도록 `A.x = C.x`라는 합성 프레디케이트가 추가됩니다.
