<div id="distributed_replica_error_cap">
  ## distributed_replica_error_cap
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" />

* 유형: 부호 없는 정수
* 기본값: 1000

각 레플리카의 오류 수는 이 값까지만 누적되므로, 하나의 레플리카에 오류가 과도하게 쌓이는 것을 방지합니다.

<div id="distributed_replica_error_half_life">
  ## distributed_replica_error_half_life
</div>

<SettingsInfoBlock type="Seconds" default_value="60" />

* 유형: 초
* 기본값: 60초

분산 테이블에서 오류 수치가 얼마나 빠르게 0으로 감소하는지를 제어합니다. 레플리카를 일정 시간 동안 사용할 수 없어 오류가 5개 누적되었고 distributed&#95;replica&#95;error&#95;half&#95;life가 1초로 설정된 경우, 마지막 오류가 발생한 후 3초가 지나면 해당 레플리카는 정상으로 간주됩니다.

<SettingsInfoBlock type="UInt64" default_value="0" />
