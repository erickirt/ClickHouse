<div id="optimize_if_chain_to_multiif">
  ## optimize_if_chain_to_multiif
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

if(cond1, then1, if(cond2, ...)) 체인을 multiIf로 대체합니다. 현재는 숫자 타입에서는 이점이 없습니다.

<div id="optimize_if_transform_strings_to_enum">
  ## optimize_if_transform_strings_to_enum
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

If 및 Transform의 String 유형 인수를 enum으로 대체합니다. 분산 쿼리에서 일관되지 않은 변경이 발생해 실패할 수 있으므로 기본적으로 비활성화되어 있습니다.
