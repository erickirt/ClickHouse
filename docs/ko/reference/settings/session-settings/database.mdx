<div id="database_atomic_wait_for_drop_and_detach_synchronously">
  ## database_atomic_wait_for_drop_and_detach_synchronously
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

모든 `DROP` 및 `DETACH` 쿼리에 `SYNC` 수정자를 적용합니다.

가능한 값:

* 0 — 쿼리가 지연된 후 실행됩니다.
* 1 — 쿼리가 지연 없이 실행됩니다.
