  ## azure_upload_part_size_multiply_factor

Azure blob storage로 한 번 쓰는 동안 azure&#95;multiply&#95;parts&#95;count&#95;threshold개의 파트가 업로드될 때마다 azure&#95;min&#95;upload&#95;part&#95;size에 이 계수를 곱합니다.

<div id="azure_upload_part_size_multiply_parts_count_threshold">
  ## azure_upload_part_size_multiply_parts_count_threshold
</div>

<SettingsInfoBlock type="UInt64" default_value="500" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "24.3"},{"label": "500"},{"label": "이 수만큼의 파트가 Azure blob storage에 업로드될 때마다 azure_min_upload_part_size에 azure_upload_part_size_multiply_factor가 곱해집니다."}]}]} />

이 수만큼의 파트가 Azure blob storage에 업로드될 때마다 azure&#95;min&#95;upload&#95;part&#95;size에 azure&#95;upload&#95;part&#95;size&#95;multiply&#95;factor가 곱해집니다.
