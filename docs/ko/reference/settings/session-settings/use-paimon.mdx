<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "0"},{"label": "파싱된 Paimon 메타데이터 파일(manifest 목록 및 manifest)의 인메모리 캐싱을 활성화하는 새로운 설정입니다. 영구적인 Paimon 테이블 엔진에서는 메타데이터 초기화 전에 활성화해야 하며, 테이블 함수에서는 쿼리별로 평가됩니다. 이후 쿼리에서 객체 스토리지의 메타데이터 파일을 반복해서 다운로드하고 역직렬화하는 일을 방지합니다."}]}]} />

활성화하면 paimon 테이블 함수와 paimon 스토리지가 paimon 메타데이터 파일 캐시를 사용할 수 있습니다.

Paimon 테이블 함수는 이 설정을 쿼리별로 평가하지만, 영구적인 Paimon 테이블 엔진은 메타데이터 초기화 시점에 이 값을 고정합니다. 이미 생성된 테이블에 대한 적용 여부를 변경하려면 해당 테이블을 삭제한 후 다시 생성해야 합니다.

가능한 값:

* 0 - 비활성화됨
* 1 - 활성화됨
