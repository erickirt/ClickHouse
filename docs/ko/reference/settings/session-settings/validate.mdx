뮤테이션 쿼리를 허용하기 전에 검증합니다. 뮤테이션은 백그라운드에서 실행되므로 잘못된 쿼리를 실행하면 뮤테이션이 중단된 채로 남아 수동 개입이 필요할 수 있습니다.

이전 버전과 호환되지 않는 버그가 발생한 경우에만 이 설정을 변경하십시오.

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "20.4"},{"label": "1"},{"label": "기본적으로 `pointInPolygon` 함수에서 유효하지 않은 다각형에 대해 잘못된 결과를 반환할 수 있는 대신 예외를 발생시킵니다"}]}]} />

다각형이 자체 교차하거나 자기 자신에 접하는 경우 [pointInPolygon](/ko/reference/functions/regular-functions/geo/coordinates#pointinpolygon) 함수에서 예외를 발생시킬지 여부를 활성화하거나 비활성화합니다.

가능한 값:

* 0 — 예외 발생이 비활성화됩니다. `pointInPolygon`은 유효하지 않은 다각형을 허용하며, 이에 대해 잘못된 결과를 반환할 수 있습니다.
* 1 — 예외 발생이 활성화됩니다.
