`MergeTree` 계열 엔진의 [CHECK TABLE](/ko/reference/statements/check-table) 쿼리 결과에 대한 상세 수준을 정의합니다.

가능한 값:

* 0 — 쿼리는 테이블의 각 개별 데이터 파트에 대한 검사 상태를 표시합니다.
* 1 — 쿼리는 테이블 전체의 검사 상태를 표시합니다.
