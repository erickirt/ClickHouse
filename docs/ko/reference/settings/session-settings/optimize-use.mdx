<div id="optimize_use_projections">
  ## optimize_use_projections
</div>

**별칭**: `allow_experimental_projection_optimization`

<SettingsInfoBlock type="Bool" default_value="1" />

`SELECT` 쿼리 처리 시 [프로젝션](/ko/reference/engines/table-engines/mergetree-family/mergetree#projections) 최적화를 활성화하거나 비활성화합니다.

가능한 값:

* 0 — 프로젝션 최적화가 비활성화됩니다.
* 1 — 프로젝션 최적화가 활성화됩니다.
