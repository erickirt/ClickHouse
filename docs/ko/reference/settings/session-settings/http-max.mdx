<div id="http_max_multipart_form_data_size">
  ## http_max_multipart_form_data_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" />

multipart/form-data 콘텐츠 크기에 대한 제한입니다. 이 설정은 URL 매개변수로 parse할 수 없으므로 사용자 프로필에 설정해야 합니다. 콘텐츠는 parse되고 external tables는 쿼리 실행이 시작되기 전에 메모리에 생성된다는 점에 유의하십시오. 또한 이 제한만이 해당 단계에 영향을 미칩니다(`max memory usage` 및 `max execution time` 제한은 HTTP 폼 데이터를 읽는 동안에는 영향을 미치지 않습니다).
