<div id="send_logs_level">
  ## send_logs_level
</div>

<SettingsInfoBlock type="LogsLevel" default_value="fatal" />

지정한 최소 수준 이상의 서버 텍스트 로그를 클라이언트로 전송합니다. 유효한 값: &#39;trace&#39;, &#39;debug&#39;, &#39;information&#39;, &#39;warning&#39;, &#39;error&#39;, &#39;fatal&#39;, &#39;none&#39;

<div id="send_logs_source_regexp">
  ## send_logs_source_regexp
</div>

로그 source 이름이 지정된 regexp와 일치하는 server 텍스트 로그를 전송합니다. 비어 있으면 모든 source를 의미합니다.
