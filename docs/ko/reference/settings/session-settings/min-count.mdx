<div id="min_count_to_compile_aggregate_expression">
  ## min_count_to_compile_aggregate_expression
</div>

<SettingsInfoBlock type="UInt64" default_value="3" />

JIT 컴파일을 시작하기 위한 동일한 집계 표현식의 최소 개수입니다. [compile&#95;aggregate&#95;expressions](/ko/reference/settings/session-settings/compile#compile_aggregate_expressions) 설정이 활성화된 경우에만 적용됩니다.

가능한 값:

* 양의 정수.
* 0 — 동일한 집계 표현식이 항상 JIT 컴파일됩니다.

<div id="min_count_to_compile_expression">
  ## min_count_to_compile_expression
</div>

<SettingsInfoBlock type="UInt64" default_value="3" />

동일한 표현식이 컴파일되기 전에 실행되는 최소 횟수입니다.
