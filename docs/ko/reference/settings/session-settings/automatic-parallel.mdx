병렬 레플리카를 자동으로 활성화하기 위한 레플리카당 읽기 바이트 수의 임계값입니다(`automatic_parallel_replicas_mode`=1인 경우에만 적용됨). 0은 임계값이 없음을 의미합니다.
읽을 총 바이트 수는 수집된 통계를 바탕으로 추정됩니다.

<div id="automatic_parallel_replicas_mode">
  ## automatic_parallel_replicas_mode
</div>

<ExperimentalBadge />

<SettingsInfoBlock type="UInt64" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.12"},{"label": "0"},{"label": "새로운 설정"}]}]} />

수집된 통계를 기반으로 병렬 레플리카를 사용하는 실행으로 자동 전환할지 설정합니다. `enable_analyzer = 1`, `enable_parallel_replicas != 0`, `parallel_replicas_local_plan = 1`이 필요하며, `cluster_for_parallel_replicas`도 지정해야 합니다.
0 - 비활성화, 1 - 활성화, 2 - 통계 수집만 활성화(병렬 레플리카를 사용하는 실행으로의 전환은 비활성화).
