  ## compatibility_ignore_auto_increment_in_create_table
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

값이 true이면 컬럼 선언에서 AUTO&#95;INCREMENT keyword를 무시하고, 그렇지 않으면 오류를 반환합니다. 이 설정은 MySQL에서 마이그레이션을 간소화합니다.
