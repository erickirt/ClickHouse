  ## read_overflow_mode_leaf
</div>

<SettingsInfoBlock type="OverflowMode" default_value="throw" />

읽은 데이터 양이 리프 제한값 중 하나를 초과할 때의 동작을 설정합니다.

가능한 옵션은 다음과 같습니다.

* `throw`: 예외를 발생시킵니다(기본값).
* `break`: 쿼리 실행을 중지하고 부분 결과를 반환합니다.
