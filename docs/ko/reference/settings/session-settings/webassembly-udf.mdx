  ## webassembly_udf_max_fuel
</div>

<ExperimentalBadge />

<SettingsInfoBlock type="UInt64" default_value="100000" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.3"},{"label": "100000"},{"label": "WebAssembly UDF 인스턴스 실행마다 CPU 명령어(연료)를 제한하는 새로운 설정."}]}]} />

WebAssembly UDF 인스턴스 실행마다 적용되는 연료 제한입니다. 각 WebAssembly 명령어는 일정량의 연료를 소비합니다. 이 값은 런타임에 전달되기 전에 1024배로 조정되므로, `webassembly_udf_max_fuel = 1`은 대략 1024 연료 단위에 해당합니다. 유한한 제한을 두지 않으려면 0으로 설정합니다. 기본값이 true인 함수별 설정 `webassembly_udf_enable_fuel`이 true인 함수에만 적용됩니다.
