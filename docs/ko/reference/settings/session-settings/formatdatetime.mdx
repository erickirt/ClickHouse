  ## formatdatetime_format_without_leading_zeros
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

함수 &#39;formatDateTime&#39;의 포매터 &#39;%c&#39;, &#39;%l&#39;, &#39;%k&#39;는 월과 시간을 앞의 0 없이 출력합니다.
