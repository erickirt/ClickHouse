디코릴레이션된 쿼리 계획에서 사용할 조인 유형을 제어합니다. 기본값은 `right`이며, 이는 디코릴레이션된 계획에 서브쿼리 입력이 오른쪽에 있는 RIGHT JOIN이 포함된다는 뜻입니다.

가능한 값:

* `left` - 디코릴레이션 과정에서 LEFT JOIN이 생성되며 입력 테이블은 왼쪽에 배치됩니다.
* `right` - 디코릴레이션 과정에서 RIGHT JOIN이 생성되며 입력 테이블은 오른쪽에 배치됩니다.
