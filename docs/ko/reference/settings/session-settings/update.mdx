  ## update_parallel_mode
</div>

동시에 실행되는 UPDATE 쿼리의 동작을 결정합니다.

가능한 값:

* `sync` - 모든 `UPDATE` 쿼리를 순차적으로 실행합니다.
* `auto` - 한 쿼리에서 업데이트되는 컬럼과 다른 쿼리의 표현식에서 사용되는 컬럼 사이에 의존성이 있는 경우에만 해당 `UPDATE` 쿼리를 순차적으로 실행합니다.
* `async` - `UPDATE` 쿼리를 동기화하지 않습니다.
