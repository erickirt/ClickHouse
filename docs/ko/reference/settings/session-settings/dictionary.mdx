  ## dictionary_lazy_load
</div>

<SettingsInfoBlock type="BoolAuto" default_value="auto" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "auto"},{"label": "개별 딕셔너리에 대해 서버 설정 `dictionaries_lazy_load`를 재정의하는 새로운 설정입니다."}]}]} />

`CREATE DICTIONARY`의 `SETTINGS` 절에서 지정할 때 딕셔너리 로드를 제어합니다. `1`은 처음 사용할 때까지 로드를 지연하고, `0`은 생성 시 딕셔너리를 로드하며, `'auto'`는 서버 설정 `dictionaries_lazy_load`를 따릅니다. 세션 또는 쿼리 수준에서 설정하면 효과가 없습니다.

<div id="dictionary_use_async_executor">
  ## dictionary_use_async_executor
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

딕셔너리 소스를 읽기 위한 파이프라인을 여러 스레드에서 실행합니다. 로컬 CLICKHOUSE 소스를 사용하는 딕셔너리에서만 지원됩니다.
