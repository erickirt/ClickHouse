<div id="quantilesExactWeightedInterpolated">
  ## quantilesExactWeightedInterpolated
</div>

도입 버전: v24.10.0

각 요소의 가중치를 고려하여 서로 다른 수준에서 선형 보간을 사용해 숫자 데이터 시퀀스의 여러 [분위수](https://en.wikipedia.org/wiki/Quantile)를 동시에 계산합니다.

이 함수는 [`quantileExactWeightedInterpolated`](/ko/reference/functions/aggregate-functions/quantileExactWeightedInterpolated)와 동일하지만, 여러 분위수 수준을 한 번에 계산할 수 있으므로 개별 분위수 함수를 각각 호출하는 것보다 더 효율적입니다.

**구문**

```sql
quantilesExactWeightedInterpolated(level1, level2, ...)(expr, weight)
```

**매개변수**

* `level` — 분위수 수준입니다. 0부터 1 사이의 상수 부동소수점 수를 하나 이상 지정합니다. `level` 값은 `[0.01, 0.99]` 범위에서 사용하는 것을 권장합니다. [`Float*`](/ko/reference/data-types/float)

**인수**

* `expr` — 컬럼 값에 대한 표현식이며, 결과는 숫자 데이터 타입, `Date`, `DateTime` 또는 `DateTime64`여야 합니다. [`(U)Int*`](/ko/reference/data-types/int-uint) 또는 [`Int128`](/ko/reference/data-types/int-uint) 또는 [`UInt128`](/ko/reference/data-types/int-uint) 또는 [`Int256`](/ko/reference/data-types/int-uint) 또는 [`UInt256`](/ko/reference/data-types/int-uint) 또는 [`Float*`](/ko/reference/data-types/float) 또는 [`Decimal*`](/ko/reference/data-types/decimal) 또는 [`Date`](/ko/reference/data-types/date) 또는 [`DateTime`](/ko/reference/data-types/datetime) 또는 [`DateTime64`](/ko/reference/data-types/datetime64)
* `weight` — 시퀀스 구성원의 가중치를 나타내는 컬럼입니다. 가중치는 값이 나타나는 횟수입니다. [`UInt*`](/ko/reference/data-types/int-uint)

**반환 값**

지정한 수준의 분위수를, 수준을 지정한 순서와 동일한 순서로 담은 배열을 반환합니다. 숫자 인수는 `Float64`를 반환하며, `Decimal`, `Date`, `DateTime`, `DateTime64` 인수는 입력 포맷을 유지합니다. [`Array(Float64)`](/ko/reference/data-types/array) 또는 [`Array(Decimal*)`](/ko/reference/data-types/array) 또는 [`Array(Date)`](/ko/reference/data-types/array) 또는 [`Array(DateTime)`](/ko/reference/data-types/array) 또는 [`Array(DateTime64)`](/ko/reference/data-types/array)

**예시**

**여러 개의 정확한 가중 보간 분위수 계산**

```sql title=Query
SELECT quantilesExactWeightedInterpolated(0.25, 0.5, 0.75)(number, 1) FROM numbers(10)
```

```response title=Response
┌─quantilesExactWeightedInterpolated(0.25, 0.5, 0.75)(number, 1)─┐
│ [2.25,4.5,6.75]                                                │
└────────────────────────────────────────────────────────────────┘
```
