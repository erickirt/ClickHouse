<div id="quantilesDD">
  ## quantilesDD
</div>

도입 버전: v24.1.0

서로 다른 수준에서 상대 오차가 보장되는 sample의 여러 근사 [분위수](https://en.wikipedia.org/wiki/Quantile)를 동시에 계산합니다.

이 함수는 [`quantileDD`](/ko/reference/functions/aggregate-functions/quantileddsketch)와 동일하지만, 단일 패스로 여러 분위수 수준을 계산할 수 있어 개별 분위수 함수를 각각 호출하는 것보다 더 효율적입니다.

**구문**

```sql
quantilesDD(relative_accuracy, level1, level2, ...)(expr)
```

**매개변수**

* `relative_accuracy` — 분위수의 상대 정확도입니다. 가능한 값은 0~1 범위입니다. 권장 값은 0.001 이상입니다. [`Float*`](/ko/reference/data-types/float)
* `level` — 분위수 수준입니다. 0~1 사이의 하나 이상의 상수 부동소수점 수입니다. `level` 값은 `[0.01, 0.99]` 범위에서 사용하는 것을 권장합니다. [`Float*`](/ko/reference/data-types/float)

**인수**

* `expr` — 컬럼 값에 대한 표현식으로, 결과는 숫자 데이터 타입, `Date` 또는 `DateTime`이어야 합니다. [`(U)Int*`](/ko/reference/data-types/int-uint) or [`Float*`](/ko/reference/data-types/float) or [`Date`](/ko/reference/data-types/date) or [`DateTime`](/ko/reference/data-types/datetime)

**반환 값**

지정한 수준의 근사 분위수를, 수준을 지정한 순서와 동일한 순서로 담은 배열입니다. `Date` 및 `DateTime` 입력의 경우 출력 포맷은 입력 포맷과 일치합니다. [`Array(Float64)`](/ko/reference/data-types/array) or [`Array(Date)`](/ko/reference/data-types/array) or [`Array(DateTime)`](/ko/reference/data-types/array)

**예시**

**DDSketch를 사용해 여러 분위수 계산하기**

```sql title=Query
SELECT quantilesDD(0.01, 0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesDD(0.01, 0.25, 0.5, 0.75)(number)────────────────┐
│ [1.9936617014173448,4.0148353330285875,5.989510371172621] │
└───────────────────────────────────────────────────────────┘
```
