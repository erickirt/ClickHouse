<div id="clickpipes">
  ## ClickHouse Cloud 的托管集成管道
</div>

ClickPipes 是一个托管集成平台，让从多种来源摄取数据变得只需点几下按钮一样简单。
ClickPipes 专为要求最严苛的工作负载而设计，其强大且可扩展的架构可确保始终如一的性能与可靠性。
ClickPipes 既可用于长期流式摄取需求，也可用于一次性数据加载作业。
