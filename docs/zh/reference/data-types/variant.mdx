иҜҘзұ»еһӢиЎЁзӨәе…¶д»–ж•°жҚ®зұ»еһӢзҡ„ unionгҖӮзұ»еһӢ `Variant(T1, T2, ..., TN)` иЎЁзӨәиҜҘзұ»еһӢзҡ„жҜҸдёҖиЎҢ
йғҪеҢ…еҗ«дёҖдёӘеҖјпјҢиҜҘеҖјзҡ„зұ»еһӢеҸҜд»ҘжҳҜ `T1`гҖҒ`T2`гҖҒвҖҰвҖҰгҖҒ`TN` дёӯзҡ„д»»ж„ҸдёҖз§ҚпјҢд№ҹеҸҜд»ҘйғҪдёҚжҳҜ (еҚі `NULL` еҖј) гҖӮ

еөҢеҘ—зұ»еһӢзҡ„йЎәеәҸж— е…ізҙ§иҰҒпјҡVariant(T1, T2) = Variant(T2, T1)гҖӮ
еөҢеҘ—зұ»еһӢеҸҜд»ҘжҳҜйҷӨ Nullable(...)гҖҒLowCardinality(Nullable(...)) е’Ң Variant(...) д№ӢеӨ–зҡ„д»»ж„Ҹзұ»еһӢгҖӮ

<Note>
  дёҚе»әи®®е°Ҷзӣёиҝ‘зҡ„зұ»еһӢз”ЁдҪңеҸҳдҪ“ (дҫӢеҰӮдёҚеҗҢзҡ„ж•°еҖјзұ»еһӢпјҢеҰӮ `Variant(UInt32, Int64)``пјҢжҲ–дёҚеҗҢзҡ„ж—Ҙжңҹзұ»еһӢпјҢеҰӮ `Variant(Date, DateTime)`пјүпјҢ
    еӣ дёәеӨ„зҗҶиҝҷзұ»зұ»еһӢзҡ„еҖјж—¶еҸҜиғҪдјҡдә§з”ҹжӯ§д№үгҖӮй»ҳи®Өжғ…еҶөдёӢпјҢеҲӣе»әиҝҷж ·зҡ„ `Variant`зұ»еһӢдјҡеҜјиҮҙејӮеёёпјҢдҪҶеҸҜд»ҘйҖҡиҝҮ setting`allow&#95;suspicious&#95;variant&#95;types&#96; еҗҜз”ЁгҖӮ
</Note>

<div id="creating-variant">
  ## еҲӣе»ә Variant
</div>

еңЁиЎЁзҡ„еҲ—е®ҡд№үдёӯдҪҝз”Ё `Variant` зұ»еһӢпјҡ

```sql
CREATE TABLE test (v Variant(UInt64, String, Array(UInt64))) ENGINE = Memory;
INSERT INTO test VALUES (NULL), (42), ('Hello, World!'), ([1, 2, 3]);
SELECT v FROM test;
```


<div id="reading-variant-nested-types-as-subcolumns">
  ## е°Ҷ Variant еөҢеҘ—зұ»еһӢиҜ»еҸ–дёәеӯҗеҲ—
</div>

Variant зұ»еһӢж”ҜжҢҒе°Ҷзұ»еһӢеҗҚз§°з”ЁдҪңеӯҗеҲ—пјҢд»Һ Variant еҲ—дёӯиҜ»еҸ–еҚ•дёӘеөҢеҘ—зұ»еһӢгҖӮ
еӣ жӯӨпјҢеҰӮжһңдҪ жңүеҲ— `variant Variant(T1, T2, T3)`пјҢе°ұеҸҜд»ҘдҪҝз”ЁиҜӯжі• `variant.T2` иҜ»еҸ–зұ»еһӢдёә `T2` зҡ„еӯҗеҲ—гҖӮ
еҰӮжһң `T2` еҸҜд»ҘеҢ…еҗ«еңЁ `Nullable` дёӯпјҢеҲҷиҜҘеӯҗеҲ—зҡ„зұ»еһӢдёә `Nullable(T2)`пјӣеҗҰеҲҷдёә `T2`гҖӮиҜҘеӯҗеҲ—зҡ„еӨ§е°ҸдёҺеҺҹе§Ӣ `Variant` еҲ—зӣёеҗҢпјҢе№¶дё”еңЁеҺҹе§Ӣ `Variant` еҲ—зҡ„зұ»еһӢдёҚжҳҜ `T2` зҡ„жүҖжңүиЎҢдёӯпјҢйғҪе°ҶеҢ…еҗ« `NULL` еҖј (еҰӮжһң `T2` дёҚиғҪеҢ…еҗ«еңЁ `Nullable` дёӯпјҢеҲҷдёәз©әеҖј) гҖӮ

д№ҹеҸҜд»ҘдҪҝз”ЁеҮҪж•° `variantElement(variant_column, type_name)` иҜ»еҸ– Variant еӯҗеҲ—гҖӮ

зӨәдҫӢпјҡ

```sql
CREATE TABLE test (v Variant(UInt64, String, Array(UInt64))) ENGINE = Memory;
INSERT INTO test VALUES (NULL), (42), ('Hello, World!'), ([1, 2, 3]);
SELECT v, v.String, v.UInt64, v.`Array(UInt64)` FROM test;
```

```sql
SELECT variantType(v) FROM test;
```

<div id="conversion-between-a-variant-column-and-other-columns">
  ## Variant еҲ—дёҺе…¶д»–еҲ—д№Ӣй—ҙзҡ„иҪ¬жҚў

`Variant` зұ»еһӢзҡ„еҲ—еҸҜиҝӣиЎҢ 4 з§ҚиҪ¬жҚўгҖӮ

<div id="converting-a-string-column-to-a-variant-column">
  ### е°Ҷ String еҲ—иҪ¬жҚўдёә Variant еҲ—
</div>

д»Һ `String` иҪ¬жҚўдёә `Variant`пјҢжҳҜйҖҡиҝҮд»Һеӯ—з¬ҰдёІеҖјдёӯи§ЈжһҗеҮә `Variant` зұ»еһӢзҡ„еҖјжқҘе®һзҺ°зҡ„пјҡ

```sql
SELECT '42'::Variant(String, UInt64) AS variant, variantType(variant) AS variant_type
```

```text
в”Ңв”Җvariantв”Җв”¬в”Җvariant_typeв”Җв”җ
в”Ӯ 42      в”Ӯ UInt64       в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

```sql
SELECT '[1, 2, 3]'::Variant(String, Array(UInt64)) as variant, variantType(variant) as variant_type
```

```text
в”Ңв”Җvariantв”Җв”¬в”Җvariant_typeв”Җв”Җв”җ
в”Ӯ [1,2,3] в”Ӯ Array(UInt64) в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

```sql
SELECT CAST(map('key1', '42', 'key2', 'true', 'key3', '2020-01-01'), 'Map(String, Variant(UInt64, Bool, Date))') AS map_of_variants, mapApply((k, v) -> (k, variantType(v)), map_of_variants) AS map_of_variant_types
```

<div id="comparing-values-of-variant-data">
  ## жҜ”иҫғ Variant зұ»еһӢзҡ„еҖј
</div>

`Variant` зұ»еһӢзҡ„еҖјеҸӘиғҪдёҺзӣёеҗҢ `Variant` зұ»еһӢзҡ„еҖјжҜ”иҫғгҖӮ

й»ҳи®Өжғ…еҶөдёӢпјҢжҜ”иҫғиҝҗз®—з¬ҰдјҡдҪҝз”Ё[Variant зҡ„й»ҳи®Өе®һзҺ°](#functions-with-variant-arguments)пјҢ
еҲҶеҲ«еҜ№жҜҸдёӘ Variant зұ»еһӢиҝӣиЎҢжҜ”иҫғгҖӮеҸҜд»ҘйҖҡиҝҮи®ҫзҪ® `use_variant_default_implementation_for_comparisons = 0` зҰҒз”ЁжӯӨиЎҢдёәпјҢ
д»ҺиҖҢж”№з”ЁдёӢйқўжүҖиҝ°зҡ„еҺҹз”ҹ Variant жҜ”иҫғи§„еҲҷгҖӮ**жіЁж„Ҹ**пјҢ`ORDER BY` е§Ӣз»ҲдҪҝз”ЁеҺҹз”ҹжҜ”иҫғгҖӮ

**еҺҹз”ҹ Variant жҜ”иҫғи§„еҲҷпјҡ**

еҜ№дәҺзұ»еһӢ `Variant(..., T1, ... T2, ...)` дёӯеә•еұӮзұ»еһӢдёә `T1` зҡ„еҖј `v1` е’Ңеә•еұӮзұ»еһӢдёә `T2` зҡ„еҖј `v2`пјҢиҝҗз®—з¬Ұ `<` зҡ„з»“жһңе®ҡд№үеҰӮдёӢпјҡ

* еҰӮжһң `T1 = T2 = T`пјҢеҲҷз»“жһңдёә `v1.T < v2.T` (еҚіжҜ”иҫғеә•еұӮеҖј) гҖӮ
* еҰӮжһң `T1 != T2`пјҢеҲҷз»“жһңдёә `T1 < T2` (еҚіжҜ”иҫғзұ»еһӢеҗҚз§°) гҖӮ

зӨәдҫӢпјҡ

```sql
SET allow_suspicious_types_in_order_by = 1;
CREATE TABLE test (v1 Variant(String, UInt64, Array(UInt32)), v2 Variant(String, UInt64, Array(UInt32))) ENGINE=Memory;
INSERT INTO test VALUES (42, 42), (42, 43), (42, 'abc'), (42, [1, 2, 3]), (42, []), (42, NULL);
```

```sql

SELECT v2, variantType(v2) AS v2_type FROM test ORDER BY v2;
```

  ### зұ»еһӢдёҚеҢ№й…Қж—¶зҡ„иЎҢдёә
</div>

и®ҫзҪ® `variant_throw_on_type_mismatch` з”ЁдәҺжҺ§еҲ¶еҪ“еҮҪж•°еә”з”ЁдәҺ `Variant` еҲ—пјҢиҖҢжҹҗдёҖиЎҢе®һйҷ…еӯҳеӮЁзҡ„зұ»еһӢдёҺиҜҘеҮҪж•°дёҚе…је®№ж—¶зҡ„еӨ„зҗҶж–№ејҸпјҡ

* `true` (й»ҳи®Ө) вҖ” еңЁйҒҮеҲ°з¬¬дёҖдёӘдёҚе…је®№зҡ„иЎҢж—¶жҠӣеҮәејӮеёё (`ILLEGAL_TYPE_OF_ARGUMENT`) гҖӮ

**зӨәдҫӢпјҡ**

```sql
CREATE TABLE test (v Variant(String, UInt64)) ENGINE = Memory;
INSERT INTO test VALUES ('hello'), (42), ('foo');

-- Default (throw on mismatch): length() does not accept UInt64, so the query throws.
SELECT length(v) FROM test;  -- throws ILLEGAL_TYPE_OF_ARGUMENT

-- With throw disabled: incompatible rows return NULL.
SET variant_throw_on_type_mismatch = false;
SELECT v, length(v) FROM test ORDER BY v::String NULLS LAST;
```
