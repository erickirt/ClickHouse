<div id="enable_mixed_granularity_parts">
  ## enable_mixed_granularity_parts
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

启用或禁用切换为使用 `index_granularity_bytes` 设置来控制粒度大小。在 19.11 版本之前，只有
`index_granularity` 设置可用于限制粒度大小。`index_granularity_bytes`
设置可在从包含大行 (数十到数百 MB) 的表中读取数据时提升 ClickHouse 的性能。
如果你的表中存在大行，可以为这些表启用此设置，
以提高 `SELECT` 查询的效率。
