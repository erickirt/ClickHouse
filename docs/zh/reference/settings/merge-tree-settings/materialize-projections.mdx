  ## materialize_projections_on_insert
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

启用后，INSERT 会创建包含投影的新 parts。
否则，可以通过显式执行 [MATERIALIZE PROJECTION](/zh/reference/statements/alter/projection#materialize-projection) 创建，
或在 合并 期间通过 [materialize&#95;projections&#95;on&#95;merge](/zh/reference/settings/merge-tree-settings/materialize-projections#materialize_projections_on_merge) 创建。

<div id="materialize_projections_on_merge">
  ## materialize_projections_on_merge
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.6"},{"label": "0"},{"label": "新设置"}]}]} />

启用后，如果参与一次合并的所有源 parts 都缺少某个投影 (例如因为它们在插入时设置了 `materialize_projections_on_insert = 0`) ，合并时会重新构建该投影，从而使合并后的 part 包含该投影。

合并仍然只会组合具有相同投影集合的 parts。要将某个投影回填到所有现有 parts，请使用显式 [MATERIALIZE PROJECTION](/zh/reference/statements/alter/projection#materialize-projection)。使用 [materialize&#95;projections&#95;on&#95;insert](/zh/reference/settings/merge-tree-settings/materialize-projections#materialize_projections_on_insert) 执行 INSERT 时，也会创建投影。
