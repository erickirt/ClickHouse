<div id="allow_experimental_replacing_merge_with_cleanup">
  ## allow_experimental_replacing_merge_with_cleanup
</div>

<ExperimentalBadge />

<SettingsInfoBlock type="Bool" default_value="0" />

允许对带有 `is_deleted`
列的 ReplacingMergeTree 执行实验性的 CLEANUP 合并。启用后，可使用 `OPTIMIZE ... FINAL CLEANUP` 手动
将一个分区中的所有 parts 合并为单个 part，并删除其中
已删除的行。

还允许通过设置 `min_age_to_force_merge_seconds`、
`min_age_to_force_merge_on_partition_only` 和
`enable_replacing_merge_with_cleanup_for_min_age_to_force_merge`
使这类合并在后台自动执行。

  ## allow_experimental_text_index_phrase_search
