<div id="materialize_skip_indexes_on_merge">
  ## materialize_skip_indexes_on_merge
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.1"},{"label": "1"},{"label": "新设置"}]}]} />

启用后，合并操作会为新的 parts 构建并存储跳过索引。
否则，这些索引只能通过显式执行 [MATERIALIZE INDEX](/zh/reference/statements/alter/skipping-index#materialize-index)
或 [在 INSERT 时](/zh/reference/settings/session-settings#materialize_skip_indexes_on_insert) 创建/存储。

另请参见 [exclude&#95;materialize&#95;skip&#95;indexes&#95;on&#95;merge](/zh/reference/settings/merge-tree-settings/exclude#exclude_materialize_skip_indexes_on_merge)，以进行更细粒度的控制。

<div id="materialize_statistics_on_merge">
  ## materialize_statistics_on_merge
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.1"},{"label": "1"},{"label": "新设置"}]}]} />

否则，这些统计信息可通过显式执行 [MATERIALIZE STATISTICS](/zh/reference/statements/alter/statistics)
或[在 INSERT 时](/zh/reference/settings/session-settings#materialize_statistics_on_insert)创建/存储
