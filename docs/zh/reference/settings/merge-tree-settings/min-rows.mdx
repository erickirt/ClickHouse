<div id="min_rows_for_compact_part">
  ## min_rows_for_compact_part
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

已废弃设置，不起任何作用。

<div id="min_rows_for_full_part_storage">
  ## min_rows_for_full_part_storage
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

仅在 ClickHouse Cloud 中可用。对于数据分片片段，使用 Full 类型存储而非 packed 所需的最少行数

<div id="min_rows_for_wide_part">
  ## min_rows_for_wide_part
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

将数据分片片段创建为 `Wide` 格式而不是 `Compact` 格式所需的最小行数。

<div id="min_rows_to_fsync_after_merge">
  ## min_rows_to_fsync_after_merge
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

合并后对分片执行 fsync 的最小行数 (0 表示禁用)
