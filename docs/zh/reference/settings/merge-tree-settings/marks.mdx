</div>

<div id="marks_compression_codec">
  ## marks_compression_codec
</div>

<SettingsInfoBlock type="String" default_value="ZSTD(3)" />

标记 使用的压缩编码。由于 标记 体积很小且会被缓存，
因此默认压缩编码为 ZSTD(3)。
