  ## max_merge_selecting_sleep_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="60000" />

在未选中任何可合并的 parts 后，再次尝试选择 parts 进行合并前的最长等待时间。较低的设置会更频繁地在
background&#95;schedule&#95;pool 中触发选择任务，从而在大规模集群中向
ZooKeeper 发送大量请求
