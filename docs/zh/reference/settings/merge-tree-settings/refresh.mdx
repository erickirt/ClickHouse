  ## refresh_parts_interval

如果该值大于 0，则会从底层文件系统刷新数据分区片段列表，以检查数据是否在底层已被更新。
只有当表位于只读磁盘上时才能设置此项 (这意味着这是一个只读副本，而数据由另一个副本写入) 。
