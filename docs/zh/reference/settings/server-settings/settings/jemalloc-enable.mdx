这些设置用于配置 ClickHouse server，由 ClickHouse 源代码自动生成。

<div id="jemalloc_enable_background_threads">
  ## jemalloc_enable_background_threads
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />启用 jemalloc 的后台线程。jemalloc 使用后台线程清理未使用的内存页。禁用此项可能会导致性能下降。

<div id="jemalloc_enable_global_profiler">
  ## jemalloc_enable_global_profiler
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />为所有线程启用 jemalloc 的内存分配分析器。Jemalloc 会对内存分配进行采样，并记录已采样分配对应的所有释放。
可以使用 SYSTEM JEMALLOC FLUSH PROFILE 刷出 profile，以用于内存分配分析。
样本也可以通过配置 jemalloc&#95;collect&#95;global&#95;profile&#95;samples&#95;in&#95;trace&#95;log 存储到 system.trace&#95;log 中，或通过查询 setting jemalloc&#95;collect&#95;profile&#95;samples&#95;in&#95;trace&#95;log 存储。
参见 [Allocation Profiling](/zh/concepts/features/performance/allocation-profiling)
