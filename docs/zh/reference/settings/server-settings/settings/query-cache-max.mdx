这些设置用于配置 ClickHouse 服务器，并由 ClickHouse 源代码自动生成。

<div id="query_cache.max_entries">
  ## query_cache.max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="1024" changeable_without_restart="No" />缓存中可存储的 SELECT 查询结果的最大数量。

<div id="query_cache.max_entry_size_in_bytes">
  ## query_cache.max_entry_size_in_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1048576" changeable_without_restart="No" />可保存到缓存中的 SELECT 查询结果的最大字节大小。

<div id="query_cache.max_entry_size_in_rows">
  ## query_cache.max_entry_size_in_rows
</div>

<SettingsInfoBlock type="UInt64" default_value="30000000" changeable_without_restart="No" />缓存中可保存的 SELECT 查询结果的最大行数。

<div id="query_cache.max_size_in_bytes">
  ## query_cache.max_size_in_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" changeable_without_restart="Yes" />缓存的最大容量 (以字节为单位) 。0 表示禁用查询缓存。
