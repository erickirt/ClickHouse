这些设置用于配置 ClickHouse server，内容由 ClickHouse 源代码自动生成。

<div id="max_backups_io_thread_pool_free_size">
  ## max_backups_io_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />如果备份 IO 线程池中的**空闲**线程数超过 `max_backup_io_thread_pool_free_size`，ClickHouse 将释放这些空闲线程占用的资源，并缩减线程池大小。必要时，仍可再次创建线程。

<div id="max_backups_io_thread_pool_size">
  ## max_backups_io_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="是" />ClickHouse 使用备份 IO 线程池中的线程执行 S3 备份 IO 操作。`max_backups_io_thread_pool_size` 用于限制该线程池中的最大线程数。
