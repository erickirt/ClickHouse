这些设置用于配置 ClickHouse 服务器，并由 ClickHouse 源代码自动生成。

<div id="merge_tree">
  ## merge_tree
</div>

用于 [MergeTree](/zh/reference/engines/table-engines/mergetree-family/mergetree) 表的细调设置。

更多信息，请参见 MergeTreeSettings.h 头文件。

**示例**

```xml
<merge_tree>
    <max_suspicious_broken_parts>5</max_suspicious_broken_parts>
</merge_tree>
```

<div id="merge_workload">
  ## merge_workload
</div>

<SettingsInfoBlock type="String" default_value="default" changeable_without_restart="Yes" />

用于调节资源在合并与其他工作负载之间的使用和共享方式。指定的值会作为所有后台合并的 `workload` 设置值使用。可被 MergeTree 设置覆盖。

**另请参阅**

* [工作负载调度](/zh/concepts/features/configuration/server-config/workload-scheduling)
