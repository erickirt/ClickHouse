这些设置用于配置 ClickHouse 服务器，并由 ClickHouse 源代码自动生成。

<div id="max_replicated_fetches_network_bandwidth_for_server">
  ## max_replicated_fetches_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />副本拉取时网络数据交换的最大速度，以每秒字节数为单位。0 表示不受限制。

<div id="max_replicated_sends_network_bandwidth_for_server">
  ## max_replicated_sends_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />用于副本发送的网络数据交换最大速度，以每秒字节数为单位。0 表示不受限制。

<div id="max_replicated_table_num_to_throw">
  ## max_replicated_table_num_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="是" />

如果复制表的数量超过该值，服务器将抛出异常。

仅统计使用以下 database engine 的表：

* Atomic
* Ordinary
* Replicated
* Lazy

<Note>
  值为 `0` 表示不受限制。
</Note>

**示例**

```xml
<max_replicated_table_num_to_throw>400</max_replicated_table_num_to_throw>
```
