这些设置用于配置 ClickHouse 服务器，由 ClickHouse 源代码自动生成。

<div id="max_snapshot_commit_thread_pool_free_size">
  ## max_snapshot_commit_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="是" />如果 快照 commit 线程池中的空闲线程数超过 `max_snapshot_commit_thread_pool_free_size`，ClickHouse 将释放这些空闲线程占用的资源，并缩减线程池大小。必要时，线程可再次创建。

<div id="max_snapshot_commit_thread_pool_size">
  ## max_snapshot_commit_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="64" changeable_without_restart="Yes" />提交快照时使用的线程数。
