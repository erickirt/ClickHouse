这些设置用于配置 ClickHouse server，由 ClickHouse 源代码自动生成。

<div id="enable_azure_sdk_logging">
  ## enable_azure_sdk_logging
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />启用 Azure SDK 的日志

<div id="enable_webterminal">
  ## enable_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />启用 `/webterminal` HTTP 端点上的 Web 终端界面。它通过 WebSocket 在浏览器中提供交互式的 `clickhouse-client` 会话。当设置为 `false` 时，对 `/webterminal` 的请求会返回 HTTP 状态 `403 Forbidden`。
