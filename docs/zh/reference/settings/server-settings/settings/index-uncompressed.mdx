这些设置用于配置 ClickHouse 服务器，并由 ClickHouse 源代码自动生成。

<div id="index_uncompressed_cache_policy">
  ## index_uncompressed_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />次级索引未压缩缓存策略名称。

<div id="index_uncompressed_cache_size">
  ## index_uncompressed_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

`MergeTree` 索引的未压缩块缓存最大容量。

<Note>
  值为 `0` 表示禁用。

  此设置可在运行时修改，并立即生效。
</Note>

<div id="index_uncompressed_cache_size_ratio">
  ## index_uncompressed_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />次级索引未压缩缓存中受保护队列 (使用 SLRU 策略时) 占缓存总大小的比例。
