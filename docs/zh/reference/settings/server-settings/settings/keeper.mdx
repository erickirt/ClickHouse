这些设置用于配置 ClickHouse server，由 ClickHouse 源代码自动生成。

<div id="keeper_hosts">
  ## keeper_hosts
</div>

动态设置。包含 ClickHouse 可能连接到的一组 [Zoo]Keeper 主机。不显示 `<auxiliary_zookeepers>` 中的信息

<div id="keeper_multiread_batch_size">
  ## keeper_multiread_batch_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />

支持批处理的发往 [Zoo]Keeper 的 MultiRead 请求，其批次的最大大小。如果设置为 0，则禁用批处理功能。仅在 ClickHouse Cloud 中可用。
