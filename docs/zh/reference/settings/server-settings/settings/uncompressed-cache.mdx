这些设置用于配置 ClickHouse 服务器，并由 ClickHouse 源代码自动生成。

<div id="uncompressed_cache_policy">
  ## uncompressed_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />未压缩缓存的策略名称。

<div id="uncompressed_cache_size">
  ## uncompressed_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

MergeTree 家族表引擎使用的未压缩数据的最大大小 (以字节为单位) 。

整个 server 共用一个未压缩缓存。内存按需分配。启用 `use_uncompressed_cache` 选项后，将使用该缓存。

对于非常短的查询，在某些情况下使用未压缩缓存会更有优势。

<Note>
  值为 `0` 表示禁用。

  此设置可在运行时修改，并会立即生效。
</Note>

<div id="uncompressed_cache_size_ratio">
  ## uncompressed_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="否" />在未压缩缓存中，受保护队列 (在 SLRU 策略下) 的大小占缓存总大小的比例。
