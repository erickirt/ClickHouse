这些设置用于配置 ClickHouse 服务器，且由 ClickHouse 源代码自动生成。

<div id="top_level_domains_list">
  ## top_level_domains_list
</div>

定义要添加的自定义顶级域名列表，其中每一项的格式均为 `<name>/path/to/file</name>`。

例如：

```xml
<top_level_domains_lists>
    <public_suffix_list>/path/to/public_suffix_list.dat</public_suffix_list>
</top_level_domains_lists>
```

另请参阅：

* 函数 [`cutToFirstSignificantSubdomainCustom`](/zh/reference/functions/regular-functions/url-functions#cutToFirstSignificantSubdomainCustom) 及其变体，
  可接受自定义 TLD 列表名称，并返回域名中从顶级子域到第一个有效子域的部分。

<div id="top_level_domains_path">
  ## top_level_domains_path
</div>

<SettingsInfoBlock type="String" default_value="/var/lib/clickhouse/top_level_domains/" changeable_without_restart="否" />

存放顶级域名的目录。

**示例**

```xml
<top_level_domains_path>/var/lib/clickhouse/top_level_domains/</top_level_domains_path>
```
