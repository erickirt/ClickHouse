这些设置用于配置 ClickHouse server，由 ClickHouse 源代码自动生成。

<div id="postgresql_port">
  ## postgresql_port
</div>

用于通过 PostgreSQL 协议与客户端通信的端口。

<Note>
  * 正整数表示要监听的端口号
  * 空值用于禁用通过 PostgreSQL 协议与客户端通信。
</Note>

**示例**

```xml
<postgresql_port>9005</postgresql_port>
```

<div id="postgresql_require_secure_transport">
  ## postgresql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="否" />如果设置为 true，则要求客户端通过 [postgresql&#95;port](/zh/reference/settings/server-settings/settings/postgresql#postgresql_port) 进行安全通信。使用 `<sslmode=disable>` 选项的连接将被拒绝。请结合 [OpenSSL](/zh/reference/settings/server-settings/settings/other#openssl) 设置使用此选项。
