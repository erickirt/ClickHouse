这些设置用于配置 ClickHouse server，由 ClickHouse 源代码自动生成。

<div id="index_mark_cache_policy">
  ## index_mark_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />次级索引标记缓存策略名称。

<div id="index_mark_cache_prewarm_ratio">
  ## index_mark_cache_prewarm_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.95" changeable_without_restart="否" />预热期间需要填充的索引标记缓存总大小占比。

<div id="index_mark_cache_size">
  ## index_mark_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="5368709120" changeable_without_restart="是" />

索引标记缓存的最大容量。

<Note>
  `0` 表示禁用。

  此设置可在运行时修改，并立即生效。
</Note>

<div id="index_mark_cache_size_ratio">
  ## index_mark_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.3" changeable_without_restart="No" />次级索引标记缓存中受保护队列 (采用 SLRU 策略时) 的大小占缓存总大小的比例。
