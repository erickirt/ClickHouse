这些设置用于配置 ClickHouse 服务器，且由 ClickHouse 源代码自动生成。

<div id="tcp_close_connection_after_queries_num">
  ## tcp_close_connection_after_queries_num
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />在关闭连接之前，单个 TCP 连接允许执行的最大查询数。设为 0 表示查询次数不受限制。

<div id="tcp_close_connection_after_queries_seconds">
  ## tcp_close_connection_after_queries_seconds
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />TCP 连接在关闭前的最长存活时间，以秒为单位。设为 0 表示连接生命周期不受限制。
