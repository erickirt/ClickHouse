这些设置用于配置 ClickHouse server，由 ClickHouse 源代码自动生成。

<div id="logger.console">
  ## logger.console
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />启用将日志输出到控制台。设置为 `<1>` 或 `<true>` 即可启用。如果 ClickHouse 未以守护进程模式运行，则默认值为 `<1>`；否则为 `<0>`。

<div id="logger.console_log_level">
  ## logger.console_log_level
</div>

<SettingsInfoBlock type="String" default_value="trace" changeable_without_restart="No" />控制台输出的日志级别。默认值为 `<level>`。
