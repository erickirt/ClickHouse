这些设置用于配置 ClickHouse 服务器，由 ClickHouse 源代码自动生成。

<div id="mysql_port">
  ## mysql_port
</div>

用于通过 MySQL 协议与客户端通信的端口。

<Note>
  * 正整数表示要监听的端口号
  * 空值用于禁用通过 MySQL 协议与客户端的通信。
</Note>

**示例**

```xml
<mysql_port>9004</mysql_port>
```

<div id="mysql_require_secure_transport">
  ## mysql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />如果设置为 true，则要求客户端通过 [mysql&#95;port](/zh/reference/settings/server-settings/settings/mysql#mysql_port) 使用安全连接。使用 `<--ssl-mode=none>` 选项的连接将被拒绝。请结合 [OpenSSL](/zh/reference/settings/server-settings/settings/other#openssl) 设置使用。
