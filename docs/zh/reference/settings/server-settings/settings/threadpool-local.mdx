这些设置用于配置 ClickHouse 服务器，并由 ClickHouse 源代码自动生成。

<div id="threadpool_local_fs_reader_pool_size">
  ## threadpool_local_fs_reader_pool_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="100" changeable_without_restart="No" />当 `local_filesystem_read_method = 'pread_threadpool'` 时，用于从本地文件系统读取的线程池线程数。

<div id="threadpool_local_fs_reader_queue_size">
  ## threadpool_local_fs_reader_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />可在线程池中调度的、用于从本地文件系统读取的作业最大数量。
