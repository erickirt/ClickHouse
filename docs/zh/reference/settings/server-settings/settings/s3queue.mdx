这些设置用于配置 ClickHouse 服务器，并由 ClickHouse 源代码自动生成。

<div id="s3queue_disable_streaming">
  ## s3queue_disable_streaming
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Yes" />即使表已创建且存在已附加的物化视图，也禁用 S3Queue 中的流式处理

<div id="s3queue_log">
  ## s3queue_log
</div>

`s3queue_log` 系统表的相关设置。

<SystemLogParameters />

默认设置如下：

```xml
<s3queue_log>
    <database>system</database>
    <table>s3queue_log</table>
    <partition_by>toYYYYMM(event_date)</partition_by>
    <flush_interval_milliseconds>7500</flush_interval_milliseconds>
</s3queue_log>
```
