这些设置用于配置 ClickHouse 服务器，由 ClickHouse 源代码自动生成。

<div id="global_profiler_cpu_time_period_ns">
  ## global_profiler_cpu_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />全局 Profiler 的 CPU 时钟定时器周期 (单位：纳秒) 。将该值设为 0 可关闭 CPU 时钟全局 Profiler。对于单个查询，建议值至少为 10000000 (每秒 100 次) ；对于集群级性能分析，建议值至少为 1000000000 (每秒一次) 。

<div id="global_profiler_real_time_period_ns">
  ## global_profiler_real_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />全局 Profiler 的实时时钟定时器周期 (以纳秒为单位) 。将该值设为 0 可关闭基于实时时钟的全局 Profiler。对于单个查询，建议值至少为 10000000 (每秒 100 次) ；对于集群级性能分析，建议值至少为 1000000000 (每秒 1 次) 。
