这些设置用于配置 ClickHouse 服务器，并且是根据 ClickHouse 源代码自动生成的。

<div id="users_config">
  ## users_config
</div>

包含以下内容的文件的路径：

* 用户配置。
* 访问权限。
* profile 设置。
* 配额设置。

**示例**

```xml
<users_config>users.xml</users_config>
```

<div id="users_to_ignore_early_memory_limit_check">
  ## users_to_ignore_early_memory_limit_check
</div>

以逗号分隔的用户列表。对于这些用户，将跳过提前内存限制检查。如果用户不在此列表中，则当总内存使用量超过限制时，查询将被拒绝。
