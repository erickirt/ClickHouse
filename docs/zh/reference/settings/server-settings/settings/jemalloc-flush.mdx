这些设置用于配置 ClickHouse 服务器，并由 ClickHouse 源代码自动生成。

<div id="jemalloc_flush_profile_interval_bytes">
  ## jemalloc_flush_profile_interval_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />当全局峰值内存占用增加到 jemalloc&#95;flush&#95;profile&#95;interval&#95;bytes 时，将刷新 jemalloc profile

<div id="jemalloc_flush_profile_on_memory_exceeded">
  ## jemalloc_flush_profile_on_memory_exceeded
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />发生总内存超限错误时，会刷新 jemalloc profile

<div id="jemalloc_flush_profile_on_memory_exceeded_interval">
  ## jemalloc_flush_profile_on_memory_exceeded_interval
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />如果该值非 0，则设置发生总内存超限错误时，两次刷新 jemalloc profile 之间的最小时间间隔 (单位为秒) 。例如，5 表示最多每 5 秒刷新一次 profile。其优先级高于 `jemalloc_flush_profile_on_memory_exceeded`。
