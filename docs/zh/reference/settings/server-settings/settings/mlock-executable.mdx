这些设置用于配置 ClickHouse 服务器，由 ClickHouse 源代码自动生成。

<div id="mlock_executable">
  ## mlock_executable
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />

启动后执行 `<mlockall>`，以降低首次查询延迟，并防止 ClickHouse 可执行文件在高 IO 负载下被换出到磁盘。

<Note>
  建议启用此选项，但这会使启动时间增加几秒。请注意，没有 &quot;CAP&#95;IPC&#95;LOCK&quot; 能力时，此设置将无法生效。
</Note>

**示例**

```xml
<mlock_executable>false</mlock_executable>
```
