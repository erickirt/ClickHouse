  ## unique_key_bitmap_cache_policy

<div id="unique_key_bitmap_cache_size_bytes">
  ## unique_key_bitmap_cache_size_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" changeable_without_restart="Yes" />用于 UNIQUE KEY 的每分片删除 bitmap 的进程内缓存最大大小 (以字节为单位) 。设为 0 可禁用该缓存。

<div id="unique_key_bitmap_cache_size_ratio">
  ## unique_key_bitmap_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />在 UNIQUE KEY 删除 bitmap cache 中，受保护队列 (在采用 SLRU policy 时) 的大小占缓存总大小的比例。
