这些设置用于配置 ClickHouse 服务器，由 ClickHouse 源代码自动生成。

<div id="max_view_num_to_throw">
  ## max_view_num_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="是" />

如果视图数量大于此值，服务器将抛出异常。

仅统计以下数据库引擎中的表：

* Atomic
* Ordinary
* Replicated
* Lazy

<Note>
  值为 `0` 表示无限制。
</Note>

**示例**

```xml
<max_view_num_to_throw>400</max_view_num_to_throw>
```

<div id="max_view_num_to_warn">
  ## max_view_num_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="Yes" />

如果已附加视图的数量超过指定值，ClickHouse 服务器 将向 `system.warnings` 表添加警告消息。

**示例**

```xml
<max_view_num_to_warn>400</max_view_num_to_warn>
```
