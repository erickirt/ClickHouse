这些设置用于配置 ClickHouse 服务器，并由 ClickHouse 源代码自动生成。

<div id="logger.async">
  ## logger.async
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />当设为 `<true>` (默认) 时，将以异步方式记录日志 (每个输出通道一个后台线程) ；否则，会在调用 LOG 的线程中记录。

<div id="logger.async_queye_max_size">
  ## logger.async_queye_max_size
</div>

<SettingsInfoBlock type="UInt64" default_value="65536" changeable_without_restart="No" />使用 async 日志时，队列中等待刷写的消息最大保留数量。超出的消息将被丢弃。
