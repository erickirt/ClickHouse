这些设置用于配置 ClickHouse 服务器，由 ClickHouse 源代码自动生成。

<div id="interserver_https_host">
  ## interserver_https_host
</div>

与 `<interserver_http_host>` 类似，不同之处在于，其他服务器可使用此主机名通过 `<HTTPS>` 访问该服务器。

**示例**

```xml
<interserver_https_host>example.clickhouse.com</interserver_https_host>
```

<div id="interserver_https_port">
  ## interserver_https_port
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />

ClickHouse 服务器之间通过 `<HTTPS>` 交换数据所使用的端口。

**示例**

```xml
<interserver_https_port>9010</interserver_https_port>
```
