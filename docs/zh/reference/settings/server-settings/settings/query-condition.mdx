这些设置用于配置 ClickHouse 服务器，并由 ClickHouse 源代码自动生成。

<div id="query_condition_cache_policy">
  ## query_condition_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />查询条件缓存策略名称。

<div id="query_condition_cache_size">
  ## query_condition_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="104857600" changeable_without_restart="Yes" />

查询条件缓存的最大容量。

<Note>
  此设置可在运行时修改，并立即生效。
</Note>

<div id="query_condition_cache_size_ratio">
  ## query_condition_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />查询条件缓存中受保护队列 (在 SLRU 策略下) 的大小占缓存总大小的比例。
