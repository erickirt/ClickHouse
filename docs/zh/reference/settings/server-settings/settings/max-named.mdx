这些设置用于配置 ClickHouse server，由 ClickHouse 源代码自动生成。

<div id="max_named_collection_num_to_throw">
  ## max_named_collection_num_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

如果命名集合的数量超过此值，服务器将抛出异常。

<Note>
  值为 `0` 表示不受限制。
</Note>

**示例**

```xml
<max_named_collection_num_to_throw>400</max_named_collection_num_to_throw>
```

<div id="max_named_collection_num_to_warn">
  ## max_named_collection_num_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="Yes" />

如果命名集合的数量超过指定值，ClickHouse server 会向 `system.warnings` 表中添加警告信息。

**示例**

```xml
<max_named_collection_num_to_warn>400</max_named_collection_num_to_warn>
```
