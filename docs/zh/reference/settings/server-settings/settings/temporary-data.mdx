这些设置项用于配置 ClickHouse 服务器，由 ClickHouse 源代码自动生成。

<div id="temporary_data_in_cache">
  ## temporary_data_in_cache
</div>

使用此选项时，临时数据将存储在特定 disk 的 cache 中。
本节中，应指定类型为 `cache` 的 disk 名称。
在这种情况下，cache 与临时数据将共享同一存储空间，并且可以清退 disk cache 以为临时数据腾出空间。

<Note>
  只能使用以下一个选项来配置临时数据 storage：`tmp_path` ,`tmp_policy`, `temporary_data_in_cache`。
</Note>

**示例**

`local_disk` 的 cache 和临时数据都将存储在 filesystem 中的 `/tiny_local_cache`，由 `tiny_local_cache` 管理。

```xml highlight={9-16,20}
<clickhouse>
<storage_configuration>
<disks>
<local_disk>
<type>local</type>
<path>/local_disk/</path>
</local_disk>

<tiny_local_cache>
<type>cache</type>
<disk>local_disk</disk>
<path>/tiny_local_cache/</path>
<max_size_rows>10M</max_size_rows>
<max_file_segment_size>1M</max_file_segment_size>
<cache_on_write_operations>1</cache_on_write_operations>
</tiny_local_cache>
</disks>
</storage_configuration>

<temporary_data_in_cache>tiny_local_cache</temporary_data_in_cache>
</clickhouse>
```
