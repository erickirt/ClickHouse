这些设置用于配置 ClickHouse 服务器，并由 ClickHouse 源代码自动生成。

<div id="max_database_num_to_throw">
  ## max_database_num_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />如果数据库数量超过此值，服务器将抛出异常。0 表示不限制。

<div id="max_database_num_to_warn">
  ## max_database_num_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="是" />

如果已附加的数据库数量超过指定值，ClickHouse 服务器会向 `system.warnings` 表添加警告消息。

**示例**

```xml
<max_database_num_to_warn>50</max_database_num_to_warn>
```

<div id="max_database_replicated_create_table_thread_pool_size">
  ## max_database_replicated_create_table_thread_pool_size
</div>

<SettingsInfoBlock type="UInt32" default_value="1" changeable_without_restart="No" />在 DatabaseReplicated 中副本恢复期间创建表所使用的线程数。0 表示线程数等于 CPU 核心数。
