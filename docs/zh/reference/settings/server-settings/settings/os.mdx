这些设置用于配置 ClickHouse 服务器，由 ClickHouse 源代码自动生成。

<div id="os_collect_psi_metrics">
  ## os_collect_psi_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />启用对 /proc/pressure/ 文件中的 PSI 指标进行统计。

<div id="os_cpu_busy_time_threshold">
  ## os_cpu_busy_time_threshold
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000" changeable_without_restart="No" />将 OS CPU 忙碌时间 (`OSCPUVirtualTimeMicroseconds` 指标) 判定为 CPU 正在执行某些有用工作的微秒阈值；如果忙碌时间低于该值，则不会视为发生 CPU 过载。
