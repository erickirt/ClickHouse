这些设置用于配置 ClickHouse server，且由 ClickHouse 源代码自动生成。

<div id="tcp_port">
  ## tcp_port
</div>

用于通过 TCP 协议与客户端进行通信的端口。

**示例**

```xml
<tcp_port>9000</tcp_port>
```

<div id="tcp_port_secure">
  ## tcp_port_secure
</div>

用于与客户端进行安全通信的 TCP 端口。请与 [OpenSSL](/zh/reference/settings/server-settings/settings/other#openssl) 设置配合使用。

**默认值**

```xml
<tcp_port_secure>9440</tcp_port_secure>
```
