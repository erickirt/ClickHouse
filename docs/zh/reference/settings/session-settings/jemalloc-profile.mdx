<div id="jemalloc_profile_text_collapsed_use_count">
  ## jemalloc_profile_text_collapsed_use_count
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.2"},{"label": "0"},{"label": "新增设置：在 collapsed jemalloc 堆内存剖析格式中，按分配次数而非字节数进行聚合"}]}]} />

使用 jemalloc 堆内存剖析的 `collapsed` 输出格式时，按分配次数而不是字节数进行聚合。当为 false (默认值) 时，每个栈按存活字节数加权；当为 true 时，则按存活分配次数加权。

<div id="jemalloc_profile_text_output_format">
  ## jemalloc_profile_text_output_format
</div>

<SettingsInfoBlock type="JemallocProfileFormat" default_value="collapsed" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.2"},{"label": "collapsed"},{"label": "用于控制 system.jemalloc_profile_text 表输出格式的新设置。可能值：'raw'、'symbolized'、'collapsed'"}]}]} />

system.jemalloc&#95;profile&#95;text 表中 jemalloc 堆内存剖析的输出格式。可选值包括：&#39;raw&#39; (原始剖析) 、&#39;symbolized&#39; (带符号的 jeprof 格式) 和 &#39;collapsed&#39; (FlameGraph 格式) 。

<div id="jemalloc_profile_text_symbolize_with_inline">
  ## jemalloc_profile_text_symbolize_with_inline
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.2"},{"label": "1"},{"label": "用于控制在对 jemalloc 堆内存剖析进行符号化时是否包含内联帧的新设置。启用后，将包含内联帧，但符号化速度会变慢；禁用后，则会跳过内联帧以获得更快的输出"}]}]} />

是否在对 jemalloc 堆内存剖析进行符号化时包含内联帧。启用后，将包含内联帧，这可能会显著减慢符号化过程；禁用后，则会跳过它们。仅影响 &#39;symbolized&#39; 和 &#39;collapsed&#39; 输出格式。
