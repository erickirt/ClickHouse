<div id="jemalloc_enable_profiler">
  ## jemalloc_enable_profiler
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.9"},{"label": "0"},{"label": "新设置"}]}]} />

为该查询启用 jemalloc 分析器。Jemalloc 会对内存分配进行采样，并记录已采样分配对应的所有释放。
可以使用 SYSTEM JEMALLOC FLUSH PROFILE 刷新这些 profile，以便进行内存分配分析。
也可以通过配置 jemalloc&#95;collect&#95;global&#95;profile&#95;samples&#95;in&#95;trace&#95;log 或查询设置 jemalloc&#95;collect&#95;profile&#95;samples&#95;in&#95;trace&#95;log，将样本存储到 system.trace&#95;log 中。
参见 [分配分析](/zh/concepts/features/performance/allocation-profiling)
