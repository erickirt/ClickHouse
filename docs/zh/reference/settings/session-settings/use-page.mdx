  ## use_page_cache_for_disks_without_file_cache

<SettingsInfoBlock type="Bool" default_value="0" />

对未启用文件系统缓存的远程磁盘，使用用户态页缓存。

<div id="use_page_cache_for_local_disks">
  ## use_page_cache_for_local_disks
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.2"},{"label": "0"},{"label": "新增设置：对本地磁盘使用用户态页缓存"}]}]} />

从本地磁盘读取时使用用户态页缓存。该设置仅用于测试，实际中通常不太可能带来性能提升。要求 `local_filesystem_read_method = 'pread'` 或 `'read'`。此设置不会禁用 OS page cache；如需禁用，可使用 `min_bytes_to_use_direct_io`。仅影响常规表，不影响 `file()` 表函数或 `File()` 表引擎。

<div id="use_page_cache_for_object_storage">
  ## use_page_cache_for_object_storage
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.2"},{"label": "0"},{"label": "新增设置：对对象存储表函数使用用户态页缓存"}]}]} />

在从对象存储表函数 (s3、azure、hdfs) 和表引擎 (S3、Azure、HDFS) 读取数据时，使用用户态页缓存。
