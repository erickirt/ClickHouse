<div id="s3_upload_part_size_multiply_factor">
  ## s3_upload_part_size_multiply_factor
</div>

<SettingsInfoBlock type="UInt64" default_value="2" />

当单次写入 S3 上传的分片数达到 s3&#95;multiply&#95;parts&#95;count&#95;threshold 时，就将 s3&#95;min&#95;upload&#95;part&#95;size 乘以该因子。

<div id="s3_upload_part_size_multiply_parts_count_threshold">
  ## s3_upload_part_size_multiply_parts_count_threshold
</div>

<SettingsInfoBlock type="UInt64" default_value="500" />

每当上传到 S3 的分片数达到该值时，s3&#95;min&#95;upload&#95;part&#95;size 都会乘以 s3&#95;upload&#95;part&#95;size&#95;multiply&#95;factor。
