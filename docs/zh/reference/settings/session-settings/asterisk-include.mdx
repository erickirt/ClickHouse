<div id="asterisk_include_alias_columns">
  ## asterisk_include_alias_columns
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

在通配符查询 (`SELECT *`) 中包含 [ALIAS](/zh/reference/statements/create/table#alias) 别名列。

可能的值：

* 0 - 禁用
* 1 - 启用

<div id="asterisk_include_materialized_columns">
  ## asterisk_include_materialized_columns
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

在通配符查询 (`SELECT *`) 中包含 [MATERIALIZED](/zh/reference/statements/create/view#materialized-view) 列。

Possible values:

* 0 - 禁用
* 1 - 启用
