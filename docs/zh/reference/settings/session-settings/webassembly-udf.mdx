<div id="webassembly_udf_max_fuel">
  ## webassembly_udf_max_fuel
</div>

<ExperimentalBadge />

<SettingsInfoBlock type="UInt64" default_value="100000" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.3"},{"label": "100000"},{"label": "用于限制每个 WebAssembly UDF 实例执行期间 CPU 指令（fuel）的新增设置。"}]}]} />

每个 WebAssembly UDF 实例执行期间的 fuel 限制。每条 WebAssembly 指令都会消耗一定数量的 fuel。该值在传递给运行时之前会先乘以 1024，因此 `webassembly_udf_max_fuel = 1` 大致相当于 1024 个 fuel 单位。设为 0 表示不设有限值限制。仅适用于每个函数的 `webassembly_udf_enable_fuel` 设置为 true 的函数；该值默认为 true。
