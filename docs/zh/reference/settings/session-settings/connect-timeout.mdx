如果在集群定义中使用了 &#39;shard&#39; 和 &#39;replica&#39; 部分，则该参数表示 Distributed 表引擎连接远程服务器时的超时时间 (以毫秒为单位) 。
如果连接失败，会多次尝试连接不同的副本。
