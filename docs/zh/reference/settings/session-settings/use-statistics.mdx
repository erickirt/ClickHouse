  ## use_statistics_for_part_pruning
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "1"},{"label": "在查询执行期间使用统计信息进行 part pruning 的新设置。"}]}]} />

在查询执行期间使用统计信息过滤 parts。

启用后，SELECT 查询中的 pruning 会使用列统计信息 (例如 MinMax 统计信息) ，在读取任何数据之前剔除不可能包含匹配数据的 parts。

可能的值：

* 0 — 已禁用。
* 1 — 已启用。
