<div id="http_headers_progress_interval_ms">
  ## http_headers_progress_interval_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="100" />

发送 HTTP 请求头 X-ClickHouse-Progress 的频率不得超过指定的时间间隔。

<div id="http_headers_read_timeout">
  ## http_headers_read_timeout
</div>

<SettingsInfoBlock type="秒" default_value="30" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "30"},{"label": "用于限制读取所有 HTTP 请求头总耗时的新设置，可防御 slowloris 攻击。"}]}]} />

读取全部 HTTP 请求头的最长时间 (以秒为单位) 。这是整个请求头解析阶段的总时限，而不是单次读取超时。它可防御 slowloris 类攻击，即客户端通过极其缓慢地发送请求头数据来长期占用连接。
