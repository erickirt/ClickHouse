  ## kafka_disable_num_consumers_limit
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

禁用基于可用 CPU 核心数的 kafka&#95;num&#95;consumers 限制。

<div id="kafka_max_wait_ms">
  ## kafka_max_wait_ms
</div>

<SettingsInfoBlock type="毫秒" default_value="5000" />

重试前从 [Kafka](/zh/reference/engines/table-engines/integrations/kafka) 读取消息时的等待时间，单位为毫秒。

可能的值：

* 正整数。
* 0 — 无限超时。

另请参见：

* [Apache Kafka](https://kafka.apache.org/)
