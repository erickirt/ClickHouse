这些设置可在 [system.settings](/zh/reference/system-tables/settings) 中查看，并由 [源代码](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) 自动生成。

<div id="max_read_buffer_size">
  ## max_read_buffer_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="1048576" />

从文件系统读取数据时使用的缓冲区最大大小。超过 256 MiB 的值会被强制限制为 256 MiB，因为读取缓冲区不需要更大。

<div id="max_read_buffer_size_local_fs">
  ## max_read_buffer_size_local_fs
</div>

<SettingsInfoBlock type="UInt64" default_value="131072" />

从本地文件系统读取数据时使用的读取缓冲区最大大小。如果设置为 0，则使用 max&#95;read&#95;buffer&#95;size。大于 256 MiB 的值会被限制为 256 MiB，因为读取缓冲区无需更大。

<div id="max_read_buffer_size_remote_fs">
  ## max_read_buffer_size_remote_fs
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

从远程文件系统读取数据时，读取缓冲区的最大大小。如果设置为 0，则使用 `max_read_buffer_size`。大于 256 MiB 的值会被限制为 256 MiB，因为读取缓冲区不需要更大。
