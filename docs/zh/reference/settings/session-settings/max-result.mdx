这些设置可在 [system.settings](/zh/reference/system-tables/settings) 中查看，并由 [源代码](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) 自动生成。

<div id="max_result_bytes">
  ## max_result_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

限制结果大小 (按字节计，未压缩) 。达到阈值后，查询会在处理完一个数据块后停止，
但不会截断结果中的最后一个块，因此结果大小可能会超过该阈值。

**注意事项**

此阈值会将内存中的结果大小计入在内。
即使结果本身较小，它也可能引用内存中更大的数据结构，
例如 LowCardinality 列的字典，以及 AggregateFunction 列的 Arena，
因此即便结果大小较小，也仍可能超过该阈值。

<Warning>
  该设置属于较底层的参数，应谨慎使用
</Warning>

<div id="max_result_rows">
  ## max_result_rows
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

限制结果中的行数。对子查询也会进行检查；在执行分布式查询的部分时，也会在远程服务器上进行检查。
当值为 `0` 时，不施加任何限制。

如果达到阈值，查询会在处理完一个数据块后停止，
但不会截断结果中的最后一个块，因此结果大小可能会
超过该阈值。
