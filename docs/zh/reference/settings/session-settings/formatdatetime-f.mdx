  ## formatdatetime_f_prints_scale_number_of_digits

<SettingsInfoBlock type="Bool" default_value="0" />

函数 `formatDateTime` 中的格式说明符 `%f` 在用于 DateTime64 时，只会输出与标度相对应位数的数字，而不是固定输出 6 位数字。

  ## formatdatetime_f_prints_single_zero
