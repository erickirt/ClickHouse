这些设置可在 [system.settings](/zh/reference/system-tables/settings) 中查看，并由 [源代码](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) 自动生成。

<div id="receive_data_timeout_ms">
  ## receive_data_timeout_ms
</div>

<SettingsInfoBlock type="毫秒" default_value="2000" />

从副本接收第一个数据包或进度为正的数据包时的连接超时时间

<div id="receive_timeout">
  ## receive_timeout
</div>

<SettingsInfoBlock type="Seconds" default_value="300" />

从网络接收数据的超时时间，单位为秒。如果在这段时间间隔内未接收到任何字节，就会抛出异常。如果在客户端设置了此项，那么套接字的 &#39;send&#95;timeout&#39; 也会在服务器端对应连接的一端进行设置。
