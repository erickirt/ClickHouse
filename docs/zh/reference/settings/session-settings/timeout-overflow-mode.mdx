<div id="timeout_overflow_mode">
  ## timeout_overflow_mode
</div>

<SettingsInfoBlock type="OverflowMode" default_value="throw" />

设置当查询运行时间超过 `max_execution_time`，或预计运行时间超过 `max_estimated_execution_time` 时应采取的操作。

可选值：

* `throw`：抛出异常 (默认值) 。
* `break`：停止执行查询并返回部分结果，就像源数据已经耗尽一样。

<div id="timeout_overflow_mode_leaf">
  ## timeout_overflow_mode_leaf
</div>

<SettingsInfoBlock type="OverflowMode" default_value="throw" />

设置叶子节点上的查询运行时间超过 `max_execution_time_leaf` 时的处理方式。

可能的值：

* `throw`：抛出异常 (默认) 。
* `break`：停止执行查询并返回部分结果，就像
  源数据已经耗尽一样。
