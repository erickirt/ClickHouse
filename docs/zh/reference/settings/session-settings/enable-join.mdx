  ## enable_join_transitive_predicates
</div>

<BetaBadge />

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.6"},{"label": "1"},{"label": "默认启用 enable_join_transitive_predicates"}]}, {"id": "row-2","items": [{"label": "26.4"},{"label": "0"},{"label": "用于推断传递性等值 JOIN 谓词、以优化 JOIN 顺序的新设置。"}]}]} />

根据现有的 JOIN 条件推断传递性等值 JOIN 谓词。
例如，给定 `A.x = B.x` 和 `B.x = C.x`，会添加一个合成的 `A.x = C.x` 谓词，
这样 JOIN 顺序优化器就可以考虑直接的 (A JOIN C) 执行计划。
