这些设置可在 [system.settings](/zh/reference/system-tables/settings) 中查看，并根据 [源文件](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) 自动生成。

<div id="send_logs_level">
  ## send_logs_level
</div>

<SettingsInfoBlock type="LogsLevel" default_value="fatal" />

将级别不低于指定最小值的服务器文本日志发送到客户端。有效值：&#39;trace&#39;、&#39;debug&#39;、&#39;information&#39;、&#39;warning&#39;、&#39;error&#39;、&#39;fatal&#39;、&#39;none&#39;

<div id="send_logs_source_regexp">
  ## send_logs_source_regexp
</div>

使用指定的正则表达式发送服务器文本日志，以匹配日志源名称。留空表示所有源。
