定义在 `MergeTree` 家族引擎中，[CHECK TABLE](/zh/reference/statements/check-table) 查询结果的详细程度。

可能的值：

* 0 — 查询会显示表中每个数据分区片段各自的检查状态。
* 1 — 查询会显示表的整体检查状态。
