<SettingsInfoBlock type="UInt64" default_value="3600" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "24.11"},{"label": "3600"},{"label": "新设置。"}]}, {"id": "row-2","items": [{"label": "24.10"},{"label": "3600"},{"label": "新设置。"}]}]} />

如果某个主机在执行 BACKUP ON CLUSTER 或 RESTORE ON CLUSTER 操作期间，在这段时间内未能在 ZooKeeper 中重新创建其临时 &#39;alive&#39; 节点，则整个备份或恢复都会被视为失败。
该值应大于主机发生故障后重新连接到 ZooKeeper 所需的任何合理时长。
零表示无限制。
