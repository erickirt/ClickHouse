这些设置可在 [system.settings](/zh/reference/system-tables/settings) 中查看，并由 [源文件](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) 自动生成。

<div id="max_distributed_connections">
  ## max_distributed_connections
</div>

<SettingsInfoBlock type="UInt64" default_value="1024" />

单个查询针对单个 Distributed 表进行分布式处理时，与远程服务器建立的最大并发连接数。建议将该值设置为不小于集群中的服务器数量。

以下参数仅在创建分布式表时 (以及在启动服务器时) 使用，因此没有必要在运行时更改它们。

<div id="max_distributed_depth">
  ## max_distributed_depth
</div>

<SettingsInfoBlock type="UInt64" default_value="5" />

限制 [Distributed](/zh/reference/engines/table-engines/special/distributed) 表递归查询的最大深度。

如果超过该值，服务器会抛出异常。

可能的值：

* 正整数。
* 0 — 深度不受限制。
