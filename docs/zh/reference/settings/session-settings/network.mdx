<div id="network_compression_method">
  ## network_compression_method
</div>

<SettingsInfoBlock type="String" default_value="LZ4" />

用于压缩客户端/服务器端以及服务器/服务器之间通信的编解码器。

可能的值：

* `NONE` — 不压缩。
* `LZ4` — 使用 LZ4 编解码器。
* `LZ4HC` — 使用 LZ4HC 编解码器。
* `ZSTD` — 使用 ZSTD 编解码器。

**另请参见**

* [network&#95;zstd&#95;compression&#95;level](/zh/reference/settings/session-settings/network#network_zstd_compression_level)

<div id="network_zstd_compression_level">
  ## network_zstd_compression_level
</div>

<SettingsInfoBlock type="Int64" default_value="1" />

调整 ZSTD 的压缩级别。仅在 [network&#95;compression&#95;method](/zh/reference/settings/session-settings/network#network_compression_method) 设置为 `ZSTD` 时使用。

可能的值：

* 1 到 15 的正整数。
