这些设置可在 [system.settings](/zh/reference/system-tables/settings) 中查看，并由 [源代码](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) 自动生成。

<div id="optimize_if_chain_to_multiif">
  ## optimize_if_chain_to_multiif
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

将 `if(cond1, then1, if(cond2, ...))` 链替换为 multiIf。当前这对数值类型没有益处。

<div id="optimize_if_transform_strings_to_enum">
  ## optimize_if_transform_strings_to_enum
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

将 If 和 Transform 中的 String 类型参数替换为枚举类型。默认禁用，因为这可能会在分布式查询中产生不一致的更改，进而导致查询失败。
