<div id="max_rand_distribution_parameter">
  ## max_rand_distribution_parameter
</div>

<SettingsInfoBlock type="Float" default_value="1000000" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "1000000"},{"label": "用于限制随机分布函数中形状参数的新设置，可防止在极端输入下出现长时间卡顿。"}]}]} />

`randChiSquared`、`randStudentT` 和 `randFisherF` 等随机分布函数中分布形状参数的最大值。这样可避免参数值过于极端时计算耗时过长。

<div id="max_rand_distribution_trials">
  ## max_rand_distribution_trials
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000000" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "1000000000"},{"label": "用于限制随机分布函数试验次数的新设置，可防止在极端输入下卡住。"}]}]} />

`randBinomial` 和 `randNegativeBinomial` 等随机分布函数允许的最大试验次数。这样可防止在试验次数很大时计算时间过长。
