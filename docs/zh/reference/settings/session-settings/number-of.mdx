这些设置可在 [system.settings](/zh/reference/system-tables/settings) 中查看，并根据 [源代码](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) 自动生成。

<div id="number_of_mutations_to_delay">
  ## number_of_mutations_to_delay
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

如果已变更的表中至少有这么多个未完成的变更，则会人为减慢该表的变更速度。0 - 禁用

<div id="number_of_mutations_to_throw">
  ## number_of_mutations_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

如果已变更的表中至少有这么多个尚未完成的变更，则抛出 &#39;Too many mutations ...&#39; 异常。0 - 禁用
