<div id="max_parser_depth">
  ## max_parser_depth
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" />

限制递归下降解析器的最大递归深度，可用于控制栈大小。

可能的值：

* 正整数。
* 0 — 递归深度不受限制。
