<div id="description">
  ## 说明
</div>

显示所有可用的词干提取器。
可在函数 [stem](/zh/reference/functions/regular-functions/nlp-functions) 中使用。

<Info>
  **可用性**

  `system.stemmers` 仅存在于使用 `libstemmer` 依赖 (`USE_LIBSTEMMER`) 编译的 ClickHouse 构建中。在未包含该依赖的构建中，此表不存在，对其发起查询会报 `UNKNOWN_TABLE` 错误。你可以通过以下命令检查当前构建是否已启用它：

  ```sql
  SELECT value FROM system.build_options WHERE name = 'USE_LIBSTEMMER';
  ```
</Info>

列：

{/*AUTOGENERATED_START*/}

* `name` ([String](/zh/reference/data-types/index)) — Snowball 词干提取器的标识符 (语言/算法)

{/*AUTOGENERATED_END*/}
