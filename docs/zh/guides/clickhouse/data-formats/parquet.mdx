Parquet жҳҜдёҖз§Қй«ҳж•Ҳзҡ„ж–Үд»¶ж јејҸпјҢеҸҜжҢүеҲ—ејҸж–№ејҸеӯҳеӮЁж•°жҚ®гҖӮ
ClickHouse ж”ҜжҢҒиҜ»еҸ–е’ҢеҶҷе…Ҙ Parquet ж–Үд»¶гҖӮ

<Tip>
  еҪ“дҪ еңЁжҹҘиҜўдёӯеј•з”Ёж–Үд»¶и·Ҝеҫ„ж—¶пјҢClickHouse е®һйҷ…е°қиҜ•иҜ»еҸ–зҡ„дҪҚзҪ®еҸ–еҶідәҺдҪ дҪҝз”Ёзҡ„жҳҜе“Әз§Қ ClickHouse зүҲжң¬гҖӮ

  еҰӮжһңдҪ дҪҝз”Ёзҡ„жҳҜ [`clickhouse-local`](/zh/concepts/features/tools-and-utilities/clickhouse-local)пјҢе®ғдјҡд»ҺзӣёеҜ№дәҺдҪ еҗҜеҠЁ ClickHouse Local ж—¶жүҖеңЁдҪҚзҪ®зҡ„и·Ҝеҫ„иҜ»еҸ–гҖӮ
  еҰӮжһңдҪ йҖҡиҝҮ `clickhouse client` дҪҝз”Ё ClickHouse Server жҲ– ClickHouse CloudпјҢе®ғдјҡд»ҺжңҚеҠЎеҷЁдёҠзҡ„ `/var/lib/clickhouse/user_files/` зӣ®еҪ•дёӢзҡ„зӣёеҜ№и·Ҝеҫ„иҜ»еҸ–гҖӮ
</Tip>

<div id="importing-from-parquet">
  ## д»Һ Parquet еҜје…Ҙ
</div>

```sql
DESCRIBE TABLE file('data.parquet', Parquet);
```

<div id="importing-to-an-existing-table">
  ## еҜје…ҘеҲ°зҺ°жңүиЎЁ
</div>

е…ҲеҲӣе»әдёҖдёӘз”ЁдәҺеҜје…Ҙ Parquet ж•°жҚ®зҡ„иЎЁпјҡ

```sql
CREATE TABLE sometable
(
    `path` String,
    `date` Date,
    `hits` UInt32
)
ENGINE = MergeTree
ORDER BY (date, path);
```

```sql
INSERT INTO sometable
FROM INFILE 'data.parquet' FORMAT Parquet;

SELECT *
FROM sometable
LIMIT 5;
```

<div id="inserting-a-local-file-to-remote-server">
  ## е°Ҷжң¬ең°ж–Үд»¶жҸ’е…ҘиҝңзЁӢжңҚеҠЎеҷЁ
</div>

еҰӮжһңдҪ жғіе°Ҷжң¬ең° Parquet ж–Үд»¶жҸ’е…ҘиҝңзЁӢ ClickHouse жңҚеҠЎеҷЁпјҢеҸҜд»ҘеғҸдёӢйқўиҝҷж ·йҖҡиҝҮз®ЎйҒ“е°Ҷж–Үд»¶еҶ…е®№дј з»ҷ `clickhouse-client`пјҡ

```sql
clickhouse client -q "INSERT INTO sometable FORMAT Parquet" < data.parquet
```

<div id="creating-new-tables-from-parquet-files">
  ## д»Һ Parquet ж–Үд»¶еҲӣе»әж–°иЎЁ

з”ұдәҺ ClickHouse дјҡиҜ»еҸ– Parquet ж–Үд»¶зҡ„ schemaпјҢеӣ жӯӨжҲ‘д»¬еҸҜд»ҘеҠЁжҖҒеҲӣе»әиЎЁпјҡ

<div id="exporting-to-parquet-format">
  ## еҜјеҮәдёә Parquet ж јејҸ
</div>

<Tip>
  еңЁ ClickHouse Cloud дёӯдҪҝз”Ё `INTO OUTFILE` ж—¶пјҢйңҖиҰҒеңЁж–Үд»¶е°ҶиҰҒеҶҷе…Ҙзҡ„йӮЈеҸ°жңәеҷЁдёҠпјҢйҖҡиҝҮ `clickhouse client` иҝҗиЎҢиҝҷдәӣе‘Ҫд»ӨгҖӮ
</Tip>

иҰҒе°Ҷд»»ж„ҸиЎЁжҲ–жҹҘиҜўз»“жһңеҜјеҮәдёә Parquet ж–Үд»¶пјҢеҸҜд»ҘдҪҝз”Ё `INTO OUTFILE` еӯҗеҸҘпјҡ

```sql
SELECT *
FROM sometable
INTO OUTFILE 'export.parquet'
FORMAT Parquet
```

иҝҷдјҡеңЁеҪ“еүҚе·ҘдҪңзӣ®еҪ•дёӯеҲӣе»ә `export.parquet` ж–Үд»¶гҖӮ
