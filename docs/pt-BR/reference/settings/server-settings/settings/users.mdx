Essas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="users_config">
  ## users_config
</div>

Caminho para o arquivo que contém:

* Configurações dos usuários.
* Permissões de acesso.
* Perfis de configuração.
* Configurações de cotas.

**Exemplo**

```xml
<users_config>users.xml</users_config>
```

<div id="users_to_ignore_early_memory_limit_check">
  ## users_to_ignore_early_memory_limit_check
</div>

Lista de usuários separada por vírgulas para ignorar uma verificação prévia do limite de memória. Se o usuário não estiver nesta lista, uma consulta será rejeitada se o uso total de memória exceder o limite.
