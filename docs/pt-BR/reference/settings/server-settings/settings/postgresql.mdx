Estas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="postgresql_port">
  ## postgresql_port
</div>

Porta para comunicação com clientes por meio do protocolo PostgreSQL.

<Note>
  * Inteiros positivos especificam o número da porta em que se deve escutar
  * Valores vazios são usados para desabilitar a comunicação com clientes por meio do protocolo PostgreSQL.
</Note>

**Exemplo**

```xml
<postgresql_port>9005</postgresql_port>
```

<div id="postgresql_require_secure_transport">
  ## postgresql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Se definido como true, a comunicação segura com os clientes via [postgresql&#95;port](/pt-BR/reference/settings/server-settings/settings/postgresql#postgresql_port) será obrigatória. Conexões com a opção `<sslmode=disable>` serão recusadas. Use essa configuração com as configurações do [OpenSSL](/pt-BR/reference/settings/server-settings/settings/other#openssl).
