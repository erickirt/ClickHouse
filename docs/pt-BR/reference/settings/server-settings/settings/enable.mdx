Essas configurações são usadas para configurar o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="enable_azure_sdk_logging">
  ## enable_azure_sdk_logging
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Habilita o logging do SDK do Azure

<div id="enable_webterminal">
  ## enable_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Habilita a interface do terminal web no endpoint HTTP `/webterminal`. Disponibiliza uma sessão interativa do `clickhouse-client` no navegador via WebSocket. Quando `false`, as solicitações para `/webterminal` retornam o código de status HTTP `403 Forbidden`.
