Essas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="asynchronous_metrics_enable_heavy_metrics">
  ## asynchronous_metrics_enable_heavy_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Habilita o cálculo de métricas assíncronas mais custosas.

<div id="asynchronous_metrics_keeper_metrics_only">
  ## asynchronous_metrics_keeper_metrics_only
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Faz com que as métricas assíncronas calculem apenas as relacionadas ao Keeper.

<div id="asynchronous_metrics_update_period_s">
  ## asynchronous_metrics_update_period_s
</div>

<SettingsInfoBlock type="UInt32" default_value="1" changeable_without_restart="No" />Período, em segundos, para atualização de métricas assíncronas.
