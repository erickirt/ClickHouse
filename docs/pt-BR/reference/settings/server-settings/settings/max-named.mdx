Essas configurações se aplicam ao servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="max_named_collection_num_to_throw">
  ## max_named_collection_num_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Sim" />

Se o número de coleções nomeadas for maior que este valor, o servidor lançará uma exceção.

<Note>
  Um valor de `0` significa que não há limite.
</Note>

**Exemplo**

```xml
<max_named_collection_num_to_throw>400</max_named_collection_num_to_throw>
```

<div id="max_named_collection_num_to_warn">
  ## max_named_collection_num_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="Yes" />

Se o número de coleções nomeadas exceder o valor especificado, o servidor ClickHouse adicionará mensagens de aviso à tabela `system.warnings`.

**Exemplo**

```xml
<max_named_collection_num_to_warn>400</max_named_collection_num_to_warn>
```
