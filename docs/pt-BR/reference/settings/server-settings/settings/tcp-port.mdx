Essas configurações são usadas para configurar o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="tcp_port">
  ## tcp_port
</div>

Porta usada para comunicação com clientes pelo protocolo TCP.

**Exemplo**

```xml
<tcp_port>9000</tcp_port>
```

<div id="tcp_port_secure">
  ## tcp_port_secure
</div>

Porta TCP para comunicação segura com os clientes. Use-a com as configurações do [OpenSSL](/pt-BR/reference/settings/server-settings/settings/other#openssl).

**Valor padrão**

```xml
<tcp_port_secure>9440</tcp_port_secure>
```
