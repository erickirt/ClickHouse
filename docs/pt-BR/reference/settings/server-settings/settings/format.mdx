Essas configurações definem o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="format_parsing_thread_pool_queue_size">
  ## format_parsing_thread_pool_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="Sim" />

O número máximo de tarefas que podem ser agendadas no pool de threads para análise da entrada.

<Note>
  Um valor de `0` significa ilimitado.
</Note>

<div id="format_schema_path">
  ## format_schema_path
</div>

<SettingsInfoBlock type="String" default_value="/var/lib/clickhouse/format_schemas/" changeable_without_restart="Não" />

O caminho para o diretório que contém os esquemas dos dados de entrada, como os esquemas do formato [CapnProto](/pt-BR/reference/formats/CapnProto).

**Exemplo**

```xml
<!-- Directory containing schema files for various input formats. -->
<format_schema_path>/var/lib/clickhouse/format_schemas/</format_schema_path>
```
