Estas configurações se aplicam ao servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="keeper_hosts">
  ## keeper_hosts
</div>

Configuração dinâmica. Contém um conjunto de hosts do [Zoo]Keeper aos quais o ClickHouse pode se conectar. Não expõe informações de `<auxiliary_zookeepers>`

<div id="keeper_multiread_batch_size">
  ## keeper_multiread_batch_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />

Tamanho máximo do lote da solicitação MultiRead para o [Zoo]Keeper com suporte a processamento em lote. Se definido como 0, o processamento em lote é desativado. Disponível somente no ClickHouse Cloud.
