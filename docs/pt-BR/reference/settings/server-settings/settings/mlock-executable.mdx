Estas configurações são usadas para configurar o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="mlock_executable">
  ## mlock_executable
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />

Execute `<mlockall>` após a inicialização para reduzir a latência das primeiras consultas e evitar que o executável do ClickHouse seja paginado em condições de alta carga de E/S.

<Note>
  É recomendável habilitar esta opção, mas isso aumentará o tempo de inicialização em até alguns segundos. Tenha em mente que essa configuração não funcionará sem a capability &quot;CAP&#95;IPC&#95;LOCK&quot;.
</Note>

**Exemplo**

```xml
<mlock_executable>false</mlock_executable>
```
