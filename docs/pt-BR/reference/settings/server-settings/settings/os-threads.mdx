Estas configurações são usadas para configurar o servidor do ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="os_threads_nice_value_distributed_cache_tcp_handler">
  ## os_threads_nice_value_distributed_cache_tcp_handler
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="Não" />

Valor nice do Linux para as threads do handler TCP do distributed cache. Valores menores significam maior prioridade de CPU.

Requer a capability CAP&#95;SYS&#95;NICE; caso contrário, não tem efeito.

Valores possíveis: -20 a 19.

<div id="os_threads_nice_value_merge_mutate">
  ## os_threads_nice_value_merge_mutate
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="No" />

Valor nice do Linux para threads de mesclagem e mutação. Valores menores significam maior prioridade de CPU.

Requer a capability CAP&#95;SYS&#95;NICE; caso contrário, não tem efeito.

Valores possíveis: -20 a 19.

<div id="os_threads_nice_value_zookeeper_client_send_receive">
  ## os_threads_nice_value_zookeeper_client_send_receive
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="No" />

Valor nice do Linux para as threads de envio e recebimento no cliente ZooKeeper. Valores mais baixos significam maior prioridade de CPU.

Requer a capability CAP&#95;SYS&#95;NICE; caso contrário, não tem efeito.

Valores possíveis: -20 a 19.
