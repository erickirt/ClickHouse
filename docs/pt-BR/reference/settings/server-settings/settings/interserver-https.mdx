Estas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="interserver_https_host">
  ## interserver_https_host
</div>

Semelhante a `<interserver_http_host>`, com a diferença de que esse hostname pode ser usado por outros servidores para acessar este servidor via `<HTTPS>`.

**Exemplo**

```xml
<interserver_https_host>example.clickhouse.com</interserver_https_host>
```

<div id="interserver_https_port">
  ## interserver_https_port
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />

Porta para a troca de dados entre servidores ClickHouse via `<HTTPS>`.

**Exemplo**

```xml
<interserver_https_port>9010</interserver_https_port>
```
