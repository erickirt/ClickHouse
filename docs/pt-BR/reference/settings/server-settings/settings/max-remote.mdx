Estas configurações definem o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="max_remote_read_connections">
  ## max_remote_read_connections
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />Número máximo de conexões remotas de leitura mantidas abertas por `ReaderExecutor` para otimizar a leitura sequencial. 0 desativa a reutilização de conexões.

<div id="max_remote_read_network_bandwidth_for_server">
  ## max_remote_read_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

A velocidade máxima de transferência de dados pela rede, em bytes por segundo, para operações de leitura.

<Note>
  Um valor de `0` (padrão) significa sem limite.
</Note>

<div id="max_remote_write_network_bandwidth_for_server">
  ## max_remote_write_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

A velocidade máxima de troca de dados pela rede, em bytes por segundo, para gravação.

<Note>
  Um valor de `0` (padrão) significa sem limite.
</Note>
