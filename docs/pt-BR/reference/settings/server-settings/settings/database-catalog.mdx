Estas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="database_catalog_drop_error_cooldown_sec">
  ## database_catalog_drop_error_cooldown_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="5" changeable_without_restart="No" />Em caso de falha ao remover uma tabela, o ClickHouse aguardará esse intervalo antes de tentar novamente a operação.

<div id="database_catalog_drop_table_concurrency">
  ## database_catalog_drop_table_concurrency
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="No" />O tamanho do threadpool usado para remover tabelas.

<div id="database_catalog_unused_dir_cleanup_period_sec">
  ## database_catalog_unused_dir_cleanup_period_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="86400" changeable_without_restart="No" />

Parâmetro de uma tarefa que limpa arquivos desnecessários do diretório `store/`.
Define o período de execução da tarefa.

<Note>
  Um valor de `0` significa &quot;nunca&quot;. O valor padrão corresponde a 1 dia.
</Note>

<div id="database_catalog_unused_dir_hide_timeout_sec">
  ## database_catalog_unused_dir_hide_timeout_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="3600" changeable_without_restart="No" />

Parâmetro de uma tarefa que limpa lixo do diretório `store/`.
Se algum subdiretório não for usado pelo clickhouse-server e esse diretório não tiver sido modificado nos últimos
[`database_catalog_unused_dir_hide_timeout_sec`](/pt-BR/reference/settings/server-settings/settings/database-catalog#database_catalog_unused_dir_hide_timeout_sec) segundos, a tarefa &quot;ocultará&quot; esse diretório
removendo todos os direitos de acesso. Isso também funciona para diretórios que o clickhouse-server não
espera ver dentro de `store/`.

<Note>
  Um valor de `0` significa &quot;imediatamente&quot;.
</Note>

<div id="database_catalog_unused_dir_rm_timeout_sec">
  ## database_catalog_unused_dir_rm_timeout_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="2592000" changeable_without_restart="No" />

Parâmetro de uma tarefa que limpa lixo do diretório `store/`.
Se algum subdiretório não for usado pelo clickhouse-server e tiver sido anteriormente &quot;ocultado&quot;
(consulte [database&#95;catalog&#95;unused&#95;dir&#95;hide&#95;timeout&#95;sec](/pt-BR/reference/settings/server-settings/settings/database-catalog#database_catalog_unused_dir_hide_timeout_sec))
e esse diretório não tiver sido modificado nos últimos
[`database_catalog_unused_dir_rm_timeout_sec`]/operations/server-configuration-parameters/settings#database&#95;catalog&#95;unused&#95;dir&#95;rm&#95;timeout&#95;sec) segundos, a tarefa removerá esse diretório.
Isso também funciona para diretórios que o clickhouse-server não
espera encontrar dentro de `store/`.

<Note>
  Um valor de `0` significa &quot;nunca&quot;. O valor padrão corresponde a 30 dias.
</Note>
