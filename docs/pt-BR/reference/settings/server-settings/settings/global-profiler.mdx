Estas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="global_profiler_cpu_time_period_ns">
  ## global_profiler_cpu_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />Período do temporizador do clock da CPU do global profiler (em nanossegundos). Defina o valor como 0 para desativar o global profiler de clock da CPU. O valor recomendado é de pelo menos 10000000 (100 vezes por segundo) para queries individuais ou 1000000000 (uma vez por segundo) para profiling em todo o cluster.

<div id="global_profiler_real_time_period_ns">
  ## global_profiler_real_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />Período do temporizador de tempo real do global profiler (em nanossegundos). Defina o valor como 0 para desativar o global profiler de tempo real. O valor recomendado é de pelo menos 10000000 (100 vezes por segundo) para consultas individuais ou 1000000000 (uma vez por segundo) para profiling em todo o cluster.
