Essas configurações servem para configurar o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="replicated_fetches_http_connection_timeout">
  ## replicated_fetches_http_connection_timeout
</div>

<SettingsInfoBlock type="Seconds" default_value="0" changeable_without_restart="No" />Tempo limite da conexão HTTP para requisições de busca de partes. Herdado do perfil padrão `http_connection_timeout` caso não seja definido explicitamente.

<div id="replicated_fetches_http_receive_timeout">
  ## replicated_fetches_http_receive_timeout
</div>

<SettingsInfoBlock type="Seconds" default_value="0" changeable_without_restart="No" />Tempo limite de recebimento HTTP para solicitações de busca de partes. Herdado do perfil padrão `http_receive_timeout` se não for definido explicitamente.

<div id="replicated_fetches_http_send_timeout">
  ## replicated_fetches_http_send_timeout
</div>

<SettingsInfoBlock type="Segundos" default_value="0" changeable_without_restart="No" />Tempo limite para envio HTTP das solicitações de busca de partes. Herdado do perfil padrão `http_send_timeout` se não for definido explicitamente.
