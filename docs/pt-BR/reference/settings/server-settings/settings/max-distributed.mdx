Essas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="max_distributed_cache_read_bandwidth_for_server">
  ## max_distributed_cache_read_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Sim" />A velocidade máxima total de leitura do cache distribuído no servidor, em bytes por segundo. Zero significa ilimitado.

<div id="max_distributed_cache_write_bandwidth_for_server">
  ## max_distributed_cache_write_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Sim" />A velocidade total máxima de gravação no cache distribuído do servidor, em bytes por segundo. Zero significa sem limite.
