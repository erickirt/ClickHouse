Essas configurações são do servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="os_collect_psi_metrics">
  ## os_collect_psi_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Habilita a coleta de métricas PSI dos arquivos /proc/pressure/.

<div id="os_cpu_busy_time_threshold">
  ## os_cpu_busy_time_threshold
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000" changeable_without_restart="No" />Limite de tempo de uso da CPU do SO, em microssegundos (métrica `OSCPUVirtualTimeMicroseconds`), para considerar que a CPU está realizando algum trabalho útil; nenhuma sobrecarga de CPU será considerada se o tempo de uso estiver abaixo desse valor.
