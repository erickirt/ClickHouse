</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Não" />Número máximo de background threads do jemalloc a serem criadas; defina como 0 para usar o valor padrão do jemalloc

<div id="jemalloc_profiler_sampling_rate">
  ## jemalloc_profiler_sampling_rate
</div>

<SettingsInfoBlock type="UInt64" default_value="19" changeable_without_restart="Não" />

Controla o `lg_prof_sample` do jemalloc — o logaritmo de base 2 do intervalo médio (em bytes) entre amostras de alocação.
O valor padrão de 19 corresponde a 512 KiB. Definir um valor menor aumenta a frequência de amostragem (mais sobrecarga, mais detalhes), e um valor maior a diminui.
Alterar esse valor chama `prof.reset`, o que zera todas as estatísticas de profiling acumuladas. Requer que o profiling esteja habilitado (`MALLOC_CONF=prof:true`).
