Essas configurações definem o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="max_pending_mutations_execution_time_to_warn">
  ## max_pending_mutations_execution_time_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="86400" changeable_without_restart="Sim" />

Se qualquer uma das mutações pendentes ultrapassar o valor especificado em segundos, o servidor ClickHouse adicionará mensagens de aviso à tabela `system.warnings`.

**Exemplo**

```xml
<max_pending_mutations_execution_time_to_warn>10000</max_pending_mutations_execution_time_to_warn>
```

<div id="max_pending_mutations_to_warn">
  ## max_pending_mutations_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="500" changeable_without_restart="Sim" />

Se o número de mutações pendentes exceder o valor especificado, o servidor ClickHouse adicionará mensagens de aviso à tabela `system.warnings`.

**Exemplo**

```xml
<max_pending_mutations_to_warn>400</max_pending_mutations_to_warn>
```
