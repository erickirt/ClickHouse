Estas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="graphite">
  ## graphite
</div>

Envio de dados para [Graphite](https://github.com/graphite-project).

Configurações:

* `host` – O servidor Graphite.
* `port` – A porta do servidor Graphite.
* `interval` – O intervalo de envio, em segundos.
* `timeout` – O timeout para envio de dados, em segundos.
* `root_path` – Prefixo das chaves.
* `metrics` – Envio de dados da tabela [system.metrics](/pt-BR/reference/system-tables/metrics).
* `events` – Envio de dados delta acumulados para o período da tabela [system.events](/pt-BR/reference/system-tables/events).
* `events_cumulative` – Envio de dados cumulativos da tabela [system.events](/pt-BR/reference/system-tables/events).
* `asynchronous_metrics` – Envio de dados da tabela [system.asynchronous&#95;metrics](/pt-BR/reference/system-tables/asynchronous_metrics).

Você pode configurar várias cláusulas `<graphite>`. Por exemplo, pode usar isso para enviar dados diferentes em intervalos diferentes.

**Exemplo**

```xml
<graphite>
    <host>localhost</host>
    <port>42000</port>
    <timeout>0.1</timeout>
    <interval>60</interval>
    <root_path>one_min</root_path>
    <metrics>true</metrics>
    <events>true</events>
    <events_cumulative>false</events_cumulative>
    <asynchronous_metrics>true</asynchronous_metrics>
</graphite>
```

<div id="graphite_rollup">
  ## graphite_rollup
</div>

Configurações para reduzir a granularidade dos dados do Graphite.

Para mais detalhes, consulte [GraphiteMergeTree](/pt-BR/reference/engines/table-engines/mergetree-family/graphitemergetree).

**Exemplo**

```xml
<graphite_rollup_example>
    <default>
        <function>max</function>
        <retention>
            <age>0</age>
            <precision>60</precision>
        </retention>
        <retention>
            <age>3600</age>
            <precision>300</precision>
        </retention>
        <retention>
            <age>86400</age>
            <precision>3600</precision>
        </retention>
    </default>
</graphite_rollup_example>
```
