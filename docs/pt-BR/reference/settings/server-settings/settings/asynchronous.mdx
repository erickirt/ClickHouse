<div id="asynchronous_insert_log">
  ## asynchronous_insert_log
</div>

Configurações da tabela de sistema [asynchronous&#95;insert&#95;log](/pt-BR/reference/system-tables/asynchronous_insert_log) para registrar inserts assíncronos.

<SystemLogParameters />

**Exemplo**

```xml
<clickhouse>
    <asynchronous_insert_log>
        <database>system</database>
        <table>asynchronous_insert_log</table>
        <flush_interval_milliseconds>7500</flush_interval_milliseconds>
        <partition_by>toYYYYMM(event_date)</partition_by>
        <max_size_rows>1048576</max_size_rows>
        <reserved_size_rows>8192</reserved_size_rows>
        <buffer_size_rows_flush_threshold>524288</buffer_size_rows_flush_threshold>
        <flush_on_crash>false</flush_on_crash>
        <!-- <engine>Engine = MergeTree PARTITION BY event_date ORDER BY event_time TTL event_date + INTERVAL 30 day</engine> -->
    </asynchronous_insert_log>
</clickhouse>
```

<div id="asynchronous_metric_log">
  ## asynchronous_metric_log
</div>

Habilitado por padrão nas implantações do ClickHouse Cloud.

Se essa configuração não estiver habilitada por padrão no seu ambiente, dependendo de como o ClickHouse foi instalado, você pode seguir a instrução abaixo para habilitá-la ou desabilitá-la.

**Habilitação**

Para ativar manualmente a coleta do histórico de métricas assíncronas em [`system.asynchronous_metric_log`](/pt-BR/reference/system-tables/asynchronous_metric_log), crie `/etc/clickhouse-server/config.d/asynchronous_metric_log.xml` com o seguinte conteúdo:

```xml
<clickhouse>
     <asynchronous_metric_log>
        <database>system</database>
        <table>asynchronous_metric_log</table>
        <flush_interval_milliseconds>7500</flush_interval_milliseconds>
        <collect_interval_milliseconds>1000</collect_interval_milliseconds>
        <max_size_rows>1048576</max_size_rows>
        <reserved_size_rows>8192</reserved_size_rows>
        <buffer_size_rows_flush_threshold>524288</buffer_size_rows_flush_threshold>
        <flush_on_crash>false</flush_on_crash>
    </asynchronous_metric_log>
</clickhouse>
```

**Desativando**

Para desativar a configuração `asynchronous_metric_log`, crie o arquivo `/etc/clickhouse-server/config.d/disable_asynchronous_metric_log.xml` com o seguinte conteúdo:

```xml
<clickhouse><asynchronous_metric_log remove="1" /></clickhouse>
```
