Essas configurações são usadas para configurar o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="dictionaries_config">
  ## dictionaries_config
</div>

O caminho para o arquivo de configuração dos dicionários.

Caminho:

* Especifique o caminho absoluto ou o caminho relativo ao arquivo de configuração do servidor.
* O caminho pode conter curingas * e ?.

Ver também:

* &quot;[Dictionaries](/pt-BR/reference/statements/create/dictionary)&quot;.

**Exemplo**

```xml
<dictionaries_config>*_dictionary.xml</dictionaries_config>
```

<div id="dictionaries_lazy_load">
  ## dictionaries_lazy_load
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />

Carregamento sob demanda de dicionários.

* Se `true`, cada dicionário é carregado na primeira utilização. Se o carregamento falhar, a função que estiver usando o dicionário lançará uma exceção.
* Se `false`, o servidor carrega todos os dicionários na inicialização.

<Note>
  O servidor aguardará, na inicialização, até que todos os dicionários concluam o carregamento antes de aceitar qualquer conexão
  (exceção: se [`wait_dictionaries_load_at_startup`](/pt-BR/reference/settings/server-settings/settings/other#wait_dictionaries_load_at_startup) estiver definido como `false`).
</Note>

**Exemplo**

```xml
<dictionaries_lazy_load>true</dictionaries_lazy_load>
```
