<div id="async_insert_queue_flush_on_shutdown">
  ## async_insert_queue_flush_on_shutdown
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="Não" />Se `true`, a fila de inserts assíncronos é esvaziada durante o desligamento controlado

<div id="async_insert_threads">
  ## async_insert_threads
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="No" />Número máximo de threads para de fato fazer o parsing e inserir dados em segundo plano. Zero significa que o modo assíncrono está desabilitado
