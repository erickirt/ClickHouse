Essas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="skip_binary_checksum_checks">
  ## skip_binary_checksum_checks
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Ignora as verificações de integridade do checksum do binário do ClickHouse

<div id="skip_check_for_incorrect_settings">
  ## skip_check_for_incorrect_settings
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />

Se definido como true, as configurações do servidor não serão verificadas para validar se estão corretas.

**Exemplo**

```xml
<skip_check_for_incorrect_settings>1</skip_check_for_incorrect_settings>
```
