Essas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="page_cache_free_memory_ratio">
  ## page_cache_free_memory_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.15" changeable_without_restart="No" />Fração do limite de memória que deve ser mantida livre, fora do cache de páginas no espaço do usuário. Análogo à configuração `min_free_kbytes` do Linux.

<div id="page_cache_history_window_ms">
  ## page_cache_history_window_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />Intervalo de espera antes que a memória liberada possa ser usada pelo cache de páginas no espaço do usuário.

<div id="page_cache_max_size">
  ## page_cache_max_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />Tamanho máximo do cache de páginas no espaço do usuário. Defina como 0 para desativar o cache. Se for maior que page&#95;cache&#95;min&#95;size, o tamanho do cache será ajustado continuamente dentro desse intervalo para usar a maior parte da memória disponível, mantendo o uso total de memória abaixo do limite (max&#95;server&#95;memory&#95;usage[&#95;to&#95;ram&#95;ratio]).

<div id="page_cache_min_size">
  ## page_cache_min_size
</div>

<SettingsInfoBlock type="UInt64" default_value="104857600" changeable_without_restart="No" />Tamanho mínimo do cache de páginas no espaço do usuário.

<div id="page_cache_policy">
  ## page_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Nome da política do cache de páginas no espaço do usuário.

<div id="page_cache_shards">
  ## page_cache_shards
</div>

<SettingsInfoBlock type="UInt64" default_value="4" changeable_without_restart="No" />Distribui o cache de páginas no espaço do usuário entre esta quantidade de shards para reduzir a contenção de mutex. Experimental, é improvável que melhore o desempenho.

<div id="page_cache_size_ratio">
  ## page_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />O tamanho da fila protegida no cache de páginas no espaço do usuário em relação ao tamanho total do cache.
