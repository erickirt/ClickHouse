Estas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="max_snapshot_commit_thread_pool_free_size">
  ## max_snapshot_commit_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Sim" />Se o número de threads ociosas no pool de threads de commit de snapshot exceder `max_snapshot_commit_thread_pool_free_size`, o ClickHouse liberará os recursos ocupados por essas threads e reduzirá o tamanho do pool. As threads poderão ser criadas novamente, se necessário.

<div id="max_snapshot_commit_thread_pool_size">
  ## max_snapshot_commit_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="64" changeable_without_restart="Yes" />O número de threads para fazer o commit do snapshot.
