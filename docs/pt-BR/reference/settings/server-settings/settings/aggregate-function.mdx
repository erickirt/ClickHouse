<div id="aggregate_function_group_array_action_when_limit_is_reached">
  ## aggregate_function_group_array_action_when_limit_is_reached
</div>

<SettingsInfoBlock type="GroupArrayActionWhenLimitReached" default_value="throw" changeable_without_restart="No" />Ação a executar quando o tamanho máximo do elemento do array for excedido em groupArray: lançar exceção `throw` ou descartar valores extras `discard`

<div id="aggregate_function_group_array_max_element_size">
  ## aggregate_function_group_array_max_element_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16777215" changeable_without_restart="No" />Tamanho máximo de um elemento do array, em bytes, para a função groupArray. Esse limite é verificado durante a serialização e ajuda a evitar um state muito grande.
