Essas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="threadpool_remote_fs_reader_pool_size">
  ## threadpool_remote_fs_reader_pool_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="250" changeable_without_restart="No" />Número de threads no pool de threads usado para leitura a partir do sistema de arquivos remoto quando `remote_filesystem_read_method = 'threadpool'`.

<div id="threadpool_remote_fs_reader_queue_size">
  ## threadpool_remote_fs_reader_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />O número máximo de jobs que podem ser agendados no pool de threads para leitura a partir do sistema de arquivos remoto.
