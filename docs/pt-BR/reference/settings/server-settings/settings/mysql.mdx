Estas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="mysql_port">
  ## mysql_port
</div>

Porta para comunicação com clientes pelo protocolo MySQL.

<Note>
  * Números inteiros positivos especificam o número da porta em que o serviço ficará escutando
  * Valores vazios são usados para desabilitar a comunicação com clientes pelo protocolo MySQL.
</Note>

**Exemplo**

```xml
<mysql_port>9004</mysql_port>
```

<div id="mysql_require_secure_transport">
  ## mysql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Se definido como true, exige comunicação segura com os clientes por [mysql&#95;port](/pt-BR/reference/settings/server-settings/settings/mysql#mysql_port). A conexão com a opção `<--ssl-mode=none>` será recusada. Use-o com as configurações do [OpenSSL](/pt-BR/reference/settings/server-settings/settings/other#openssl).
