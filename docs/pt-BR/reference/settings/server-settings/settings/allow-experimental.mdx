Estas configurações são usadas para configurar o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="allow_experimental_executable_udf_drivers">
  ## allow_experimental_executable_udf_drivers
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Não" />Habilita suporte experimental a drivers para funções executáveis definidas pelo usuário, declaradas por meio de `user_defined_executable_function_drivers_config`. Um driver converte um trecho de código fornecido pelo usuário em `CREATE FUNCTION ... ENGINE = DriverName(...) AS '...'` em uma UDF executável.

<div id="allow_experimental_webassembly_udf">
  ## allow_experimental_webassembly_udf
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Habilita o suporte experimental a UDFs de WebAssembly

<div id="allow_experimental_webterminal">
  ## allow_experimental_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="Não" />Nome anterior (experimental) de `enable_webterminal`. Ainda é aceito por compatibilidade retroativa quando `enable_webterminal` não está definido. Esta configuração está obsoleta.
