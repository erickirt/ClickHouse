Essas configurações definem o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="tcp_close_connection_after_queries_num">
  ## tcp_close_connection_after_queries_num
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />Número máximo de consultas permitidas por conexão TCP antes de a conexão ser fechada. Defina como 0 para consultas ilimitadas.

<div id="tcp_close_connection_after_queries_seconds">
  ## tcp_close_connection_after_queries_seconds
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />Tempo máximo de uma conexão TCP, em segundos, antes de ser fechada. Defina como 0 para duração ilimitada da conexão.
