Essas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="logger.console">
  ## logger.console
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Habilita o logging no console. Defina como `<1>` ou `<true>` para ativar. O padrão é `<1>` se o ClickHouse não estiver em execução no modo daemon; caso contrário, `<0>`.

<div id="logger.console_log_level">
  ## logger.console_log_level
</div>

<SettingsInfoBlock type="String" default_value="trace" changeable_without_restart="Não" />Nível de log para saída no console. O padrão é `<level>`.
