Essas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="threadpool_local_fs_reader_pool_size">
  ## threadpool_local_fs_reader_pool_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="100" changeable_without_restart="No" />O número de threads no pool de threads para leitura do sistema de arquivos local quando `local_filesystem_read_method = 'pread_threadpool'`.

<div id="threadpool_local_fs_reader_queue_size">
  ## threadpool_local_fs_reader_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />O número máximo de jobs que podem ser agendados no pool de threads para ler do sistema de arquivos local.
