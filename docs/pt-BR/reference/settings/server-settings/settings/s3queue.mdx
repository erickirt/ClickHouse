Essas configurações configuram o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="s3queue_disable_streaming">
  ## s3queue_disable_streaming
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Yes" />Desabilita o streaming no S3Queue mesmo que a tabela tenha sido criada e existam views materializadas anexadas

<div id="s3queue_log">
  ## s3queue_log
</div>

Configurações da tabela de sistema `s3queue_log`.

<SystemLogParameters />

As configurações padrão são:

```xml
<s3queue_log>
    <database>system</database>
    <table>s3queue_log</table>
    <partition_by>toYYYYMM(event_date)</partition_by>
    <flush_interval_milliseconds>7500</flush_interval_milliseconds>
</s3queue_log>
```
