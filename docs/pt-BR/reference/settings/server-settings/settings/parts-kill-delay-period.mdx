Estas configurações definem o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="parts_kill_delay_period">
  ## parts_kill_delay_period
</div>

<SettingsInfoBlock type="UInt64" default_value="30" changeable_without_restart="No" />

Período para a remoção completa de partes no SharedMergeTree. Disponível apenas no ClickHouse Cloud

<div id="parts_kill_delay_period_random_add">
  ## parts_kill_delay_period_random_add
</div>

<SettingsInfoBlock type="UInt64" default_value="10" changeable_without_restart="No" />

Adiciona a kill&#95;delay&#95;period um valor com distribuição uniforme de 0 a x segundos para evitar o efeito de thundering herd e o consequente DoS do ZooKeeper em caso de um número muito grande de tabelas. Disponível apenas no ClickHouse Cloud
