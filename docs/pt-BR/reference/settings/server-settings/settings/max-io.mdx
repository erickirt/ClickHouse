Estas configurações são usadas para configurar o servidor ClickHouse e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="max_io_thread_pool_free_size">
  ## max_io_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Sim" />

Se o número de threads **ociosas** no pool de threads de E/S exceder `max_io_thread_pool_free_size`, o ClickHouse liberará os recursos usados pelas threads inativas e reduzirá o tamanho do pool. As threads poderão ser criadas novamente, se necessário.

<div id="max_io_thread_pool_size">
  ## max_io_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="100" changeable_without_restart="Sim" />

O ClickHouse usa threads do pool de threads de E/S para executar algumas operações de E/S (por exemplo, para interagir com o S3). `max_io_thread_pool_size` limita o número máximo de threads no pool.
