  ## max_parser_depth
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" />

Limita a profundidade máxima de recursão no parser de descida recursiva. Permite controlar o tamanho da pilha.

Valores possíveis:

* Inteiro positivo.
* 0 — A profundidade de recursão é ilimitada.
