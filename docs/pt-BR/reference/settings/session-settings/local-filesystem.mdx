<div id="local_filesystem_read_method">
  ## local_filesystem_read_method
</div>

<SettingsInfoBlock type="String" default_value="pread_threadpool" />

Método de leitura de dados do sistema de arquivos local; um de: read, pread, mmap, io&#95;uring, pread&#95;threadpool.

O método &#39;io&#95;uring&#39; é experimental e não funciona com Log, TinyLog, StripeLog, File, Set e Join, nem com outras tabelas com arquivos aos quais é possível acrescentar dados, na presença de leituras e gravações concorrentes.
Se você ler vários artigos sobre &#39;io&#95;uring&#39; na internet, não se deixe impressionar por eles. Não é um método melhor para leitura de arquivos, exceto no caso de uma grande quantidade de pequenas solicitações de E/S, o que não ocorre no ClickHouse. Não há motivo para habilitar &#39;io&#95;uring&#39;.

<div id="local_filesystem_read_prefetch">
  ## local_filesystem_read_prefetch
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Deve usar prefetching ao ler dados do sistema de arquivos local.
