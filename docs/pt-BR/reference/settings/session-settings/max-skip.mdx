<div id="max_skip_unavailable_shards_num">
  ## max_skip_unavailable_shards_num
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.3"},{"label": "0"},{"label": "Nova configuração para limitar o número de shards que podem ser ignorados sem aviso quando skip_unavailable_shards estiver habilitado."}]}]} />

Quando `skip_unavailable_shards` está habilitado, limita o número máximo de shards que podem ser ignorados sem aviso.
Se o número de shards indisponíveis exceder esse valor, uma exceção é lançada em vez de serem ignorados sem aviso.

Um valor de 0 significa que não há limite (comportamento padrão — todos os shards indisponíveis podem ser ignorados).

<div id="max_skip_unavailable_shards_ratio">
  ## max_skip_unavailable_shards_ratio
</div>

<SettingsInfoBlock type="Float" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.3"},{"label": "0"},{"label": "Nova configuração para limitar a razão de shards que podem ser ignorados silenciosamente quando skip_unavailable_shards está habilitado."}]}]} />

Quando `skip_unavailable_shards` está habilitado, limita a razão máxima (de 0 a 1) de shards que podem ser ignorados silenciosamente.
Se a razão entre shards indisponíveis e o total de shards exceder esse valor, uma exceção é lançada em vez de serem ignorados silenciosamente.

Um valor de 0 significa que não há limite (comportamento padrão — todos os shards indisponíveis podem ser ignorados).
