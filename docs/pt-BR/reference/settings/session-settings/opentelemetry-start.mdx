Probabilidade de iniciar um rastreamento para uma solicitação ao ZooKeeper, com ou sem um rastreamento pai.

Valores possíveis:

* &#39;auto&#39; - Equivale à configuração opentelemetry&#95;start&#95;trace&#95;probability
* 0 — O rastreamento está desabilitado
* 0 a 1 — Probabilidade (por exemplo, 1.0 = sempre habilitado)

<div id="opentelemetry_start_trace_probability">
  ## opentelemetry_start_trace_probability
</div>

<SettingsInfoBlock type="Float" default_value="0" />

Define a probabilidade de o ClickHouse iniciar um rastreamento para consultas executadas (se nenhum [contexto de rastreamento](https://www.w3.org/TR/trace-context/) pai for fornecido).

Valores possíveis:

* 0 — O rastreamento para todas as consultas executadas é desabilitado (se nenhum contexto de rastreamento pai for fornecido).
* Número de ponto flutuante positivo no intervalo [0..1]. Por exemplo, se o valor da configuração for `0,5`, o ClickHouse pode iniciar um rastreamento, em média, para metade das consultas.
* 1 — O rastreamento para todas as consultas executadas é habilitado.
