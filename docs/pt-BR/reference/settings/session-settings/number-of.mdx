<div id="number_of_mutations_to_delay">
  ## number_of_mutations_to_delay
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Se a tabela alterada contiver pelo menos esse número de mutações não concluídas, reduza artificialmente a velocidade das mutações da tabela. 0 - desabilitado

<div id="number_of_mutations_to_throw">
  ## number_of_mutations_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Se a tabela alterada contiver pelo menos essa quantidade de mutações não concluídas, a exceção &#39;Too many mutations ...&#39; será lançada. 0 - desabilitado
