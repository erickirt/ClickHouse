<div id="compatibility">
  ## compatibility
</div>

A configuração `compatibility` faz com que o ClickHouse use as configurações padrão de uma versão anterior do ClickHouse, especificada no valor da própria configuração.

Se houver configurações definidas com valores diferentes do padrão, esses valores serão mantidos (somente as configurações que não foram modificadas são afetadas pela configuração `compatibility`).

Essa configuração recebe um número de versão do ClickHouse como string, como `22.3` ou `22.8`. Um valor vazio significa que essa configuração está desabilitada.

Desabilitada por padrão.

<Note>
  No ClickHouse Cloud, a configuração de compatibilidade padrão no nível do serviço deve ser definida pelo ClickHouse Cloud Support. Abra [um chamado](https://clickhouse.cloud/support) para solicitar essa definição.
  No entanto, a configuração `compatibility` pode ser substituída nos níveis de usuário, função, perfil, consulta ou sessão usando os mecanismos padrão de configuração do ClickHouse, como `SET compatibility = '22.3'` em uma sessão ou `SETTINGS compatibility = '22.3'` em uma consulta.
</Note>
