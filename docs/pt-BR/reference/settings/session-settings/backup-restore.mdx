Se um host, durante uma operação BACKUP ON CLUSTER ou RESTORE ON CLUSTER, não recriar seu nó efêmero &#39;alive&#39; no ZooKeeper dentro desse período, todo o backup ou restauração será considerado falho.
Esse valor deve ser maior do que qualquer tempo razoável para um host se reconectar ao ZooKeeper após uma falha.
Zero significa ilimitado.
