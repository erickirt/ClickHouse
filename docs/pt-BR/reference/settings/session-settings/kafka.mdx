<div id="kafka_disable_num_consumers_limit">
  ## kafka_disable_num_consumers_limit
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Desativa o limite de kafka&#95;num&#95;consumers, que depende do número de núcleos de CPU disponíveis.

<div id="kafka_max_wait_ms">
  ## kafka_max_wait_ms
</div>

<SettingsInfoBlock type="Milissegundos" default_value="5000" />

O tempo de espera, em milissegundos, para ler mensagens do [Kafka](/pt-BR/reference/engines/table-engines/integrations/kafka) antes de uma nova tentativa.

Valores possíveis:

* Inteiro positivo.
* 0 — timeout infinito.
