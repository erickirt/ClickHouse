<div id="timeout_overflow_mode">
  ## timeout_overflow_mode
</div>

<SettingsInfoBlock type="OverflowMode" default_value="throw" />

Define o que fazer caso a consulta seja executada por mais tempo do que `max_execution_time` ou caso o
tempo estimado de execução seja maior que `max_estimated_execution_time`.

Valores possíveis:

* `throw`: lança uma exceção (padrão).
* `break`: interrompe a execução da consulta e retorna o resultado parcial, como se os
  dados de origem tivessem se esgotado.

<div id="timeout_overflow_mode_leaf">
  ## timeout_overflow_mode_leaf
</div>

<SettingsInfoBlock type="OverflowMode" default_value="throw" />

Define o que acontece quando a consulta em um nó leaf é executada por mais tempo que `max_execution_time_leaf`.

Valores possíveis:

* `throw`: lança uma exceção (padrão).
* `break`: interrompe a execução da consulta e retorna o resultado parcial, como se os
  dados de origem tivessem se esgotado.
