<div id="max_download_buffer_size">
  ## max_download_buffer_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10485760" />

O tamanho máximo do buffer para downloads paralelos (por exemplo, para o engine URL) por thread.

<div id="max_download_threads">
  ## max_download_threads
</div>

<SettingsInfoBlock type="MaxThreads" default_value="4" />

O número máximo de threads para download de dados (por exemplo, para o engine URL).
