  ## materialize_skip_indexes_on_insert
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "24.6"},{"label": "1"},{"label": "Adicionada nova configuração para permitir desabilitar a materialização de skip indexes no insert"}]}]} />

Define se os `INSERT`s criam e armazenam skip indexes. Se desabilitado, os skip indexes só serão criados e armazenados [durante merges](/pt-BR/reference/settings/merge-tree-settings#materialize_skip_indexes_on_merge) ou por [MATERIALIZE INDEX](/pt-BR/reference/statements/alter/skipping-index#materialize-index) explícito.
