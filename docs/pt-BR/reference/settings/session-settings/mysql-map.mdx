  ## mysql_map_fixed_string_to_text_in_show_columns
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "24.2"},{"label": "1"},{"label": "Reduz o esforço de configuração para conectar o ClickHouse a ferramentas de BI."}]}]} />

Quando ativada, o tipo de dado [FixedString](/pt-BR/reference/data-types/fixedstring) do ClickHouse será exibido como `TEXT` em [SHOW COLUMNS](/pt-BR/reference/statements/show#show_columns).

Tem efeito apenas quando a conexão é feita por meio do protocolo wire do MySQL.

* 0 - Use `BLOB`.
* 1 - Use `TEXT`.
