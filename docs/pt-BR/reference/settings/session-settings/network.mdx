<div id="network_compression_method">
  ## network_compression_method
</div>

<SettingsInfoBlock type="String" default_value="LZ4" />

O codec usado para comprimir a comunicação cliente/servidor e servidor/servidor.

Valores possíveis:

* `NONE` — sem compressão.
* `LZ4` — usa o codec LZ4.
* `LZ4HC` — usa o codec LZ4HC.
* `ZSTD` — usa o codec ZSTD.

**Veja também**

* [network&#95;zstd&#95;compression&#95;level](/pt-BR/reference/settings/session-settings/network#network_zstd_compression_level)

<div id="network_zstd_compression_level">
  ## network_zstd_compression_level
</div>

<SettingsInfoBlock type="Int64" default_value="1" />

Ajusta o nível de compressão ZSTD. Usado somente quando [network&#95;compression&#95;method](/pt-BR/reference/settings/session-settings/network#network_compression_method) está definido como `ZSTD`.

Valores possíveis:

* Número inteiro positivo de 1 a 15.
