Estas configurações estão disponíveis em [system.settings](/pt-BR/reference/system-tables/settings) e são geradas automaticamente a partir do [código-fonte](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp).

<div id="optimize_if_chain_to_multiif">
  ## optimize_if_chain_to_multiif
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Substitui cadeias de `if(cond1, then1, if(cond2, ...))` por `multiIf`. No momento, isso não é vantajoso para tipos numéricos.

<div id="optimize_if_transform_strings_to_enum">
  ## optimize_if_transform_strings_to_enum
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Substitui argumentos do tipo String em If e Transform por enum. Desabilitado por padrão, pois isso pode causar uma alteração inconsistente em consulta distribuída, o que levaria à falha dela.
