  ## use_paimon_metadata_files_cache
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "0"},{"label": "Nova configuração para habilitar o cache em memória de arquivos de metadados do Paimon convertidos (manifest lists e manifests). Para motores de tabela Paimon persistentes, ela deve ser habilitada antes da inicialização dos metadados; as funções de tabela a avaliam por consulta. Evita downloads repetidos e a desserialização de arquivos de metadados do armazenamento de objetos em consultas subsequentes."}]}]} />

Se ativada, a função de tabela paimon e o storage paimon podem usar o cache de arquivos de metadados do Paimon.

As funções de tabela Paimon avaliam essa configuração por consulta, enquanto os motores de tabela Paimon persistentes a definem na inicialização dos metadados: para mudar essa decisão em uma tabela já criada, ela deve ser removida e recriada.

Valores possíveis:

* 0 - Desabilitado
* 1 - Habilitado
