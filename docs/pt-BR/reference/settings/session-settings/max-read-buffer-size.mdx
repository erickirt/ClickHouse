Essas configurações estão disponíveis em [system.settings](/pt-BR/reference/system-tables/settings) e são geradas automaticamente a partir do [código-fonte](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp).

<div id="max_read_buffer_size">
  ## max_read_buffer_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="1048576" />

O tamanho máximo do buffer usado para ler do sistema de arquivos. Valores acima de 256 MiB são limitados a 256 MiB, pois um buffer de leitura nunca precisa ser maior.

<div id="max_read_buffer_size_local_fs">
  ## max_read_buffer_size_local_fs
</div>

<SettingsInfoBlock type="UInt64" default_value="131072" />

O tamanho máximo do buffer de leitura do sistema de arquivos local. Se definido como 0, `max_read_buffer_size` será usado. Valores acima de 256 MiB são limitados a 256 MiB, pois um buffer de leitura nunca precisa ser maior.

<div id="max_read_buffer_size_remote_fs">
  ## max_read_buffer_size_remote_fs
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

O tamanho máximo do buffer de leitura do sistema de arquivos remoto. Se definido como 0, será usado o max&#95;read&#95;buffer&#95;size. Valores acima de 256 MiB são limitados a 256 MiB, pois um buffer de leitura nunca precisa ser maior.
