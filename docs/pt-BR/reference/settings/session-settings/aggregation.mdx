<div id="aggregation_in_order_max_block_bytes">
  ## aggregation_in_order_max_block_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="50000000" />

Tamanho máximo, em bytes, do bloco acumulado durante a aggregation in order da chave primária. Um bloco menor permite paralelizar melhor o estágio final de merge da agregação.

<div id="aggregation_memory_efficient_merge_threads">
  ## aggregation_memory_efficient_merge_threads
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Número de threads a serem usadas para mesclar, no modo com uso eficiente de memória, resultados intermediários de agregação. Quanto maior o valor, mais memória é consumida. 0 significa o mesmo que &#39;max&#95;threads&#39;.
