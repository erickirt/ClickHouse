<div id="stream_flush_interval_ms">
  ## stream_flush_interval_ms
</div>

<SettingsInfoBlock type="Milissegundos" default_value="7500" />

Aplica-se a tabelas com streaming em caso de timeout ou quando uma thread gera [max&#95;insert&#95;block&#95;size](/pt-BR/reference/settings/session-settings/max-insert#max_insert_block_size) linhas.

O valor padrão é 7500.

Quanto menor o valor, mais frequentemente os dados são gravados na tabela. Definir um valor muito baixo prejudica o desempenho.

<div id="stream_poll_timeout_ms">
  ## stream_poll_timeout_ms
</div>

<SettingsInfoBlock type="Milissegundos" default_value="500" />

Tempo limite para fazer polling de dados de/para armazenamentos de streaming.
