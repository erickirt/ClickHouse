<div id="cloud_mode_engine">
  ## cloud_mode_engine
</div>

<SettingsInfoBlock type="UInt64" default_value="1" />

A família de engines permitida no Cloud.

* 0 - permitir tudo
* 1 - reescrever DDLs para usar *ReplicatedMergeTree
* 2 - reescrever DDLs para usar SharedMergeTree
* 3 - reescrever DDLs para usar SharedMergeTree, exceto quando um disco remoto for especificado explicitamente
* 4 - igual ao 3, além de usar Alias em vez de Distributed (a tabela Alias apontará para a tabela de destino da tabela Distributed, assim usará a tabela local correspondente)

UInt64 para minimizar a parte pública

Valor padrão na Cloud: `2`.
