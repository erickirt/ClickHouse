<div id="database_atomic_wait_for_drop_and_detach_synchronously">
  ## database_atomic_wait_for_drop_and_detach_synchronously
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Adiciona o modificador `SYNC` a todas as consultas `DROP` e `DETACH`.

Valores possíveis:

* 0 — As consultas serão executadas com atraso.
* 1 — As consultas serão executadas sem atraso.
