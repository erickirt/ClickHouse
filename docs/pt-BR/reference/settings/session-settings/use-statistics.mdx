Usa estatísticas para filtrar partes durante a execução da consulta.

Quando habilitada, a eliminação de partes em consultas SELECT usa estatísticas de colunas (por exemplo, estatísticas MinMax) para descartar partes que não podem conter dados correspondentes antes que qualquer dado seja lido.

Possible values:

* 0 — Desabilitado.
* 1 — Habilitado.
