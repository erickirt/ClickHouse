Essas configurações estão disponíveis em [system.settings](/pt-BR/reference/system-tables/settings) e são geradas automaticamente a partir do [código-fonte](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp).

<div id="external_storage_connect_timeout_sec">
  ## external_storage_connect_timeout_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="10" />

Tempo limite para conexão, em segundos. No momento, compatível apenas com MySQL

<div id="external_storage_max_read_bytes">
  ## external_storage_max_read_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Limita o número máximo de bytes quando a tabela com mecanismo externo deve gravar os dados do histórico. Atualmente, há suporte apenas para o mecanismo de tabela MySQL, o mecanismo de banco de dados e o Dicionário. Se for igual a 0, esta configuração ficará desabilitada

<div id="external_storage_max_read_rows">
  ## external_storage_max_read_rows
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Limita o número máximo de linhas quando uma tabela com mecanismo externo deve gravar os dados do histórico. Atualmente, há suporte apenas para mecanismo de tabela MySQL, mecanismo de banco de dados e Dicionário. Se for igual a 0, essa configuração será desabilitada

<div id="external_storage_rw_timeout_sec">
  ## external_storage_rw_timeout_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="300" />

Tempo limite de leitura/gravação em segundos. No momento, compatível apenas com MySQL
