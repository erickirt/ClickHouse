<div id="update_parallel_mode">
  ## update_parallel_mode
</div>

<SettingsInfoBlock type="UpdateParallelMode" default_value="auto" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.5"},{"label": "auto"},{"label": "Nova configuração"}]}]} />

Determina o comportamento de consultas `UPDATE` concorrentes.

Valores possíveis:

* `sync` - execute todas as consultas `UPDATE` sequencialmente.
* `auto` - execute sequencialmente apenas as consultas `UPDATE` com dependências entre colunas atualizadas em uma consulta e colunas usadas nas expressões de outra consulta.
* `async` - não sincronize as consultas `UPDATE`.
