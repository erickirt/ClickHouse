<div id="receive_data_timeout_ms">
  ## receive_data_timeout_ms
</div>

<SettingsInfoBlock type="Milissegundos" default_value="2000" />

Tempo limite de conexão para o recebimento do primeiro pacote de dados ou de um pacote com progresso positivo da réplica

<div id="receive_timeout">
  ## receive_timeout
</div>

<SettingsInfoBlock type="Segundos" default_value="300" />

Tempo limite para receber dados da rede, em segundos. Se nenhum byte for recebido nesse intervalo, uma exceção será lançada. Se você definir essa configuração no cliente, o `send_timeout` do socket também será definido na outra ponta da conexão correspondente no servidor.
