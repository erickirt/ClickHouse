<SettingsInfoBlock type="UInt64" default_value="30" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "30"},{"label": "Nova configuração para controlar a compactação de manifestos para tabelas Iceberg."}]}]} />

Número mínimo de arquivos de manifesto necessário para acionar a compactação apenas de manifestos via OPTIMIZE TABLE ... MANIFEST.
Se o número atual de arquivos de manifesto for menor ou igual a esse limite, a compactação será ignorada.
Exige que `allow_experimental_iceberg_compaction` esteja habilitado.
