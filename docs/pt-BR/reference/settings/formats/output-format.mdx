<div id="output_format_float_precision">

Quando diferente de zero, formata a saída de valores de ponto flutuante (`Float32`, `Float64`, `BFloat16`) com no máximo esta quantidade de dígitos após o separador decimal (os zeros à direita são removidos).

Intervalo válido: de 0 a 100.
