Essas configurações são geradas automaticamente a partir do [código-fonte](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/FormatFactorySettings.h).

<div id="type_json_allow_duplicated_key_with_literal_and_nested_object">
  ## type_json_allow_duplicated_key_with_literal_and_nested_object
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Quando habilitada, a análise de JSONs como `{"a" : 42, "a" : {"b" : 42}}`, em que alguma chave é duplicada, mas uma delas é um objeto aninhado, é permitida.

<div id="type_json_skip_duplicated_paths">
  ## type_json_skip_duplicated_paths
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Quando habilitado, durante o parsing de um objeto JSON para o tipo JSON, caminhos duplicados serão ignorados e apenas o primeiro será inserido, sem gerar uma exceçã

<div id="type_json_skip_invalid_typed_paths">
  ## type_json_skip_invalid_typed_paths
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Quando ativada, os campos com valores que não podem ser convertidos para o tipo declarado em colunas do tipo JSON com caminhos tipados são ignorados em vez de gerar um erro. Os campos ignorados são tratados como ausentes e usarão valores padrão/NULL com base na definição do caminho tipado.

Essa configuração se aplica apenas a colunas do tipo JSON (por exemplo, JSON(a Int64, b String)) em que caminhos específicos têm tipos declarados. Ela não se aplica a formatos de entrada JSON comuns, como JSONEachRow, ao inserir dados em colunas tipadas comuns.

Valores possíveis:

* 0 — Desabilitar (gerar erro em caso de incompatibilidade de tipo).
* 1 — Habilitar (ignorar o campo em caso de incompatibilidade de tipo).

<div id="type_json_use_partial_match_to_skip_paths_by_regexp">
  ## type_json_use_partial_match_to_skip_paths_by_regexp
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Quando ativada, durante a análise de um objeto JSON como tipo JSON, as expressões regulares especificadas com SKIP REGEXP exigirão correspondência parcial para ignorar um caminho. Quando desativada, será exigida correspondência completa.
