<div id="enable_mixed_granularity_parts">
  ## enable_mixed_granularity_parts
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Ativa ou desativa a transição para o controle do tamanho do grânulo com a
configuração `index_granularity_bytes`. Antes da versão 19.11, existia apenas a
configuração `index_granularity` para restringir o tamanho do grânulo. A
configuração `index_granularity_bytes` melhora o desempenho do ClickHouse ao
selecionar dados de tabelas com linhas grandes (dezenas ou centenas de megabytes).
Se você tiver tabelas com linhas grandes, poderá ativar essa configuração para as tabelas
a fim de melhorar a eficiência das consultas `SELECT`.
