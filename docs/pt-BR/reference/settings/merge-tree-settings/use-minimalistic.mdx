Essas configurações estão disponíveis em [system.merge&#95;tree&#95;settings](/pt-BR/reference/system-tables/merge_tree_settings) e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="use_minimalistic_checksums_in_zookeeper">
  ## use_minimalistic_checksums_in_zookeeper
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Use o formato pequeno (dezenas de bytes) para os checksum de partes no ZooKeeper em vez dos
comuns (dezenas de KB). Antes de habilitar, verifique se todas as réplicas oferecem suporte ao
novo formato.

<div id="use_minimalistic_part_header_in_zookeeper">
  ## use_minimalistic_part_header_in_zookeeper
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Método de armazenamento dos cabeçalhos das partes de dados no ZooKeeper. Se ativado, o ZooKeeper
armazena menos dados. Para mais detalhes, consulte [aqui](/pt-BR/reference/settings/server-settings/settings#use_minimalistic_part_header_in_zookeeper).
