<div id="zero_copy_concurrent_part_removal_max_postpone_ratio">
  ## zero_copy_concurrent_part_removal_max_postpone_ratio
</div>

<SettingsInfoBlock type="Float" default_value="0.05" />

Percentual máximo de partes de nível superior cuja remoção deve ser adiada para obter
intervalos independentes menores. Recomenda-se não alterar.

<div id="zero_copy_concurrent_part_removal_max_split_times">
  ## zero_copy_concurrent_part_removal_max_split_times
</div>

<SettingsInfoBlock type="UInt64" default_value="5" />

Profundidade máxima de recursão para dividir intervalos independentes de partes desatualizadas em
subintervalos menores. Não é recomendável alterar.

<div id="zero_copy_merge_mutation_min_parts_size_sleep_before_lock">
  ## zero_copy_merge_mutation_min_parts_size_sleep_before_lock
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" />

Se a replicação zero copy estiver habilitada, aguarde um intervalo aleatório antes de tentar adquirir o bloqueio, dependendo do tamanho das partes para merge ou mutation

<div id="zero_copy_merge_mutation_min_parts_size_sleep_no_scale_before_lock">
  ## zero_copy_merge_mutation_min_parts_size_sleep_no_scale_before_lock
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.3"},{"label": "0"},{"label": "Nova configuração"}]}]} />

Se a replicação zero copy estiver habilitada, aguarde um intervalo aleatório de até 500 ms
antes de tentar adquirir o bloqueio para merge ou mutation.
