<div id="cache_populated_by_fetch">
  ## cache_populated_by_fetch
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<Note>
  Esta configuração se aplica apenas ao ClickHouse Cloud.
</Note>

Quando `cache_populated_by_fetch` está desabilitada (configuração padrão), novas
partes de dados são carregadas no cache do sistema de arquivos somente quando é executada uma consulta que exige
essas partes.

Se estiver habilitada, `cache_populated_by_fetch` fará com que todos os nós carreguem
novas partes de dados do armazenamento para o cache do sistema de arquivos, sem precisar de uma consulta
para acionar essa ação.
