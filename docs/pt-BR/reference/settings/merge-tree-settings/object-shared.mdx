  ## object_shared_data_serialization_version
</div>

<SettingsInfoBlock type="MergeTreeObjectSharedDataSerializationVersion" default_value="advanced" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.8"},{"label": "map"},{"label": "Adiciona uma configuração para controlar as versões de serialização JSON"}]}, {"id": "row-2","items": [{"label": "25.12"},{"label": "advanced"},{"label": "Habilita por padrão a versão avançada de serialização de dados compartilhados"}]}]} />

Versão de serialização para dados compartilhados no tipo de dados JSON.

Valores possíveis:

* `map` - armazena dados compartilhados como `Map(String, String)`
* `map_with_buckets` - armazena dados compartilhados como várias colunas `Map(String, String)` separadas. O uso de buckets melhora a leitura de paths individuais dos dados compartilhados.
* `advanced` - serialização especial de dados compartilhados projetada para melhorar significativamente a leitura de paths individuais dos dados compartilhados.
  Observe que essa serialização aumenta o tamanho em disco dos dados compartilhados, porque armazenamos muitas informações adicionais.

O número de buckets para as serializações `map_with_buckets` e `advanced` é determinado pelas configurações
[object&#95;shared&#95;data&#95;buckets&#95;for&#95;compact&#95;part](/pt-BR/reference/settings/merge-tree-settings/object-shared#object_shared_data_buckets_for_compact_part)/[object&#95;shared&#95;data&#95;buckets&#95;for&#95;wide&#95;part](/pt-BR/reference/settings/merge-tree-settings/object-shared#object_shared_data_buckets_for_wide_part).

  ## object_shared_data_serialization_version_for_zero_level_parts

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.8"},{"label": "map"},{"label": "Adicionar uma configuração para controlar as versões de serialização JSON para partes de nível zero"}]}, {"id": "row-2","items": [{"label": "25.12"},{"label": "map_with_buckets"},{"label": "Habilitar por padrão a versão de serialização de dados compartilhados map_with_buckets para partes de nível zero"}]}]} />

Essa configuração permite especificar uma versão de serialização diferente dos
dados compartilhados no tipo JSON para partes de nível zero criadas durante inserções.
Recomenda-se não usar a serialização de dados compartilhados `advanced` para partes de nível zero, porque isso pode aumentar
significativamente o tempo de inserção.
