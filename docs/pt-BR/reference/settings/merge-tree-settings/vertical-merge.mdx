algoritmo de mesclagem vertical.

  ## vertical_merge_algorithm_min_columns_to_activate
</div>

<SettingsInfoBlock type="UInt64" default_value="11" />

Número mínimo de colunas não PK para ativar o algoritmo de mesclagem vertical.

<div id="vertical_merge_algorithm_min_rows_to_activate">
  ## vertical_merge_algorithm_min_rows_to_activate
</div>

<SettingsInfoBlock type="UInt64" default_value="131072" />

Soma mínima (aproximada) de linhas nas
partes em mesclagem para ativar o algoritmo de mesclagem vertical.

<div id="vertical_merge_optimize_ttl_delete">
  ## vertical_merge_optimize_ttl_delete
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.3"},{"label": "1"},{"label": "Permitir o algoritmo de mesclagem vertical para mesclagens que precisam remover linhas expiradas por TTL"}]}]} />
