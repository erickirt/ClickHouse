<div id="allow_experimental_replacing_merge_with_cleanup">
  ## allow_experimental_replacing_merge_with_cleanup
</div>

<ExperimentalBadge />

<SettingsInfoBlock type="Bool" default_value="0" />

Permite merges experimentais de CLEANUP para ReplacingMergeTree com a
coluna `is_deleted`. Quando habilitada, essa configuração permite usar `OPTIMIZE ... FINAL CLEANUP` para
mesclar manualmente todas as partes de uma partição em uma única parte e remover
quaisquer linhas excluídas.

Também permite habilitar a execução automática desses merges em segundo plano
com as configurações `min_age_to_force_merge_seconds`,
`min_age_to_force_merge_on_partition_only` e
`enable_replacing_merge_with_cleanup_for_min_age_to_force_merge`.
