<div id="add_minmax_index_for_block_number_column">
  ## add_minmax_index_for_block_number_column
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.5"},{"label": "0"},{"label": "Nova configuração."}]}]} />

Quando habilitada, um índice min-max implícito (skipping) é adicionado à coluna virtual persistente `_block_number`.
Requer `enable_block_number_column = 1` para ter efeito. O índice é construído apenas durante operações de mesclagem,
não durante inserções: no momento da inserção, o número do bloco é provisório e indexaria uma constante.

Quando ativada, um índice min-max implícito (skipping) é adicionado à coluna virtual persistente `_block_offset`.
Requer `enable_block_offset_column = 1` para ter efeito. O índice é criado apenas durante os merges,
não durante as inserções.
