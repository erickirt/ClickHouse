<div id="min_rows_for_compact_part">
  ## min_rows_for_compact_part
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Configuração obsoleta, sem efeito.

<div id="min_rows_for_full_part_storage">
  ## min_rows_for_full_part_storage
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Disponível apenas no ClickHouse Cloud. Número mínimo de linhas para usar o tipo Full
de armazenamento para a data part em vez do packed

<div id="min_rows_for_wide_part">
  ## min_rows_for_wide_part
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Número mínimo de linhas para criar uma data part no formato `Wide` em vez de `Compact`.

<div id="min_rows_to_fsync_after_merge">
  ## min_rows_to_fsync_after_merge
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Número mínimo de linhas para executar `fsync` na part após o merge (0 - desativado)
