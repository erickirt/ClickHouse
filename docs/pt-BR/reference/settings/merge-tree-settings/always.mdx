Essas configurações estão disponíveis em [system.merge&#95;tree&#95;settings](/pt-BR/reference/system-tables/merge_tree_settings) e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="always_fetch_merged_part">
  ## always_fetch_merged_part
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Se true, esta réplica nunca mescla partes e sempre baixa partes mescladas
de outras réplicas.

Valores possíveis:

* true, false

<div id="always_use_copy_instead_of_hardlinks">
  ## always_use_copy_instead_of_hardlinks
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Sempre copie os dados em vez de criar hardlinks durante mutações/substituições/desanexações
e assim por diante.
