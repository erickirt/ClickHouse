Essas configurações estão disponíveis em [system.merge&#95;tree&#95;settings](/pt-BR/reference/system-tables/merge_tree_settings) e são geradas automaticamente a partir do código-fonte do ClickHouse.

<div id="min_delay_to_insert_ms">
  ## min_delay_to_insert_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="10" />

Atraso mínimo para a inserção de dados em uma tabela MergeTree, em milissegundos, se houver muitas partes não mescladas em uma única partição.

<div id="min_delay_to_mutate_ms">
  ## min_delay_to_mutate_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="10" />

Atraso mínimo para mutar uma tabela MergeTree, em milissegundos, se houver muitas
mutações não concluídas
