  ## refresh_parts_interval
</div>

<SettingsInfoBlock type="Segundos" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.4"},{"label": "0"},{"label": "Uma nova configuração"}]}]} />

Se for maior que zero, atualize a lista de partes de dados a partir do sistema de arquivos subjacente para verificar se os dados foram atualizados internamente.
Isso só pode ser definido se a tabela estiver localizada em discos somente leitura (o que significa que esta é uma réplica somente leitura, enquanto os dados estão sendo gravados por outra réplica).
