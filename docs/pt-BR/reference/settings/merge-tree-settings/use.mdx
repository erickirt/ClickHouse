<div id="use_adaptive_write_buffer_for_dynamic_subcolumns">
  ## use_adaptive_write_buffer_for_dynamic_subcolumns
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Permite usar buffers de gravação adaptativos ao gravar subcolunas dinâmicas para
reduzir o uso de memória

<div id="use_async_block_ids_cache">
  ## use_async_block_ids_cache
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Configuração obsoleta, sem efeito.

<div id="use_compact_variant_discriminators_serialization">
  ## use_compact_variant_discriminators_serialization
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Habilita o modo compacto para a serialização binária de discriminadores no tipo de
dados Variant.
Esse modo permite usar significativamente menos memória para armazenar discriminadores
em partes quando há predominantemente uma variante ou muitos valores NULL.

<div id="use_const_adaptive_granularity">
  ## use_const_adaptive_granularity
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Sempre use granularidade constante para a parte inteira. Isso permite compactar na
memória os valores da granularidade do índice. Pode ser útil em cargas de
trabalho extremamente grandes com tabelas estreitas.

<div id="use_metadata_cache">
  ## use_metadata_cache
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Configuração obsoleta, sem efeito.
