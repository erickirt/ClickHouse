<div id="index_granularity">
  ## index_granularity
</div>

<SettingsInfoBlock type="UInt64" default_value="8192" />

Número máximo de linhas entre as marcas de um índice. Ou seja, quantas linhas
correspondem a um valor de chave primária.

<div id="index_granularity_bytes">
  ## index_granularity_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="10485760" />

Tamanho máximo dos grânulos de dados em bytes.

Para restringir o tamanho do grânulo apenas com base no número de linhas, defina como `0` (não recomendado).
