<div id="check_sample_column_is_correct">
  ## check_sample_column_is_correct
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Habilita a verificação, durante a criação da tabela, de que o tipo de dados de uma coluna de amostragem ou de uma expressão de amostragem está correto. O tipo de dados deve ser um dos [tipos inteiros](/pt-BR/reference/data-types/int-uint): `UInt8`, `UInt16`,
`UInt32`, `UInt64`.

Valores possíveis:

* `true`  — A verificação está habilitada.
* `false` — A verificação é desabilitada durante a criação da tabela.

Valor padrão: `true`.

Por padrão, o servidor ClickHouse verifica, durante a criação da tabela, o tipo de dados de uma coluna de amostragem ou de uma expressão de amostragem. Se você já tiver tabelas com expressões de amostragem incorretas e não quiser que o servidor lance uma exceção durante a inicialização, defina `check_sample_column_is_correct` como `false`.
