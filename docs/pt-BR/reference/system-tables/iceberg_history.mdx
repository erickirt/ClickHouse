<div id="description">
  ## Descrição
</div>

Esta tabela de sistema contém o histórico de snapshots das tabelas Iceberg existentes no ClickHouse. Ela ficará vazia se você não tiver nenhuma tabela Iceberg no ClickHouse.

<div id="columns">
  ## Colunas
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/pt-BR/reference/data-types/index)) — nome do banco de dados.
* `table` ([String](/pt-BR/reference/data-types/index)) — Nome da tabela.
* `made_current_at` ([Nullable(DateTime64(3))](/pt-BR/reference/data-types/index)) — Data e hora em que este snapshot se tornou o snapshot atual.
* `snapshot_id` ([UInt64](/pt-BR/reference/data-types/index)) — ID do snapshot usado para identificar um snapshot.
* `parent_id` ([UInt64](/pt-BR/reference/data-types/index)) — ID pai deste snapshot.
* `is_current_ancestor` ([UInt8](/pt-BR/reference/data-types/index)) — Flag que indica se este snapshot é um ancestral do snapshot atual.
* `operation` ([Enum8(&#39;UNKNOWN&#39; = -1, &#39;APPEND&#39; = 0, &#39;OVERWRITE&#39; = 1, &#39;DELETE&#39; = 2, &#39;REPLACE&#39; = 3)](/pt-BR/reference/data-types/index)) — Operação do snapshot (APPEND, OVERWRITE, DELETE, REPLACE e UNKNOWN). O status UNKNOWN significa que não foi possível ler o resumo ou que ele está vazio (é opcional para v1). O campo &#39;operation&#39; correto é obrigatório
* `summary` ([Map(String, String)](/pt-BR/reference/data-types/index)) — Campos do resumo

{/*AUTOGENERATED_END*/}
