<div id="description">
  ## Descrição
</div>

Mostra todos os stemmers disponíveis.
Eles podem ser usados na função [stem](/pt-BR/reference/functions/regular-functions/nlp-functions).

<Info>
  **Disponibilidade**

  `system.stemmers` está presente apenas em compilações do ClickHouse com a dependência `libstemmer` (`USE_LIBSTEMMER`) habilitada. Em compilações sem ela, a tabela não existe, e consultas a ela falharão com `UNKNOWN_TABLE`. Você pode verificar se ela está habilitada na sua compilação com:

  ```sql
  SELECT value FROM system.build_options WHERE name = 'USE_LIBSTEMMER';
  ```
</Info>

Colunas:

{/*AUTOGENERATED_START*/}

* `name` ([String](/pt-BR/reference/data-types/index)) — Identificador do stemmer Snowball (idioma/algoritmo)

{/*AUTOGENERATED_END*/}
