<div id="quantilesExactLow">
  ## quantilesExactLow
</div>

Introduzido em: v20.8.0

Calcula exatamente vários [quantis](https://en.wikipedia.org/wiki/Quantile) de uma sequência de dados numéricos em diferentes níveis simultaneamente. Para um número par de elementos no nível `0.5`, retorna o menor dos dois valores centrais.

Esta função é equivalente a [`quantileExactLow`](/pt-BR/reference/functions/aggregate-functions/quantileExactLow), mas permite calcular vários níveis de quantil em uma única passagem, o que é mais eficiente do que chamar funções de quantil separadamente.

**Sintaxe**

```sql
quantilesExactLow(level1, level2, ...)(expr)
```

**Parâmetros**

* `level` — Níveis de quantis. Um ou mais números constantes de ponto flutuante de 0 a 1. Recomendamos usar valores de `level` no intervalo `[0.01, 0.99]`. [`Float*`](/pt-BR/reference/data-types/float)

**Argumentos**

* `expr` — Expressão sobre os valores da coluna que resulta em tipos de dados numéricos, `Date`, `DateTime` ou `DateTime64`. [`(U)Int*`](/pt-BR/reference/data-types/int-uint) ou [`Float*`](/pt-BR/reference/data-types/float) ou [`Decimal*`](/pt-BR/reference/data-types/decimal) ou [`Date`](/pt-BR/reference/data-types/date) ou [`DateTime`](/pt-BR/reference/data-types/datetime) ou [`DateTime64`](/pt-BR/reference/data-types/datetime64)

**Valor retornado**

Array de quantis dos níveis especificados, na mesma ordem em que foram especificados. Para tipos de dados numéricos, o formato de saída corresponde ao formato de entrada. [`Array((U)Int*)`](/pt-BR/reference/data-types/array) ou [`Array(Float*)`](/pt-BR/reference/data-types/array) ou [`Array(Decimal*)`](/pt-BR/reference/data-types/array) ou [`Array(Date)`](/pt-BR/reference/data-types/array) ou [`Array(DateTime)`](/pt-BR/reference/data-types/array) ou [`Array(DateTime64)`](/pt-BR/reference/data-types/array)

**Exemplos**

**Calculando vários quantis exatos baixos**

```sql title=Query
SELECT quantilesExactLow(0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesExactLow(0.25, 0.5, 0.75)(number)─┐
│ [2,4,7]                                    │
└────────────────────────────────────────────┘
```
