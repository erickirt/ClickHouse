## varSamp {#varSamp}

Introduced in: v1.1.0

Calculate the variância amostral of a data set.

The variância amostral is calculated using the formula:

$$
\frac{\Sigma{(x - \bar{x})^2}}{n-1}
$$

<br/>

Where:
- $x$ is each individual data point in the data set
- $\bar{x}$ is the arithmetic mean of the data set
- $n$ is the number of data points in the data set

The function assumes that the input data set represents a sample from a larger population. If you want to calculate the variance of the entire population (when you have the complete data set), you should use [`varPop`](/reference/functions/aggregate-functions/varPop) instead.

<Note>
Esta função utiliza um algoritmo numericamente instável. Se você precisar de [estabilidade numérica](https://en.wikipedia.org/wiki/Numerical_stability) nos cálculos, use a função [`varSampStable`](/reference/functions/aggregate-functions/varSampStable). Ela é mais lenta, mas apresenta menor erro computacional.
</Note>
    

**Syntax**

```sql
varSamp(x)
```

**Aliases**: `VAR_SAMP`

**Arguments**

- `x` — The population for which you want to calculate the variância amostral. [`(U)Int*`](/reference/data-types/int-uint) or [`Float*`](/reference/data-types/float) or [`Decimal*`](/reference/data-types/decimal)

**Returned value**

Returns the variância amostral of the input data set `x`. [`Float64`](/reference/data-types/float)

**Examples**

**Computing variância amostral**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x Float64
)
ENGINE = Memory;

INSERT INTO test_data VALUES (10.5), (12.3), (9.8), (11.2), (10.7);

SELECT round(varSamp(x),3) AS var_samp FROM test_data;
```

```response title=Response
┌─var_samp─┐
│    0.865 │
└──────────┘
```
