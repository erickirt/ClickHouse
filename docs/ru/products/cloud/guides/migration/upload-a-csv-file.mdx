ClickHouse Cloud предоставляет простой способ импортировать файлы и поддерживает
следующие форматы:

| Формат                          |
| ------------------------------- |
| `CSV`                           |
| `CSVWithNamesAndTypes`          |
| `CSVWithNames`                  |
| `JSONEachRow`                   |
| `TabSeparated`                  |
| `TabSeparatedWithNames`         |
| `TabSeparatedWithNamesAndTypes` |

<Steps>
  <Step title="Загрузите файл" id="upload-file">
    На главной странице Cloud выберите свой сервис, как показано ниже:

    <Image img="/images/cloud/migrate/csv_01.webp" alt="upload_file_02" />

    Если ваш сервис бездействует, его нужно будет вывести из спящего режима.

    Выберите `Data sources` на вкладке слева, как показано ниже:

    <Image img="/images/cloud/migrate/csv_02.webp" alt="upload_file_03" />

    Затем в правой части страницы источников данных выберите `Upload a file`:

    <Image img="/images/cloud/migrate/csv_03.webp" alt="upload_file_04" />

    Откроется диалоговое окно выбора файла, в котором вы сможете выбрать файл для
    вставки данных в таблицу вашего сервиса Cloud.

    <Image img="/images/cloud/migrate/csv_04.webp" alt="upload_file_05" />
  </Step>

  <Step title="Настройте таблицу" id="configure-table">
    После загрузки файла вы сможете настроить таблицу, в которую хотите
    вставить данные. Также будет показан предварительный просмотр таблицы с первыми тремя строками.

    <Image img="/images/cloud/migrate/csv_08.webp" alt="upload_file_08" />

    Теперь вы можете выбрать целевую таблицу. Доступны следующие варианты:

    * новая таблица
    * существующая таблица

    <br />

    Вы можете указать, в какую базу данных загрузить данные, а в случае
    новой таблицы — имя таблицы, которая будет создана. Также вы сможете выбрать ключ сортировки:

    <Image img="/images/cloud/migrate/csv_05.webp" alt="upload_file_05" />

    Столбцы, считанные из файла, отображаются как `Source field`, и для каждого поля
    вы можете изменить:

    * автоматически определённый тип
    * значение по умолчанию
    * сделать ли столбец [Nullable](/ru/reference/data-types/nullable)

    <Image img="/images/cloud/migrate/csv_06.webp" alt="upload_file_06" />

    <Info>
      **Исключение полей**

      Вы также можете удалить поле, если не хотите включать его в импорт
    </Info>

    Вы можете указать, какой движок таблицы хотите использовать:

    * `MergeTree`
    * `ReplacingMergeTree`
    * `SummingMergeTree`
    * `Null`

    <br />

    Вы можете указать выражение ключа партиционирования и выражение первичного
    ключа.

    <Image img="/images/cloud/migrate/csv_07.webp" alt="upload_file_07" />

    Нажмите `Import to ClickHouse` (показано выше), чтобы импортировать данные. Импорт будет поставлен в очередь, что
    обозначается значком статуса `queued` в столбце `Status`, как показано ниже. Вы также можете нажать
    `Open as query` (показано выше), чтобы открыть запрос на вставку в SQL-консоли. Этот запрос выполнит вставку
    файла, который был загружен в S3 бакет с помощью табличной функции `URL`.

    <Image img="/images/cloud/migrate/csv_09.webp" alt="upload_file_09" />

    Если задание завершится с ошибкой, вы увидите значок статуса `failed` в столбце `Status` на
    вкладке `Data upload history`. Вы можете нажать `View Details`, чтобы получить больше информации
    о причине сбоя загрузки. В зависимости от сообщения об ошибке для неудавшейся вставки
    вам может потребоваться изменить настройки таблицы или очистить данные.

    <Image img="/images/cloud/migrate/csv_10.webp" alt="upload_file_11" />
  </Step>
</Steps>
