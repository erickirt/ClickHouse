<div id="clickpipes">
  ## Управляемые интеграционные конвейеры для ClickHouse Cloud
</div>

ClickPipes — это управляемая интеграционная платформа, которая делает загрузку данных из самых разных источников такой же простой, как нажатие нескольких кнопок.
Надёжная и масштабируемая архитектура ClickPipes, созданная для самых требовательных рабочих нагрузок, обеспечивает стабильную производительность и надёжную работу.
ClickPipes можно использовать как для долгосрочного стриминга, так и для разовой загрузки данных.
