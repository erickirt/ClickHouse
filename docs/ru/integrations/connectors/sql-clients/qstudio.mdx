QStudio — это бесплатный SQL GUI, который позволяет запускать SQL-скрипты, удобно просматривать таблицы, строить графики и экспортировать результаты. Он работает на любой операционной системе и с любой базой данных.

QStudio подключается к ClickHouse через JDBC.

<Steps>
  <Step title="Соберите данные для подключения к ClickHouse" id="1-gather-your-clickhouse-details">
    QStudio использует JDBC поверх HTTP(S) для подключения к ClickHouse; вам понадобятся:

    * конечная точка
    * номер порта
    * имя пользователя
    * пароль

    <ConnectionDetails />
  </Step>

  <Step title="Скачайте QStudio" id="2-download-qstudio">
    QStudio доступен по адресу https://www.timestored.com/qstudio/download/
  </Step>

  <Step title="Добавьте базу данных" id="3-add-a-database">
    * При первом запуске QStudio нажмите в меню **Server-&gt;Add Server** или кнопку добавления сервера на панели инструментов.
    * Затем укажите параметры:

    <Image img="/images/integrations/sql-clients/qstudio-add-connection.webp" size="lg" border alt="Экран настройки подключения к базе данных в QStudio, показывающий параметры подключения к ClickHouse" />

    1. Тип сервера: Clickhouse.com
    2. Обратите внимание: в поле Host ОБЯЗАТЕЛЬНО нужно указать https://
       Host: https://abc.def.clickhouse.cloud
       Port: 8443
    3. Username: default
       Password: `XXXXXXXXXXX`
    4. Нажмите Add

    Если QStudio обнаружит, что ClickHouse JDBC driver у вас не установлен, он предложит скачать его за вас:
  </Step>

  <Step title="Выполните запрос к ClickHouse" id="4-query-clickhouse">
    * Откройте редактор запросов и выполните запрос. Запросы можно выполнять так:

    * Ctrl + e — выполняет выделенный текст

    * Ctrl + Enter — выполняет текущую строку

    * Пример запроса:

    <Image img="/images/integrations/sql-clients/qstudio-running-query.webp" size="lg" border alt="Интерфейс QStudio, показывающий выполнение примера SQL-запроса к базе данных ClickHouse" />
  </Step>
</Steps>
