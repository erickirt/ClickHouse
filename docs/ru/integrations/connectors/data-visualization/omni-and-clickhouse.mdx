Omni может подключаться к ClickHouse Cloud или ClickHouse в собственной инфраструктуре через официальный источник данных ClickHouse.

<Steps>
  <Step title="Соберите сведения о подключении" id="1-gather-your-connection-details">
    <ConnectionDetails />
  </Step>

  <Step title="Создайте источник данных ClickHouse" id="2-create-a-clickhouse-data-source">
    Перейдите в Admin -&gt; Connections и нажмите кнопку &quot;Add Connection&quot; в правом верхнем углу.

    <Image size="lg" img="/images/integrations/data-visualization/omni_01.webp" alt="Интерфейс администратора Omni с кнопкой Add Connection в разделе Connections" border />

    <br />

    Выберите `ClickHouse`. Введите учетные данные в форму.

    <Image size="lg" img="/images/integrations/data-visualization/omni_02.webp" alt="Интерфейс настройки подключения Omni для ClickHouse с полями формы для учетных данных" border />

    <br />

    Теперь вы можете выполнять запросы к данным ClickHouse и визуализировать их в Omni.
  </Step>
</Steps>
