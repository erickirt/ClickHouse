Эти настройки доступны в [system.merge&#95;tree&#95;settings](/ru/reference/system-tables/merge_tree_settings) и автоматически генерируются на основе исходного кода ClickHouse.

<div id="merge_selecting_sleep_ms">
  ## merge_selecting_sleep_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="5000" />

Минимальное время ожидания перед повторной попыткой выбрать части для слияния,
если ни одна часть не была выбрана. Меньшее значение приведёт к
частому запуску задач выбора в background&#95;schedule&#95;pool, что вызовет большое
количество запросов к ZooKeeper в кластерах большого масштаба

<div id="merge_selecting_sleep_slowdown_factor">
  ## merge_selecting_sleep_slowdown_factor
</div>

<SettingsInfoBlock type="Float" default_value="1.2" />

Время ожидания для задачи выбора слияния умножается на этот коэффициент, когда
объединять нечего, и уменьшается, когда слияние назначено
