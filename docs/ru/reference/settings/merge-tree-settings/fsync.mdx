Эти настройки доступны в [system.merge&#95;tree&#95;settings](/ru/reference/system-tables/merge_tree_settings) и автоматически создаются на основе исходного кода ClickHouse.

<div id="fsync_after_insert">
  ## fsync_after_insert
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Выполняет fsync для каждой вставленной части. Существенно снижает производительность
вставки, не рекомендуется использовать с частями Wide.

<div id="fsync_part_directory">
  ## fsync_part_directory
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Выполнять fsync для каталога части после всех операций с частью (записи, переименования и т. д.).
