Эти настройки доступны в [system.merge&#95;tree&#95;settings](/ru/reference/system-tables/merge_tree_settings) и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="async_block_ids_cache_update_wait_ms">
  ## async_block_ids_cache_update_wait_ms
</div>

<SettingsInfoBlock type="Milliseconds" default_value="100" />

Устаревший алиас `deduplication_hashes_cache_update_wait_ms`, сохранённый на один релиз для обратной
совместимости. Учитывается только в том случае, если для `deduplication_hashes_cache_update_wait_ms` оставлено
значение по умолчанию; в одном из будущих релизов этот параметр будет удалён.

<div id="async_insert">
  ## async_insert
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Если значение равно true, данные из запроса INSERT помещаются в очередь, а затем в фоновом режиме сбрасываются в таблицу.
