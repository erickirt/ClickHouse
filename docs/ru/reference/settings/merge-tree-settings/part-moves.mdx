<div id="part_moves_between_shards_delay_seconds">
  ## part_moves_between_shards_delay_seconds
</div>

<ExperimentalBadge />

<SettingsInfoBlock type="UInt64" default_value="30" />

Время ожидания перед и после перемещения частей между сегментами.

<div id="part_moves_between_shards_enable">
  ## part_moves_between_shards_enable
</div>

<ExperimentalBadge />

<SettingsInfoBlock type="UInt64" default_value="0" />

Экспериментальная/незавершённая возможность перемещения частей между сегментами. Не учитывает
выражения распределения по сегментам.
