<div id="merge_workload">
  ## merge_workload
</div>

Используется для управления тем, как ресурсы используются и распределяются между слияниями и
другими рабочими нагрузками. Указанное значение используется в качестве значения настройки `workload` для
фоновых слияний этой таблицы. Если оно не указано (пустая строка),
вместо него используется настройка сервера `merge_workload`.
