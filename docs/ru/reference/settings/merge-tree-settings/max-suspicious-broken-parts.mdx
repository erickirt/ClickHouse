Эти настройки доступны в [system.merge&#95;tree&#95;settings](/ru/reference/system-tables/merge_tree_settings) и автоматически генерируются на основе исходного кода ClickHouse.

<div id="max_suspicious_broken_parts">
  ## max_suspicious_broken_parts
</div>

<SettingsInfoBlock type="UInt64" default_value="100" />

Если количество повреждённых частей в одной партиции превышает значение
`max_suspicious_broken_parts`, автоматическое удаление не допускается.

Возможные значения:

* Любое положительное целое число.

<div id="max_suspicious_broken_parts_bytes">
  ## max_suspicious_broken_parts_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" />

Максимальный суммарный размер всех повреждённых частей; при превышении этого значения автоматическое удаление запрещено.

Возможные значения:

* Любое положительное целое число.
