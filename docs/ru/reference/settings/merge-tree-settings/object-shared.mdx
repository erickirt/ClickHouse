Версия сериализации общих данных в типе данных JSON.

Возможные значения:

* `map` - хранить общие данные как `Map(String, String)`
* `map_with_buckets` - хранить общие данные в виде нескольких отдельных столбцов `Map(String, String)`. Использование бакетов улучшает чтение отдельных путей из общих данных.
* `advanced` - специальная сериализация общих данных, разработанная для значительного улучшения чтения отдельных путей из общих данных.
  Обратите внимание, что эта сериализация увеличивает объем общих данных на диске, поскольку мы сохраняем много дополнительной информации.

Количество бакетов для сериализаций `map_with_buckets` и `advanced` определяется настройками
[object&#95;shared&#95;data&#95;buckets&#95;for&#95;compact&#95;part](/ru/reference/settings/merge-tree-settings/object-shared#object_shared_data_buckets_for_compact_part)/[object&#95;shared&#95;data&#95;buckets&#95;for&#95;wide&#95;part](/ru/reference/settings/merge-tree-settings/object-shared#object_shared_data_buckets_for_wide_part).

существенно увеличить время вставки.
