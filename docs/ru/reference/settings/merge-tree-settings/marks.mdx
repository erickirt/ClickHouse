Эти настройки доступны в [system.merge&#95;tree&#95;settings](/ru/reference/system-tables/merge_tree_settings) и автоматически генерируются на основе исходного кода ClickHouse.

<div id="marks_compress_block_size">
  ## marks_compress_block_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="65536" />

Размер блока сжатия marks, то есть фактический размер сжимаемого блока.

<div id="marks_compression_codec">
  ## marks_compression_codec
</div>

<SettingsInfoBlock type="String" default_value="ZSTD(3)" />

Кодек сжатия, используемый для marks; они достаточно малы и кэшируются,
поэтому по умолчанию используется ZSTD(3).
