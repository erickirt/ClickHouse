Эти настройки доступны в [system.merge&#95;tree&#95;settings](/ru/reference/system-tables/merge_tree_settings) и автоматически генерируются из исходного кода ClickHouse.

<div id="min_delay_to_insert_ms">
  ## min_delay_to_insert_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="10" />

Минимальная задержка при вставке данных в таблицу семейства MergeTree в миллисекундах, если в одной
партиции много неслитых частей.

<div id="min_delay_to_mutate_ms">
  ## min_delay_to_mutate_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="10" />

Минимальная задержка мутации таблицы семейства MergeTree в миллисекундах при большом количестве
незавершённых мутаций
