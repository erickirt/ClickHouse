Эти настройки доступны в [system.merge&#95;tree&#95;settings](/ru/reference/system-tables/merge_tree_settings) и автоматически формируются на основе исходного кода ClickHouse.

<div id="dead_blobs_to_delay_insert">
  ## dead_blobs_to_delay_insert
</div>

<SettingsInfoBlock type="UInt64" default_value="100000" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "100000"},{"label": "Новая настройка для искусственного замедления вставок, когда в очередях dead blobs на дисках таблицы накапливается слишком много blob-объектов, ожидающих удаления."}]}]} />

Если число blob-объектов, ожидающих удаления в очередях dead blobs на дисках таблицы, превышает
значение `dead_blobs_to_delay_insert`, выполнение `INSERT` искусственно замедляется (вплоть до `max_delay_to_insert`).

Очередь dead blobs принадлежит диску и является общей для всех таблиц на нём (включая blob-объекты уже
удалённых таблиц), поэтому порог следует задавать для всего диска, а не для отдельной таблицы.

<Tip>
  Полезно, если сервер не успевает достаточно быстро очищать blob-объекты.
</Tip>

Возможные значения:

* Любое положительное целое число.
* 0 — отключено.

<div id="dead_blobs_to_throw_insert">
  ## dead_blobs_to_throw_insert
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "1000000"},{"label": "Новая настройка, которая отклоняет вставки, если в очередях dead blobs на дисках таблицы накапливается слишком много blob-объектов, ожидающих удаления."}]}]} />

Если количество blob-объектов, ожидающих удаления в очередях dead blobs на дисках таблицы, превышает значение
`dead_blobs_to_throw_insert`, выполнение `INSERT` прерывается со следующей ошибкой:

> &quot;Слишком много dead blobs в очереди (N) на дисках таблицы. Очистка blob-объектов выполняется
> значительно медленнее, чем вставки&quot;

Очередь dead blobs относится к диску и является общей для всех таблиц на нём (включая blob-объекты уже
удалённых таблиц), поэтому порог следует задавать для всего диска, а не для отдельной таблицы.

Возможные значения:

* Любое положительное целое число.
* 0 — отключено.
