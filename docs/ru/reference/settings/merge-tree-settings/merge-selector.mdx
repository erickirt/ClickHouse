Алгоритм выбора частей для распределения слияний

<div id="merge_selector_base">
  ## merge_selector_base
</div>

<SettingsInfoBlock type="Float" default_value="5" />

Влияет на усиление записи для
выбранных слияний (настройка экспертного уровня; не изменяйте её, если не понимаете,
как она работает). Работает для селекторов слияний Simple и StochasticSimple

<div id="merge_selector_blurry_base_scale_factor">
  ## merge_selector_blurry_base_scale_factor
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Определяет, когда срабатывает эта логика в зависимости от количества частей в
партиции. Чем больше коэффициент, тем позже будет реакция.

<div id="merge_selector_enable_heuristic_to_lower_max_parts_to_merge_at_once">
  ## merge_selector_enable_heuristic_to_lower_max_parts_to_merge_at_once
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "1"},{"label": "Включено по умолчанию"}]}, {"id": "row-2","items": [{"label": "25.12"},{"label": "0"},{"label": "новая настройка"}]}]} />

Включает эвристику для простого селектора слияний, которая снижает максимальный лимит на выбор слияния.
Это увеличивает число параллельных слияний, что может помочь с ошибками TOO&#95;MANY&#95;PARTS,
но при этом также увеличивает усиление записи.

<div id="merge_selector_enable_heuristic_to_remove_small_parts_at_right">
  ## merge_selector_enable_heuristic_to_remove_small_parts_at_right
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Включает эвристику выбора частей для слияния, которая удаляет части с правой
стороны диапазона, если их размер меньше указанной доли (0.01) от sum&#95;size.
Работает для селекторов слияний Simple и StochasticSimple

<div id="merge_selector_heuristic_to_lower_max_parts_to_merge_at_once_exponent">
  ## merge_selector_heuristic_to_lower_max_parts_to_merge_at_once_exponent
</div>

<ExperimentalBadge />

<SettingsInfoBlock type="UInt64" default_value="5" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.12"},{"label": "5"},{"label": "Новая настройка"}]}]} />

Управляет значением экспоненты, используемым в формулах, задающих кривую уменьшения. Уменьшение экспоненты
уменьшает размер слияний, что приводит к увеличению усиления записи. Верно и обратное.

<div id="merge_selector_window_size">
  ## merge_selector_window_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" />

Сколько частей обрабатывать за один раз.
