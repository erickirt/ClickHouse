Эти настройки доступны в [system.merge&#95;tree&#95;settings](/ru/reference/system-tables/merge_tree_settings) и автоматически генерируются из исходного кода ClickHouse.

<div id="min_rows_for_compact_part">
  ## min_rows_for_compact_part
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Устаревшая настройка, не имеет эффекта.

<div id="min_rows_for_full_part_storage">
  ## min_rows_for_full_part_storage
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Доступно только в ClickHouse Cloud. Минимальное количество строк для использования полного типа
хранения для части данных вместо упакованного типа

<div id="min_rows_for_wide_part">
  ## min_rows_for_wide_part
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Минимальное количество строк, необходимое для создания части данных в формате `Wide` вместо `Compact`.

<div id="min_rows_to_fsync_after_merge">
  ## min_rows_to_fsync_after_merge
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Минимальное количество строк, при котором после слияния для части выполняется fsync (0 — отключено)
