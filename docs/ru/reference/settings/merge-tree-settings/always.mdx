Эти настройки доступны в [system.merge&#95;tree&#95;settings](/ru/reference/system-tables/merge_tree_settings) и автоматически генерируются на основе исходного кода ClickHouse.

<div id="always_fetch_merged_part">
  ## always_fetch_merged_part
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Если true, эта реплика никогда не выполняет слияние частей и всегда загружает слитые части
с других реплик.

Возможные значения:

* true, false

<div id="always_use_copy_instead_of_hardlinks">
  ## always_use_copy_instead_of_hardlinks
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Всегда копировать данные вместо создания жёстких ссылок при мутациях/заменах/отсоединении
и т. д.
