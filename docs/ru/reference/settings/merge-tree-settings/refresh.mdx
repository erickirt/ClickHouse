  ## refresh_parts_interval
</div>

<SettingsInfoBlock type="Seconds" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.4"},{"label": "0"},{"label": "Новая настройка"}]}]} />

Если значение больше нуля, обновляет список частей данных из нижележащей файловой системы, чтобы проверить, не были ли данные обновлены напрямую в обход системы.
Может быть задана только в том случае, если таблица расположена на дисках только для чтения (это означает, что данная реплика доступна только для чтения, а запись данных выполняется другой репликой).
