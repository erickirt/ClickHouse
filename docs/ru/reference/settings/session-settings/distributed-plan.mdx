  ## distributed_plan_force_exchange_kind
</div>

<ExperimentalBadge />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.5"},{"label": ""},{"label": "Новая экспериментальная настройка."}]}]} />

Принудительно использовать указанный тип операторов Exchange между стадиями распределённого запроса.

Возможные значения:

* &#39;&#39; - не принудительно использовать какой-либо тип операторов Exchange, предоставить выбор оптимизатору,
* &#39;Persisted&#39; - использовать временные файлы в Объектном хранилище,
* &#39;Streaming&#39; - передавать данные Exchange по сети.
