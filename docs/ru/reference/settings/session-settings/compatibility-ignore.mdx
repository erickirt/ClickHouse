  ## compatibility_ignore_auto_increment_in_create_table
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Если установлено значение `true`, игнорирует ключевое слово AUTO&#95;INCREMENT в объявлении столбца; в противном случае возвращает ошибку. Это упрощает миграцию из MySQL
