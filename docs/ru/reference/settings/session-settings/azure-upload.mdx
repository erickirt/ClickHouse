  ## azure_upload_part_size_multiply_factor

<VersionHistory rows={[{"id": "row-1","items": [{"label": "24.3"},{"label": "2"},{"label": "Умножайте azure_min_upload_part_size на этот коэффициент каждый раз, когда в Azure Blob Storage загружается azure_multiply_parts_count_threshold частей в рамках одной операции записи."}]}]} />

Умножайте azure&#95;min&#95;upload&#95;part&#95;size на этот коэффициент каждый раз, когда в Azure Blob Storage загружается azure&#95;multiply&#95;parts&#95;count&#95;threshold частей в рамках одной операции записи.

  ## azure_upload_part_size_multiply_parts_count_threshold
</div>

<SettingsInfoBlock type="UInt64" default_value="500" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "24.3"},{"label": "500"},{"label": "После загрузки в Azure Blob Storage такого количества частей значение azure_min_upload_part_size умножается на azure_upload_part_size_multiply_factor."}]}]} />

После загрузки в Azure Blob Storage такого количества частей значение azure&#95;min&#95;upload&#95;part&#95;size умножается на azure&#95;upload&#95;part&#95;size&#95;multiply&#95;factor.
