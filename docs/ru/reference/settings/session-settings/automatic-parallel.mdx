Пороговый объём байт для чтения на реплику, при котором автоматически включаются параллельные реплики (применяется только при `automatic_parallel_replicas_mode`=1). 0 означает отсутствие порога.
Общий объём байт для чтения оценивается на основе собранной статистики.

<div id="automatic_parallel_replicas_mode">
  ## automatic_parallel_replicas_mode
</div>

<ExperimentalBadge />

<SettingsInfoBlock type="UInt64" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.12"},{"label": "0"},{"label": "Новая настройка"}]}]} />

Включает автоматическое переключение на выполнение запроса с параллельными репликами на основе собранной статистики. Требует `enable_analyzer = 1`, `enable_parallel_replicas != 0`, `parallel_replicas_local_plan = 1` и указания `cluster_for_parallel_replicas`.
0 — выключено, 1 — включено, 2 — включен только сбор статистики (переключение на выполнение с параллельными репликами отключено).
