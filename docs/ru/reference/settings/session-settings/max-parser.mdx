<div id="max_parser_depth">
  ## max_parser_depth
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" />

Ограничивает максимальную глубину рекурсии в рекурсивном парсере нисходящего разбора. Позволяет управлять размером стека.

Возможные значения:

* Положительное целое число.
* 0 — глубина рекурсии не ограничена.
