<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "0"},{"label": "Новая настройка для включения кэширования в памяти разобранных файлов метаданных Paimon (manifest list и manifests). Если настройка включена, табличная функция paimon и хранилище paimon могут использовать кэш файлов метаданных paimon.

Табличные функции Paimon применяют эту настройку для каждого запроса, тогда как постоянные движки таблиц Paimon фиксируют её значение при инициализации метаданных: чтобы изменить это для уже созданной таблицы, её необходимо удалить и создать заново.

Возможные значения:

* 0 - Отключено
* 1 - Включено
