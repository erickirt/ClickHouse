Эти настройки доступны в [system.settings](/ru/reference/system-tables/settings) и автоматически сгенерированы на основе [исходного кода](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp).

<div id="min_count_to_compile_aggregate_expression">
  ## min_count_to_compile_aggregate_expression
</div>

<SettingsInfoBlock type="UInt64" default_value="3" />

Минимальное количество одинаковых агрегатных выражений, необходимое для запуска JIT-компиляции. Работает только при включенной настройке [compile&#95;aggregate&#95;expressions](/ru/reference/settings/session-settings/compile#compile_aggregate_expressions).

Возможные значения:

* Положительное целое число.
* 0 — одинаковые агрегатные выражения всегда JIT-компилируются.

<div id="min_count_to_compile_expression">
  ## min_count_to_compile_expression
</div>

<SettingsInfoBlock type="UInt64" default_value="3" />

Минимальное число выполнений одного и того же выражения, после которого оно компилируется.
