<div id="optimize_move_to_prewhere">
  ## optimize_move_to_prewhere
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Включает или отключает автоматическую оптимизацию [PREWHERE](/ru/reference/statements/select/prewhere) в запросах [SELECT](/ru/reference/statements/select/index).

Работает только для [таблиц семейства *MergeTree](/ru/reference/engines/table-engines/mergetree-family/index).

Возможные значения:

* 0 — Автоматическая оптимизация `PREWHERE` отключена.
* 1 — Автоматическая оптимизация `PREWHERE` включена.

<div id="optimize_move_to_prewhere_if_final">
  ## optimize_move_to_prewhere_if_final
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Включает или отключает автоматическую оптимизацию [PREWHERE](/ru/reference/statements/select/prewhere) в запросах [SELECT](/ru/reference/statements/select/index) с модификатором [FINAL](/ru/reference/statements/select/from#final-modifier).

Работает только для таблиц семейства [*MergeTree](/ru/reference/engines/table-engines/mergetree-family/index).

Возможные значения:

* 0 — Автоматическая оптимизация `PREWHERE` в запросах `SELECT` с модификатором `FINAL` отключена.
* 1 — Автоматическая оптимизация `PREWHERE` в запросах `SELECT` с модификатором `FINAL` включена.
