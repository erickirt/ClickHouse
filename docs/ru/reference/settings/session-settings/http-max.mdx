<div id="http_max_multipart_form_data_size">
  ## http_max_multipart_form_data_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" />

Ограничение на размер содержимого multipart/form-data. Этот параметр нельзя задать через URL-параметры, его следует задавать в профиле пользователя. Обратите внимание, что содержимое разбирается, а внешние таблицы создаются в памяти до начала выполнения запроса. Это единственное ограничение, которое влияет на эту стадию (ограничения на максимальное использование памяти и максимальное время выполнения не действуют при чтении данных HTTP-формы).
