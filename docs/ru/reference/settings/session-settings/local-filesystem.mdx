<div id="local_filesystem_read_method">
  ## local_filesystem_read_method
</div>

<SettingsInfoBlock type="String" default_value="pread_threadpool" />

Метод чтения данных из локальной файловой системы; одно из следующих значений: read, pread, mmap, io&#95;uring, pread&#95;threadpool.

Метод &#39;io&#95;uring&#39; экспериментальный и не работает для Log, TinyLog, StripeLog, File, Set и JOIN, а также других таблиц с файлами, допускающими дописывание, при одновременных чтении и записи.
Если вы читали в интернете разные статьи об &#39;io&#95;uring&#39;, не стоит ими обольщаться. Это не более эффективный метод чтения файлов, кроме случаев с большим количеством мелких запросов IO, что для ClickHouse нехарактерно. Нет причин включать &#39;io&#95;uring&#39;.

<div id="local_filesystem_read_prefetch">
  ## local_filesystem_read_prefetch
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Использовать предвыборку при чтении данных из локальной файловой системы.
