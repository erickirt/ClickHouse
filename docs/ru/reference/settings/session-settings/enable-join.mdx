  ## enable_join_transitive_predicates
</div>

<BetaBadge />

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.6"},{"label": "1"},{"label": "Параметр enable_join_transitive_predicates включен по умолчанию"}]}, {"id": "row-2","items": [{"label": "26.4"},{"label": "0"},{"label": "Новая настройка для вывода транзитивных предикатов эквисоединения при оптимизации порядка JOIN."}]}]} />

Выводит транзитивные предикаты эквисоединения из существующих условий JOIN.
Например, если заданы `A.x = B.x` и `B.x = C.x`, добавляется синтетический предикат `A.x = C.x`,
чтобы оптимизатор порядка JOIN мог учитывать прямые планы (A JOIN C).
