<div id="materialize_skip_indexes_on_insert">
  ## materialize_skip_indexes_on_insert
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "24.6"},{"label": "1"},{"label": "Добавлена новая настройка, позволяющая отключать материализацию индексов пропуска данных при вставке"}]}]} />

Определяет, создаются и сохраняются ли индексы пропуска данных при INSERT. Если параметр отключён, индексы пропуска данных будут создаваться и сохраняться только [во время слияний](/ru/reference/settings/merge-tree-settings#materialize_skip_indexes_on_merge) или с помощью явной команды [MATERIALIZE INDEX](/ru/reference/statements/alter/skipping-index#materialize-index).

См. также [exclude&#95;materialize&#95;skip&#95;indexes&#95;on&#95;insert](/ru/reference/settings/session-settings/other#exclude_materialize_skip_indexes_on_insert).

Определяет, будут ли команды INSERT строить и вставлять статистику. Если параметр отключён, статистика будет строиться и сохраняться во время слияний или с помощью явной команды MATERIALIZE STATISTICS
