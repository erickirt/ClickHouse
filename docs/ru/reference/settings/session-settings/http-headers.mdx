  ## http_headers_progress_interval_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="100" />

Не отправлять HTTP-заголовки X-ClickHouse-Progress чаще указанного интервала.

<div id="http_headers_read_timeout">
  ## http_headers_read_timeout
</div>

<SettingsInfoBlock type="Секунды" default_value="30" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "30"},{"label": "Новая настройка для ограничения общего времени чтения всех заголовков HTTP-запроса, защищающая от атак типа slowloris."}]}]} />

Максимальное время в секундах для чтения всех заголовков HTTP-запроса. Это общее ограничение по времени для всего этапа разбора заголовков, а не тайм-аут для отдельной операции чтения. Защищает от атак типа slowloris, при которых клиент медленно передаёт данные заголовков, чтобы удерживать соединения открытыми.
