<div id="number_of_mutations_to_delay">
  ## number_of_mutations_to_delay
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Если в таблице есть как минимум столько незавершённых мутаций, мутации этой таблицы искусственно замедляются. 0 — отключено

<div id="number_of_mutations_to_throw">
  ## number_of_mutations_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Если в таблице есть как минимум столько незавершённых мутаций, сгенерировать исключение &#39;Too many mutations ...&#39;. 0 — отключено
