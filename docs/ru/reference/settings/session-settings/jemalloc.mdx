<div id="jemalloc_enable_profiler">
  ## jemalloc_enable_profiler
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.9"},{"label": "0"},{"label": "новая настройка"}]}]} />

Включает профилировщик jemalloc для запроса. Jemalloc будет выполнять выборку выделений памяти и всех освобождений для выделений, попавших в выборку.
Профили можно сбрасывать с помощью SYSTEM JEMALLOC FLUSH PROFILE и использовать для анализа выделения памяти.
Образцы также можно сохранять в system.trace&#95;log с помощью параметра config jemalloc&#95;collect&#95;global&#95;profile&#95;samples&#95;in&#95;trace&#95;log или настройки запроса jemalloc&#95;collect&#95;profile&#95;samples&#95;in&#95;trace&#95;log.
См. [Профилирование выделения памяти](/ru/concepts/features/performance/allocation-profiling)
