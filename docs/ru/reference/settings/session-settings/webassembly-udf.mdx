<div id="webassembly_udf_max_fuel">
  ## webassembly_udf_max_fuel
</div>

<ExperimentalBadge />

<SettingsInfoBlock type="UInt64" default_value="100000" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.3"},{"label": "100000"},{"label": "Новая настройка для ограничения числа инструкций CPU (fuel) за одно выполнение экземпляра WebAssembly UDF."}]}]} />

Ограничение fuel на одно выполнение экземпляра WebAssembly UDF. Каждая инструкция WebAssembly потребляет некоторое количество fuel. Перед передачей в среду выполнения значение умножается на 1024, поэтому `webassembly_udf_max_fuel = 1` соответствует примерно 1024 единицам fuel. Установите 0, чтобы снять ограничение. Применяется только к функциям, у которых для настройки `webassembly_udf_enable_fuel` задано значение true; это значение используется по умолчанию.
