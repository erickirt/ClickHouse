<div id="describe_include_subcolumns">
  ## describe_include_subcolumns
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Включает описание подстолбцов в запросе [DESCRIBE](/ru/reference/statements/describe-table). Например, элементов [Tuple](/ru/reference/data-types/tuple) или подстолбцов типов данных [Map](/ru/reference/data-types/map#reading-subcolumns-of-map), [Nullable](/ru/reference/data-types/nullable#finding-null) и [Array](/ru/reference/data-types/array#array-size).

Возможные значения:

* 0 — Подстолбцы не включаются в запросы `DESCRIBE`.
* 1 — Подстолбцы включаются в запросы `DESCRIBE`.

**Пример**

См. пример для оператора [DESCRIBE](/ru/reference/statements/describe-table).

<div id="describe_include_virtual_columns">
  ## describe_include_virtual_columns
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Если значение true, виртуальные столбцы таблицы будут включены в результат запроса DESCRIBE
