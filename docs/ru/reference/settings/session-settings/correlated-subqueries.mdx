Управляет типом JOIN в декоррелированном плане запроса. Значение по умолчанию — `right`, то есть декоррелированный план будет содержать RIGHT JOIN, где вход подзапроса находится справа.

Возможные значения:

* `left` - В процессе декорреляции будут создаваться LEFT JOIN, и входная таблица будет находиться слева.
* `right` - В процессе декорреляции будут создаваться RIGHT JOIN, и входная таблица будет находиться справа.
