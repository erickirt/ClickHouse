Эти настройки доступны в [system.settings](/ru/reference/system-tables/settings) и автоматически генерируются на основе [исходного файла](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp).

<div id="stream_flush_interval_ms">
  ## stream_flush_interval_ms
</div>

<SettingsInfoBlock type="Milliseconds" default_value="7500" />

Применяется к таблицам со стримингом при тайм-ауте или когда поток генерирует [max&#95;insert&#95;block&#95;size](/ru/reference/settings/session-settings/max-insert#max_insert_block_size) строк.

Значение по умолчанию — 7500.

Чем меньше значение, тем чаще данные сбрасываются в таблицу. Слишком низкое значение приводит к снижению производительности.

<div id="stream_poll_timeout_ms">
  ## stream_poll_timeout_ms
</div>

<SettingsInfoBlock type="Milliseconds" default_value="500" />

Тайм-аут опроса данных при чтении из/записи в стриминговые хранилища.
