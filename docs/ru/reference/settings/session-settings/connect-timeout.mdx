<div id="connect_timeout">
  ## connect_timeout
</div>

<SettingsInfoBlock type="Секунды" default_value="10" />

Тайм-аут соединения при отсутствии реплик.

<div id="connect_timeout_with_failover_ms">
  ## connect_timeout_with_failover_ms
</div>

<SettingsInfoBlock type="Milliseconds" default_value="1000" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "23.4"},{"label": "1000"},{"label": "Увеличено значение тайм-аута подключения по умолчанию из-за async-подключения"}]}]} />

Тайм-аут в миллисекундах для подключения к удалённому серверу при использовании движка таблицы Distributed, если в определении cluster используются секции &#39;shard&#39; и &#39;replica&#39;.
Если подключение не удаётся, выполняется несколько попыток подключения к различным репликам.
