<div id="iceberg_manifest_min_count_to_compact">
  ## iceberg_manifest_min_count_to_compact
</div>

<ExperimentalBadge />

<SettingsInfoBlock type="UInt64" default_value="30" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "30"},{"label": "Новая настройка для управления компактацией файлов манифеста для таблиц Iceberg."}]}]} />

Минимальное количество файлов манифеста, необходимое для запуска компактации только файлов манифеста через OPTIMIZE TABLE ... MANIFEST.
Если текущее количество файлов манифеста меньше или равно этому порогу, компактация пропускается.
Необходимо, чтобы был включен allow&#95;experimental&#95;iceberg&#95;compaction.

</div>

<ExperimentalBadge />

<SettingsInfoBlock type="UInt64" default_value="259200" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "259200"},{"label": "Новая настройка: пороговое значение возраста файлов-сирот по умолчанию"}]}]} />

Пороговое значение возраста по умолчанию в секундах для удаления файлов-сирот в таблицах Iceberg. Файлы новее этого значения не считаются сиротами. Используется, если в вызове процедуры remove&#95;orphan&#95;files() опущен аргумент older&#95;than. Значение по умолчанию — 259200 (3 дня).
