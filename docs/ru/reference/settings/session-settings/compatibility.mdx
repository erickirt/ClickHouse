<div id="compatibility">
  ## compatibility
</div>

Параметр `compatibility` заставляет ClickHouse использовать настройки по умолчанию из более ранней версии ClickHouse, номер которой задается значением этого параметра.

Если для каких-либо настроек заданы значения, отличные от значений по умолчанию, они сохраняются (параметр `compatibility` влияет только на настройки, которые не были изменены).

Этот параметр принимает номер версии ClickHouse в виде строки, например `22.3`, `22.8`. Пустое значение означает, что параметр отключен.

По умолчанию отключен.

<Note>
  В ClickHouse Cloud значение параметра compatibility по умолчанию на уровне сервиса должно быть задано службой поддержки ClickHouse Cloud. Пожалуйста, [создайте обращение](https://clickhouse.cloud/support), чтобы его настроили.
  Однако параметр compatibility можно переопределить на уровне пользователя, роли, профиля, запроса или сеанса с помощью стандартных механизмов настройки ClickHouse, например `SET compatibility = '22.3'` в сеансе или `SETTINGS compatibility = '22.3'` в запросе.
</Note>
