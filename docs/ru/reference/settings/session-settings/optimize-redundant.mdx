  ## optimize_redundant_comparisons
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "1"},{"label": "Новая настройка для обнаружения конфликтующих и избыточных условий сравнения для одного и того же выражения в цепочках AND."}]}]} />

Обнаруживает конфликтующие и избыточные условия сравнения для одного и того же выражения в цепочках AND. Например, `a < 1 AND a > 5` будет переписано в `false`.
