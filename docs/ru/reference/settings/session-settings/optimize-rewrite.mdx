<div id="optimize_rewrite_aggregate_function_with_if">
  ## optimize_rewrite_aggregate_function_with_if
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Переписывает агрегатные функции, у которых аргументом является выражение if, если они логически эквивалентны.
Например, `avg(if(cond, col, null))` можно переписать в `avgOrNullIf(cond, col)`. Это может повысить производительность.

<Note>
  Поддерживается только при использовании анализатор (`enable_analyzer = 1`).
</Note>

  ## optimize_rewrite_array_exists_to_has
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Переписывать функции arrayExists() в has(), когда они логически эквивалентны. Например, arrayExists(x -&gt; x = 1, arr) можно переписать в has(arr, 1)

<div id="optimize_rewrite_has_to_in">
  ## optimize_rewrite_has_to_in
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.6"},{"label": "1"},{"label": "новая настройка"}]}]} />

Переписывает функции `has` в `IN`, если первый аргумент — константный массив. Например, `has([1, 2, 3], x)` можно переписать как `x IN [1, 2, 3]` для повышения производительности при использовании константных массивов
