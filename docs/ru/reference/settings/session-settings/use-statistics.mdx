  ## use_statistics_for_part_pruning
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "1"},{"label": "Новая настройка для использования статистики при отсечении частей во время выполнения запроса."}]}]} />

Использует статистику для отсеивания частей во время выполнения запроса.

Когда включено, отсечение частей в запросах SELECT использует статистику столбцов (например, статистику MinMax), чтобы исключать части, которые не могут содержать подходящие данные, до чтения каких-либо данных.

Возможные значения:

* 0 — Отключено.
* 1 — Включено.
