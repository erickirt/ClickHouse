Эти настройки предназначены для ClickHouse server и автоматически генерируются на основе исходного кода ClickHouse.

<div id="interserver_https_host">
  ## interserver_https_host
</div>

Аналогично `<interserver_http_host>`, но это имя хоста может использоваться другими серверами для доступа к данному серверу по `<HTTPS>`.

**Пример**

```xml
<interserver_https_host>example.clickhouse.com</interserver_https_host>
```

<div id="interserver_https_port">
  ## interserver_https_port
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />

Порт для обмена данными между серверами ClickHouse по протоколу `<HTTPS>`.

**Пример**

```xml
<interserver_https_port>9010</interserver_https_port>
```
