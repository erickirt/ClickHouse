Эти настройки относятся к серверу ClickHouse и автоматически сгенерированы из исходного кода ClickHouse.

<div id="max_dictionary_num_to_throw">
  ## max_dictionary_num_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Если количество словарей превышает это значение, сервер сгенерирует исключение.

Учитываются только таблицы для следующих движков баз данных:

* Atomic
* Ordinary
* Replicated
* Lazy

<Note>
  Значение `0` означает отсутствие ограничения.
</Note>

**Пример**

```xml
<max_dictionary_num_to_throw>400</max_dictionary_num_to_throw>
```

<div id="max_dictionary_num_to_warn">
  ## max_dictionary_num_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="Yes" />

Если количество присоединённых словарей превышает указанное значение, сервер ClickHouse добавит предупреждения в таблицу `system.warnings`.

**Пример**

```xml
<max_dictionary_num_to_warn>400</max_dictionary_num_to_warn>
```
