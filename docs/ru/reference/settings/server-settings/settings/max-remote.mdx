Эти настройки используются для настройки сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="max_remote_read_connections">
  ## max_remote_read_connections
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />Максимальное количество открытых удалённых соединений для чтения, которые `ReaderExecutor` поддерживает активными для оптимизации последовательного чтения. Значение 0 отключает повторное использование соединений.

<div id="max_remote_read_network_bandwidth_for_server">
  ## max_remote_read_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Максимальная скорость передачи данных по сети при чтении, в байтах в секунду.

<Note>
  Значение `0` (по умолчанию) означает отсутствие ограничений.
</Note>

<div id="max_remote_write_network_bandwidth_for_server">
  ## max_remote_write_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Максимальная скорость обмена данными по сети в байтах в секунду при записи.

<Note>
  Значение `0` (по умолчанию) означает отсутствие ограничений.
</Note>
