Эти настройки задают параметры сервера ClickHouse и автоматически генерируются на основе исходного кода ClickHouse.

<div id="replicated_fetches_http_connection_timeout">
  ## replicated_fetches_http_connection_timeout
</div>

<SettingsInfoBlock type="Секунды" default_value="0" changeable_without_restart="Нет" />Тайм-аут HTTP-соединения для запросов на загрузку частей данных. Если значение не задано явно, оно наследуется из настройки `http_connection_timeout` профиля по умолчанию.

<div id="replicated_fetches_http_receive_timeout">
  ## replicated_fetches_http_receive_timeout
</div>

<SettingsInfoBlock type="Секунды" default_value="0" changeable_without_restart="Нет" />Тайм-аут получения данных по HTTP для запросов на загрузку частей. Наследуется из профиля по умолчанию `http_receive_timeout`, если не задан явно.

<div id="replicated_fetches_http_send_timeout">
  ## replicated_fetches_http_send_timeout
</div>

<SettingsInfoBlock type="Секунды" default_value="0" changeable_without_restart="Нет" />Тайм-аут отправки для HTTP-запросов на загрузку частей. Наследуется из профиля по умолчанию `http_send_timeout`, если не задан явно.
