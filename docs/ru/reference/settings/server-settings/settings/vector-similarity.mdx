Эти настройки предназначены для конфигурации сервера ClickHouse и автоматически сгенерированы из исходного кода ClickHouse.

<div id="vector_similarity_index_cache_max_entries">
  ## vector_similarity_index_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000" changeable_without_restart="No" />Размер кэша индекса векторного сходства в записях. Ноль означает, что кэш отключён.

<div id="vector_similarity_index_cache_policy">
  ## vector_similarity_index_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Название политики кэша индекса векторного сходства.

<div id="vector_similarity_index_cache_size">
  ## vector_similarity_index_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="5368709120" changeable_without_restart="Yes" />Размер кэша для индексов векторного сходства. Ноль означает, что кэш отключен.

<Note>
  Этот параметр можно изменить во время выполнения, и изменения вступят в силу немедленно.
</Note>

<div id="vector_similarity_index_cache_size_ratio">
  ## vector_similarity_index_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />Размер защищённой очереди (в случае политики SLRU) в кэше индекса векторного сходства относительно общего размера кэша.
