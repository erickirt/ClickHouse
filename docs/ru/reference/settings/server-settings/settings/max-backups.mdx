Эти настройки определяют конфигурацию сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="max_backups_io_thread_pool_free_size">
  ## max_backups_io_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />Если количество **бездействующих** потоков в пуле потоков ввода-вывода резервного копирования превышает `max_backup_io_thread_pool_free_size`, ClickHouse освободит ресурсы, занятые бездействующими потоками, и уменьшит размер пула. При необходимости потоки будут созданы снова.

<div id="max_backups_io_thread_pool_size">
  ## max_backups_io_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="Yes" />ClickHouse использует потоки из пула потоков ввода-вывода резервного копирования для операций ввода-вывода резервных копий в S3. `max_backups_io_thread_pool_size` ограничивает максимальное число потоков в этом пуле.
