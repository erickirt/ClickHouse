Эти настройки относятся к серверу ClickHouse и автоматически сгенерированы из исходного кода ClickHouse.

<div id="query_cache.max_entries">
  ## query_cache.max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="1024" changeable_without_restart="No" />Максимальное количество результатов SELECT-запросов, хранящихся в кэше.

<div id="query_cache.max_entry_size_in_bytes">
  ## query_cache.max_entry_size_in_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1048576" changeable_without_restart="No" />Максимальный размер в байтах, который могут иметь результаты SELECT-запросов для сохранения в кэше.

<div id="query_cache.max_entry_size_in_rows">
  ## query_cache.max_entry_size_in_rows
</div>

<SettingsInfoBlock type="UInt64" default_value="30000000" changeable_without_restart="No" />Максимальное количество строк в результатах SELECT-запроса, которые можно сохранить в кэше.

<div id="query_cache.max_size_in_bytes">
  ## query_cache.max_size_in_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" changeable_without_restart="Yes" />Максимальный размер кэша в байтах. Значение 0 означает, что кэш запросов отключён.
