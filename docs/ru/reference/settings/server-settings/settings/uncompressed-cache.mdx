Эти настройки конфигурируют сервер ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="uncompressed_cache_policy">
  ## uncompressed_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Название политики кэша несжатых данных.

<div id="uncompressed_cache_size">
  ## uncompressed_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Максимальный размер (в байтах) кэша несжатых данных, используемого движками таблиц семейства MergeTree.

Для сервера используется один общий кэш. Память выделяется по мере необходимости. Кэш используется, если включена опция `use_uncompressed_cache`.

В отдельных случаях кэш несжатых данных полезен для очень коротких запросов.

<Note>
  Значение `0` означает, что кэш отключен.

  Этот параметр можно изменять во время выполнения, и он вступит в силу немедленно.
</Note>

<div id="uncompressed_cache_size_ratio">
  ## uncompressed_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />Размер защищённой очереди (в случае политики SLRU) в кэше несжатых данных относительно общего размера кэша.
