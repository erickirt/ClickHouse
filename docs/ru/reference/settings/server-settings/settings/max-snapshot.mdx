Эти настройки настраивают сервер ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="max_snapshot_commit_thread_pool_free_size">
  ## max_snapshot_commit_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />Если число бездействующих потоков в пуле потоков для коммита снимков превышает `max_snapshot_commit_thread_pool_free_size`, ClickHouse освободит ресурсы, занимаемые этими потоками, и уменьшит размер пула. При необходимости потоки будут созданы снова.

<div id="max_snapshot_commit_thread_pool_size">
  ## max_snapshot_commit_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="64" changeable_without_restart="Да" />Количество потоков для коммита снимка.
