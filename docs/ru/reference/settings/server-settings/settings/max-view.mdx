Эти настройки настраивают сервер ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="max_view_num_to_throw">
  ## max_view_num_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Если число представлений превышает это значение, сервер сгенерирует исключение.

Подсчитываются только таблицы в базах данных со следующими движками:

* Atomic
* Ordinary
* Replicated
* Lazy

<Note>
  Значение `0` означает отсутствие ограничений.
</Note>

**Пример**

```xml
<max_view_num_to_throw>400</max_view_num_to_throw>
```

<div id="max_view_num_to_warn">
  ## max_view_num_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="Yes" />

Если число подключённых представлений превышает указанное значение, сервер ClickHouse добавит предупреждения в таблицу `system.warnings`.

**Пример**

```xml
<max_view_num_to_warn>400</max_view_num_to_warn>
```
