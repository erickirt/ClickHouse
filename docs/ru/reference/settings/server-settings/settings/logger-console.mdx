Эти настройки настраивают сервер ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="logger.console">
  ## logger.console
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Включает вывод логов в консоль. Установите `<1>` или `<true>`, чтобы включить эту опцию. По умолчанию используется `<1>`, если ClickHouse не запущен в режиме демона, в противном случае — `<0>`.

<div id="logger.console_log_level">
  ## logger.console_log_level
</div>

<SettingsInfoBlock type="String" default_value="trace" changeable_without_restart="No" />Уровень логирования для вывода в консоль. По умолчанию — `<level>`.
