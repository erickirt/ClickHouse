Эти настройки задают параметры сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="os_collect_psi_metrics">
  ## os_collect_psi_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Включает сбор PSI-метрик из файлов /proc/pressure/.

<div id="os_cpu_busy_time_threshold">
  ## os_cpu_busy_time_threshold
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000" changeable_without_restart="No" />Порог времени занятости CPU ОС в микросекундах (метрика OSCPUVirtualTimeMicroseconds), начиная с которого считается, что CPU выполняет полезную работу; если время занятости ниже этого значения, перегрузка CPU не учитывается.
