Эти настройки задают параметры сервера ClickHouse и автоматически генерируются на основе исходного кода ClickHouse.

<div id="max_format_parsing_thread_pool_free_size">
  ## max_format_parsing_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Максимальное количество бездействующих резервных потоков, которые могут оставаться в пуле потоков для разбора входных данных.

<div id="max_format_parsing_thread_pool_size">
  ## max_format_parsing_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="100" changeable_without_restart="Да" />

Максимальное общее число потоков, используемых для разбора входных данных.
