Эти настройки задают конфигурацию сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="os_threads_nice_value_distributed_cache_tcp_handler">
  ## os_threads_nice_value_distributed_cache_tcp_handler
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="No" />

Значение nice в Linux для потоков обработчика TCP distributed cache. Чем меньше значение, тем выше приоритет CPU.

Требуется привилегия CAP&#95;SYS&#95;NICE, иначе это no-op.

Возможные значения: от -20 до 19.

<div id="os_threads_nice_value_merge_mutate">
  ## os_threads_nice_value_merge_mutate
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="No" />

Значение nice в Linux для потоков слияния и мутаций. Чем ниже значение, тем выше приоритет CPU.

Требуется привилегия CAP&#95;SYS&#95;NICE, иначе no-op.

Возможные значения: от -20 до 19.

<div id="os_threads_nice_value_zookeeper_client_send_receive">
  ## os_threads_nice_value_zookeeper_client_send_receive
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="No" />

Значение nice в Linux для потоков отправки и приёма в клиенте ZooKeeper. Чем ниже значение, тем выше приоритет CPU.

Требуется привилегия CAP&#95;SYS&#95;NICE, иначе операция не даёт эффекта.

Possible values: от -20 до 19.
