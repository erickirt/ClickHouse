Эти настройки задают конфигурацию сервера ClickHouse и автоматически сгенерированы из исходного кода ClickHouse.

<div id="tcp_port">
  ## tcp_port
</div>

Порт для взаимодействия с клиентами по протоколу TCP.

**Пример**

```xml
<tcp_port>9000</tcp_port>
```

<div id="tcp_port_secure">
  ## tcp_port_secure
</div>

TCP-порт для защищённого взаимодействия с клиентами. Используйте его вместе с настройками [OpenSSL](/ru/reference/settings/server-settings/settings/other#openssl).

**Значение по умолчанию**

```xml
<tcp_port_secure>9440</tcp_port_secure>
```
