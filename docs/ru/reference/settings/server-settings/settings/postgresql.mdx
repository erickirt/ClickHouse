Эти настройки конфигурируют сервер ClickHouse и автоматически генерируются на основе исходного кода ClickHouse.

<div id="postgresql_port">
  ## postgresql_port
</div>

Порт для взаимодействия с клиентами по протоколу PostgreSQL.

<Note>
  * Положительные целые числа задают номер порта для прослушивания
  * Пустые значения используются для отключения взаимодействия с клиентами по протоколу PostgreSQL.
</Note>

**Пример**

```xml
<postgresql_port>9005</postgresql_port>
```

<div id="postgresql_require_secure_transport">
  ## postgresql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Если установлено значение true, требуется защищённое взаимодействие с клиентами через [postgresql&#95;port](/ru/reference/settings/server-settings/settings/postgresql#postgresql_port). Подключение с параметром `<sslmode=disable>` будет отклонено. Используйте эту настройку вместе с настройками [OpenSSL](/ru/reference/settings/server-settings/settings/other#openssl).
