Эти настройки задают конфигурацию сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="parts_killer_max_condemned_parts_per_batch">
  ## parts_killer_max_condemned_parts_per_batch
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="100000" changeable_without_restart="No" />

Максимальное количество частей, помеченных на удаление, которые таблица SharedMergeTree загружает из Keeper за один цикл очистки. Ограничение размера батча позволяет удерживать операции listing и multi-get в пределах тайм-аута Keeper, благодаря чему очистка продолжает выполняться даже при очень большом накопившемся количестве частей, помеченных на удаление. Значение должно быть больше нуля: при значении `0` Keeper будет возвращать пустой listing в каждом цикле, и очистка будет навсегда заблокирована. Доступно только в ClickHouse Cloud

<div id="parts_killer_pool_size">
  ## parts_killer_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="128" changeable_without_restart="No" />

Потоки очистки, используемые процессом удаления частей shared merge tree. Доступно только в ClickHouse Cloud
