Эти настройки задают конфигурацию сервера ClickHouse и автоматически сгенерированы из исходного кода ClickHouse.

<div id="background_buffer_flush_schedule_pool_size">
  ## background_buffer_flush_schedule_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="Только увеличение" />Максимальное количество потоков, используемых для фонового выполнения операций сброса для [таблиц движка Buffer](/ru/reference/engines/table-engines/special/buffer).

<div id="background_common_pool_size">
  ## background_common_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="8" changeable_without_restart="No" />Максимальное количество потоков, используемых в фоновом режиме для выполнения различных операций (в основном сборки мусора) для [таблиц семейства *MergeTree](/ru/reference/engines/table-engines/mergetree-family/index).

<div id="background_distributed_schedule_pool_size">
  ## background_distributed_schedule_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="Только увеличение" />Максимальное количество потоков, используемых для выполнения распределённой отправки данных.

<div id="background_fetches_pool_size">
  ## background_fetches_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="No" />Максимальное количество потоков, используемых в фоновом режиме для загрузки частей данных с другой реплики для [таблиц семейства *MergeTree](/ru/reference/engines/table-engines/mergetree-family/index).

<div id="background_message_broker_schedule_pool_size">
  ## background_message_broker_schedule_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="Только увеличение" />Максимальное количество потоков, используемых для выполнения фоновых операций, связанных с потоковой передачей сообщений.

<div id="background_move_pool_size">
  ## background_move_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="8" changeable_without_restart="No" />Максимальное количество потоков, используемых в фоновом режиме для перемещения частей данных на другой диск или том в таблицах семейства *MergeTree.

<div id="background_pool_size">
  ## background_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="No" />

Задаёт количество потоков, выполняющих фоновые слияния и мутации для таблиц с движками MergeTree.

<Note>
  * Этот параметр также можно применять при запуске сервера из конфигурации профиля `default` для обеспечения обратной совместимости при запуске сервер ClickHouse.
  * Во время выполнения можно только увеличить количество потоков.
  * Чтобы уменьшить количество потоков, необходимо перезапустить сервер.
  * Настраивая этот параметр, вы управляете нагрузкой на CPU и диск.
</Note>

<Danger>
  Меньший размер пула требует меньше ресурсов CPU и диска, но фоновые процессы выполняются медленнее, что со временем может повлиять на производительность запросов.
</Danger>

Перед изменением этого параметра также обратите внимание на связанные настройки MergeTree, такие как:

* [`number_of_free_entries_in_pool_to_lower_max_size_of_merge`](/ru/reference/settings/merge-tree-settings#number_of_free_entries_in_pool_to_lower_max_size_of_merge).
* [`number_of_free_entries_in_pool_to_execute_mutation`](/ru/reference/settings/merge-tree-settings#number_of_free_entries_in_pool_to_execute_mutation).
* [`number_of_free_entries_in_pool_to_execute_optimize_entire_partition`](/ru/reference/settings/merge-tree-settings#number_of_free_entries_in_pool_to_execute_optimize_entire_partition)

**Пример**

```xml
<background_pool_size>16</background_pool_size>
```
