Эти настройки определяют конфигурацию сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="storage_connections_hard_limit">
  ## storage_connections_hard_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="200000" changeable_without_restart="No" />При попытке создания соединения при достижении этого предела будет генерироваться исключение. Установите 0, чтобы отключить жесткое ограничение. Это ограничение применяется к подключениям к хранилищам.

<div id="storage_connections_rcvbuf">
  ## storage_connections_rcvbuf
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />Размер опции SO&#95;RCVBUF для подключений к хранилищу (репликация, распределённые запросы). Если задано значение больше 0, оно переопределяет автотюнинг TCP ядра для буфера приёма. 0 = значение ядра по умолчанию (автотюнинг). Примечание: если снова установить для этого параметра значение 0, автотюнинг будет восстановлен только для новых подключений; существующие подключения из пула сохранят фиксированные размеры буфера до пересоздания.

<div id="storage_connections_sndbuf">
  ## storage_connections_sndbuf
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />Размер параметра SO&#95;SNDBUF для подключений к хранилищу (репликация, распределённые запросы). Если задано значение больше 0, оно отключает автотюнинг TCP ядра для буфера отправки. 0 = значение ядра по умолчанию (автотюнинг). Примечание: если вернуть этот параметр к 0, автотюнинг восстановится только для вновь создаваемых подключений; существующие соединения из пула сохранят фиксированный размер буфера, пока не будут пересозданы.

<div id="storage_connections_soft_limit">
  ## storage_connections_soft_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="100" changeable_without_restart="No" />Для соединений сверх этого предела время жизни значительно сокращается. Предел применяется к подключениям к хранилищам.

<div id="storage_connections_store_limit">
  ## storage_connections_store_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />Соединения сверх этого лимита сбрасываются после использования. Установите значение 0, чтобы отключить кэш соединений. Лимит применяется к подключениям к хранилищам.

<div id="storage_connections_warn_limit">
  ## storage_connections_warn_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="500" changeable_without_restart="No" />Предупреждающие сообщения записываются в журнал, если число используемых соединений превышает этот предел. Предел применяется к подключениям к хранилищам.
