Эти настройки задают конфигурацию сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="max_database_num_to_throw">
  ## max_database_num_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />Если количество баз данных превышает это значение, сервер сгенерирует исключение. 0 означает отсутствие ограничений.

<div id="max_database_num_to_warn">
  ## max_database_num_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="Yes" />

Если число присоединённых баз данных превышает указанное значение, сервер ClickHouse добавит предупреждения в таблицу `system.warnings`.

**Пример**

```xml
<max_database_num_to_warn>50</max_database_num_to_warn>
```

<div id="max_database_replicated_create_table_thread_pool_size">
  ## max_database_replicated_create_table_thread_pool_size
</div>

<SettingsInfoBlock type="UInt32" default_value="1" changeable_without_restart="No" />Количество потоков для создания таблиц во время восстановления реплики в DatabaseReplicated. Ноль означает, что количество потоков равно числу ядер.
