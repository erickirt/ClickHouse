Эти настройки определяют конфигурацию сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="s3queue_disable_streaming">
  ## s3queue_disable_streaming
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Да" />Отключает стриминг в S3Queue, даже если таблица создана и к ней присоединены материализованные представления

<div id="s3queue_log">
  ## s3queue_log
</div>

Параметры системной таблицы `s3queue_log`.

<SystemLogParameters />

По умолчанию используются следующие настройки:

```xml
<s3queue_log>
    <database>system</database>
    <table>s3queue_log</table>
    <partition_by>toYYYYMM(event_date)</partition_by>
    <flush_interval_milliseconds>7500</flush_interval_milliseconds>
</s3queue_log>
```
