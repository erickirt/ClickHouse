Эти настройки конфигурируют сервер ClickHouse и автоматически генерируются на основе исходного кода ClickHouse.

<div id="mlock_executable">
  ## mlock_executable
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />

Выполняет `<mlockall>` после запуска, чтобы уменьшить задержку первых запросов и предотвратить выгрузку исполняемого файла ClickHouse в подкачку при высокой нагрузке на IO.

<Note>
  Этот параметр рекомендуется включить, однако это может увеличить время запуска на несколько секунд. Имейте в виду, что без capability &quot;CAP&#95;IPC&#95;LOCK&quot; этот параметр работать не будет.
</Note>

**Пример**

```xml
<mlock_executable>false</mlock_executable>
```
