Эти настройки задают конфигурацию сервера ClickHouse и автоматически генерируются на основе исходного кода ClickHouse.

<div id="skip_binary_checksum_checks">
  ## skip_binary_checksum_checks
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Пропускает проверки контрольной суммы бинарного файла ClickHouse

<div id="skip_check_for_incorrect_settings">
  ## skip_check_for_incorrect_settings
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />

Если задано значение true, корректность настроек сервера не проверяется.

**Пример**

```xml
<skip_check_for_incorrect_settings>1</skip_check_for_incorrect_settings>
```
