Эти настройки задают конфигурацию сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="max_distributed_cache_read_bandwidth_for_server">
  ## max_distributed_cache_read_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />Максимальная суммарная скорость чтения из distributed cache на сервере, в байтах в секунду. Ноль означает отсутствие ограничений.

<div id="max_distributed_cache_write_bandwidth_for_server">
  ## max_distributed_cache_write_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />Максимальная суммарная скорость записи в distributed cache на сервере, в байтах в секунду. Ноль означает отсутствие ограничений.
