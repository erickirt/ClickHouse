Эти настройки предназначены для конфигурации сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="backups">
  ## резервные копии
</div>

Настройки резервного копирования, используемые при выполнении команд [`BACKUP` и `RESTORE`](/ru/concepts/features/backup-restore/overview).

Следующие настройки можно задать с помощью под-тегов:


<div id="backups_io_thread_pool_queue_size">
  ## backups_io_thread_pool_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Максимальное количество задач, которые можно поставить в очередь пула потоков ввода-вывода Backups. Из-за текущей логики резервного копирования в S3 рекомендуется оставлять эту очередь неограниченной.

<Note>
  Значение `0` (по умолчанию) означает неограниченную очередь.
</Note>
