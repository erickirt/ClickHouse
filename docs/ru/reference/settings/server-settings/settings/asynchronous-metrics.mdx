Эти настройки настраивают сервер ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="asynchronous_metrics_enable_heavy_metrics">
  ## asynchronous_metrics_enable_heavy_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Включает расчёт тяжёлых асинхронных метрик.

<div id="asynchronous_metrics_keeper_metrics_only">
  ## asynchronous_metrics_keeper_metrics_only
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Ограничивает вычисление асинхронных метрик только метриками, связанными с Keeper.

<div id="asynchronous_metrics_update_period_s">
  ## asynchronous_metrics_update_period_s
</div>

<SettingsInfoBlock type="UInt32" default_value="1" changeable_without_restart="No" />Период обновления асинхронных метрик в секундах.
