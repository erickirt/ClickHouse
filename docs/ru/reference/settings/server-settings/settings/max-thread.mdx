Эти настройки задают конфигурацию сервера ClickHouse и автоматически генерируются на основе исходного кода ClickHouse.

<div id="max_thread_pool_free_size">
  ## max_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />

Если количество **бездействующих** потоков в Global Thread pool превышает [`max_thread_pool_free_size`](/ru/reference/settings/server-settings/settings/max-thread#max_thread_pool_free_size), ClickHouse освобождает ресурсы, занятые некоторыми потоками, и размер пула уменьшается. При необходимости потоки могут быть созданы повторно.

**Пример**

```xml
<max_thread_pool_free_size>1200</max_thread_pool_free_size>
```

<div id="max_thread_pool_size">
  ## max_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />

ClickHouse использует потоки из Global Thread pool для обработки запросов. Если для обработки запроса нет бездействующего потока, в пуле создаётся новый. `max_thread_pool_size` ограничивает максимальное количество потоков в пуле.

**Пример**

```xml
<max_thread_pool_size>12000</max_thread_pool_size>
```
