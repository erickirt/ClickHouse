Эти настройки определяют конфигурацию сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="iceberg_metadata_files_cache_max_entries">
  ## iceberg_metadata_files_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />Максимальный размер кэша файлов метаданных Iceberg в записях. Ноль означает, что кэш отключён.

<div id="iceberg_metadata_files_cache_policy">
  ## iceberg_metadata_files_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="Нет" />Название политики кэширования метаданных Iceberg.

<div id="iceberg_metadata_files_cache_size">
  ## iceberg_metadata_files_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="134217728" changeable_without_restart="Yes" />Максимальный размер кэша метаданных Iceberg в байтах. Ноль означает, что кэш отключён.

<div id="iceberg_metadata_files_cache_size_ratio">
  ## iceberg_metadata_files_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />Размер защищённой очереди (в случае политики SLRU) в кэше метаданных Iceberg относительно общего размера кэша.
