Эти настройки задают конфигурацию сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="index_uncompressed_cache_policy">
  ## index_uncompressed_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Название политики кэша несжатых данных вторичных индексов.

<div id="index_uncompressed_cache_size">
  ## index_uncompressed_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Максимальный размер кэша для несжатых блоков индексов `MergeTree`.

<Note>
  Значение `0` означает, что кэш отключён.

  Этот параметр можно изменять во время выполнения, и изменения вступят в силу немедленно.
</Note>

<div id="index_uncompressed_cache_size_ratio">
  ## index_uncompressed_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />Размер защищённой очереди (в случае политики SLRU) в кэше несжатых данных вторичных индексов по отношению к общему размеру кэша.
