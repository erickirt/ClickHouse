Эти настройки относятся к серверу ClickHouse и автоматически генерируются на основе исходного кода ClickHouse.

<div id="shutdown_wait_backups_and_restores">
  ## shutdown_wait_backups_and_restores
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Если задано значение true, ClickHouse будет ждать завершения выполняющихся резервных копий и восстановления перед остановкой.

<div id="shutdown_wait_unfinished">
  ## shutdown_wait_unfinished
</div>

<SettingsInfoBlock type="UInt64" default_value="5" changeable_without_restart="No" />Время задержки в секундах перед ожиданием завершения незавершённых запросов

<div id="shutdown_wait_unfinished_queries">
  ## shutdown_wait_unfinished_queries
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Если установить значение `true`, ClickHouse перед завершением работы будет ждать окончания выполняющихся запросов.
