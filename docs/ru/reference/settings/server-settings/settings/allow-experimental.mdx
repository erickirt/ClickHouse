Эти настройки используются для настройки сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="allow_experimental_executable_udf_drivers">
  ## allow_experimental_executable_udf_drivers
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Включает экспериментальную поддержку драйверов для исполняемых пользовательских функций, определяемых через `user_defined_executable_function_drivers_config`. Драйвер преобразует фрагмент пользовательского кода, указанный в `CREATE FUNCTION ... ENGINE = DriverName(...) AS '...'`, в исполняемую UDF, готовую к выполнению.

<div id="allow_experimental_webassembly_udf">
  ## allow_experimental_webassembly_udf
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Включает поддержку WebAssembly UDF в экспериментальном режиме

<div id="allow_experimental_webterminal">
  ## allow_experimental_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Прежнее (экспериментальное) название `enable_webterminal`. По-прежнему поддерживается для обратной совместимости, если `enable_webterminal` не задан. Настройка устарела.
