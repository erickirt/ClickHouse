Эти настройки задают конфигурацию сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="global_profiler_cpu_time_period_ns">
  ## global_profiler_cpu_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />Период таймера CPU clock для глобального профилировщика (в наносекундах). Чтобы отключить глобальный профилировщик CPU clock, установите значение 0. Рекомендуемое значение — не менее 10000000 (100 раз в секунду) для отдельных запросов или 1000000000 (раз в секунду) для профилирования в масштабе всего кластера.

<div id="global_profiler_real_time_period_ns">
  ## global_profiler_real_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />Период таймера реального времени глобального профилировщика (в наносекундах). Установите значение 0, чтобы отключить глобальный профилировщик реального времени. Рекомендуемое значение — не менее 10000000 (100 раз в секунду) для отдельных запросов или 1000000000 (один раз в секунду) для профилирования в масштабе всего кластера.
