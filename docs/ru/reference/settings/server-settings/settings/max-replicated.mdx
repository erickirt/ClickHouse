Эти настройки предназначены для конфигурации сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="max_replicated_fetches_network_bandwidth_for_server">
  ## max_replicated_fetches_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />Максимальная скорость сетевого обмена данными в байтах в секунду для загрузок из реплик. Ноль означает отсутствие ограничений.

<div id="max_replicated_sends_network_bandwidth_for_server">
  ## max_replicated_sends_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Нет" />Максимальная скорость обмена данными по сети в байтах в секунду при реплицированных отправках. Ноль означает, что ограничение отсутствует.

<div id="max_replicated_table_num_to_throw">
  ## max_replicated_table_num_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Если количество реплицируемых таблиц превышает это значение, сервер сгенерирует исключение.

Учитываются только таблицы с движками баз данных:

* Atomic
* Ordinary
* Replicated
* Lazy

<Note>
  Значение `0` означает отсутствие ограничений.
</Note>

**Пример**

```xml
<max_replicated_table_num_to_throw>400</max_replicated_table_num_to_throw>
```
