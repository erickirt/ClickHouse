  ## database_replicated_drop_broken_tables

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Удалять неожиданно обнаруженные таблицы из баз данных Replicated вместо их перемещения в отдельную локальную базу данных
