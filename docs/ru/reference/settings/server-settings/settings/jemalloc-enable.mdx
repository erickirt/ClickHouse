Эти настройки задают параметры сервера ClickHouse и автоматически сгенерированы из исходного кода ClickHouse.

<div id="jemalloc_enable_background_threads">
  ## jemalloc_enable_background_threads
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Включает фоновые потоки jemalloc. Jemalloc использует фоновые потоки для очистки неиспользуемых страниц памяти. Их отключение может привести к снижению производительности.

<div id="jemalloc_enable_global_profiler">
  ## jemalloc_enable_global_profiler
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Включает профилировщик выделения памяти jemalloc для всех потоков. Jemalloc будет выполнять выборочное профилирование выделений памяти и всех освобождений для отобранных выделений.
Профили можно сбрасывать на диск с помощью SYSTEM JEMALLOC FLUSH PROFILE, которую можно использовать для анализа выделений памяти.
Образцы также можно сохранять в system.trace&#95;log с помощью параметра конфигурации jemalloc&#95;collect&#95;global&#95;profile&#95;samples&#95;in&#95;trace&#95;log или настройки запроса jemalloc&#95;collect&#95;profile&#95;samples&#95;in&#95;trace&#95;log.
См. [Профилирование выделений памяти](/ru/concepts/features/performance/allocation-profiling)
