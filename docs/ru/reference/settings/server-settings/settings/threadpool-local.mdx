Эти настройки задают конфигурацию сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="threadpool_local_fs_reader_pool_size">
  ## threadpool_local_fs_reader_pool_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="100" changeable_without_restart="No" />Количество потоков в пуле потоков, используемом для чтения из локальной файловой системы, когда `local_filesystem_read_method = 'pread_threadpool'`.

<div id="threadpool_local_fs_reader_queue_size">
  ## threadpool_local_fs_reader_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />Максимальное количество задач, которые могут быть поставлены в очередь пула потоков для чтения из локальной файловой системы.
