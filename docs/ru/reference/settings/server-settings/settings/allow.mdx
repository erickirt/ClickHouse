Эти настройки используются для конфигурации сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="allow_feature_tier">
  ## allow_feature_tier
</div>

<SettingsInfoBlock type="UInt32" default_value="0" changeable_without_restart="Yes" />

Определяет, может ли пользователь изменять настройки, относящиеся к различным уровням возможностей.

* `0` - Разрешено изменять любые настройки (экспериментальные, бета, продакшн).
* `1` - Разрешено изменять только настройки возможностей уровней бета и продакшн. Изменения экспериментальных настроек отклоняются.
* `2` - Разрешено изменять только настройки возможностей уровня продакшн. Изменения экспериментальных настроек и настроек уровня бета отклоняются.

Это эквивалентно установке ограничения «только для чтения» для всех возможностей `EXPERIMENTAL` / `BETA`.

<Note>
  Значение `0` означает, что можно изменять все настройки.
</Note>

<div id="allow_impersonate_user">
  ## allow_impersonate_user
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Включает или отключает возможность IMPERSONATE (EXECUTE AS target&#95;user). Эта настройка устарела.

<div id="allow_implicit_no_password">
  ## allow_implicit_no_password
</div>

Запрещает создавать пользователя без пароля, если только явно не указано &#39;IDENTIFIED WITH no&#95;password&#39;.

```xml
<allow_implicit_no_password>1</allow_implicit_no_password>
```

<div id="allow_no_password">
  ## allow_no_password
</div>

Определяет, разрешён ли небезопасный тип пароля no&#95;password.

```xml
<allow_no_password>1</allow_no_password>
```

<div id="allow_plaintext_password">
  ## allow_plaintext_password
</div>

Определяет, разрешены ли типы паролей в незашифрованном виде (небезопасные).

```xml
<allow_plaintext_password>1</allow_plaintext_password>
```

<div id="allow_use_jemalloc_memory">
  ## allow_use_jemalloc_memory
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="Нет" />Разрешает использовать jemalloc для управления памятью.
