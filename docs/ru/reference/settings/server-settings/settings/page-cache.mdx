Эти настройки задают конфигурацию сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="page_cache_free_memory_ratio">
  ## page_cache_free_memory_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.15" changeable_without_restart="No" />Доля лимита памяти, которую следует оставлять свободной от кэша страниц в пространстве пользователя. Аналогично настройке Linux `min_free_kbytes`.

<div id="page_cache_history_window_ms">
  ## page_cache_history_window_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />Задержка перед тем, как освобождённая память сможет использоваться кэшем страниц в пространстве пользователя.

<div id="page_cache_max_size">
  ## page_cache_max_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />Максимальный размер кэша страниц в пространстве пользователя. Установите значение 0, чтобы отключить кэш. Если значение больше `page_cache_min_size`, размер кэша будет непрерывно корректироваться в этом диапазоне, чтобы использовать большую часть доступной памяти, сохраняя общее использование памяти ниже предела (`max_server_memory_usage[_to_ram_ratio]`).

<div id="page_cache_min_size">
  ## page_cache_min_size
</div>

<SettingsInfoBlock type="UInt64" default_value="104857600" changeable_without_restart="No" />Минимальный размер кэша страниц в пространстве пользователя.

<div id="page_cache_policy">
  ## page_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Название политики кэша страниц в пространстве пользователя.

<div id="page_cache_shards">
  ## page_cache_shards
</div>

<SettingsInfoBlock type="UInt64" default_value="4" changeable_without_restart="No" />Разбить кэш страниц в пространстве пользователя на такое количество сегментов, чтобы снизить конкуренцию за мьютексы. Экспериментальная настройка; вряд ли повысит производительность.

<div id="page_cache_size_ratio">
  ## page_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />Размер защищённой очереди в кэше страниц в пространстве пользователя относительно общего размера кэша.
