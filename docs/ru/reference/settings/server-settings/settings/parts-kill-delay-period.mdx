Эти настройки задают конфигурацию сервера ClickHouse и автоматически генерируются на основе исходного кода ClickHouse.

<div id="parts_kill_delay_period">
  ## parts_kill_delay_period
</div>

<SettingsInfoBlock type="UInt64" default_value="30" changeable_without_restart="No" />

Период, по истечении которого части в SharedMergeTree полностью удаляются. Доступно только в ClickHouse Cloud

<div id="parts_kill_delay_period_random_add">
  ## parts_kill_delay_period_random_add
</div>

<SettingsInfoBlock type="UInt64" default_value="10" changeable_without_restart="No" />

Добавляет к kill&#95;delay&#95;period равномерно распределённое значение от 0 до x секунд, чтобы избежать эффекта «стадного поведения» и последующей DoS-атаки на ZooKeeper при очень большом количестве таблиц. Доступно только в ClickHouse Cloud
