Эти настройки относятся к серверу ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="threadpool_remote_fs_reader_pool_size">
  ## threadpool_remote_fs_reader_pool_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="250" changeable_without_restart="No" />Количество потоков в пуле, используемом для чтения из удалённой файловой системы, когда `remote_filesystem_read_method = 'threadpool'`.

<div id="threadpool_remote_fs_reader_queue_size">
  ## threadpool_remote_fs_reader_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />Максимальное количество задач, которые можно поставить в очередь пула потоков для чтения из удалённой файловой системы.
