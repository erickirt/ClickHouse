Эти настройки задают конфигурацию сервера ClickHouse и автоматически генерируются на основе исходного кода ClickHouse.

<div id="query_condition_cache_policy">
  ## query_condition_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Название политики кэша условий запроса.

<div id="query_condition_cache_size">
  ## query_condition_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="104857600" changeable_without_restart="Yes" />

Максимальный размер кэша условий запроса.

<Note>
  Этот параметр можно изменить во время работы, и изменения вступят в силу немедленно.
</Note>

<div id="query_condition_cache_size_ratio">
  ## query_condition_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="Нет" />Размер защищённой очереди (в случае политики SLRU) в кэше условий запроса относительно общего размера кэша.
