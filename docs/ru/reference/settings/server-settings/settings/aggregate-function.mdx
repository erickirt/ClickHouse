Эти настройки задают конфигурацию сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="aggregate_function_group_array_action_when_limit_is_reached">
  ## aggregate_function_group_array_action_when_limit_is_reached
</div>

<SettingsInfoBlock type="GroupArrayActionWhenLimitReached" default_value="throw" changeable_without_restart="No" />Действие при превышении максимального размера элемента массива в groupArray: `throw` — сгенерировать исключение, `discard` — отбросить лишние значения

<div id="aggregate_function_group_array_max_element_size">
  ## aggregate_function_group_array_max_element_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16777215" changeable_without_restart="No" />Максимальный размер элемента массива в байтах для функции groupArray. Это ограничение проверяется при сериализации и помогает избежать слишком большого размера состояния.
