Эти настройки задают конфигурацию сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="users_config">
  ## users_config
</div>

Путь к файлу, содержащему:

* Конфигурации пользователей.
* Права доступа.
* Профили настроек.
* Настройки квот.

**Пример**

```xml
<users_config>users.xml</users_config>
```

<div id="users_to_ignore_early_memory_limit_check">
  ## users_to_ignore_early_memory_limit_check
</div>

Разделённый запятыми список пользователей, для которых ранняя проверка лимита памяти не выполняется. Если пользователя нет в этом списке, запрос будет отклонён, если общее использование памяти превысит лимит.
