Эти настройки задают конфигурацию сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="mysql_port">
  ## mysql_port
</div>

Порт для связи с клиентами по протоколу MySQL.

<Note>
  * Положительные целые числа задают номер порта для прослушивания
  * Пустые значения используются для отключения связи с клиентами по протоколу MySQL.
</Note>

**Пример**

```xml
<mysql_port>9004</mysql_port>
```

<div id="mysql_require_secure_transport">
  ## mysql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Если задано значение true, для взаимодействия с клиентами через [mysql&#95;port](/ru/reference/settings/server-settings/settings/mysql#mysql_port) требуется защищённое соединение. Подключения с параметром `<--ssl-mode=none>` будут отклонены. Используйте этот параметр вместе с настройками [OpenSSL](/ru/reference/settings/server-settings/settings/other#openssl).
