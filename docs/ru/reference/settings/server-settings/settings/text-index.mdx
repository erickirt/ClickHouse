Эти настройки предназначены для конфигурирования сервера ClickHouse и автоматически сгенерированы из исходного кода ClickHouse.

<div id="text_index_header_cache_max_entries">
  ## text_index_header_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="100000" changeable_without_restart="No" />Размер кэша заголовка текстового индекса в записях. Ноль означает, что кэш отключён.

<div id="text_index_header_cache_policy">
  ## text_index_header_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Имя политики кэширования заголовка текстового индекса.

<div id="text_index_header_cache_size">
  ## text_index_header_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" changeable_without_restart="Yes" />Размер кэша для заголовков текстового индекса. Ноль означает, что кэш отключён.

<Note>
  Этот параметр можно изменять во время выполнения, и изменения вступят в силу немедленно.
</Note>

<div id="text_index_header_cache_size_ratio">
  ## text_index_header_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />Доля защищённой очереди (в случае политики SLRU) в кэше заголовка текстового индекса относительно общего размера кэша.
