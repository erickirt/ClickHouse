Эти настройки задают конфигурацию сервера ClickHouse и автоматически сгенерированы из исходного кода ClickHouse.

<div id="enable_azure_sdk_logging">
  ## enable_azure_sdk_logging
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Нет" />Включает логирование Azure SDK

<div id="enable_webterminal">
  ## enable_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="Нет" />Включает интерфейс веб-терминала на HTTP-конечной точке `/webterminal`. Предоставляет интерактивный сеанс клиента ClickHouse в браузере через WebSocket. Если установлено значение `false`, запросы к `/webterminal` возвращают HTTP-статус `403 Forbidden`.
