Эти настройки задают параметры сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="http_connections_hard_limit">
  ## http_connections_hard_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="200000" changeable_without_restart="No" />При попытке создать новое соединение после достижения этого предела генерируется исключение. Установите значение 0, чтобы отключить жёсткое ограничение. Этот предел применяется к HTTP-соединениям, не относящимся ни к одному диску или хранилищу.

<div id="http_connections_rcvbuf">
  ## http_connections_rcvbuf
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Да" />Размер параметра SO&#95;RCVBUF для обычных HTTP-соединений. Если задано значение больше 0, оно отключает TCP автонастройку ядра для буфера приёма и использует указанное значение. 0 = значение ядра по умолчанию (автонастройка). Примечание: если снова установить для этого параметра значение 0, автонастройка будет восстановлена только для новых соединений; существующие соединения из пула сохранят фиксированные размеры буфера до пересоздания.

<div id="http_connections_sndbuf">
  ## http_connections_sndbuf
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />Размер параметра SO&#95;SNDBUF для обычных HTTP-соединений. Если задано значение больше 0, оно переопределяет автонастройку TCP в ядре для буфера отправки. 0 = значение ядра по умолчанию (автонастройка). Примечание: если снова установить для этого параметра значение 0, автонастройка восстановится только для новых соединений; существующие соединения из пула сохранят фиксированные размеры буфера, пока не будут пересозданы.

<div id="http_connections_soft_limit">
  ## http_connections_soft_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="100" changeable_without_restart="No" />Для HTTP-соединений, превышающих этот предел, время жизни значительно сокращается. Предел применяется к HTTP-соединениям, которые не относятся ни к какому диску или хранилищу.

<div id="http_connections_store_limit">
  ## http_connections_store_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />Соединения сверх этого лимита сбрасываются после использования. Установите 0, чтобы отключить кэш соединений. Лимит применяется к HTTP-соединениям, которые не относятся ни к одному диску или хранилищу.

<div id="http_connections_warn_limit">
  ## http_connections_warn_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="500" changeable_without_restart="No" />Предупреждающие сообщения записываются в журнал, если число используемых соединений превышает этот предел. Этот предел применяется к HTTP-соединениям, которые не принадлежат ни к какому диску или хранилищу.
