Эти настройки определяют конфигурацию сервера ClickHouse и автоматически генерируются из исходного кода ClickHouse.

<div id="tcp_close_connection_after_queries_num">
  ## tcp_close_connection_after_queries_num
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />Максимальное количество запросов в рамках одного TCP-соединения до его закрытия. Установите 0, чтобы снять ограничение на число запросов.

<div id="tcp_close_connection_after_queries_seconds">
  ## tcp_close_connection_after_queries_seconds
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />Максимальное время жизни TCP-соединения в секундах до его закрытия. Значение 0 означает неограниченное время жизни подключения.
