Эти настройки определяют конфигурацию сервера ClickHouse и автоматически сгенерированы на основе исходного кода ClickHouse.

<div id="keeper_hosts">
  ## keeper_hosts
</div>

Динамическая настройка. Содержит набор хостов [Zoo]Keeper, к которым ClickHouse потенциально может подключиться. Не раскрывает информацию из `<auxiliary_zookeepers>`

<div id="keeper_multiread_batch_size">
  ## keeper_multiread_batch_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />

Максимальный размер батча для запроса MultiRead к [Zoo]Keeper с поддержкой батчинга. Если задано значение 0, батчинг отключается. Доступно только в ClickHouse Cloud.
