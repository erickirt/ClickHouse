Эти настройки относятся к серверу ClickHouse и автоматически сгенерированы из исходного кода ClickHouse.

<div id="merges_mutations_memory_usage_soft_limit">
  ## merges_mutations_memory_usage_soft_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />

Задаёт ограничение на объём оперативной памяти, который можно использовать для выполнения операций слияния и мутаций.
Если ClickHouse достигает заданного ограничения, он не будет планировать новые фоновые операции слияния или мутации, но продолжит выполнять уже запланированные задачи.

<Note>
  Значение `0` означает отсутствие ограничений.
</Note>

**Пример**

```xml
<merges_mutations_memory_usage_soft_limit>0</merges_mutations_memory_usage_soft_limit>
```

<div id="merges_mutations_memory_usage_to_ram_ratio">
  ## merges_mutations_memory_usage_to_ram_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />

Значение `merges_mutations_memory_usage_soft_limit` по умолчанию рассчитывается по формуле `memory_amount * merges_mutations_memory_usage_to_ram_ratio`.
