Эти настройки задают конфигурацию сервера ClickHouse и автоматически сгенерированы из исходного кода ClickHouse.

<div id="primary_index_cache_policy">
  ## primary_index_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Название политики кэширования первичного индекса.

<div id="primary_index_cache_prewarm_ratio">
  ## primary_index_cache_prewarm_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.95" changeable_without_restart="No" />Доля общего размера кэша меток, заполняемая при предварительном прогреве.

<div id="primary_index_cache_size">
  ## primary_index_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="5368709120" changeable_without_restart="Yes" />Максимальный размер кэша первичного индекса (индекса таблиц семейства MergeTree).

<div id="primary_index_cache_size_ratio">
  ## primary_index_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />Размер защищённой очереди (при использовании политики SLRU) в кэше первичного индекса относительно общего размера кэша.
