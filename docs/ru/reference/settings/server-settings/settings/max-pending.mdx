Эти настройки определяют конфигурацию сервера ClickHouse и автоматически сгенерированы из исходного кода ClickHouse.

<div id="max_pending_mutations_execution_time_to_warn">
  ## max_pending_mutations_execution_time_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="86400" changeable_without_restart="Да" />

Если время выполнения любой из ожидающих мутаций превышает указанное значение в секундах, сервер ClickHouse добавит предупреждения в таблицу `system.warnings`.

**Пример**

```xml
<max_pending_mutations_execution_time_to_warn>10000</max_pending_mutations_execution_time_to_warn>
```

<div id="max_pending_mutations_to_warn">
  ## max_pending_mutations_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="500" changeable_without_restart="Yes" />

Если число ожидающих мутаций превышает указанное значение, сервер ClickHouse добавит предупреждения в таблицу `system.warnings`.

**Пример**

```xml
<max_pending_mutations_to_warn>400</max_pending_mutations_to_warn>
```
