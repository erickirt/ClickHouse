<div id="description">
  ## Описание
</div>

Показывает все доступные стеммеры.
Их можно использовать в функции [stem](/ru/reference/functions/regular-functions/nlp-functions).

<Info>
  **Доступность**

  `system.stemmers` доступна только в сборках ClickHouse, скомпилированных с зависимостью `libstemmer` (`USE_LIBSTEMMER`). В сборках без нее эта таблица отсутствует, и запросы к ней завершаются ошибкой `UNKNOWN_TABLE`. Проверить, включена ли эта зависимость в вашей сборке, можно так:

  ```sql
  SELECT value FROM system.build_options WHERE name = 'USE_LIBSTEMMER';
  ```
</Info>

Столбцы:

{/*AUTOGENERATED_START*/}

* `name` ([String](/ru/reference/data-types/index)) — идентификатор Snowball-стеммера (язык/алгоритм)

{/*AUTOGENERATED_END*/}
