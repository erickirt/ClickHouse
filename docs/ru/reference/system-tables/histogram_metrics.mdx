<div id="description">
  ## Описание
</div>

Эта таблица содержит метрики гистограммы, которые можно мгновенно вычислять и экспортировать в формате Prometheus. Данные в ней всегда актуальны. Заменяет устаревшую `system.latency_log`.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `metric` ([String](/ru/reference/data-types/string)) — Название метрики.
* `value` ([Float64](/ru/reference/data-types/float)) — Значение метрики.
* `description` ([String](/ru/reference/data-types/string)) — Описание метрики.
* `labels` ([Map(String, String)](/ru/reference/data-types/map)) — Метки метрик.

**Псевдонимы:**

* `name` — Псевдоним для `metric`.

{/*AUTOGENERATED_END*/}

<div id="example">
  ## Пример
</div>

Вы можете использовать такой запрос, чтобы экспортировать все метрики гистограммы в формате Prometheus.

```sql
SELECT
  metric AS name,
  toFloat64(value) AS value,
  description AS help,
  labels,
  'histogram' AS type
FROM system.histogram_metrics
FORMAT Prometheus
```

<div id="metric_descriptions">
  ## Описания метрик
</div>

| Метрика                                                      | Описание                                                                                                                                 |
| ------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------- |
| `keeper_response_time_ms_bucket`                             | Время ответа Keeper в миллисекундах.                                                                                                     |
| `keeper_client_queue_duration_milliseconds_bucket`           | Время, которое запросы тратят на ожидание постановки в очередь и нахождение в очереди перед обработкой клиентом Keeper, в миллисекундах. |
| `keeper_receive_request_time_milliseconds_bucket`            | Время на получение и разбор запроса от клиента в обработчике TCP Keeper, в миллисекундах.                                                |
| `keeper_dispatcher_requests_queue_time_milliseconds_bucket`  | Время, которое запрос проводит в очереди запросов диспетчера Keeper, в миллисекундах.                                                    |
| `keeper_write_pre_commit_time_milliseconds_bucket`           | Время на предварительную обработку запроса на запись перед фиксацией в Raft, в миллисекундах.                                            |
| `keeper_write_commit_time_milliseconds_bucket`               | Время на обработку запроса на запись после фиксации в Raft, в миллисекундах.                                                             |
| `keeper_dispatcher_responses_queue_time_milliseconds_bucket` | Время, которое ответ проводит в очереди ответов диспетчера Keeper, в миллисекундах.                                                      |
| `keeper_send_response_time_milliseconds_bucket`              | Время на отправку ответа клиенту в обработчике TCP Keeper (включая ожидание в очереди и запись в сокет), в миллисекундах.                |
| `keeper_read_wait_for_write_time_milliseconds_bucket`        | Время, в течение которого запрос на чтение ожидает завершения зависимого от него запроса на запись, в миллисекундах.                     |
| `keeper_read_process_time_milliseconds_bucket`               | Время на обработку запроса на чтение в Keeper, в миллисекундах.                                                                          |
| `keeper_batch_size_elements_bucket`                          | Размер батча, отправляемого в Raft, в элементах.                                                                                         |
| `keeper_batch_size_bytes_bucket`                             | Размер батча, отправляемого в Raft, в байтах.                                                                                            |
| `filesystem_cache_evicted_segment_hits_bucket`               | Распределение числа попаданий в кэш для сегментов файла в момент их вытеснения, с меткой имени кэша.                                     |
| `filesystem_cache_evicted_segment_size_bytes_bucket`         | Распределение размеров в байтах для вытесненных сегментов файла, с меткой имени кэша.                                                    |
| `filesystem_cache_evicted_segment_hits_by_user_bucket`       | Распределение числа попаданий в кэш для вытесненных сегментов файла, с метками имени кэша и идентификатора пользователя.                 |
| `filesystem_cache_evicted_segment_size_bytes_by_user_bucket` | Распределение размеров в байтах для вытесненных сегментов файла, с метками имени кэша и идентификатора пользователя.                     |
