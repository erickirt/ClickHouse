<div id="description">
  ## Описание
</div>

Содержит информацию о [узлах планировщика](/ru/concepts/features/configuration/server-config/workload-scheduling#hierarchy), находящихся на локальном сервере, и их статусе.
Эту таблицу можно использовать для мониторинга. Таблица содержит по одной строке для каждого узла планировщика.

<div id="columns">
  ## Столбцы
</div>

{/*AUTOGENERATED_START*/}

* `resource` ([String](/ru/reference/data-types/index)) — Имя ресурса
* `path` ([String](/ru/reference/data-types/index)) — Путь к узлу планировщика в этой иерархии планирования ресурсов
* `type` ([String](/ru/reference/data-types/index)) — Тип узла планировщика.
* `weight` ([Float64](/ru/reference/data-types/index)) — Вес узла, используемый родительским узлом типа `fair`.
* `priority` ([Int64](/ru/reference/data-types/index)) — Приоритет узла, используемый родительским узлом типа &#39;priority&#39; (меньшее значение означает более высокий приоритет).
* `precedence` ([Int64](/ru/reference/data-types/index)) — Приоритет узла, используемый родительскими узлами с разделением пространства (memory) для упорядочивания допуска и вытеснения (меньшее значение означает более высокий приоритет).
* `is_active` ([Nullable(UInt8)](/ru/reference/data-types/index)) — Указывает, активен ли этот узел в данный момент — есть ли у него запросы ресурсов, ожидающие извлечения из очереди, и удовлетворены ли ограничения.
* `active_children` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Количество дочерних узлов в активном состоянии.
* `dequeued_requests` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Общее количество запросов ресурсов, извлечённых из очереди этого узла.
* `canceled_requests` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Общее количество запросов ресурсов, отменённых для этого узла.
* `rejected_requests` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Общее количество запросов ресурсов, отклонённых для этого узла.
* `dequeued_cost` ([Nullable(Int64)](/ru/reference/data-types/index)) — Сумма стоимостей (например, размера в байтах) всех запросов, извлечённых из очереди этого узла.
* `canceled_cost` ([Nullable(Int64)](/ru/reference/data-types/index)) — Сумма стоимостей (например, размера в байтах) всех запросов, отменённых для этого узла.
* `rejected_cost` ([Nullable(Int64)](/ru/reference/data-types/index)) — Сумма стоимостей (например, размера в байтах) всех запросов, отклонённых для этого узла.
* `busy_periods` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Общее количество деактиваций этого узла.
* `throughput` ([Nullable(Float64)](/ru/reference/data-types/index)) — Текущая средняя пропускная способность (стоимость запросов, извлечённых из очереди, в секунду).
* `vruntime` ([Nullable(Float64)](/ru/reference/data-types/index)) — Только для дочерних узлов `fair`. Виртуальное время выполнения узла, используемое алгоритмом SFQ для выбора следующего дочернего узла для обработки по принципу max-min fairness.
* `system_vruntime` ([Nullable(Float64)](/ru/reference/data-types/index)) — Только для узлов `fair`. Виртуальное время выполнения, показывающее `vruntime` последнего обработанного запроса ресурсов. Используется при активации дочернего узла как новое значение `vruntime`.
* `queue_length` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Только для узлов `fifo`. Текущее количество запросов ресурсов в очереди.
* `queue_cost` ([Nullable(Int64)](/ru/reference/data-types/index)) — Только для узлов `fifo`. Сумма стоимостей (например, размера в байтах) всех запросов, находящихся в очереди.
* `budget` ([Nullable(Int64)](/ru/reference/data-types/index)) — Только для узлов `fifo`. Количество доступных &#39;единиц стоимости&#39; для новых запросов ресурсов. Может появляться в случае расхождения между оценочной и фактической стоимостью запросов ресурсов (например, после сбоя чтения/записи)
* `is_satisfied` ([Nullable(UInt8)](/ru/reference/data-types/index)) — Только для узлов-ограничений (например, `inflight_limit`). Равно `1`, если все ограничения этого узла соблюдены.
* `inflight_requests` ([Nullable(Int64)](/ru/reference/data-types/index)) — Только для узлов `inflight_limit`. Количество запросов ресурсов, извлечённых из очереди этого узла и находящихся в данный момент в состоянии обработки.
* `inflight_cost` ([Nullable(Int64)](/ru/reference/data-types/index)) — Только для узлов `inflight_limit`. Сумма стоимостей (например, байтов) всех запросов ресурсов, извлечённых из очереди этого узла и находящихся в данный момент в состоянии обработки.
* `max_requests` ([Nullable(Int64)](/ru/reference/data-types/index)) — Только для узлов `inflight_limit`. Верхний предел для inflight&#95;requests, при превышении которого нарушается ограничение.
* `max_cost` ([Nullable(Int64)](/ru/reference/data-types/index)) — Только для узлов `inflight_limit`. Верхний предел для inflight&#95;cost, при превышении которого нарушается ограничение.
* `max_speed` ([Nullable(Float64)](/ru/reference/data-types/index)) — Только для узлов `bandwidth_limit`. Верхний предел пропускной способности в токенах в секунду.
* `max_burst` ([Nullable(Float64)](/ru/reference/data-types/index)) — Только для узлов `bandwidth_limit`. Верхний предел количества токенов, доступных в throttler token-bucket.
* `throttling_us` ([Nullable(Int64)](/ru/reference/data-types/index)) — Только для узлов `bandwidth_limit`. Общее количество микросекунд, в течение которых этот узел находился в состоянии throttling.
* `tokens` ([Nullable(Float64)](/ru/reference/data-types/index)) — Только для узлов `bandwidth_limit`. Количество токенов, доступных в данный момент в throttler token-bucket.
* `allocated` ([Nullable(Int64)](/ru/reference/data-types/index)) — Только для узлов с разделяемым пространством. Текущий объём ресурса, выделенный в рамках этого узла.
* `allocations` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Только для узлов с разделяемым пространством. Текущее количество активных выделений ресурсов в рамках этого узла.
* `updates` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Только для узлов с разделяемым пространством. Общее количество обновлений, прошедших через этот узел.
* `increases` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Только для узлов с разделяемым пространством. Общее количество запросов на увеличение, одобренных этим узлом.
* `decreases` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Только для узлов с разделяемым пространством. Общее количество запросов на уменьшение, одобренных этим узлом.
* `admits` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Только для узлов с разделяемым пространством. Общее количество выделений ресурсов, допущенных этим узлом.
* `removes` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Только для узлов с разделяемым пространством. Общее количество выделений, удалённых из этого узла.
* `killers` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Только для узлов с разделяемым пространством. Общее количество вытеснений, инициированных этим узлом из-за запросов на увеличение.
* `victims` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Только для узлов с разделяемым пространством. Общее количество выделений, вытесненных из этого узла.
* `rejects` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Только для узлов `allocation_queue`. Общее количество выделений ресурсов, отклонённых этим узлом.
* `pending` ([Nullable(UInt64)](/ru/reference/data-types/index)) — Только для узлов `allocation_queue`. Текущее количество ожидающих выделений ресурсов в этом узле.

{/*AUTOGENERATED_END*/}
