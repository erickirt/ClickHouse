<div id="quantilesBFloat16">
  ## quantilesBFloat16
</div>

Добавленный в: v21.7.0

Одновременно вычисляет несколько приблизительных [квантилей](https://en.wikipedia.org/wiki/Quantile) для выборки из чисел типа bfloat16 на разных уровнях.

Эта функция эквивалентна [`quantileBFloat16`](/ru/reference/functions/aggregate-functions/quantileBFloat16), но позволяет вычислять несколько уровней квантилей за один проход, что эффективнее, чем вызывать отдельные функции квантилей.

**Синтаксис**

```sql
quantilesBFloat16(level1, level2, ...)(expr)
```

**Параметры**

* `level` — Уровни квантилей. Одно или несколько постоянных чисел с плавающей запятой от 0 до 1. Рекомендуется использовать значения `level` в диапазоне `[0.01, 0.99]`. [`Float*`](/ru/reference/data-types/float)

**Аргументы**

* `expr` — Выражение над значениями столбца, результатом которого являются числовые типы данных, `Date` или `DateTime`. [`(U)Int*`](/ru/reference/data-types/int-uint) или [`Float*`](/ru/reference/data-types/float) или [`Date`](/ru/reference/data-types/date) или [`DateTime`](/ru/reference/data-types/datetime)

**Возвращаемое значение**

Массив приближённых квантилей указанных уровней в том же порядке, в котором были заданы эти уровни. Для входных значений `Date` и `DateTime` выходной формат совпадает с входным. [`Array(Float64)`](/ru/reference/data-types/array) или [`Array(Date)`](/ru/reference/data-types/array) или [`Array(DateTime)`](/ru/reference/data-types/array)

**Примеры**

**Вычисление нескольких квантилей bfloat16**

```sql title=Query
SELECT quantilesBFloat16(0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesBFloat16(0.25, 0.5, 0.75)(number)─┐
│ [2,4,7]                                    │
└────────────────────────────────────────────┘
```
