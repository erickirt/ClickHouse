<div id="quantilesBFloat16Weighted">
  ## quantilesBFloat16Weighted
</div>

Добавленный в: v21.10.0

Как [`quantilesBFloat16`](/ru/reference/functions/aggregate-functions/quantilesBFloat16), но с учётом веса каждого значения.

Одновременно вычисляет несколько приближённых [квантилей](https://en.wikipedia.org/wiki/Quantile) выборки, состоящей из чисел [bfloat16](https://en.wikipedia.org/wiki/Bfloat16_floating-point_format), на разных уровнях за один проход.

**Синтаксис**

```sql
quantilesBFloat16Weighted(level1, level2, ...)(expr, weight)
```

**Параметры**

* `level` — Уровни квантилей. Одно или несколько константных чисел с плавающей запятой в диапазоне от 0 до 1. Рекомендуется использовать значения `level` из диапазона `[0.01, 0.99]`. [`Float*`](/ru/reference/data-types/float)

**Аргументы**

* `expr` — Выражение над значениями столбца, результатом которого являются числовые типы данных, `Date` или `DateTime`. [`(U)Int*`](/ru/reference/data-types/int-uint) or [`Float*`](/ru/reference/data-types/float) or [`Date`](/ru/reference/data-types/date) or [`DateTime`](/ru/reference/data-types/datetime)
* `weight` — Столбец с весами элементов последовательности. Вес — это количество вхождений значения. [`UInt*`](/ru/reference/data-types/int-uint)

**Возвращаемое значение**

Массив приближённых квантилей указанных уровней в том же порядке, в котором были указаны уровни. Для входных данных `Date` и `DateTime` формат вывода совпадает с входным форматом. [`Array(Float64)`](/ru/reference/data-types/array) or [`Array(Date)`](/ru/reference/data-types/array) or [`Array(DateTime)`](/ru/reference/data-types/array)

**Примеры**

**Вычисление нескольких взвешенных квантилей bfloat16**

```sql title=Query
SELECT quantilesBFloat16Weighted(0.25, 0.5, 0.75)(number, 1) FROM numbers(10)
```

```response title=Response
┌─quantilesBFloat16Weighted(0.25, 0.5, 0.75)(number, 1)─┐
│ [2,4,7]                                               │
└───────────────────────────────────────────────────────┘
```
