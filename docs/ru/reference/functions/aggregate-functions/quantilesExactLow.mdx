<div id="quantilesExactLow">
  ## quantilesExactLow
</div>

Добавленный в: v20.8.0

Точно вычисляет несколько [квантилей](https://en.wikipedia.org/wiki/Quantile) числовой последовательности данных для разных уровней одновременно. Для чётного числа элементов на уровне `0.5` возвращает меньшее из двух средних значений.

Эта функция эквивалентна [`quantileExactLow`](/ru/reference/functions/aggregate-functions/quantileExactLow), но позволяет вычислять несколько уровней квантилей за один проход, что эффективнее, чем вызывать отдельные функции квантилей.

**Синтаксис**

```sql
quantilesExactLow(level1, level2, ...)(expr)
```

**Параметры**

* `level` — Уровни квантилей. Одно или несколько константных чисел с плавающей запятой от 0 до 1. Рекомендуется использовать значения `level` в диапазоне `[0.01, 0.99]`. [`Float*`](/ru/reference/data-types/float)

**Аргументы**

* `expr` — Выражение над значениями столбца, возвращающее значения числовых типов данных, `Date`, `DateTime` или `DateTime64`. [`(U)Int*`](/ru/reference/data-types/int-uint) или [`Float*`](/ru/reference/data-types/float) или [`Decimal*`](/ru/reference/data-types/decimal) или [`Date`](/ru/reference/data-types/date) или [`DateTime`](/ru/reference/data-types/datetime) или [`DateTime64`](/ru/reference/data-types/datetime64)

**Возвращаемое значение**

Массив квантилей указанных уровней в том же порядке, в котором эти уровни были заданы. Для числовых типов данных формат вывода соответствует входному формату. [`Array((U)Int*)`](/ru/reference/data-types/array) или [`Array(Float*)`](/ru/reference/data-types/array) или [`Array(Decimal*)`](/ru/reference/data-types/array) или [`Array(Date)`](/ru/reference/data-types/array) или [`Array(DateTime)`](/ru/reference/data-types/array) или [`Array(DateTime64)`](/ru/reference/data-types/array)

**Примеры**

**Вычисление нескольких точных нижних квантилей**

```sql title=Query
SELECT quantilesExactLow(0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesExactLow(0.25, 0.5, 0.75)(number)─┐
│ [2,4,7]                                    │
└────────────────────────────────────────────┘
```
