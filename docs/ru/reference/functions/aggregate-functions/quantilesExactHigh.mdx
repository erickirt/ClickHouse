<div id="quantilesExactHigh">
  ## quantilesExactHigh
</div>

Добавленный в: v20.8.0

Точно вычисляет сразу несколько [квантилей](https://en.wikipedia.org/wiki/Quantile) числовой последовательности данных на разных уровнях. Для чётного количества элементов при уровне `0.5` возвращает большее из двух средних значений.

Эта функция эквивалентна [`quantileExactHigh`](/ru/reference/functions/aggregate-functions/quantileExactHigh), но позволяет вычислять несколько уровней квантилей за один проход, что эффективнее, чем вызывать отдельные функции квантилей.

**Синтаксис**

```sql
quantilesExactHigh(level1, level2, ...)(expr)
```

**Параметры**

* `level` — Уровни квантилей. Одно или несколько константных чисел с плавающей запятой от 0 до 1. Рекомендуется использовать значения `level` в диапазоне `[0.01, 0.99]`. [`Float*`](/ru/reference/data-types/float)

**Аргументы**

* `expr` — Выражение над значениями столбца, результатом которого являются числовые типы данных, `Date`, `DateTime` или `DateTime64`. [`(U)Int*`](/ru/reference/data-types/int-uint) или [`Float*`](/ru/reference/data-types/float) или [`Decimal*`](/ru/reference/data-types/decimal) или [`Date`](/ru/reference/data-types/date) или [`DateTime`](/ru/reference/data-types/datetime) или [`DateTime64`](/ru/reference/data-types/datetime64)

**Возвращаемое значение**

Массив квантилей указанных уровней в том же порядке, в котором эти уровни были заданы. Для числовых типов данных формат вывода соответствует входному формату. [`Array((U)Int*)`](/ru/reference/data-types/array) или [`Array(Float*)`](/ru/reference/data-types/array) или [`Array(Decimal*)`](/ru/reference/data-types/array) или [`Array(Date)`](/ru/reference/data-types/array) или [`Array(DateTime)`](/ru/reference/data-types/array) или [`Array(DateTime64)`](/ru/reference/data-types/array)

**Примеры**

**Вычисление нескольких точных высоких квантилей**

```sql title=Query
SELECT quantilesExactHigh(0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesExactHigh(0.25, 0.5, 0.75)(number)─┐
│ [2,5,7]                                     │
└─────────────────────────────────────────────┘
```
