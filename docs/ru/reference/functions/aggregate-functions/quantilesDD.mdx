<div id="quantilesDD">
  ## quantilesDD
</div>

Добавленный в: v24.1.0

Вычисляет сразу несколько приближённых [квантилей](https://en.wikipedia.org/wiki/Quantile) для выборки на разных уровнях с гарантиями относительной погрешности.

Эта функция эквивалентна [`quantileDD`](/ru/reference/functions/aggregate-functions/quantileddsketch), но позволяет вычислять несколько уровней квантилей за один проход, что эффективнее, чем вызывать отдельные функции для вычисления квантилей.

**Синтаксис**

```sql
quantilesDD(relative_accuracy, level1, level2, ...)(expr)
```

**Параметры**

* `relative_accuracy` — Относительная точность квантилей. Возможные значения — в диапазоне от 0 до 1. Рекомендуемое значение — 0.001 или выше. [`Float*`](/ru/reference/data-types/float)
* `level` — Уровни квантилей. Одно или несколько константных чисел с плавающей запятой от 0 до 1. Рекомендуется использовать значения `level` в диапазоне `[0.01, 0.99]`. [`Float*`](/ru/reference/data-types/float)

**Аргументы**

* `expr` — Выражение над значениями столбца, результат которого имеет числовой тип данных, `Date` или `DateTime`. [`(U)Int*`](/ru/reference/data-types/int-uint) или [`Float*`](/ru/reference/data-types/float) или [`Date`](/ru/reference/data-types/date) или [`DateTime`](/ru/reference/data-types/datetime)

**Возвращаемое значение**

Массив приближённых квантилей указанных уровней в том же порядке, в котором были заданы уровни. Для входных данных `Date` и `DateTime` выходной формат соответствует входному. [`Array(Float64)`](/ru/reference/data-types/array) или [`Array(Date)`](/ru/reference/data-types/array) или [`Array(DateTime)`](/ru/reference/data-types/array)

**Примеры**

**Вычисление нескольких квантилей с помощью DDSketch**

```sql title=Query
SELECT quantilesDD(0.01, 0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesDD(0.01, 0.25, 0.5, 0.75)(number)────────────────┐
│ [1.9936617014173448,4.0148353330285875,5.989510371172621] │
└───────────────────────────────────────────────────────────┘
```
