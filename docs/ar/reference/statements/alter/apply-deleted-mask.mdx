```sql
ALTER TABLE [db].name [ON CLUSTER cluster] APPLY DELETED MASK [IN PARTITION partition_id]
```

يطبّق هذا الأمر القناع الذي أنشأته عملية [lightweight delete](/ar/reference/statements/delete)، ويزيل قسرًا الصفوف المعلَّمة على أنها محذوفة من القرص. هذا الأمر عبارة عن mutation ثقيلة، وهو مكافئ دلاليًا للاستعلام `ALTER TABLE [db].name DELETE WHERE _row_exists = 0`.

<Note>
  لا يعمل إلا مع الجداول ضمن عائلة [`MergeTree`](/ar/reference/engines/table-engines/mergetree-family/mergetree) (بما في ذلك الجداول [المكررة](/ar/reference/engines/table-engines/mergetree-family/replication)).
</Note>
