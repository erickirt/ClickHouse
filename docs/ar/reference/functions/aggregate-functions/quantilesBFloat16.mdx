<div id="quantilesBFloat16">
  ## quantilesBFloat16
</div>

أُضيفت في: v21.7.0

تحسب هذه الدالة عدة [كميات مئينية](https://en.wikipedia.org/wiki/Quantile) تقريبية لعينة تتكون من أعداد [bfloat16](https://en.wikipedia.org/wiki/Bfloat16_floating-point_format) ذات فاصلة عائمة، وذلك عند مستويات مختلفة في الوقت نفسه.

هذه الدالة مكافئة لـ [`quantileBFloat16`](/ar/reference/functions/aggregate-functions/quantileBFloat16)، لكنها تتيح حساب عدة مستويات مئينية في تمريرة واحدة، مما يجعلها أكثر كفاءة من استدعاء دوال quantile الفردية.

**البنية**

```sql
quantilesBFloat16(level1, level2, ...)(expr)
```

**المعلمات**

* `level` — مستويات المئينات. عدد ثابت واحد أو أكثر من الأعداد ذات الفاصلة العائمة من 0 إلى 1. نوصي باستخدام قيم `level` ضمن النطاق `[0.01, 0.99]`. [`Float*`](/ar/reference/data-types/float)

**الوسائط**

* `expr` — تعبير على قيم العمود ينتج قيمًا من نوع بيانات رقمي أو `Date` أو `DateTime`. [`(U)Int*`](/ar/reference/data-types/int-uint) أو [`Float*`](/ar/reference/data-types/float) أو [`Date`](/ar/reference/data-types/date) أو [`DateTime`](/ar/reference/data-types/datetime)

**القيمة المُعادة**

مصفوفة من المئينات التقريبية للمستويات المحددة، وبالترتيب نفسه الذي حُدِّدت به هذه المستويات. بالنسبة إلى مُدخلات `Date` و`DateTime`، يطابق تنسيق الإخراج تنسيق الإدخال. [`Array(Float64)`](/ar/reference/data-types/array) أو [`Array(Date)`](/ar/reference/data-types/array) أو [`Array(DateTime)`](/ar/reference/data-types/array)

**أمثلة**

**حساب عدة مئينات bfloat16**

```sql title=Query
SELECT quantilesBFloat16(0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesBFloat16(0.25, 0.5, 0.75)(number)─┐
│ [2,4,7]                                    │
└────────────────────────────────────────────┘
```
