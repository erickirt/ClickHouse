<div id="quantilesDeterministic">
  ## quantilesDeterministic
</div>

أُضيفت في: v1.1.0

تحسب هذه الدالة عدة [كوانتايلات](https://en.wikipedia.org/wiki/Quantile) تقريبية لتسلسل بيانات رقمي عند مستويات مختلفة في الوقت نفسه، باستخدام أخذ عينات الخزان الحتمي.

تعادل هذه الدالة [`quantileDeterministic`](/ar/reference/functions/aggregate-functions/quantileDeterministic)، لكنها تتيح حساب عدة مستويات من quantile في تمريرة واحدة، مما يجعلها أكثر كفاءة من استدعاء دوال quantile بشكل منفصل.

**الصيغة**

```sql
quantilesDeterministic(level1, level2, ...)(expr, determinator)
```

**المعلمات**

* `level` — مستويات الكوانتايل. عدد ثابت واحد أو أكثر من أعداد الفاصلة العائمة من 0 إلى 1. نوصي باستخدام قيم `level` ضمن النطاق `[0.01, 0.99]`. [`Float*`](/ar/reference/data-types/float)

**الوسائط**

* `expr` — تعبير على قيم العمود ينتج أنواع بيانات رقمية أو `Date` أو `DateTime`. [`(U)Int*`](/ar/reference/data-types/int-uint) أو [`Int128`](/ar/reference/data-types/int-uint) أو [`UInt128`](/ar/reference/data-types/int-uint) أو [`Int256`](/ar/reference/data-types/int-uint) أو [`UInt256`](/ar/reference/data-types/int-uint) أو [`Float*`](/ar/reference/data-types/float) أو [`Date`](/ar/reference/data-types/date) أو [`DateTime`](/ar/reference/data-types/datetime)
* `determinator` — عدد تُستخدم قيمة التجزئة الخاصة به بدلًا من مولّد أرقام عشوائية في خوارزمية أخذ العينات بالخزان لجعل نتيجة أخذ العينات حتمية. [`UInt*`](/ar/reference/data-types/int-uint)

**القيمة المعادة**

مصفوفة من الكوانتايلات التقريبية للمستويات المحددة، بنفس ترتيب تحديد هذه المستويات. [`Array(Float64)`](/ar/reference/data-types/array) أو [`Array(Date)`](/ar/reference/data-types/array) أو [`Array(DateTime)`](/ar/reference/data-types/array)

**أمثلة**

**حساب عدة كوانتايلات حتمية**

```sql title=Query
SELECT quantilesDeterministic(0.25, 0.5, 0.75)(number, number) FROM numbers(10)
```

```response title=Response
┌─quantilesDeterministic(0.25, 0.5, 0.75)(number, number)─┐
│ [2.25,4.5,6.75]                                         │
└─────────────────────────────────────────────────────────┘
```
