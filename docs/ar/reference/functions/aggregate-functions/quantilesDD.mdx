<div id="quantilesDD">
  ## quantilesDD
</div>

أُضيفت في: v24.1.0

تحسب هذه الدالة عدة [كوانتايلات](https://en.wikipedia.org/wiki/Quantile) تقريبية لعينة واحدة، مع ضمانات للخطأ النسبي، عند مستويات مختلفة في الوقت نفسه.

تعادل هذه الدالة [`quantileDD`](/ar/reference/functions/aggregate-functions/quantileddsketch)، لكنها تتيح حساب عدة مستويات كوانتايل في تمريرة واحدة، ما يجعلها أكثر كفاءة من استدعاء كل دالة كوانتايل على حدة.

**الصياغة**

```sql
quantilesDD(relative_accuracy, level1, level2, ...)(expr)
```

**المعلمات**

* `relative_accuracy` — الدقة النسبية للكوانتايلات. القيم الممكنة تقع ضمن النطاق من 0 إلى 1. القيمة الموصى بها هي 0.001 أو أعلى. [`Float*`](/ar/reference/data-types/float)
* `level` — مستويات الكوانتايلات. عدد ثابت واحد أو أكثر من الأعداد ذات الفاصلة العائمة من 0 إلى 1. نوصي باستخدام قيم `level` ضمن النطاق `[0.01, 0.99]`. [`Float*`](/ar/reference/data-types/float)

**الوسائط**

* `expr` — تعبير على قيم العمود ينتج أنواع بيانات رقمية أو `Date` أو `DateTime`. [`(U)Int*`](/ar/reference/data-types/int-uint) أو [`Float*`](/ar/reference/data-types/float) أو [`Date`](/ar/reference/data-types/date) أو [`DateTime`](/ar/reference/data-types/datetime)

**القيمة المعادة**

مصفوفة من الكوانتايلات التقريبية للمستويات المحددة، وبالترتيب نفسه الذي حُدِّدت به المستويات. بالنسبة إلى مدخلات `Date` و`DateTime`، فإن تنسيق الإخراج يطابق تنسيق الإدخال. [`Array(Float64)`](/ar/reference/data-types/array) أو [`Array(Date)`](/ar/reference/data-types/array) أو [`Array(DateTime)`](/ar/reference/data-types/array)

**أمثلة**

**حساب عدة كوانتايلات باستخدام DDSketch**

```sql title=Query
SELECT quantilesDD(0.01, 0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesDD(0.01, 0.25, 0.5, 0.75)(number)────────────────┐
│ [1.9936617014173448,4.0148353330285875,5.989510371172621] │
└───────────────────────────────────────────────────────────┘
```
