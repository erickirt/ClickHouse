  ## min_relative_delay_to_close
</div>

<SettingsInfoBlock type="UInt64" default_value="300" />

الحد الأدنى للتأخير مقارنةً بالنسخ المتماثلة الأخرى لإغلاق الخدمة، والتوقف عن معالجة
الطلبات، وعدم إرجاع Ok أثناء فحص الحالة.

هذه القيمة.
