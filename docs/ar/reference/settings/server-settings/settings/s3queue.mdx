  ## s3queue_disable_streaming
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Yes" />عطّل التدفق في S3Queue حتى إذا كان الجدول قد أُنشئ وكانت هناك عروض مادية مرفقة

<div id="s3queue_log">
  ## s3queue_log
</div>

إعدادات جدول النظام `s3queue_log`.

<SystemLogParameters />

الإعدادات الافتراضية هي:

```xml
<s3queue_log>
    <database>system</database>
    <table>s3queue_log</table>
    <partition_by>toYYYYMM(event_date)</partition_by>
    <flush_interval_milliseconds>7500</flush_interval_milliseconds>
</s3queue_log>
```
