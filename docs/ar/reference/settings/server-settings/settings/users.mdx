تضبط هذه الإعدادات خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشيفرة المصدرية لـ ClickHouse.

<div id="users_config">
  ## users_config
</div>

مسار الملف الذي يحتوي على:

* إعدادات المستخدمين.
* حقوق الوصول.
* ملفات تعريف الإعدادات.
* إعدادات الحصص.

**مثال**

```xml
<users_config>users.xml</users_config>
```

<div id="users_to_ignore_early_memory_limit_check">
  ## users_to_ignore_early_memory_limit_check
</div>

قائمة مفصولة بفواصل بأسماء المستخدمين الذين يُتجاهَل معهم الفحص المبكر لحد الذاكرة. إذا لم يكن المستخدم ضمن هذه القائمة، فسيُرفَض الاستعلام إذا تجاوز إجمالي استخدام الذاكرة الحد المسموح به.
