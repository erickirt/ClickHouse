تُستخدَم هذه الإعدادات لتهيئة خادم ClickHouse، وتُولَّد تلقائيًا من الشيفرة المصدرية لـ ClickHouse.

<div id="mysql_port">
  ## mysql_port
</div>

المنفذ المستخدم للتواصل مع العملاء عبر بروتوكول MySQL.

<Note>
  * تحدد الأعداد الصحيحة الموجبة رقم المنفذ المطلوب الاستماع عليه
  * تُستخدم القيم الفارغة لتعطيل التواصل مع العملاء عبر بروتوكول MySQL.
</Note>

**مثال**

```xml
<mysql_port>9004</mysql_port>
```

<div id="mysql_require_secure_transport">
  ## mysql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />إذا ضُبط على true، فسيكون الاتصال الآمن مع العملاء عبر [mysql&#95;port](/ar/reference/settings/server-settings/settings/mysql#mysql_port) مطلوبًا. سيُرفض أي اتصال يستخدم الخيار `<--ssl-mode=none>`. استخدمه مع إعدادات [OpenSSL](/ar/reference/settings/server-settings/settings/other#openssl).
