تضبط هذه الإعدادات خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشيفرة المصدرية لـ ClickHouse.

<div id="replicated_fetches_http_connection_timeout">
  ## replicated_fetches_http_connection_timeout
</div>

<SettingsInfoBlock type="ثوانٍ" default_value="0" changeable_without_restart="No" />مهلة اتصال HTTP لطلبات جلب الأجزاء. تُورَّث من ملف التعريف default `http_connection_timeout` إذا لم يتم تعيينها صراحةً.

<div id="replicated_fetches_http_receive_timeout">
  ## replicated_fetches_http_receive_timeout
</div>

<SettingsInfoBlock type="ثوانٍ" default_value="0" changeable_without_restart="No" />مهلة استقبال HTTP لطلبات جلب الأجزاء. تُورَّث من ملف التعريف default `http_receive_timeout` إذا لم تُعيَّن صراحةً.

<div id="replicated_fetches_http_send_timeout">
  ## replicated_fetches_http_send_timeout
</div>

<SettingsInfoBlock type="ثوانٍ" default_value="0" changeable_without_restart="No" />مهلة إرسال HTTP لطلبات جلب الأجزاء. يُورَّث من ملف التعريف default `http_send_timeout` إذا لم يتم تعيينه صراحةً.
