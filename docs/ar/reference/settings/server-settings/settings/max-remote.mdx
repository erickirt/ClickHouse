تُهيِّئ هذه الإعدادات خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشيفرة المصدرية لـ ClickHouse.

<div id="max_remote_read_connections">
  ## max_remote_read_connections
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />الحد الأقصى لعدد اتصالات القراءة البعيدة المفتوحة التي يُبقي `ReaderExecutor` عليها نشطة لتحسين القراءة المتسلسلة. تؤدي القيمة 0 إلى تعطيل إعادة استخدام الاتصال.

<div id="max_remote_read_network_bandwidth_for_server">
  ## max_remote_read_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

الحد الأقصى لسرعة تبادل البيانات عبر الشبكة، بالبايت في الثانية، لعمليات القراءة.

<Note>
  تعني القيمة `0` (default) عدم وجود حد.
</Note>

<div id="max_remote_write_network_bandwidth_for_server">
  ## max_remote_write_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

الحد الأقصى لمعدل تبادل البيانات عبر الشبكة، بالبايت في الثانية، لعمليات الكتابة.

<Note>
  تعني القيمة `0` (الافتراضية) عدم وجود حد.
</Note>
