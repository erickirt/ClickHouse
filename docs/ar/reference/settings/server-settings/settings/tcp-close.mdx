تُعِدّ هذه الإعدادات خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشيفرة المصدرية لـ ClickHouse.

<div id="tcp_close_connection_after_queries_num">
  ## tcp_close_connection_after_queries_num
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />الحد الأقصى لعدد الاستعلامات المسموح بها لكل اتصال TCP قبل إغلاقه. اضبط القيمة على 0 للسماح بعدد غير محدود من الاستعلامات.

<div id="tcp_close_connection_after_queries_seconds">
  ## tcp_close_connection_after_queries_seconds
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />الحد الأقصى لعمر اتصال TCP، بالثواني، قبل إغلاقه. اضبطه على 0 لجعل عمر الاتصال غير محدود.
