تضبط هذه الإعدادات خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشفرة المصدرية لـ ClickHouse.

<div id="query_cache.max_entries">
  ## query_cache.max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="1024" changeable_without_restart="No" />الحد الأقصى لعدد نتائج استعلامات SELECT المخزّنة في ذاكرة التخزين المؤقت.

<div id="query_cache.max_entry_size_in_bytes">
  ## query_cache.max_entry_size_in_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1048576" changeable_without_restart="No" />الحد الأقصى لحجم نتائج استعلام SELECT، بالبايت، الذي يمكن حفظه في ذاكرة التخزين المؤقت.

<div id="query_cache.max_entry_size_in_rows">
  ## query_cache.max_entry_size_in_rows
</div>

<SettingsInfoBlock type="UInt64" default_value="30000000" changeable_without_restart="لا" />الحد الأقصى لعدد الصفوف التي يمكن أن تتضمنها نتائج استعلامات `SELECT` حتى تُحفَظ في ذاكرة التخزين المؤقت.

<div id="query_cache.max_size_in_bytes">
  ## query_cache.max_size_in_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" changeable_without_restart="نعم" />الحد الأقصى لحجم ذاكرة التخزين المؤقت للاستعلامات، بالبايت. وتعني القيمة 0 أن ذاكرة التخزين المؤقت للاستعلامات معطّلة.
