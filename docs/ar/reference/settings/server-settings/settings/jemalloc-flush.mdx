<div id="jemalloc_flush_profile_interval_bytes">
  ## jemalloc_flush_profile_interval_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />سيتم تفريغ ملف تعريف jemalloc بعد زيادة ذروة استخدام الذاكرة الإجمالية بمقدار jemalloc&#95;flush&#95;profile&#95;interval&#95;bytes

<div id="jemalloc_flush_profile_on_memory_exceeded">
  ## jemalloc_flush_profile_on_memory_exceeded
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />سيتم تفريغ ملف تعريف jemalloc عند حدوث أخطاء تجاوز إجمالي الذاكرة

<div id="jemalloc_flush_profile_on_memory_exceeded_interval">
  ## jemalloc_flush_profile_on_memory_exceeded_interval
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />إذا كانت قيمته غير صفرية، فإنه يحدّد الحد الأدنى للفاصل الزمني، بالثواني، بين عمليات تفريغ ملفات تعريف jemalloc عند حدوث أخطاء تجاوز إجمالي الذاكرة. فعلى سبيل المثال، تعني القيمة 5 تنفيذ عملية تفريغ واحدة فقط لملف التعريف كل 5 ثوانٍ كحد أقصى. وله أولوية على `jemalloc_flush_profile_on_memory_exceeded`.
