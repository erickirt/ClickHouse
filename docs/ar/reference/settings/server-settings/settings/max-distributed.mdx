<div id="max_distributed_cache_read_bandwidth_for_server">

<div id="max_distributed_cache_write_bandwidth_for_server">
  ## max_distributed_cache_write_bandwidth_for_server

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />الحد الأقصى لإجمالي سرعة الكتابة إلى distributed cache على الخادم، محسوبًا بالبايت في الثانية. يشير الصفر إلى عدم وجود حد.
