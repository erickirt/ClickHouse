تُهيِّئ هذه الإعدادات خادم ClickHouse، وقد أُنشئت تلقائيًا من الشفرة المصدرية لـClickHouse.

<div id="mlock_executable">
  ## mlock_executable
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />

نفّذ `<mlockall>` بعد بدء التشغيل لتقليل زمن استجابة الاستعلامات الأولى ومنع ترحيل الملف التنفيذي لـ ClickHouse من الذاكرة تحت حمل IO المرتفع.

<Note>
  يُوصى بتمكين هذا الخيار، لكنه سيزيد وقت بدء التشغيل لبضع ثوانٍ. ضع في اعتبارك أن هذا الإعداد لن يعمل بدون capability ‏&quot;CAP&#95;IPC&#95;LOCK&quot;.
</Note>

**مثال**

```xml
<mlock_executable>false</mlock_executable>
```
