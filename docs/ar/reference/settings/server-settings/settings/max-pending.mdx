تُستخدم هذه الإعدادات لتهيئة خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشيفرة المصدرية لـ ClickHouse.

<div id="max_pending_mutations_execution_time_to_warn">
  ## max_pending_mutations_execution_time_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="86400" changeable_without_restart="نعم" />

إذا تجاوزت أي من عمليات mutation المعلّقة القيمة المحددة بالثواني، فسيضيف ClickHouse server رسائل تحذير إلى جدول `system.warnings`.

**مثال**

```xml
<max_pending_mutations_execution_time_to_warn>10000</max_pending_mutations_execution_time_to_warn>
```

<div id="max_pending_mutations_to_warn">
  ## max_pending_mutations_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="500" changeable_without_restart="Yes" />

إذا تجاوز عدد عمليات mutation المعلّقة القيمة المحددة، فسيضيف ClickHouse server رسائل تحذير إلى جدول `system.warnings`.

**مثال**

```xml
<max_pending_mutations_to_warn>400</max_pending_mutations_to_warn>
```
