تُستخدم هذه الإعدادات لتهيئة خادم ClickHouse، وقد أُنشئت تلقائيًا من الشيفرة المصدرية لـ ClickHouse.

<div id="max_snapshot_commit_thread_pool_free_size">
  ## max_snapshot_commit_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="نعم" />إذا تجاوز عدد الخيوط الخاملة في مجمّع خيوط commit الخاص بـ snapshot القيمة `max_snapshot_commit_thread_pool_free_size`، فسيُحرِّر ClickHouse الموارد التي تشغلها هذه الخيوط ويُقلِّص حجم المجمّع. ويمكن إنشاء الخيوط مجددًا عند الحاجة.

<div id="max_snapshot_commit_thread_pool_size">
  ## max_snapshot_commit_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="64" changeable_without_restart="Yes" />عدد خيوط التنفيذ المخصصة لعملية commit snapshot.
