<div id="logger.async">
  ## logger.async
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />عندما تكون القيمة `<true>` (الافتراضية)، سيتم التسجيل بشكل غير متزامن (خيط خلفي واحد لكل قناة إخراج). وإلا فسيتم التسجيل داخل الخيط الذي يستدعي LOG.

  ## logger.async_queye_max_size
</div>
