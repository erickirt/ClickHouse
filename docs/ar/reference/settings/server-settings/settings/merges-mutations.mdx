تُهيِّئ هذه الإعدادات خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشفرة المصدرية لـ ClickHouse.

<div id="merges_mutations_memory_usage_soft_limit">
  ## merges_mutations_memory_usage_soft_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />

يحدد الحد الأقصى لمقدار ذاكرة RAM المسموح باستخدامه لتنفيذ عمليات الدمج والتعديل.
إذا بلغ ClickHouse الحد المحدد، فلن يجدول أي عمليات دمج أو تعديل جديدة في الخلفية، لكنه سيواصل تنفيذ المهام التي جرى جدولتها بالفعل.

<Note>
  تعني القيمة `0` عدم وجود حد.
</Note>

**مثال**

```xml
<merges_mutations_memory_usage_soft_limit>0</merges_mutations_memory_usage_soft_limit>
```

<div id="merges_mutations_memory_usage_to_ram_ratio">
  ## merges_mutations_memory_usage_to_ram_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />

تُحتسَب القيمة الافتراضية لـ `merges_mutations_memory_usage_soft_limit` وفقًا للصيغة `memory_amount * merges_mutations_memory_usage_to_ram_ratio`.
