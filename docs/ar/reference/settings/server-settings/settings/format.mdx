تُستخدم هذه الإعدادات لتهيئة خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشفرة المصدرية لـ ClickHouse.

<div id="format_parsing_thread_pool_queue_size">
  ## format_parsing_thread_pool_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="Yes" />

الحد الأقصى لعدد المهام التي يمكن جدولتها في مجموعة مؤشرات الترابط لتحليل الإدخال.

<Note>
  تعني القيمة `0` عدم وجود حد.
</Note>

<div id="format_schema_path">
  ## format_schema_path
</div>

<SettingsInfoBlock type="String" default_value="/var/lib/clickhouse/format_schemas/" changeable_without_restart="No" />

المسار إلى الدليل الذي يحتوي على مخططات بيانات الإدخال، مثل مخططات تنسيق [CapnProto](/ar/reference/formats/CapnProto).

**مثال**

```xml
<!-- Directory containing schema files for various input formats. -->
<format_schema_path>/var/lib/clickhouse/format_schemas/</format_schema_path>
```
