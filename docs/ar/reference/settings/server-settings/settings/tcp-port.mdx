تُستخدم هذه الإعدادات لتهيئة خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشفرة المصدرية لـ ClickHouse.

<div id="tcp_port">
  ## tcp_port
</div>

المنفذ المستخدم للتواصل مع البرامج العميلة عبر بروتوكول TCP.

**مثال**

```xml
<tcp_port>9000</tcp_port>
```

<div id="tcp_port_secure">
  ## tcp_port_secure
</div>

منفذ TCP للاتصال الآمن بالعملاء. استخدمه مع إعدادات [OpenSSL](/ar/reference/settings/server-settings/settings/other#openssl).

**القيمة الافتراضية**

```xml
<tcp_port_secure>9440</tcp_port_secure>
```
