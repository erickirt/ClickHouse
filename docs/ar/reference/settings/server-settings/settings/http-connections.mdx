تُستخدم هذه الإعدادات لتهيئة خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشيفرة المصدرية لـ ClickHouse.

<div id="http_connections_hard_limit">
  ## http_connections_hard_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="200000" changeable_without_restart="No" />يتم طرح استثناء عند محاولة الإنشاء إذا تم بلوغ هذا الحد. اضبطه على 0 لتعطيل التقييد الصارم. ينطبق هذا الحد على اتصالات HTTP التي لا تنتمي إلى أي قرص أو وحدة تخزين.

<div id="http_connections_rcvbuf">
  ## http_connections_rcvbuf
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="نعم" />حجم خيار SO&#95;RCVBUF لاتصالات HTTP العامة. إذا ضُبطت هذه القيمة على قيمة أكبر من 0، فإنها تتجاوز الضبط التلقائي في نواة TCP لمخزن الاستقبال المؤقت. 0 = القيمة الافتراضية للنواة (الضبط التلقائي). ملاحظة: تؤدي إعادة هذا الإعداد إلى 0 إلى استعادة الضبط التلقائي للاتصالات المُنشأة حديثًا فقط؛ أما الاتصالات المجمّعة الحالية فتحتفظ بأحجام مخزن مؤقت ثابتة إلى أن يُعاد إنشاؤها.

<div id="http_connections_sndbuf">
  ## http_connections_sndbuf
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />حجم الخيار SO&#95;SNDBUF لاتصالات HTTP العامة. إذا ضُبطت هذه القيمة على قيمة أكبر من 0، فإنها تتجاوز الضبط التلقائي في نواة TCP للمخزن المؤقت للإرسال. 0 = القيمة الافتراضية للنواة (الضبط التلقائي). ملاحظة: تؤدي إعادة هذا الإعداد إلى 0 إلى استعادة الضبط التلقائي للاتصالات المُنشأة حديثًا فقط؛ أما الاتصالات المجمّعة الحالية فتحتفظ بأحجام مخزن مؤقت ثابتة إلى أن يُعاد إنشاؤها.

<div id="http_connections_soft_limit">
  ## http_connections_soft_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="100" changeable_without_restart="لا" />يكون للاتصالات التي تتجاوز هذا الحد عمر أقصر بكثير. ينطبق هذا الحد على اتصالات HTTP التي لا تتبع أي قرص أو وحدة تخزين.

<div id="http_connections_store_limit">
  ## http_connections_store_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />تُعاد تعيين الاتصالات التي تتجاوز هذا الحد بعد استخدامها. اضبط القيمة على 0 لتعطيل ذاكرة التخزين المؤقت للاتصالات. ينطبق هذا الحد على اتصالات HTTP التي لا تتبع أي قرص أو وحدة تخزين.

<div id="http_connections_warn_limit">
  ## http_connections_warn_limit
</div>

<SettingsInfoBlock type="UInt64" default_value="500" changeable_without_restart="No" />تُسجَّل رسائل تحذير في السجلات إذا كان عدد الاتصالات قيد الاستخدام أعلى من هذا الحد. وينطبق هذا الحد على اتصالات HTTP التي لا تنتمي إلى أي قرص أو وحدة تخزين.
