  ## enable_azure_sdk_logging
</div>

<div id="enable_webterminal">
  ## enable_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />فعِّل واجهة طرفية الويب عند نقطة نهاية HTTP ‏`/webterminal`. يوفّر جلسة `clickhouse-client` تفاعلية داخل المتصفح عبر WebSocket. عند تعيين القيمة إلى `false`، تُرجع الطلبات إلى `/webterminal` حالة HTTP ‏`403 Forbidden`.
