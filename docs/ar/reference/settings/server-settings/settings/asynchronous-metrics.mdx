<div id="asynchronous_metrics_enable_heavy_metrics">
  ## asynchronous_metrics_enable_heavy_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />تمكين احتساب المقاييس غير المتزامنة الثقيلة.

<div id="asynchronous_metrics_keeper_metrics_only">
  ## asynchronous_metrics_keeper_metrics_only
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />اجعل المقاييس غير المتزامنة تقتصر على احتساب المقاييس المرتبطة بـ Keeper فقط.

<div id="asynchronous_metrics_update_period_s">
  ## asynchronous_metrics_update_period_s
</div>

<SettingsInfoBlock type="UInt32" default_value="1" changeable_without_restart="No" />فترة تحديث المقاييس غير المتزامنة، بالثواني.
