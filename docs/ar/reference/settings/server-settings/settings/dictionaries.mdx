تُهيِّئ هذه الإعدادات خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشفرة المصدرية لـ ClickHouse.

<div id="dictionaries_config">
  ## dictionaries_config
</div>

مسار ملف إعدادات القاموس.

المسار:

* حدِّد المسار المطلق أو المسار النسبي بالنسبة إلى ملف إعدادات الخادم.
* يمكن أن يتضمّن المسار محارف البدل * و ?.

انظر أيضًا:

* &quot;[قاموس](/ar/reference/statements/create/dictionary)&quot;.

**مثال**

```xml
<dictionaries_config>*_dictionary.xml</dictionaries_config>
```

<div id="dictionaries_lazy_load">
  ## dictionaries_lazy_load
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />

التحميل الكسول للقواميس.

* إذا كانت القيمة `true`، فسيُحمَّل كل قاموس عند أول استخدام. وإذا فشل التحميل، فستُطلق الدالة التي تستخدم القاموس استثناءً.
* إذا كانت القيمة `false`، فسيُحمِّل الخادم جميع القواميس عند بدء التشغيل.

<Note>
  سينتظر الخادم عند بدء التشغيل حتى تكتمل عملية تحميل جميع القواميس قبل استقبال أي اتصالات
  (استثناء: إذا كان [`wait_dictionaries_load_at_startup`](/ar/reference/settings/server-settings/settings/other#wait_dictionaries_load_at_startup) مضبوطًا على `false`).
</Note>

**مثال**

```xml
<dictionaries_lazy_load>true</dictionaries_lazy_load>
```

<div id="dictionaries_lib_path">
  ## dictionaries_lib_path
</div>

<SettingsInfoBlock type="String" default_value="/var/lib/clickhouse/dictionaries_lib/" changeable_without_restart="No" />
