تُضبط هذه الإعدادات خادم ClickHouse، وهي مُولَّدة تلقائيًا من الشيفرة المصدرية لـ ClickHouse.

<div id="postgresql_port">
  ## postgresql_port
</div>

المنفذ المستخدم للتواصل مع العملاء عبر بروتوكول PostgreSQL.

<Note>
  * تحدد الأعداد الصحيحة الموجبة رقم المنفذ الذي سيتم الاستماع إليه
  * تُستخدم القيم الفارغة لتعطيل التواصل مع العملاء عبر بروتوكول PostgreSQL.
</Note>

**مثال**

```xml
<postgresql_port>9005</postgresql_port>
```

<div id="postgresql_require_secure_transport">
  ## postgresql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />إذا تم تعيينه إلى true، فسيُشترط استخدام اتصال آمن مع العملاء عبر [postgresql&#95;port](/ar/reference/settings/server-settings/settings/postgresql#postgresql_port). سيُرفض أي اتصال يستخدم الخيار `<sslmode=disable>`. استخدم هذا الإعداد مع إعدادات [OpenSSL](/ar/reference/settings/server-settings/settings/other#openssl).
