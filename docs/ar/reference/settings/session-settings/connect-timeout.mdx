  ## connect_timeout_with_failover_ms

<VersionHistory rows={[{"id": "row-1","items": [{"label": "23.4"},{"label": "1000"},{"label": "زيادة مهلة الاتصال الافتراضية بسبب الاتصال غير المتزامن"}]}]} />

مهلة الاتصال، بالمللي ثانية، للاتصال بخادم بعيد ضمن محرك الجدول Distributed، إذا استُخدم القسمان &#39;shard&#39; و &#39;replica&#39; في تعريف العنقود.
إذا تعذّر الاتصال، تُجرى عدة محاولات للاتصال بنسخ متماثلة مختلفة.
