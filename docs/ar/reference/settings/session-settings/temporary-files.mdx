<div id="temporary_files_codec">
  ## temporary_files_codec
</div>

<SettingsInfoBlock type="String" default_value="LZ4" />

يضبط برنامج ترميز الضغط للملفات المؤقتة المستخدمة في عمليات الفرز وعمليات الـ join على القرص.

القيم الممكنة:

* LZ4 — يُطبَّق ضغط [LZ4](https://en.wikipedia.org/wiki/LZ4_\(compression_algorithm\)).
* NONE — لا يُطبَّق أي ضغط.
