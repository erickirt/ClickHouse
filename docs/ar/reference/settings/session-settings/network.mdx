<div id="network_compression_method">
  ## network_compression_method
</div>

<SettingsInfoBlock type="String" default_value="LZ4" />

ترميز الضغط المستخدم لاتصالات client/server وserver/server.

القيم الممكنة:

* `NONE` — بدون ضغط.
* `LZ4` — استخدام ترميز `LZ4`.
* `LZ4HC` — استخدام ترميز `LZ4HC`.
* `ZSTD` — استخدام ترميز `ZSTD`.

**انظر أيضًا**

* [network&#95;zstd&#95;compression&#95;level](/ar/reference/settings/session-settings/network#network_zstd_compression_level)

<div id="network_zstd_compression_level">
  ## network_zstd_compression_level
</div>

<SettingsInfoBlock type="Int64" default_value="1" />

يحدّد مستوى ضغط ZSTD. لا يُستخدم إلا عندما تكون قيمة [network&#95;compression&#95;method](/ar/reference/settings/session-settings/network#network_compression_method) هي `ZSTD`.

القيم الممكنة:

* عدد صحيح موجب من 1 إلى 15.
