<div id="describe_include_subcolumns">
  ## describe_include_subcolumns
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

يتيح عرض الأعمدة الفرعية في استعلام [DESCRIBE](/ar/reference/statements/describe-table). على سبيل المثال، أعضاء [Tuple](/ar/reference/data-types/tuple) أو الأعمدة الفرعية لنوع البيانات [Map](/ar/reference/data-types/map#reading-subcolumns-of-map) أو [Nullable](/ar/reference/data-types/nullable#finding-null) أو [Array](/ar/reference/data-types/array#array-size).

القيم الممكنة:

* 0 — لا يتم تضمين الأعمدة الفرعية في استعلامات `DESCRIBE`.
* 1 — يتم تضمين الأعمدة الفرعية في استعلامات `DESCRIBE`.

**مثال**

اطّلع على مثال لتعليمة [DESCRIBE](/ar/reference/statements/describe-table).
