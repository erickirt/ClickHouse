<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "0"},{"label": "إعداد جديد لتمكين التخزين المؤقت داخل الذاكرة لملفات البيانات الوصفية المحلَّلة الخاصة بـ Paimon (manifest lists وmanifests). بالنسبة إلى محركات جداول Paimon الدائمة، يجب تمكينه قبل تهيئة البيانات الوصفية؛ أما دوال الجداول فتقيّمه لكل استعلام. يجنّب ذلك عمليات التنزيل المتكررة وفك التسلسل لملفات البيانات الوصفية من تخزين الكائنات في الاستعلامات اللاحقة."}]}]} />

إذا كان هذا الإعداد مفعّلًا، فقد تستخدم دالة الجدول paimon وتخزين paimon ذاكرة التخزين المؤقت لملفات البيانات الوصفية الخاصة بـ paimon.

تقيّم دوال جداول Paimon هذا الإعداد لكل استعلام، بينما تعتمد محركات جداول Paimon الدائمة قيمته عند تهيئة البيانات الوصفية: ولتغيير هذا القرار لجدول أُنشئ بالفعل، يجب حذفه ثم إعادة إنشائه.

القيم الممكنة:

* 0 - معطّل
* 1 - مفعّل
