<VersionHistory rows={[{"id": "row-1","items": [{"label": "24.2"},{"label": "1"},{"label": "تقليل الجهد اللازم لتهيئة اتصال ClickHouse بأدوات BI."}]}]} />

عند التمكين، سيُعرض نوع بيانات ClickHouse ‏[FixedString](/ar/reference/data-types/fixedstring) باعتباره `TEXT` في [SHOW COLUMNS](/ar/reference/statements/show#show_columns).

لا يسري هذا إلا عند إنشاء الاتصال عبر MySQL wire protocol.

* 0 - استخدم `BLOB`.
* 1 - استخدم `TEXT`.
