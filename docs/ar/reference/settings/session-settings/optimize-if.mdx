<div id="optimize_if_chain_to_multiif">
  ## optimize_if_chain_to_multiif
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

استبدِل سلاسل if(cond1, then1, if(cond2, ...)) بـ multiIf. لا يحقق هذا حاليًا فائدة للأنواع الرقمية.

<div id="optimize_if_transform_strings_to_enum">
  ## optimize_if_transform_strings_to_enum
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

يستبدل الوسيطات من النوع String في الدالتين If وTransform بنوع enum. وهو معطّل افتراضيًا لأنه قد يتسبب في تغيير غير متسق في استعلام موزّع، مما يؤدي إلى فشله.
