<div id="aggregation_in_order_max_block_bytes">
  ## aggregation_in_order_max_block_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="50000000" />

الحد الأقصى لحجم الـ block، بالبايت، الذي يتم تجميعه أثناء تجميع in order وفق ترتيب المفتاح الأساسي. يتيح تقليل حجم الـ block زيادة التوازي في مرحلة الدمج النهائية للتجميع.

<div id="aggregation_memory_efficient_merge_threads">
  ## aggregation_memory_efficient_merge_threads
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

عدد الخيوط المستخدمة لدمج نتائج التجميع الوسيطة في الوضع الموفّر للذاكرة. وكلما زاد هذا العدد، ازداد استهلاك الذاكرة. وتعني القيمة 0 أنها تساوي &#39;max&#95;threads&#39;.
