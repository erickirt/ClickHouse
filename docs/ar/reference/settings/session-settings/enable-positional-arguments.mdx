<div id="enable_positional_arguments">
  ## enable_positional_arguments
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "22.7"},{"label": "1"},{"label": "تفعيل ميزة الوسيطات الموضعية افتراضيًا"}]}]} />

يُفعِّل أو يعطّل دعم الوسيطات الموضعية في عبارات [GROUP BY](/ar/reference/statements/select/group-by) و[LIMIT BY](/ar/reference/statements/select/limit-by) و[ORDER BY](/ar/reference/statements/select/order-by).

القيم الممكنة:

* 0 — لا يتم دعم الوسيطات الموضعية.
* 1 — يتم دعم الوسيطات الموضعية: يمكن استخدام أرقام الأعمدة بدلًا من أسماء الأعمدة.

**مثال**

الاستعلام:

```sql
CREATE TABLE positional_arguments(one Int, two Int, three Int) ENGINE=Memory();

INSERT INTO positional_arguments VALUES (10, 20, 30), (20, 20, 10), (30, 10, 20);

SELECT * FROM positional_arguments ORDER BY 2,3;
```

النتيجة:

```text
┌─one─┬─two─┬─three─┐
│  30 │  10 │   20  │
│  20 │  20 │   10  │
│  10 │  20 │   30  │
└─────┴─────┴───────┘
```

يؤدي هذا الإعداد إلى تمكين أو تعطيل دعم الوسائط الموضعية في تعريفات PROJECTION. راجع أيضًا الإعداد [enable&#95;positional&#95;arguments](/ar/reference/settings/session-settings/enable-positional-arguments#enable_positional_arguments).

<Note>
  هذا إعداد مخصّص للمستخدمين الخبراء، ويجب ألا تغيّره إذا كنت لا تزال في بداية استخدام ClickHouse.
</Note>

القيم الممكنة:

* 0 — الوسائط الموضعية غير مدعومة.
* 1 — الوسائط الموضعية مدعومة: يمكن استخدام أرقام الأعمدة بدلًا من أسماء الأعمدة.
