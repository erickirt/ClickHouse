  ## send_logs_level
</div>

  ## send_logs_source_regexp
</div>

أرسل السجل النصي الخاص بالخادم باستخدام regexp المحدد لمطابقة اسم مصدر السجلات. وتركه فارغًا يعني جميع المصادر.
