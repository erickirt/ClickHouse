  ## max_parser_depth
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" />

يحدّد الحد الأقصى لعمق الاستدعاء التعاودي في المحلّل التنازلي التعاودي. ويتيح ذلك التحكم في حجم المكدس.

* عدد صحيح موجب.
* 0 — عمق الاستدعاء التعاودي غير محدود.
