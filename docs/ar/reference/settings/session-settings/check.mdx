<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.12"},{"label": "0"},{"label": "تم تغيير هذا الإعداد لجعل CHECK TABLE أكثر فائدة"}]}]} />

يحدّد مستوى التفصيل في نتيجة استعلام [CHECK TABLE](/ar/reference/statements/check-table) لمحركات عائلة `MergeTree`.

القيم الممكنة:

* 0 — يعرض الاستعلام حالة الفحص لكل data part على حدة في الجدول.
* 1 — يعرض الاستعلام الحالة العامة لفحص الجدول.
