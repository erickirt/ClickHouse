تُستضاف جميع النماذج المستخدمة حاليًا على AWS Bedrock، وليس من قِبل مطوّر النموذج المعني.
تُوفَّر هذه الروابط لراحتك. وتقع على عاتقك مسؤولية ضمان الامتثال لشروط النماذج المعمول بها وأي سياسات استخدام سارية.
