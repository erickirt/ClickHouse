<div id="quantilesExactHigh">
  ## quantilesExactHigh
</div>

Introducido en: v20.8.0

Calcula exactamente varios [cuantiles](https://en.wikipedia.org/wiki/Quantile) de una secuencia de datos numéricos en distintos niveles de forma simultánea. Para un número par de elementos en el nivel `0.5`, devuelve el mayor de los dos valores centrales.

Esta función es equivalente a [`quantileExactHigh`](/es/reference/functions/aggregate-functions/quantileExactHigh), pero permite calcular varios niveles de cuantiles en una sola pasada, lo que resulta más eficiente que invocar funciones de cuantiles por separado.

**Sintaxis**

```sql
quantilesExactHigh(level1, level2, ...)(expr)
```

**Parámetros**

* `level` — Niveles de cuantiles. Uno o varios números constantes de coma flotante entre 0 y 1. Recomendamos usar valores de `level` en el rango de `[0.01, 0.99]`. [`Float*`](/es/reference/data-types/float)

**Argumentos**

* `expr` — Expresión sobre los valores de la columna cuyo resultado son tipos de datos numéricos, `Date`, `DateTime` o `DateTime64`. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal) o [`Date`](/es/reference/data-types/date) o [`DateTime`](/es/reference/data-types/datetime) o [`DateTime64`](/es/reference/data-types/datetime64)

**Valor devuelto**

Array de cuantiles de los niveles especificados en el mismo orden en que se indicaron dichos niveles. Para los tipos de datos numéricos, el formato de salida coincide con el formato de entrada. [`Array((U)Int*)`](/es/reference/data-types/array) o [`Array(Float*)`](/es/reference/data-types/array) o [`Array(Decimal*)`](/es/reference/data-types/array) o [`Array(Date)`](/es/reference/data-types/array) o [`Array(DateTime)`](/es/reference/data-types/array) o [`Array(DateTime64)`](/es/reference/data-types/array)

**Ejemplos**

**Cálculo de varios cuantiles altos exactos**

```sql title=Query
SELECT quantilesExactHigh(0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesExactHigh(0.25, 0.5, 0.75)(number)─┐
│ [2,5,7]                                     │
└─────────────────────────────────────────────┘
```
