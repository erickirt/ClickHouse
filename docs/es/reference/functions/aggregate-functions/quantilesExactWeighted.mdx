<div id="quantilesExactWeighted">
  ## quantilesExactWeighted
</div>

Introducido en: v1.1.0

Calcula con exactitud varios [cuantiles](https://en.wikipedia.org/wiki/Quantile) de una secuencia de datos numéricos en distintos niveles de forma simultánea, teniendo en cuenta el peso de cada elemento.

Esta función es equivalente a [`quantileExactWeighted`](/es/reference/functions/aggregate-functions/quantileExactWeighted), pero permite calcular varios niveles de cuantil en una sola pasada, lo que resulta más eficiente que llamar a funciones de cuantil por separado.

**Sintaxis**

```sql
quantilesExactWeighted(level1, level2, ...)(expr, weight)
```

**Parámetros**

* `level` — Niveles de los cuantiles. Uno o más números constantes de coma flotante entre 0 y 1. Recomendamos usar valores de `level` dentro del intervalo `[0.01, 0.99]`. [`Float*`](/es/reference/data-types/float)

**Argumentos**

* `expr` — Expresión sobre los valores de la columna cuyo resultado es de tipo de dato numérico, `Date`, `DateTime` o `DateTime64`. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Int128`](/es/reference/data-types/int-uint) o [`UInt128`](/es/reference/data-types/int-uint) o [`Int256`](/es/reference/data-types/int-uint) o [`UInt256`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal) o [`Date`](/es/reference/data-types/date) o [`DateTime`](/es/reference/data-types/datetime) o [`DateTime64`](/es/reference/data-types/datetime64)
* `weight` — Columna con los pesos de los elementos de la secuencia. El peso es el número de apariciones del valor. [`UInt*`](/es/reference/data-types/int-uint)

**Valor devuelto**

Array de cuantiles de los niveles especificados, en el mismo orden en que se especificaron. Para los tipos de datos numéricos, el formato de salida coincide con el formato de entrada. [`Array((U)Int*)`](/es/reference/data-types/array) o [`Array(Int128)`](/es/reference/data-types/array) o [`Array(UInt128)`](/es/reference/data-types/array) o [`Array(Int256)`](/es/reference/data-types/array) o [`Array(UInt256)`](/es/reference/data-types/array) o [`Array(Float*)`](/es/reference/data-types/array) o [`Array(Decimal*)`](/es/reference/data-types/array) o [`Array(Date)`](/es/reference/data-types/array) o [`Array(DateTime)`](/es/reference/data-types/array) o [`Array(DateTime64)`](/es/reference/data-types/array)

**Ejemplos**

**Cálculo de varios cuantiles ponderados exactos**

```sql title=Query
SELECT quantilesExactWeighted(0.25, 0.5, 0.75)(number, 1) FROM numbers(10)
```

```response title=Response
┌─quantilesExactWeighted(0.25, 0.5, 0.75)(number, 1)─┐
│ [2,4,7]                                            │
└────────────────────────────────────────────────────┘
```
