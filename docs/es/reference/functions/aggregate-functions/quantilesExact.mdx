<div id="quantilesExact">
  ## quantilesExact
</div>

Introducido en: v1.1.0

Calcula exactamente varios [cuantiles](https://en.wikipedia.org/wiki/Quantile) de una secuencia de datos numéricos en distintos niveles de forma simultánea.

Esta función es equivalente a [`quantileExact`](/es/reference/functions/aggregate-functions/quantileExact), pero permite calcular varios niveles de cuantiles en una sola pasada, lo que resulta más eficiente que llamar a funciones de cuantiles por separado.

Para obtener valores exactos, todos los valores proporcionados se combinan en un array, que luego se ordena parcialmente. Por lo tanto, la función consume `O(n)` de memoria, donde `n` es el número de valores proporcionados.

**Sintaxis**

```sql
quantilesExact(level1, level2, ...)(expr)
```

**Parámetros**

* `level` — Niveles de cuantiles. Uno o más números constantes de coma flotante entre 0 y 1. Recomendamos usar valores de `level` en el rango `[0.01, 0.99]`. [`Float*`](/es/reference/data-types/float)

**Argumentos**

* `expr` — Expresión sobre los valores de la columna que produce tipos de datos numéricos, `Date`, `DateTime` o `DateTime64`. [`(U)Int*`](/es/reference/data-types/int-uint) o [`Int128`](/es/reference/data-types/int-uint) o [`UInt128`](/es/reference/data-types/int-uint) o [`Int256`](/es/reference/data-types/int-uint) o [`UInt256`](/es/reference/data-types/int-uint) o [`Float*`](/es/reference/data-types/float) o [`Decimal*`](/es/reference/data-types/decimal) o [`Date`](/es/reference/data-types/date) o [`DateTime`](/es/reference/data-types/datetime) o [`DateTime64`](/es/reference/data-types/datetime64)

**Valor devuelto**

Array de cuantiles de los niveles especificados, en el mismo orden en que se indicaron. Para los tipos de datos numéricos, el formato de salida coincide con el formato de entrada. [`Array((U)Int*)`](/es/reference/data-types/array) o [`Array(Int128)`](/es/reference/data-types/array) o [`Array(UInt128)`](/es/reference/data-types/array) o [`Array(Int256)`](/es/reference/data-types/array) o [`Array(UInt256)`](/es/reference/data-types/array) o [`Array(Float*)`](/es/reference/data-types/array) o [`Array(Decimal*)`](/es/reference/data-types/array) o [`Array(Date)`](/es/reference/data-types/array) o [`Array(DateTime)`](/es/reference/data-types/array) o [`Array(DateTime64)`](/es/reference/data-types/array)

**Ejemplos**

**Cálculo de múltiples cuantiles exactos**

```sql title=Query
SELECT quantilesExact(0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesExact(0.25, 0.5, 0.75)(number)─┐
│ [2,5,7]                                 │
└─────────────────────────────────────────┘
```
