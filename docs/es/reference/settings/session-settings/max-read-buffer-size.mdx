  ## max_read_buffer_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="1048576" />

El tamaño máximo del búfer de lectura desde el sistema de archivos. Los valores superiores a 256 MiB se limitan a 256 MiB, ya que un búfer de lectura nunca necesita ser más grande.

<div id="max_read_buffer_size_local_fs">
  ## max_read_buffer_size_local_fs
</div>

<SettingsInfoBlock type="UInt64" default_value="131072" />

El tamaño máximo del búfer de lectura desde el sistema de archivos local. Si se establece en 0, se usará max&#95;read&#95;buffer&#95;size. Los valores superiores a 256 MiB se limitan a 256 MiB, ya que un búfer de lectura nunca necesita ser mayor.

<div id="max_read_buffer_size_remote_fs">
  ## max_read_buffer_size_remote_fs
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

El tamaño máximo del búfer para leer desde un sistema de archivos remoto. Si se establece en 0, se usará max&#95;read&#95;buffer&#95;size. Los valores superiores a 256 MiB se limitan a 256 MiB, ya que un búfer de lectura nunca necesita ser mayor.
