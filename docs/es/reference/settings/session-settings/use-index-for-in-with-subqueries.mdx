  ## use_index_for_in_with_subqueries
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Intenta usar un índice si hay una subconsulta o una expresión de tabla en la parte derecha del operador IN.

<div id="use_index_for_in_with_subqueries_max_values">
  ## use_index_for_in_with_subqueries_max_values
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

El tamaño máximo del conjunto en el lado derecho del operador IN para utilizar el índice de la tabla para el filtrado. Permite evitar la degradación del rendimiento y un mayor uso de memoria debidos a la preparación de estructuras de datos adicionales para consultas grandes. Cero significa que no hay límite.
