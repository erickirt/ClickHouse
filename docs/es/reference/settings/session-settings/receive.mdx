<div id="receive_data_timeout_ms">
  ## receive_data_timeout_ms
</div>

<SettingsInfoBlock type="Milisegundos" default_value="2000" />

Tiempo de espera de la conexión para recibir el primer paquete de datos o un paquete con progreso positivo desde la réplica

<div id="receive_timeout">
  ## receive_timeout
</div>

<SettingsInfoBlock type="Seconds" default_value="300" />

Tiempo de espera para recibir datos de la red, en segundos. Si no se recibe ningún byte en este intervalo, se genera una excepción. Si establece esta configuración en el client, también se establecerá &#39;send&#95;timeout&#39; para el socket en el extremo de conexión correspondiente en el server.
