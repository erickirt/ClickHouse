<div id="s3_upload_part_size_multiply_factor">
  ## s3_upload_part_size_multiply_factor
</div>

<SettingsInfoBlock type="UInt64" default_value="2" />

Multiplica s3&#95;min&#95;upload&#95;part&#95;size por este factor cada vez que se hayan cargado s3&#95;multiply&#95;parts&#95;count&#95;threshold partes a partir de una sola escritura en S3.

<div id="s3_upload_part_size_multiply_parts_count_threshold">
  ## s3_upload_part_size_multiply_parts_count_threshold
</div>

<SettingsInfoBlock type="UInt64" default_value="500" />

Cada vez que se haya cargado en S3 esta cantidad de partes, s3&#95;min&#95;upload&#95;part&#95;size se multiplica por s3&#95;upload&#95;part&#95;size&#95;multiply&#95;factor.
