<div id="optimize_rewrite_aggregate_function_with_if">
  ## optimize_rewrite_aggregate_function_with_if
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Reescribe las funciones de agregación con una expresión `if` como argumento cuando son lógicamente equivalentes.
Por ejemplo, `avg(if(cond, col, null))` puede reescribirse como `avgOrNullIf(cond, col)`. Esto puede mejorar el rendimiento.

<Note>
  Compatible solo con el analizador (`enable_analyzer = 1`).
</Note>

<div id="optimize_rewrite_has_to_in">
  ## optimize_rewrite_has_to_in
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.6"},{"label": "1"},{"label": "Nueva SETTING"}]}]} />

Reescribe las funciones `has` como `IN` cuando el primer argumento es un Array constante. Por ejemplo, `has([1, 2, 3], x)` puede reescribirse como `x IN [1, 2, 3]` para mejorar el rendimiento con Arrays constantes
