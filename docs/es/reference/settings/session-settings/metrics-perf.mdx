<div id="metrics_perf_events_enabled">
  ## metrics_perf_events_enabled
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Si está habilitado, se medirán algunos de los eventos de perf durante toda la ejecución de las consultas.

<div id="metrics_perf_events_list">
  ## metrics_perf_events_list
</div>

Lista separada por comas de métricas de perf que se medirán durante toda la ejecución de las consultas. Si está vacía, se medirán todos los eventos. Consulta PerfEventInfo en el código fuente para ver los eventos disponibles.
