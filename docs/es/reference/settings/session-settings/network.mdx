<div id="network_compression_method">
  ## network_compression_method
</div>

<SettingsInfoBlock type="String" default_value="LZ4" />

El códec para comprimir la comunicación entre cliente y servidor, y entre servidores.

Valores posibles:

* `NONE` — sin compresión.
* `LZ4` — utiliza el códec LZ4.
* `LZ4HC` — utiliza el códec LZ4HC.
* `ZSTD` — utiliza el códec ZSTD.

**Véase también**

* [network&#95;zstd&#95;compression&#95;level](/es/reference/settings/session-settings/network#network_zstd_compression_level)

<div id="network_zstd_compression_level">
  ## network_zstd_compression_level
</div>

<SettingsInfoBlock type="Int64" default_value="1" />

Ajusta el nivel de compresión de ZSTD. Se usa solo cuando [network&#95;compression&#95;method](/es/reference/settings/session-settings/network#network_compression_method) se establece en `ZSTD`.

Valores posibles:

* Entero positivo de 1 a 15.
