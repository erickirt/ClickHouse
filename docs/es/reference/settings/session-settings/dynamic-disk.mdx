  ## dynamic_disk_allow_from_env

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.5"},{"label": "0"},{"label": "Nueva opción para permitir sustituciones `from_env` en la configuración dinámica de discos (la función `disk()`). Deshabilitada de forma predeterminada por seguridad."}]}]} />

Permite usar sustituciones `from_env` en la configuración dinámica de discos (es decir, en los argumentos de la función `disk()`).
Está deshabilitada de forma predeterminada para evitar que los usuarios lean variables de entorno arbitrarias al definir el almacenamiento de la tabla.
