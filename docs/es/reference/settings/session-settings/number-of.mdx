<div id="number_of_mutations_to_delay">
  ## number_of_mutations_to_delay
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Si la tabla mutada contiene al menos esa cantidad de mutaciones sin finalizar, las mutaciones de la tabla se ralentizan artificialmente. 0 - deshabilitado

<div id="number_of_mutations_to_throw">
  ## number_of_mutations_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Si la tabla mutada contiene al menos esa cantidad de mutaciones sin finalizar, lanza la excepción &#39;Too many mutations ...&#39;. 0 - deshabilitado
