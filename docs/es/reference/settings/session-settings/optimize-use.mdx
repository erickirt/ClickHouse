<div id="optimize_use_projections">
  ## optimize_use_projections
</div>

**Aliases**: `allow_experimental_projection_optimization`

<SettingsInfoBlock type="Bool" default_value="1" />

Habilita o deshabilita la optimización de [proyecciones](/es/reference/engines/table-engines/mergetree-family/mergetree#projections) al procesar consultas `SELECT`.

Valores posibles:

* 0 — Optimización de proyecciones deshabilitada.
* 1 — Optimización de proyecciones habilitada.
