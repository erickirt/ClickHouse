  ## http_headers_progress_interval_ms

<div id="http_headers_read_timeout">
  ## http_headers_read_timeout
</div>

<SettingsInfoBlock type="Segundos" default_value="30" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "30"},{"label": "Nueva configuración para limitar el tiempo total de lectura de los encabezados de la solicitud HTTP, como protección frente a ataques slowloris."}]}]} />

Tiempo máximo, en segundos, para leer todos los encabezados de la solicitud HTTP. Es un plazo total para toda la fase de análisis de encabezados, no un tiempo de espera por lectura. Protege frente a ataques de tipo slowloris, en los que un cliente envía lentamente los datos de los encabezados para mantener abiertas las conexiones.
