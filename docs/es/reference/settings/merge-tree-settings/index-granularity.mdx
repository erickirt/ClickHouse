Estas opciones de configuración están disponibles en [system.merge&#95;tree&#95;settings](/es/reference/system-tables/merge_tree_settings) y se autogeneran a partir del código fuente de ClickHouse.

<div id="index_granularity">
  ## index_granularity
</div>

<SettingsInfoBlock type="UInt64" default_value="8192" />

Número máximo de filas de datos entre las marcas de un índice. Es decir, cuántas filas
se corresponden con un valor de clave primaria.

<div id="index_granularity_bytes">
  ## index_granularity_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="10485760" />

Tamaño máximo de los gránulos de datos, en bytes.

Para restringir el tamaño del gránulo únicamente por el número de filas, establézcalo en `0` (no se recomienda).
