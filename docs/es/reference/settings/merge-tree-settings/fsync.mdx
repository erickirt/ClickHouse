Estos ajustes están disponibles en [system.merge&#95;tree&#95;settings](/es/reference/system-tables/merge_tree_settings) y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="fsync_after_insert">
  ## fsync_after_insert
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Realiza `fsync` para cada parte insertada. Reduce significativamente el rendimiento de
las inserciones; no se recomienda usarlo con partes wide.

<div id="fsync_part_directory">
  ## fsync_part_directory
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Realiza fsync en el directorio de la parte después de todas las operaciones de la parte (escrituras, cambios de nombre, etc.).
