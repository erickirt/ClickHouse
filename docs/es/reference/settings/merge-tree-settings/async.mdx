<div id="async_block_ids_cache_update_wait_ms">
  ## async_block_ids_cache_update_wait_ms
</div>

<SettingsInfoBlock type="Milisegundos" default_value="100" />

Alias obsoleto de `deduplication_hashes_cache_update_wait_ms`, mantenido durante una versión para conservar la
compatibilidad con versiones anteriores. Solo se tiene en cuenta cuando `deduplication_hashes_cache_update_wait_ms`
se deja con su valor predeterminado; esta configuración se eliminará en una versión futura.

<div id="async_insert">
  ## async_insert
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Si es true, los datos de la consulta INSERT se almacenan en la cola y más tarde se escriben en
la tabla en segundo plano.
