<div id="cleanup_delay_period">
  ## cleanup_delay_period
</div>

<SettingsInfoBlock type="UInt64" default_value="30" />

Período mínimo para limpiar los logs antiguos de la cola, los hash de bloques y las partes.

<div id="cleanup_delay_period_random_add">
  ## cleanup_delay_period_random_add
</div>

<SettingsInfoBlock type="UInt64" default_value="10" />

Añade a cleanup&#95;delay&#95;period un valor con distribución uniforme de 0 a x segundos
para evitar el efecto de avalancha y el posterior DoS de ZooKeeper en caso de
haber un número muy grande de tablas.
