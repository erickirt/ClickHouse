  ## add_minmax_index_for_block_number_column
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.5"},{"label": "0"},{"label": "Nueva configuración."}]}]} />

Si se habilita, se añade un índice implícito min-max (de omisión) para la columna virtual persistente `_block_number`.
Requiere `enable_block_number_column = 1` para surtir efecto. El índice se construye solo durante las fusiones,
no durante las inserciones: en el momento de la inserción, el número de bloque es provisional e indexaría una constante.
