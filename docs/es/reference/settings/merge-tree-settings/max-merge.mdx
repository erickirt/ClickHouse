  ## max_merge_delayed_streams_for_parallel_write

El número máximo de flujos (columnas) que pueden volcarse en paralelo
(análogo a max&#95;insert&#95;delayed&#95;streams&#95;for&#95;parallel&#95;write para merges). Solo
funciona con merges Vertical.

<div id="max_merge_selecting_sleep_ms">
  ## max_merge_selecting_sleep_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="60000" />

Tiempo máximo de espera antes de volver a intentar seleccionar partes para
fusionar cuando no se haya seleccionado ninguna. Un valor más bajo activará con
frecuencia las tareas de selección en background&#95;schedule&#95;pool, lo que genera una gran cantidad de
peticiones a ZooKeeper en clústeres de gran escala
