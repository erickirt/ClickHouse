<div id="use_minimalistic_checksums_in_zookeeper">
  ## use_minimalistic_checksums_in_zookeeper
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Use el formato reducido (decenas de bytes) para las sumas de comprobación de las partes en ZooKeeper en lugar de
las habituales (decenas de KB). Antes de habilitarlo, compruebe que todas las réplicas admitan el
nuevo formato.

<div id="use_minimalistic_part_header_in_zookeeper">
  ## use_minimalistic_part_header_in_zookeeper
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Método de almacenamiento de las cabeceras de las partes de datos en ZooKeeper. Si está activado, ZooKeeper
almacena menos datos. Para obtener más información, consulte [aquí](/es/reference/settings/server-settings/settings#use_minimalistic_part_header_in_zookeeper).
