Estas opciones de configuración están disponibles en [system.merge&#95;tree&#95;settings](/es/reference/system-tables/merge_tree_settings) y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="always_fetch_merged_part">
  ## always_fetch_merged_part
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Si es true, esta réplica nunca fusiona partes y siempre descarga las partes fusionadas
de otras réplicas.

Valores posibles:

* true, false

<div id="always_use_copy_instead_of_hardlinks">
  ## always_use_copy_instead_of_hardlinks
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Copiar siempre los datos en lugar de usar enlaces físicos durante las mutaciones, los replaces, los detaches, etc.
