<div id="use_compact_variant_discriminators_serialization">
  ## use_compact_variant_discriminators_serialization
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Habilita el modo compacto para la serialización binaria de discriminadores en el
tipo de dato Variant.
Este modo permite usar mucha menos memoria para almacenar discriminadores
en las partes cuando hay sobre todo una sola variante o muchos valores NULL.

<div id="use_const_adaptive_granularity">
  ## use_const_adaptive_granularity
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Usa siempre una granularidad constante en toda la parte. Esto permite comprimir en
memoria los valores de la granularidad del índice. Puede ser útil en cargas de
trabajo extremadamente grandes con tablas estrechas.
