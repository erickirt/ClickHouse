Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="os_collect_psi_metrics">
  ## os_collect_psi_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Habilita la recopilación de métricas PSI a partir de los archivos de /proc/pressure/.

<div id="os_cpu_busy_time_threshold">
  ## os_cpu_busy_time_threshold
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000" changeable_without_restart="No" />Umbral del tiempo de uso de la CPU del SO en microsegundos (métrica OSCPUVirtualTimeMicroseconds) para considerar que la CPU está realizando algún trabajo útil; no se considerará que hay sobrecarga de CPU si el tiempo de uso está por debajo de este valor.
