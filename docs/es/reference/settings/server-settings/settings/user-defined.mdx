Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="user_defined_executable_functions_config">
  ## user_defined_executable_functions_config
</div>

La ruta del archivo de configuración de las funciones ejecutables definidas por el usuario.

Ruta:

* Especifique la ruta absoluta o la ruta relativa al archivo de configuración del servidor.
* La ruta puede contener comodines * y ?.

Véase también:

* &quot;[Executable User Defined Functions](/es/reference/functions/regular-functions/udf#executable-user-defined-functions).&quot;.

**Ejemplo**

```xml
<user_defined_executable_functions_config>*_function.xml</user_defined_executable_functions_config>
```

<div id="user_defined_path">
  ## user_defined_path
</div>

El directorio que contiene archivos definidos por el usuario. Se utiliza para las funciones definidas por el usuario en SQL [Funciones definidas por el usuario en SQL](/es/reference/functions/regular-functions/udf).

**Ejemplo**

```xml
<user_defined_path>/var/lib/clickhouse/user_defined/</user_defined_path>
```
