Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="background_buffer_flush_schedule_pool_size">
  ## background_buffer_flush_schedule_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="Increase only" />El número máximo de hilos que se utilizarán para realizar operaciones de flush en segundo plano para las [tablas con motor Buffer](/es/reference/engines/table-engines/special/buffer).

<div id="background_common_pool_size">
  ## background_common_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="8" changeable_without_restart="No" />El número máximo de hilos que se usarán para realizar diversas operaciones (principalmente, la recolección de basura) en segundo plano en las tablas [*MergeTree-engine](/es/reference/engines/table-engines/mergetree-family/index).

<div id="background_distributed_schedule_pool_size">
  ## background_distributed_schedule_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="Increase only" />El número máximo de hilos que se utilizarán para realizar envíos distribuidos.

<div id="background_fetches_pool_size">
  ## background_fetches_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="No" />El número máximo de hilos que se utilizarán en segundo plano para recuperar partes de datos desde otra réplica en tablas [*MergeTree-engine](/es/reference/engines/table-engines/mergetree-family/index).

<div id="background_message_broker_schedule_pool_size">
  ## background_message_broker_schedule_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="Increase only" />El número máximo de hilos que se utilizarán para ejecutar operaciones en segundo plano relacionadas con la transmisión de mensajes.

<div id="background_move_pool_size">
  ## background_move_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="8" changeable_without_restart="No" />El número máximo de hilos que se utilizarán en segundo plano para mover partes de datos a otro disco o volumen en las tablas con *MergeTree-engine.

<div id="background_pool_size">
  ## background_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="No" />

Establece el número de hilos que realizan merges y mutaciones en segundo plano para tablas con motores MergeTree.

<Note>
  * Esta configuración también puede aplicarse al iniciar el servidor desde la configuración del perfil `default`, para mantener la compatibilidad con versiones anteriores durante el arranque del servidor de ClickHouse.
  * Solo puede aumentar el número de hilos en tiempo de ejecución.
  * Para reducir el número de hilos, debe reiniciar el servidor.
  * Al ajustar esta configuración, administra la carga de CPU y disco.
</Note>

<Danger>
  Un tamaño de grupo menor utiliza menos recursos de CPU y disco, pero los procesos en segundo plano avanzan más lentamente, lo que con el tiempo podría afectar el rendimiento de las consultas.
</Danger>

Antes de cambiarla, consulte también los ajustes de MergeTree relacionados, como:

* [`number_of_free_entries_in_pool_to_lower_max_size_of_merge`](/es/reference/settings/merge-tree-settings#number_of_free_entries_in_pool_to_lower_max_size_of_merge).
* [`number_of_free_entries_in_pool_to_execute_mutation`](/es/reference/settings/merge-tree-settings#number_of_free_entries_in_pool_to_execute_mutation).
* [`number_of_free_entries_in_pool_to_execute_optimize_entire_partition`](/es/reference/settings/merge-tree-settings#number_of_free_entries_in_pool_to_execute_optimize_entire_partition)

**Ejemplo**

```xml
<background_pool_size>16</background_pool_size>
```
