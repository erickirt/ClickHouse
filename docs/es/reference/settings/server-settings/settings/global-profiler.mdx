Estos ajustes configuran el servidor ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="global_profiler_cpu_time_period_ns">
  ## global_profiler_cpu_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />Período del temporizador del reloj de CPU del perfilador global (en nanosegundos). Establezca el valor en 0 para desactivar el perfilador global del reloj de CPU. El valor recomendado es de al menos 10000000 (100 veces por segundo) para consultas individuales o 1000000000 (una vez por segundo) para el perfilado en todo el clúster.

<div id="global_profiler_real_time_period_ns">
  ## global_profiler_real_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />Período del temporizador de tiempo real del perfilador global (en nanosegundos). Establezca el valor 0 para desactivar el perfilador global de tiempo real. El valor recomendado es de al menos 10000000 (100 veces por segundo) para consultas individuales o 1000000000 (una vez por segundo) para el perfilado en todo el clúster.
