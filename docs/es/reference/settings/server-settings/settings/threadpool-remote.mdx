Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="threadpool_remote_fs_reader_pool_size">
  ## threadpool_remote_fs_reader_pool_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="250" changeable_without_restart="No" />Número de hilos del grupo de hilos utilizado para leer desde un sistema de archivos remoto cuando `remote_filesystem_read_method = 'threadpool'`.

<div id="threadpool_remote_fs_reader_queue_size">
  ## threadpool_remote_fs_reader_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />La cantidad máxima de trabajos que se pueden programar en el grupo de hilos para leer desde un sistema de archivos remoto.
