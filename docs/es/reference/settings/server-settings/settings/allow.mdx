Estos ajustes corresponden a la configuración del servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="allow_feature_tier">
  ## allow_feature_tier
</div>

<SettingsInfoBlock type="UInt32" default_value="0" changeable_without_restart="Yes" />

Controla si el usuario puede cambiar configuraciones relacionadas con los distintos niveles de funcionalidades.

* `0` - Se permiten cambios en cualquier configuración (experimental, beta, producción).
* `1` - Solo se permiten cambios en configuraciones de funcionalidades beta y de producción. Se rechazan los cambios en configuraciones experimentales.
* `2` - Solo se permiten cambios en configuraciones de funcionalidades de producción. Se rechazan los cambios en configuraciones experimentales o beta.

Esto equivale a aplicar una restricción readonly a todas las funcionalidades `EXPERIMENTAL` / `BETA`.

<Note>
  Un valor de `0` significa que se pueden cambiar todas las configuraciones.
</Note>

<div id="allow_impersonate_user">
  ## allow_impersonate_user
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Habilita o deshabilita la funcionalidad IMPERSONATE (EXECUTE AS target&#95;user). Esta configuración está obsoleta.

<div id="allow_implicit_no_password">
  ## allow_implicit_no_password
</div>

Prohíbe crear un usuario sin contraseña, a menos que se especifique explícitamente &#39;IDENTIFIED WITH no&#95;password&#39;.

```xml
<allow_implicit_no_password>1</allow_implicit_no_password>
```

<div id="allow_no_password">
  ## allow_no_password
</div>

Indica si se permite o no el tipo de contraseña no segura `no_password`.

```xml
<allow_no_password>1</allow_no_password>
```

<div id="allow_plaintext_password">
  ## allow_plaintext_password
</div>

Indica si se permiten o no los tipos de contraseña en texto plano (inseguros).

```xml
<allow_plaintext_password>1</allow_plaintext_password>
```

<div id="allow_use_jemalloc_memory">
  ## allow_use_jemalloc_memory
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Permite usar la memoria de jemalloc.
