Estas opciones configuran el servidor ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="max_remote_read_connections">
  ## max_remote_read_connections
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />Número máximo de conexiones remotas de lectura abiertas que `ReaderExecutor` mantiene para optimizar la lectura secuencial. 0 desactiva la reutilización de conexiones.

<div id="max_remote_read_network_bandwidth_for_server">
  ## max_remote_read_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

La velocidad máxima de intercambio de datos por la red durante la lectura, en bytes por segundo.

<Note>
  Un valor de `0` (predeterminado) significa que no hay límite.
</Note>

<div id="max_remote_write_network_bandwidth_for_server">
  ## max_remote_write_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

El ancho de banda máximo para el intercambio de datos a través de la red, en bytes por segundo, para escritura.

<Note>
  Un valor de `0` (predeterminado) significa que no tiene límite.
</Note>
