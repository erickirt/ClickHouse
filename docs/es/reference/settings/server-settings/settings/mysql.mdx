Estas opciones de configuración del servidor de ClickHouse se generan automáticamente a partir del código fuente de ClickHouse.

<div id="mysql_port">
  ## mysql_port
</div>

Puerto para comunicarse con clientes mediante el protocolo MySQL.

<Note>
  * Los enteros positivos especifican el número de puerto en el que escuchar
  * Los valores vacíos se usan para deshabilitar la comunicación con clientes mediante el protocolo MySQL.
</Note>

**Ejemplo**

```xml
<mysql_port>9004</mysql_port>
```

<div id="mysql_require_secure_transport">
  ## mysql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Si se establece en true, se requiere una comunicación segura con los client a través de [mysql&#95;port](/es/reference/settings/server-settings/settings/mysql#mysql_port). Se rechazará cualquier conexión con la opción `<--ssl-mode=none>`. Úselo junto con la configuración de [OpenSSL](/es/reference/settings/server-settings/settings/other#openssl).
