Estos ajustes configuran el servidor de ClickHouse y se han generado automáticamente a partir del código fuente de ClickHouse.

<div id="skip_binary_checksum_checks">
  ## skip_binary_checksum_checks
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Omite las comprobaciones de integridad por suma de comprobación del binario de ClickHouse

<div id="skip_check_for_incorrect_settings">
  ## skip_check_for_incorrect_settings
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />

Si se establece en true, no se comprobará que el ajuste del servidor sea correcta.

**Ejemplo**

```xml
<skip_check_for_incorrect_settings>1</skip_check_for_incorrect_settings>
```
