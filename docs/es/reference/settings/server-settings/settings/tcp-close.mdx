Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="tcp_close_connection_after_queries_num">
  ## tcp_close_connection_after_queries_num
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />Número máximo de consultas permitidas por conexión TCP antes de cerrar la conexión. Establezca 0 para permitir un número ilimitado de consultas.

<div id="tcp_close_connection_after_queries_seconds">
  ## tcp_close_connection_after_queries_seconds
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />Tiempo máximo de una conexión TCP, en segundos, antes de que se cierre. Establézcalo en 0 para una duración de la conexión ilimitada.
