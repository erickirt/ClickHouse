Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="threadpool_local_fs_reader_pool_size">
  ## threadpool_local_fs_reader_pool_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="100" changeable_without_restart="No" />El número de hilos del grupo de subprocesos para leer desde el sistema de archivos local cuando `local_filesystem_read_method = 'pread_threadpool'`.

<div id="threadpool_local_fs_reader_queue_size">
  ## threadpool_local_fs_reader_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />El número máximo de trabajos que se pueden programar en el grupo de subprocesos para leer desde el sistema de archivos local.
