Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="max_snapshot_commit_thread_pool_free_size">
  ## max_snapshot_commit_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Sí" />Si el número de hilos inactivos del grupo de hilos de commit de instantáneas supera `max_snapshot_commit_thread_pool_free_size`, ClickHouse liberará los recursos que ocupan esos hilos inactivos y reducirá el tamaño del grupo. Los hilos pueden volver a crearse si es necesario.

<div id="max_snapshot_commit_thread_pool_size">
  ## max_snapshot_commit_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="64" changeable_without_restart="Sí" />El número de hilos para hacer commit de instantáneas.
