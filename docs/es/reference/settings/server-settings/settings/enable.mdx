Estas opciones de configuración corresponden al servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="enable_azure_sdk_logging">
  ## enable_azure_sdk_logging
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Habilita el logging del SDK de Azure

<div id="enable_webterminal">
  ## enable_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Habilita la interfaz de terminal web en el endpoint HTTP `/webterminal`. Proporciona una sesión interactiva de `clickhouse-client` en el navegador a través de WebSocket. Si es `false`, las solicitudes a `/webterminal` devuelven el estado HTTP `403 Forbidden`.
