Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="users_config">
  ## users_config
</div>

Ruta al archivo que contiene:

* Configuraciones de usuario.
* Permisos de acceso.
* Perfiles de configuración.
* Configuración de cuotas.

**Ejemplo**

```xml
<users_config>users.xml</users_config>
```

<div id="users_to_ignore_early_memory_limit_check">
  ## users_to_ignore_early_memory_limit_check
</div>

Lista de usuarios separada por comas para omitir una comprobación anticipada del límite de memoria. Si el usuario no está en esta lista, se rechazará una consulta si el uso total de memoria supera el límite.
