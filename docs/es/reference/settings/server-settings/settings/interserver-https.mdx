Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="interserver_https_host">
  ## interserver_https_host
</div>

Similar a `<interserver_http_host>`, salvo que este nombre de host puede ser usado por otros servidores para acceder a este servidor a través de `<HTTPS>`.

**Ejemplo**

```xml
<interserver_https_host>example.clickhouse.com</interserver_https_host>
```

<div id="interserver_https_port">
  ## interserver_https_port
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />

Puerto para intercambiar datos entre servidores de ClickHouse a través de `<HTTPS>`.

**Ejemplo**

```xml
<interserver_https_port>9010</interserver_https_port>
```
