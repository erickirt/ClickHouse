Estas opciones configuran el servidor ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="query_cache.max_entries">
  ## query_cache.max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="1024" changeable_without_restart="No" />El número máximo de resultados de consultas SELECT almacenados en caché.

<div id="query_cache.max_entry_size_in_bytes">
  ## query_cache.max_entry_size_in_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1048576" changeable_without_restart="No" />El tamaño máximo en bytes que pueden tener los resultados de una consulta SELECT para poder guardarse en la caché.

<div id="query_cache.max_entry_size_in_rows">
  ## query_cache.max_entry_size_in_rows
</div>

<SettingsInfoBlock type="UInt64" default_value="30000000" changeable_without_restart="No" />El número máximo de filas que pueden tener los resultados de las consultas SELECT para guardarse en la caché.

<div id="query_cache.max_size_in_bytes">
  ## query_cache.max_size_in_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" changeable_without_restart="Sí" />El tamaño máximo de la caché en bytes. 0 significa que la caché de consultas está desactivada.
