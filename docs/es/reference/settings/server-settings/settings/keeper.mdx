Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="keeper_hosts">
  ## keeper_hosts
</div>

Configuración dinámica. Contiene un conjunto de hosts de [Zoo]Keeper a los que ClickHouse podría conectarse. No expone información de `<auxiliary_zookeepers>`

<div id="keeper_multiread_batch_size">
  ## keeper_multiread_batch_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />

Tamaño máximo del lote para la solicitud MultiRead a [Zoo]Keeper con compatibilidad para lotes. Si se establece en 0, se desactiva el procesamiento por lotes. Disponible solo en ClickHouse Cloud.
