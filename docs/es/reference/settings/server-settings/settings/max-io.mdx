Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="max_io_thread_pool_free_size">
  ## max_io_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Sí" />

Si el número de **hilos** inactivos en el grupo de hilos de IO supera `max_io_thread_pool_free_size`, ClickHouse liberará los recursos ocupados por esos hilos inactivos y reducirá el tamaño del grupo. Los hilos pueden volver a crearse si es necesario.

<div id="max_io_thread_pool_size">
  ## max_io_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="100" changeable_without_restart="Yes" />

ClickHouse utiliza hilos del grupo de hilos de IO para realizar algunas operaciones de IO (p. ej., para interactuar con S3). `max_io_thread_pool_size` limita el número máximo de hilos del grupo.
