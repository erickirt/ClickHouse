Estas opciones de configuración sirven para configurar servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="format_parsing_thread_pool_queue_size">
  ## format_parsing_thread_pool_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="Yes" />

El número máximo de trabajos que se pueden programar en el pool de hilos para el análisis de la entrada.

<Note>
  Un valor de `0` significa ilimitado.
</Note>

<div id="format_schema_path">
  ## format_schema_path
</div>

<SettingsInfoBlock type="String" default_value="/var/lib/clickhouse/format_schemas/" changeable_without_restart="No" />

La ruta del directorio que contiene los esquemas de los datos de entrada, como los esquemas del formato [CapnProto](/es/reference/formats/CapnProto).

**Ejemplo**

```xml
<!-- Directory containing schema files for various input formats. -->
<format_schema_path>/var/lib/clickhouse/format_schemas/</format_schema_path>
```
