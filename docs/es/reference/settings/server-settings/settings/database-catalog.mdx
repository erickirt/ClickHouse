Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="database_catalog_drop_error_cooldown_sec">
  ## database_catalog_drop_error_cooldown_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="5" changeable_without_restart="No" />En caso de que falle la eliminación de una tabla, ClickHouse esperará este intervalo antes de reintentar la operación.

<div id="database_catalog_drop_table_concurrency">
  ## database_catalog_drop_table_concurrency
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="No" />El tamaño del grupo de subprocesos usado para la eliminación de tablas.

<div id="database_catalog_unused_dir_cleanup_period_sec">
  ## database_catalog_unused_dir_cleanup_period_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="86400" changeable_without_restart="No" />

Parámetro de una tarea que limpia residuos del directorio `store/`.
Establece el período de planificación de la tarea.

<Note>
  Un valor de `0` significa &quot;nunca&quot;. El valor predeterminado corresponde a 1 día.
</Note>

<div id="database_catalog_unused_dir_hide_timeout_sec">
  ## database_catalog_unused_dir_hide_timeout_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="3600" changeable_without_restart="No" />

Parámetro de una tarea que limpia residuos del directorio `store/`.
Si algún subdirectorio no es utilizado por clickhouse-server y este directorio no se ha modificado durante los últimos
[`database_catalog_unused_dir_hide_timeout_sec`](/es/reference/settings/server-settings/settings/database-catalog#database_catalog_unused_dir_hide_timeout_sec) segundos, la tarea &quot;ocultará&quot; este directorio
eliminando todos los permisos de acceso. También funciona para directorios que clickhouse-server no
espera ver dentro de `store/`.

<Note>
  Un valor de `0` significa &quot;inmediatamente&quot;.
</Note>

<div id="database_catalog_unused_dir_rm_timeout_sec">
  ## database_catalog_unused_dir_rm_timeout_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="2592000" changeable_without_restart="No" />

Parámetro de una tarea que limpia residuos del directorio `store/`.
Si algún subdirectorio no está siendo utilizado por clickhouse-server y anteriormente fue &quot;ocultado&quot;
(consulte [database&#95;catalog&#95;unused&#95;dir&#95;hide&#95;timeout&#95;sec](/es/reference/settings/server-settings/settings/database-catalog#database_catalog_unused_dir_hide_timeout_sec))
y este directorio no se ha modificado durante los últimos
[`database_catalog_unused_dir_rm_timeout_sec`]/operations/server-configuration-parameters/settings#database&#95;catalog&#95;unused&#95;dir&#95;rm&#95;timeout&#95;sec) segundos, la tarea eliminará este directorio.
También funciona para los directorios que clickhouse-server no
espera encontrar dentro de `store/`.

<Note>
  Un valor de `0` significa &quot;nunca&quot;. El valor predeterminado corresponde a 30 días.
</Note>
