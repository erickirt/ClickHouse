Estas opciones de configuración configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="max_local_read_bandwidth_for_server">
  ## max_local_read_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Sí" />

La velocidad máxima de lectura local en bytes por segundo.

<Note>
  Un valor de `0` significa que no hay límite.
</Note>

<div id="max_local_write_bandwidth_for_server">
  ## max_local_write_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Sí" />

La velocidad máxima de escritura local en bytes por segundo.

<Note>
  Un valor de `0` significa sin límite.
</Note>
