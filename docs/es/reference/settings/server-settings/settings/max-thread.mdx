Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="max_thread_pool_free_size">
  ## max_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />

Si el número de hilos **inactivos** en el Global Thread pool es mayor que [`max_thread_pool_free_size`](/es/reference/settings/server-settings/settings/max-thread#max_thread_pool_free_size), ClickHouse libera los recursos que ocupan algunos hilos y reduce el tamaño del grupo. Los hilos pueden volver a crearse si es necesario.

**Ejemplo**

```xml
<max_thread_pool_free_size>1200</max_thread_pool_free_size>
```

<div id="max_thread_pool_size">
  ## max_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />

ClickHouse utiliza hilos del Global Thread pool para procesar consultas. Si no hay ningún hilo inactivo para procesar una consulta, se crea un hilo nuevo en el pool. `max_thread_pool_size` limita el número máximo de hilos en el pool.

**Ejemplo**

```xml
<max_thread_pool_size>12000</max_thread_pool_size>
```
