Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="tcp_port">
  ## tcp_port
</div>

Puerto para comunicarse con los clientes a través del protocolo TCP.

**Ejemplo**

```xml
<tcp_port>9000</tcp_port>
```

<div id="tcp_port_secure">
  ## tcp_port_secure
</div>

Puerto TCP para la comunicación segura con los client. Úselo con la configuración de [OpenSSL](/es/reference/settings/server-settings/settings/other#openssl).

**Valor predeterminado**

```xml
<tcp_port_secure>9440</tcp_port_secure>
```
