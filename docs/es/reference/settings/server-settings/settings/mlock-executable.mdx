Estos ajustes configuran el servidor de ClickHouse y se generan automáticamente a partir del código fuente de ClickHouse.

<div id="mlock_executable">
  ## mlock_executable
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />

Ejecute `<mlockall>` después del inicio para reducir la latencia de las primeras consultas y evitar que el ejecutable de ClickHouse se intercambie a disco con una carga alta de IO.

<Note>
  Se recomienda habilitar esta opción, pero aumentará el tiempo de inicio en hasta unos pocos segundos. Tenga en cuenta que esta configuración no funcionará sin la capacidad &quot;CAP&#95;IPC&#95;LOCK&quot;.
</Note>

**Ejemplo**

```xml
<mlock_executable>false</mlock_executable>
```
