These settings configure the ClickHouse server and are autogenerated from ClickHouse source.

## tcp_port {#tcp_port}

Port for communicating with clients over the TCP protocol.

**Example**

```xml
<tcp_port>9000</tcp_port>
```

## tcp_port_secure {#tcp_port_secure}

TCP port for secure communication with clients. Use it with [OpenSSL](/reference/settings/server-settings/settings/other#openssl) settings.

**Default value**

```xml
<tcp_port_secure>9440</tcp_port_secure>
```
