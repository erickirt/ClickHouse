## max_remote_read_connections {#max_remote_read_connections} 

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />Maximum number of open remote read connections kept alive by `ReaderExecutor` for sequential read optimization. 0 disables connection reuse.

## max_remote_read_network_bandwidth_for_server {#max_remote_read_network_bandwidth_for_server} 

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

<Note>
A value of `0` (default) means unlimited.
</Note>
