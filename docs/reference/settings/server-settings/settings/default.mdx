These settings configure the ClickHouse server and are autogenerated from ClickHouse source.

## default_database {#default_database} 

<SettingsInfoBlock type="String" default_value="default" changeable_without_restart="No" />The default database name.

## default_password_type {#default_password_type} 

Sets the password type to be automatically set for in queries like `CREATE USER u IDENTIFIED BY 'p'`.

Accepted values are:
- `plaintext_password`
- `sha256_password`
- `double_sha1_password`
- `bcrypt_password`

```xml
<default_password_type>sha256_password</default_password_type>
```

## default_profile {#default_profile} 

Default settings profile. Settings profiles are located in the file specified in the setting `user_config`.

**Example**

```xml
<default_profile>default</default_profile>
```

## default_session_timeout {#default_session_timeout} 

Default session timeout, in seconds.

```xml
<default_session_timeout>60</default_session_timeout>
```
