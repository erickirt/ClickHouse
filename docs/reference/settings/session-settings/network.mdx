## network_compression_method {#network_compression_method}

<SettingsInfoBlock type="String" default_value="LZ4" />

The codec for compressing the client/server and server/server communication.

Possible values:

- `NONE` — no compression.
- `LZ4` — use the LZ4 codec.
- `LZ4HC` — use the LZ4HC codec.
- `ZSTD` — use the ZSTD codec.

**See Also**

- [network_zstd_compression_level](/reference/settings/session-settings/network#network_zstd_compression_level)

## network_zstd_compression_level {#network_zstd_compression_level}

<SettingsInfoBlock type="Int64" default_value="1" />

Adjusts the level of ZSTD compression. Used only when [network_compression_method](/reference/settings/session-settings/network#network_compression_method) is set to `ZSTD`.

Possible values:

- Positive integer from 1 to 15.
