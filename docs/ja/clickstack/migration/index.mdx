このセクションでは、さまざまなオブザーバビリティソリューションからClickStackへ移行するための包括的なガイドを提供しています。各ガイドには、運用の継続性を維持しながら、データ、エージェント、ワークフローを移行するための詳細な手順が含まれています。

| Technology                                           | Description                                                                                                      |
| ---------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| [Elastic Stack](/ja/clickstack/migration/elastic/index) | Elastic Stack から ClickStack への移行に関する完全ガイド。データの移行、エージェントの切り替え、検索機能を網羅しています                                        |
| [Datadog](/ja/clickstack/migration/datadog)             | ClickStack OpenTelemetry collector の Datadog receiver を使用して、既存の Datadog エージェントと SDKs から ClickStack にテレメトリーを転送します |
