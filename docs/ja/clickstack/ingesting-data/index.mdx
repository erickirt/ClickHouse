マネージド版およびオープンソース版の ClickStack では、オブザーバビリティデータを ClickHouse インスタンスに取り込むための方法が複数用意されています。ログ、メトリクス、トレース、またはセッションデータを収集する場合は、OpenTelemetry (OTel) collector を統一されたインジェストポイントとして使用することも、特定のユースケース向けにプラットフォーム固有のインテグレーションを活用することもできます。
