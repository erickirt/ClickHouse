<div id="description">
  ## 説明
</div>

`Form` フォーマットは、`key1=value1&key2=value2` のような形式でデータが表現される application/x-www-form-urlencoded フォーマットの単一レコードを読み取るために使用できます。

<div id="example-usage">
  ## 使用例
</div>

URL エンコードされたデータを含むファイル `data.tmp` が `user_files` パスに配置されているとします。

```text title="data.tmp"
t_page=116&c.e=ls7xfkpm&c.tti.m=raf&rt.start=navigation&rt.bmr=390%2C11%2C10
```

```sql title="Query"
SELECT * FROM file(data.tmp, Form) FORMAT vertical;
```

```response title="Response"
Row 1:
──────
t_page:   116
c.e:      ls7xfkpm
c.tti.m:  raf
rt.start: navigation
rt.bmr:   390,11,10
```
