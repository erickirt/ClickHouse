<div id="http_headers_progress_interval_ms">
  ## http_headers_progress_interval_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="100" />

HTTPヘッダー X-ClickHouse-Progress は、指定された間隔より短い頻度では送信しません。

<div id="http_headers_read_timeout">
  ## http_headers_read_timeout
</div>

<SettingsInfoBlock type="Seconds" default_value="30" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "30"},{"label": "slowloris 攻撃を防ぐため、HTTPリクエストヘッダーの読み取りにかかる合計時間を制限する新しい設定。"}]}]} />

すべてのHTTPリクエストヘッダーを読み取る最大時間 (秒) です。これはヘッダーのパース段階全体に対する合計デッドラインであり、個々の読み取りごとのタイムアウトではありません。クライアントがヘッダーデータを少しずつ送信して接続を開いたまま維持しようとする、slowloris 型の攻撃を防ぎます。
