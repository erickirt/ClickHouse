これらの設定は [system.settings](/ja/reference/system-tables/settings) で確認でき、[source](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) から自動生成されています。

<div id="optimize_if_chain_to_multiif">
  ## optimize_if_chain_to_multiif
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

if(cond1, then1, if(cond2, ...)) の連鎖を multiIf に置き換えます。現在のところ、数値型に対してはメリットがありません。

<div id="optimize_if_transform_strings_to_enum">
  ## optimize_if_transform_strings_to_enum
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

If と Transform の String 型のargumentを enum に置き換えます。分散クエリで不整合な変更が発生して失敗するおそれがあるため、既定では無効です。
