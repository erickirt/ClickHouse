これらの設定は [system.settings](/ja/reference/system-tables/settings) で確認でき、[source](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) から自動生成されています。

<div id="max_read_buffer_size">
  ## max_read_buffer_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="1048576" />

ファイルシステムから読み取るためのバッファの最大サイズです。読み取りバッファをそれ以上大きくする必要はないため、256 MiB を超える値は 256 MiB に制限されます。

<div id="max_read_buffer_size_local_fs">
  ## max_read_buffer_size_local_fs
</div>

<SettingsInfoBlock type="UInt64" default_value="131072" />

local ファイルシステム から読み込むためのバッファの最大サイズです。0 に設定すると、max&#95;read&#95;buffer&#95;size が使用されます。読み取りバッファをそれ以上大きくする必要はないため、256 MiB を超える値は 256 MiB に制限されます。

<div id="max_read_buffer_size_remote_fs">
  ## max_read_buffer_size_remote_fs
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

リモートファイルシステムから読み込むための読み取りバッファの最大サイズです。0 に設定すると、max&#95;read&#95;buffer&#95;size が使用されます。読み取りバッファをそれ以上大きくする必要はないため、256 MiB を超える値は 256 MiB に制限されます。
