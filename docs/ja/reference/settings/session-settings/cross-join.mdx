CROSS JOIN で圧縮するブロックの最小サイズ。値が 0 の場合、このしきい値は無効になります。ブロックは、2 つのしきい値 (行数またはバイト数) のいずれかに達すると圧縮されます。

CROSS JOIN でブロックを圧縮するための最小行数です。値がゼロの場合、このしきい値は無効になります。行数またはバイト数のしきい値のいずれかに達すると、このブロックは圧縮されます。
