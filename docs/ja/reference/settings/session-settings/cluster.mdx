`true` に設定すると、クラスター関数でアーカイブを処理する際のパフォーマンスが向上します。互換性を確保し、以前のバージョンでアーカイブを使用するクラスター関数を利用している場合に 25.7+ へのアップグレード中にエラーが発生するのを避けるため、`false` に設定してください。
