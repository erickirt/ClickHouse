<div id="distributed_plan_force_exchange_kind">
  ## distributed_plan_force_exchange_kind
</div>

<ExperimentalBadge />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.5"},{"label": ""},{"label": "新しい実験的な設定"}]}]} />

分散クエリのステージ間で、指定した種類の Exchange 演算子を強制します。

設定可能な値:

* &#39;&#39; - どの種類の Exchange 演算子も強制せず、オプティマイザに選択させます。
* &#39;Persisted&#39; - オブジェクトストレージ内の一時ファイルを使用します。
* &#39;Streaming&#39; - Exchange データをネットワーク経由でストリーミングします。
