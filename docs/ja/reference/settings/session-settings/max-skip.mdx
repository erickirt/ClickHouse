<div id="max_skip_unavailable_shards_num">
  ## max_skip_unavailable_shards_num
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.3"},{"label": "0"},{"label": "skip_unavailable_shards が有効な場合に、黙ってスキップできる分片の数を制限するための新しい設定。"}]}]} />

`skip_unavailable_shards` が有効な場合、黙ってスキップできる分片の最大数を制限します。
使用不可の分片の数がこの値を超えると、黙ってスキップする代わりに例外が発生します。

値が 0 の場合は制限なしを意味します (デフォルトの動作です。すべての使用不可の分片をスキップできます) 。

<div id="max_skip_unavailable_shards_ratio">
  ## max_skip_unavailable_shards_ratio
</div>

<SettingsInfoBlock type="Float" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.3"},{"label": "0"},{"label": "skip_unavailable_shards が有効な場合に、暗黙的にスキップできる分片の比率を制限する新しい設定。"}]}]} />

`skip_unavailable_shards` が有効な場合、暗黙的にスキップできる分片の最大比率 (0～1) を制限します。
使用不可の分片の比率が全分片に対してこの値を超えると、暗黙的にスキップする代わりに例外がスローされます。

値が 0 の場合は制限なしを意味します (既定の動作 — すべての使用不可の分片をスキップできます) 。
