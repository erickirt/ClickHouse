これらの設定は [system.settings](/ja/reference/system-tables/settings) で確認でき、[ソースコード](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) から自動生成されています。

<div id="regexp_dict_allow_hyperscan">
  ## regexp_dict_allow_hyperscan
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Hyperscanライブラリを使用するregexp&#95;tree Dictionaryを許可します。

<div id="regexp_dict_flag_case_insensitive">
  ## regexp_dict_flag_case_insensitive
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

regexp&#95;tree Dictionary で、大文字と小文字を区別しないマッチングを使用します。個々の式では、(?i) および (?-i) を使って上書きできます。

<div id="regexp_dict_flag_dotall">
  ## regexp_dict_flag_dotall
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

regexp&#95;tree Dictionary で、&#39;.&#39; が改行文字にも一致するようにします。
