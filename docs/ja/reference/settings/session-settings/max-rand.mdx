これらの設定は [system.settings](/ja/reference/system-tables/settings) で参照でき、[ソースコード](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) から自動生成されています。

<div id="max_rand_distribution_parameter">
  ## max_rand_distribution_parameter
</div>

<SettingsInfoBlock type="Float" default_value="1000000" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "1000000"},{"label": "極端な入力によるハングを防ぐため、乱数分布関数の形状パラメータを制限する新しい設定。"}]}]} />

`randChiSquared`、`randStudentT`、`randFisherF` などの乱数分布関数における分布の形状パラメータの最大値です。これにより、パラメータ値が極端な場合に計算時間が著しく長くなるのを防ぎます。

<div id="max_rand_distribution_trials">
  ## max_rand_distribution_trials
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000000" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "1000000000"},{"label": "極端な入力でハングしないよう、乱数分布関数の試行回数を制限する新しい設定。"}]}]} />

`randBinomial` や `randNegativeBinomial` などの乱数分布関数で許可される試行回数の最大値です。これにより、試行回数が非常に多い場合に計算時間が極端に長くなるのを防ぎます。
