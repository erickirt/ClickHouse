<div id="min_count_to_compile_aggregate_expression">
  ## min_count_to_compile_aggregate_expression
</div>

<SettingsInfoBlock type="UInt64" default_value="3" />

JITコンパイルを開始するのに必要な、同一の集約式の最小数です。[compile&#95;aggregate&#95;expressions](/ja/reference/settings/session-settings/compile#compile_aggregate_expressions) 設定が有効な場合にのみ機能します。

設定可能な値:

* 正の整数。
* 0 — 同一の集約式は常にJITコンパイルされます。

<div id="min_count_to_compile_expression">
  ## min_count_to_compile_expression
</div>

<SettingsInfoBlock type="UInt64" default_value="3" />

同じ式がコンパイルされるまでに実行される最小回数。
