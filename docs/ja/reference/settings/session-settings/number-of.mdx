  ## number_of_mutations_to_delay
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

mutation 済みのテーブルに、この数以上の未完了の mutation がある場合、そのテーブルの mutation を意図的に遅くします。0 - 無効

<div id="number_of_mutations_to_throw">
  ## number_of_mutations_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

mutation 済みのテーブルに未完了の mutation が少なくともこの数だけある場合、&#39;Too many mutations ...&#39; 例外をスローします。0 は無効
