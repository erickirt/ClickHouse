<div id="odbc_bridge_connection_pool_size">
  ## odbc_bridge_connection_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16" />

ODBC bridge における各接続設定文字列の接続プールサイズ。

<div id="odbc_bridge_use_connection_pooling">
  ## odbc_bridge_use_connection_pooling
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

ODBC bridge で接続プーリングを使用します。false に設定すると、毎回新しい接続が作成されます。
