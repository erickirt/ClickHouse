<div id="materialize_skip_indexes_on_insert">
  ## materialize_skip_indexes_on_insert
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "24.6"},{"label": "1"},{"label": "INSERT 時のスキップ索引のマテリアライズを無効化できる新しい設定を追加"}]}]} />

INSERT 時にスキップ索引を構築して保存するかどうかを指定します。無効にすると、スキップ索引は[マージ中](/ja/reference/settings/merge-tree-settings#materialize_skip_indexes_on_merge)または明示的な [MATERIALIZE INDEX](/ja/reference/statements/alter/skipping-index#materialize-index) によってのみ構築・保存されます。

関連項目: [exclude&#95;materialize&#95;skip&#95;indexes&#95;on&#95;insert](/ja/reference/settings/session-settings/other#exclude_materialize_skip_indexes_on_insert)。

  ## materialize_statistics_on_insert
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "0"},{"label": "デフォルトで INSERT 時の統計情報の構築を無効化し、代わりにマージに依存する"}]}, {"id": "row-2","items": [{"label": "24.6"},{"label": "1"},{"label": "INSERT 時の統計情報のマテリアライズを無効にできる新しい設定を追加"}]}]} />

INSERT 時に統計情報を構築して挿入するかどうかを指定します。無効にすると、統計情報はマージ中または明示的な MATERIALIZE STATISTICS によって構築され、保存されます
