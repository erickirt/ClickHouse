BACKUP ON CLUSTER または RESTORE ON CLUSTER の実行中に、あるホストがこの時間内に ZooKeeper 上の一時的な &#39;alive&#39; ノードを再作成できなかった場合、バックアップまたは復元全体は失敗したと見なされます。
この値は、障害発生後にホストが ZooKeeper に再接続するまでにかかりうる妥当な時間よりも大きくする必要があります。
0 は無制限を意味します。
