<div id="local_filesystem_read_method">
  ## local_filesystem_read_method
</div>

<SettingsInfoBlock type="String" default_value="pread_threadpool" />

ローカルファイルシステムからデータを読み取る方法です。次のいずれかを指定します: read, pread, mmap, io&#95;uring, pread&#95;threadpool。

&#39;io&#95;uring&#39; メソッドは実験的機能であり、Log、TinyLog、StripeLog、File、Set、Join、および追記可能なファイルを持つその他のテーブルでは、読み取りと書き込みが同時実行される場合に動作しません。
インターネット上で &#39;io&#95;uring&#39; に関するさまざまな記事を読んでも、うのみにしないでください。大量の小さな I/O リクエストを扱う場合を除き、これはより優れたファイル読み取り方法ではありませんが、ClickHouse ではそのような状況には当てはまりません。&#39;io&#95;uring&#39; を有効にする理由はありません。

<div id="local_filesystem_read_prefetch">
  ## local_filesystem_read_prefetch
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

ローカルファイルシステムからデータを読み取る際に、プリフェッチを使用します。
