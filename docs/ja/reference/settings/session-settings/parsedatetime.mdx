  ## parsedatetime_e_requires_space_padding

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.5"},{"label": "0"},{"label": "MySQL DATE_FORMAT\/STR_TO_DATE との互換性を改善"}]}]} />

関数 `parseDateTime` のフォーマッタ `%e` では、1 桁の日は空白で埋められている必要があります。たとえば、`' 2'` は受け付けられますが、`'2'` はエラーになります。
