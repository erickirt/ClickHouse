<div id="dead_blobs_to_delay_insert">
  ## dead_blobs_to_delay_insert
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "0"},{"label": "同名の `MergeTree` 設定をクエリ単位でオーバーライドするための新しい設定。"}]}]} />

宛先テーブルのディスク上の dead blobs キューに、削除待ちのブロブが少なくともこの数だけある場合、テーブルへの insert を意図的に遅くします。0 の場合はテーブル設定を使用します。

<div id="dead_blobs_to_throw_insert">
  ## dead_blobs_to_throw_insert
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "0"},{"label": "同名の `MergeTree` 設定をクエリごとにオーバーライドするための新しい設定。"}]}]} />

宛先テーブルのディスクの dead blobs キューに削除待ちのブロブがこの数を超えて存在する場合は、&#39;Too many dead blobs ...&#39; 例外をスローします。0 の場合はテーブル設定を使用します。
