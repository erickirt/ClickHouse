<div id="max_parser_depth">
  ## max_parser_depth
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" />

再帰下降パーサーの最大再帰深度を制限します。スタックサイズの制御に使用できます。

設定可能な値:

* 正の整数。
* 0 — 再帰深度は無制限です。
