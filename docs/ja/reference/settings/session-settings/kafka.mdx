これらの設定は [system.settings](/ja/reference/system-tables/settings) で参照でき、[ソースコード](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) から自動生成されています。

<div id="kafka_disable_num_consumers_limit">
  ## kafka_disable_num_consumers_limit
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

利用可能な CPU コア数に応じて適用される kafka&#95;num&#95;consumers の制限を無効にします。

<div id="kafka_max_wait_ms">
  ## kafka_max_wait_ms
</div>

<SettingsInfoBlock type="Milliseconds" default_value="5000" />

再試行するまでに [Kafka](/ja/reference/engines/table-engines/integrations/kafka) からメッセージを読み取る際の待機時間 (ミリ秒) です。

設定可能な値:

* 正の整数。
* 0 — 無期限のタイムアウト。

関連項目:

* [Apache Kafka](https://kafka.apache.org/)
