これらの設定は [system.settings](/ja/reference/system-tables/settings) で参照でき、[ソースコード](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) から自動生成されています。

<div id="totals_auto_threshold">
  ## totals_auto_threshold
</div>

<SettingsInfoBlock type="Float" default_value="0.5" />

`totals_mode = 'auto'` のしきい値です。
「WITH TOTALS modifier」セクションを参照してください。

<div id="totals_mode">
  ## totals_mode
</div>

<SettingsInfoBlock type="TotalsMode" default_value="after_having_exclusive" />

HAVING がある場合、および max&#95;rows&#95;to&#95;group&#95;by と group&#95;by&#95;overflow&#95;mode = &#39;any&#39; が指定されている場合に、TOTALS をどのように計算するかを指定します。
「WITH TOTALS modifier」セクションを参照してください。
