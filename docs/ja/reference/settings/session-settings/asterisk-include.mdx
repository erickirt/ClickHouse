これらの設定は [system.settings](/ja/reference/system-tables/settings) で参照でき、[ソースコード](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp) から自動生成されています。

<div id="asterisk_include_alias_columns">
  ## asterisk_include_alias_columns
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

ワイルドカードクエリ (`SELECT *`) に [ALIAS](/ja/reference/statements/create/table#alias) カラムを含めます。

設定可能な値:

* 0 - 無効
* 1 - 有効

<div id="asterisk_include_materialized_columns">
  ## asterisk_include_materialized_columns
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

ワイルドカードクエリ (`SELECT *`) で [MATERIALIZED](/ja/reference/statements/create/view#materialized-view) カラムを含めます。

設定可能な値:

* 0 - 無効
* 1 - 有効

<div id="asterisk_include_virtual_columns">
  ## asterisk_include_virtual_columns
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "0"},{"label": "新しい設定"}]}]} />

ワイルドカードクエリ (`SELECT *`) の結果に仮想カラムを含めます。

設定可能な値:

* 0 - 無効
* 1 - 有効
