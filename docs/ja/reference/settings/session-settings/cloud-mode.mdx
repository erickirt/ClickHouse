  ## cloud_mode
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Cloudモード

Cloud でのデフォルト値: `1`.

<div id="cloud_mode_engine">
  ## cloud_mode_engine
</div>

<SettingsInfoBlock type="UInt64" default_value="1" />

Cloud で許可されるエンジンファミリー。

* 0 - すべてを許可
* 1 - DDLs を書き換えて *ReplicatedMergeTree を使用
* 2 - DDLs を書き換えて SharedMergeTree を使用
* 3 - remote disk が明示的に指定されている場合を除き、DDLs を書き換えて SharedMergeTree を使用
* 4 - 3 と同じですが、さらに Distributed の代わりに Alias を使用します (Alias table は Distributed table の宛先テーブルを指すため、対応するローカルテーブルが使用されます)

public 部分を最小限に抑えるための UInt64

Cloud でのデフォルト値: `2`.
