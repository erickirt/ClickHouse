  ## optimize_redundant_comparisons
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "1"},{"label": "AND の連鎖内で、同じ式に対する競合する比較条件や冗長な比較条件を検出するための新しい設定。"}]}]} />

AND の連鎖内で、同じ式に対する競合する比較条件や冗長な比較条件を検出します。たとえば、`a < 1 AND a > 5` は `false` に書き換えられます。

<div id="optimize_redundant_functions_in_order_by">
  ## optimize_redundant_functions_in_order_by
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

関数の引数も ORDER BY に含まれている場合、ORDER BY からその関数を削除します
