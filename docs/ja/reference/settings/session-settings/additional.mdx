<div id="additional_result_filter">
  ## additional_result_filter
</div>

`SELECT` クエリの結果に適用する追加のフィルタ式です。
この設定は、いかなるサブクエリにも適用されません。

**例**

```sql
INSERT INTO table_1 VALUES (1, 'a'), (2, 'bb'), (3, 'ccc'), (4, 'dddd');
SElECT * FROM table_1;
```

```response
┌─x─┬─y────┐
│ 1 │ a    │
│ 2 │ bb   │
│ 3 │ ccc  │
│ 4 │ dddd │
└───┴──────┘
```

```sql
SELECT *
FROM table_1
SETTINGS additional_result_filter = 'x != 2'
```

```response
┌─x─┬─y────┐
│ 1 │ a    │
│ 3 │ ccc  │
│ 4 │ dddd │
└───┴──────┘
```

  ## additional_table_filters
</div>

指定したテーブルから読み取った後に適用される



```sql
SELECT *
FROM table_1
```

```response

│ 1 │ a    │
│ 3 │ ccc  │
│ 4 │ dddd │
```
