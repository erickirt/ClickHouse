これらの設定は [system.merge&#95;tree&#95;settings](/ja/reference/system-tables/merge_tree_settings) で参照でき、ClickHouse のソースコードから自動生成されています。

<div id="merge_selecting_sleep_ms">
  ## merge_selecting_sleep_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="5000" />

パーツが選択されなかった場合に、マージするパーツの選択を再試行するまでの最小待機時間です。設定値を小さくすると、background&#95;schedule&#95;pool での選択タスクが頻繁にトリガーされるため、大規模なクラスターでは ZooKeeper への大量のリクエストが発生します

<div id="merge_selecting_sleep_slowdown_factor">
  ## merge_selecting_sleep_slowdown_factor
</div>

<SettingsInfoBlock type="Float" default_value="1.2" />

マージ対象がない場合、merge selecting task の sleep time はこの係数倍され、
マージが割り当てられた場合はこの係数で除算されます
