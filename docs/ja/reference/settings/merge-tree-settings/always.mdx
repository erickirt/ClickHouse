<div id="always_fetch_merged_part">
  ## always_fetch_merged_part
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

true の場合、このレプリカはパーツのマージを行わず、常に他のレプリカからマージ済みパーツをダウンロードします。

設定可能な値:

* true, false
