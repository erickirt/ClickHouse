これらの設定は [system.merge&#95;tree&#95;settings](/ja/reference/system-tables/merge_tree_settings) で参照でき、ClickHouse のソースから自動生成されています。

<div id="index_granularity">
  ## index_granularity
</div>

<SettingsInfoBlock type="UInt64" default_value="8192" />

索引の mark 間に含められるデータ行の最大数。つまり、1 つの主キー値に対応する行数です。

<div id="index_granularity_bytes">
  ## index_granularity_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="10485760" />

データgranuleの最大サイズ (バイト単位) 。

granuleのサイズを行数のみで制限するには、`0` に設定します (非推奨) 。
