<div id="max_merge_delayed_streams_for_parallel_write">
  ## max_merge_delayed_streams_for_parallel_write
</div>

<SettingsInfoBlock type="UInt64" default_value="40" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "25.4"},{"label": "40"},{"label": "新しい設定"}]}]} />

並列にフラッシュできるストリーム (カラム) の最大数
(マージにおける max&#95;insert&#95;delayed&#95;streams&#95;for&#95;parallel&#95;write に相当) 。Vertical
マージでのみ有効です。

<div id="max_merge_selecting_sleep_ms">
  ## max_merge_selecting_sleep_ms
</div>

<SettingsInfoBlock type="UInt64" default_value="60000" />

パーツが選択されなかった場合に、マージするパーツの選択を再試行するまでの最大待機時間です。設定値を小さくすると、background&#95;schedule&#95;pool での選択タスクが頻繁にトリガーされるため、大規模なクラスターでは ZooKeeper への大量のリクエストが発生します。
