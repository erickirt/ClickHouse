<div id="materialize_projections_on_insert">
  ## materialize_projections_on_insert
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.6"},{"label": "1"},{"label": "新しい設定"}]}]} />

有効にすると、INSERT によってプロジェクションを含む新しいパーツが作成されます。
無効の場合は、明示的な [MATERIALIZE PROJECTION](/ja/reference/statements/alter/projection#materialize-projection)
または [materialize&#95;projections&#95;on&#95;merge](/ja/reference/settings/merge-tree-settings/materialize-projections#materialize_projections_on_merge) を使用したマージ時に作成できます。

<div id="materialize_projections_on_merge">
  ## materialize_projections_on_merge
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.6"},{"label": "0"},{"label": "新しい設定"}]}]} />

有効にすると、マージ時に、その元となるすべてのパーツに存在しないプロジェクション (たとえば `materialize_projections_on_insert = 0` で
挿入されたために存在しない場合) が再構築され、マージ後のパーツにそのプロジェクションが含まれるようになります。

マージでは、引き続き同じプロジェクションのセットを持つパーツ同士だけが結合されます。既存のすべてのパーツにプロジェクションを
補完するには、明示的に [MATERIALIZE PROJECTION](/ja/reference/statements/alter/projection#materialize-projection) を使用してください。プロジェクションは、
[materialize&#95;projections&#95;on&#95;insert](/ja/reference/settings/merge-tree-settings/materialize-projections#materialize_projections_on_insert) による INSERT 時にも作成されます。
