<div id="async_block_ids_cache_update_wait_ms">
  ## async_block_ids_cache_update_wait_ms
</div>

<SettingsInfoBlock type="Milliseconds" default_value="100" />

`deduplication_hashes_cache_update_wait_ms` の非推奨エイリアスです。後方互換性のため、1 リリースの間だけ維持されます。`deduplication_hashes_cache_update_wait_ms` がデフォルトのままの場合にのみ有効で、この設定は将来のリリースで削除されます。

<div id="async_insert">
  ## async_insert
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

true の場合、INSERT クエリのデータはキューに格納され、後でバックグラウンドで
テーブルにフラッシュされます。
