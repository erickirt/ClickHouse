これらの設定は [system.merge&#95;tree&#95;settings](/ja/reference/system-tables/merge_tree_settings) で確認でき、ClickHouse のソースコードから自動生成されています。

<div id="fsync_after_insert">
  ## fsync_after_insert
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

挿入された各パーツに対して fsync を実行します。これにより
insert のパフォーマンスが大幅に低下するため、wide パーツでの使用は推奨されません。

<div id="fsync_part_directory">
  ## fsync_part_directory
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

すべてのパーツに関する操作 (書き込み、リネームなど) が完了した後、パーツディレクトリに対して fsync を実行します。
