<div id="cache_populated_by_fetch">
  ## cache_populated_by_fetch
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<Note>
  この設定は ClickHouse Cloud でのみ適用されます。
</Note>

`cache_populated_by_fetch` が無効な場合 (デフォルト設定) 、新しいデータ
パーツは、それらのパーツを必要とするクエリが実行されたときにのみファイルシステムキャッシュに読み込まれます。

有効にすると、`cache_populated_by_fetch` により、こうした動作をトリガーするクエリがなくても、すべてのノードがストレージ上の新しいデータ
パーツをそれぞれのファイルシステムキャッシュに読み込むようになります。
