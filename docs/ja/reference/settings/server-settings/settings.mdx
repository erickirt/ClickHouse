<a id="server-settings" />

このセクションでは、サーバー設定について説明します。これらの設定は、
セッションレベルまたはクエリレベルでは変更できません。

ClickHouse の設定ファイルの詳細については、[&quot;&quot;設定ファイル&quot;&quot;](/ja/concepts/features/configuration/server-config/configuration-files)を参照してください。

その他の設定については、&quot;&quot;[Settings](/ja/concepts/features/configuration/settings/overview)&quot;&quot; セクションで説明しています。
設定について確認する前に、[設定ファイル](/ja/concepts/features/configuration/server-config/configuration-files)
セクションを読み、substitutions (`incl` 属性と `optional` 属性) の使用に注意することをお勧めします。
