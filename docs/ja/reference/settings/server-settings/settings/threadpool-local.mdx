これらの設定は ClickHouse server を設定するもので、ClickHouse のソースから自動生成されています。

<div id="threadpool_local_fs_reader_pool_size">
  ## threadpool_local_fs_reader_pool_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="100" changeable_without_restart="No" />`local_filesystem_read_method = 'pread_threadpool'` の場合に、ローカルファイルシステムから読み取るためのスレッドプール内のスレッド数です。

<div id="threadpool_local_fs_reader_queue_size">
  ## threadpool_local_fs_reader_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />ローカルファイルシステムからの読み取り用スレッドプールでスケジュールできるジョブの最大数。
