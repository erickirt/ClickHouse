これらの設定は ClickHouse server の構成に使用され、ClickHouse のソースコードから自動生成されています。

<div id="parquet_metadata_cache_max_entries">
  ## parquet_metadata_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="5000" changeable_without_restart="No" />Parquet メタデータキャッシュの最大エントリ数。0 は無効を意味します。

<div id="parquet_metadata_cache_policy">
  ## parquet_metadata_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Parquet メタデータキャッシュのポリシー名。

<div id="parquet_metadata_cache_size">
  ## parquet_metadata_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="536870912" changeable_without_restart="Yes" />Parquet メタデータキャッシュの最大サイズ (バイト単位) 。0 は無効を意味します。

<div id="parquet_metadata_cache_size_ratio">
  ## parquet_metadata_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />Parquet メタデータキャッシュにおける protected queue (SLRU ポリシーの場合) のサイズが、cache の総サイズに占める比率です。
