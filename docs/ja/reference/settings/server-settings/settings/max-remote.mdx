これらの設定はClickHouseサーバーを構成するもので、ClickHouseのソースコードから自動生成されています。

<div id="max_remote_read_connections">
  ## max_remote_read_connections
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />`ReaderExecutor` がシーケンシャル読み取りの最適化のために維持する、開いたままのリモート読み取り connection の最大数です。0 を指定すると、connection の再利用は無効になります。

<div id="max_remote_read_network_bandwidth_for_server">
  ## max_remote_read_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

読み取り時の、ネットワーク経由のデータ転送速度の上限です。単位は 1 秒あたりのバイト数です。

<Note>
  `0` (デフォルト) は、無制限を意味します。
</Note>

<div id="max_remote_write_network_bandwidth_for_server">
  ## max_remote_write_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

書き込み時のネットワーク経由のデータ転送速度の上限 (バイト/秒) 。

<Note>
  `0` (デフォルト) は無制限を意味します。
</Note>
