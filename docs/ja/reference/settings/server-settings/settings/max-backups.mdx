これらの設定はClickHouseサーバーを構成するもので、ClickHouseのソースコードから自動生成されています。

<div id="max_backups_io_thread_pool_free_size">
  ## max_backups_io_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />Backups I/O スレッドプール内の**アイドル状態**のスレッド数が `max_backup_io_thread_pool_free_size` を超えると、ClickHouse は待機中のスレッドが使用しているリソースを解放し、プールサイズを縮小します。必要に応じてスレッドは再作成されます。

<div id="max_backups_io_thread_pool_size">
  ## max_backups_io_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="Yes" />ClickHouse は、S3 バックアップの I/O 処理に Backups I/O スレッドプールのスレッドを使用します。`max_backups_io_thread_pool_size` は、このプール内のスレッドの最大数を制限します。
