これらの設定はClickHouse サーバーを設定するためのもので、ClickHouseのソースコードから自動生成されています。

<div id="temporary_data_in_cache">
  ## temporary_data_in_cache
</div>

このオプションを使用すると、一時データは指定したディスクの cache に保存されます。
このセクションでは、type が `cache` のディスク名を指定する必要があります。
この場合、cache と一時データは同じ領域を共有するため、一時データを作成する際にディスク cache が追い出されることがあります。

<Note>
  一時データの保存先を設定するには、`tmp_path`、`tmp_policy`、`temporary_data_in_cache` のいずれか 1 つだけを使用できます。
</Note>

**例**

`local_disk` の cache と一時データは、どちらも `tiny_local_cache` が管理する filesystem 上の `/tiny_local_cache` に保存されます。

```xml highlight={9-16,20}
<clickhouse>
<storage_configuration>
<disks>
<local_disk>
<type>local</type>
<path>/local_disk/</path>
</local_disk>

<tiny_local_cache>
<type>cache</type>
<disk>local_disk</disk>
<path>/tiny_local_cache/</path>
<max_size_rows>10M</max_size_rows>
<max_file_segment_size>1M</max_file_segment_size>
<cache_on_write_operations>1</cache_on_write_operations>
</tiny_local_cache>
</disks>
</storage_configuration>

<temporary_data_in_cache>tiny_local_cache</temporary_data_in_cache>
</clickhouse>
```
