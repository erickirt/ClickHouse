これらの設定は ClickHouse サーバー を設定するためのもので、ClickHouse のソースコードから自動生成されています。

<div id="mlock_executable">
  ## mlock_executable
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />

起動後に `<mlockall>` を実行して、初回クエリのレイテンシを低減し、I/O 負荷が高いときに ClickHouse 実行可能ファイルがページアウトされるのを防ぎます。

<Note>
  このオプションを有効にすることを推奨しますが、起動時間が最大で数秒長くなります。また、この設定は &quot;CAP&#95;IPC&#95;LOCK&quot; ケーパビリティがないと機能しない点に注意してください。
</Note>

**例**

```xml
<mlock_executable>false</mlock_executable>
```
