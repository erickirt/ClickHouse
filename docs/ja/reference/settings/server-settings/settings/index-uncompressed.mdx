これらの設定はClickHouse serverを設定するもので、ClickHouseのソースコードから自動生成されています。

<div id="index_uncompressed_cache_policy">
  ## index_uncompressed_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />セカンダリ索引の非圧縮cacheポリシー名。

<div id="index_uncompressed_cache_size">
  ## index_uncompressed_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

`MergeTree` の索引用の非圧縮ブロック cache の最大サイズ。

<Note>
  `0` は無効を意味します。

  この設定は実行時に変更でき、即座に反映されます。
</Note>

<div id="index_uncompressed_cache_size_ratio">
  ## index_uncompressed_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />セカンダリ索引の非圧縮cacheにおいて、保護キュー (SLRU policy の場合) がcache全体に占めるサイズの比率。
