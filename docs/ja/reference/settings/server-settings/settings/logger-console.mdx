これらの設定は ClickHouseサーバーの構成に使用され、ClickHouse のソースコードから自動生成されています。

<div id="logger.console">
  ## logger.console
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />コンソールへのログ出力を有効にします。有効にするには、`<1>` または `<true>` に設定します。デフォルト値は、ClickHouse がデーモンモードで実行されていない場合は `<1>`、それ以外の場合は `<0>` です。

<div id="logger.console_log_level">
  ## logger.console_log_level
</div>

<SettingsInfoBlock type="String" default_value="trace" changeable_without_restart="No" />コンソール出力のログレベルです。デフォルトは `<level>` です。
