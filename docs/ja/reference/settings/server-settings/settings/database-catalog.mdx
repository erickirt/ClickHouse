これらの設定はClickHouseサーバーの設定項目であり、ClickHouseのソースコードから自動生成されています。

<div id="database_catalog_drop_error_cooldown_sec">
  ## database_catalog_drop_error_cooldown_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="5" changeable_without_restart="No" />テーブルの削除に失敗した場合、ClickHouse はこの時間が経過するまで待機してから、操作を再試行します。

<div id="database_catalog_drop_table_concurrency">
  ## database_catalog_drop_table_concurrency
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="No" />テーブルの削除に使用されるスレッドプールのサイズ。

<div id="database_catalog_unused_dir_cleanup_period_sec">
  ## database_catalog_unused_dir_cleanup_period_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="86400" changeable_without_restart="No" />

`store/` ディレクトリ内の不要なファイルやディレクトリをクリーンアップするタスクのパラメータです。
このタスクの実行間隔を設定します。

<Note>
  `0` は「実行しない」ことを意味します。デフォルト値は 1 日に相当します。
</Note>

<div id="database_catalog_unused_dir_hide_timeout_sec">
  ## database_catalog_unused_dir_hide_timeout_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="3600" changeable_without_restart="No" />

`store/` ディレクトリ内の不要なファイルやディレクトリをクリーンアップするタスクのパラメータです。
あるサブディレクトリが clickhouse-server に使用されておらず、そのディレクトリが直近
[`database_catalog_unused_dir_hide_timeout_sec`](/ja/reference/settings/server-settings/settings/database-catalog#database_catalog_unused_dir_hide_timeout_sec) 秒間変更されていない場合、このタスクはすべてのアクセス権を削除することで、そのディレクトリを「非表示」にします。これは、clickhouse-server が
`store/` 内に存在することを想定していないディレクトリにも適用されます。

<Note>
  `0` は「即時」を意味します。
</Note>

<div id="database_catalog_unused_dir_rm_timeout_sec">
  ## database_catalog_unused_dir_rm_timeout_sec
</div>

<SettingsInfoBlock type="UInt64" default_value="2592000" changeable_without_restart="No" />

`store/` ディレクトリ内の不要な残存物をクリーンアップするタスクのパラメータです。
あるサブディレクトリが clickhouse-server に使用されておらず、以前に &quot;非表示&quot; にされていて
([database&#95;catalog&#95;unused&#95;dir&#95;hide&#95;timeout&#95;sec](/ja/reference/settings/server-settings/settings/database-catalog#database_catalog_unused_dir_hide_timeout_sec) を参照)
さらに、このディレクトリが直近
[`database_catalog_unused_dir_rm_timeout_sec`]/operations/server-configuration-parameters/settings#database&#95;catalog&#95;unused&#95;dir&#95;rm&#95;timeout&#95;sec) 秒間変更されていない場合、このタスクはこのディレクトリを削除します。
これは、clickhouse-server が `store/` 内に存在することを
想定していないディレクトリにも適用されます。

<Note>
  値 `0` は &quot;実行しない&quot; ことを意味します。デフォルト値は 30 日に相当します。
</Note>
