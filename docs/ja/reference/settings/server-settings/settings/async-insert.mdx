<div id="async_insert_queue_flush_on_shutdown">
  ## async_insert_queue_flush_on_shutdown
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />true の場合、非同期挿入キューは正常終了時にフラッシュされます

<div id="async_insert_threads">
  ## async_insert_threads
</div>

<SettingsInfoBlock type="UInt64" default_value="16" changeable_without_restart="No" />バックグラウンドで実際にデータを解析して挿入するスレッドの最大数。0 は非同期モードが無効であることを示します
