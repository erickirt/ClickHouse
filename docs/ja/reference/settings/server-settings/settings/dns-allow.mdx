<div id="dns_allow_resolve_names_to_ipv4">
  ## dns_allow_resolve_names_to_ipv4

<div id="dns_allow_resolve_names_to_ipv6">
  ## dns_allow_resolve_names_to_ipv6
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="Yes" />名前を IPv6 アドレスに解決することを許可します。
