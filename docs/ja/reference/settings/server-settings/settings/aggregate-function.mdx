これらの設定はClickHouse serverを設定するためのもので、ClickHouseのソースから自動生成されています。

<div id="aggregate_function_group_array_action_when_limit_is_reached">
  ## aggregate_function_group_array_action_when_limit_is_reached
</div>

<SettingsInfoBlock type="GroupArrayActionWhenLimitReached" default_value="throw" changeable_without_restart="No" />groupArray で最大 配列要素サイズ を超えた場合の動作です。`throw` で例外を発生させるか、`discard` で超過分の値を破棄します

<div id="aggregate_function_group_array_max_element_size">
  ## aggregate_function_group_array_max_element_size
</div>

<SettingsInfoBlock type="UInt64" default_value="16777215" changeable_without_restart="No" />groupArray 関数における配列要素 1 つあたりの最大サイズ (バイト単位) 。この制限はシリアライゼーション時にチェックされ、state サイズが大きくなりすぎるのを防ぐのに役立ちます。
