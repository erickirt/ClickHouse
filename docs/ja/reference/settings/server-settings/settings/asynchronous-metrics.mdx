これらの設定はClickHouseサーバーの構成に使用され、ClickHouseのソースコードから自動生成されています。

<div id="asynchronous_metrics_enable_heavy_metrics">
  ## asynchronous_metrics_enable_heavy_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />負荷の高い非同期メトリクスの計算を有効にします。

<div id="asynchronous_metrics_keeper_metrics_only">
  ## asynchronous_metrics_keeper_metrics_only
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />非同期メトリクスで計算する対象を、Keeper 関連のメトリクスのみにします。

<div id="asynchronous_metrics_update_period_s">
  ## asynchronous_metrics_update_period_s
</div>

<SettingsInfoBlock type="UInt32" default_value="1" changeable_without_restart="No" />非同期メトリクスの更新間隔 (秒) 。
