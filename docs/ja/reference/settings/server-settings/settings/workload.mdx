これらの設定はClickHouseサーバーを構成するもので、ClickHouseのソースコードから自動生成されています。

<div id="workload_path">
  ## workload_path
</div>

すべての `CREATE WORKLOAD` および `CREATE RESOURCE` クエリの保存先として使用されるディレクトリです。デフォルトでは、サーバーの作業ディレクトリ配下にある `/workload/` フォルダーが使用されます。

**例**

```xml
<workload_path>/var/lib/clickhouse/workload/</workload_path>
```

<div id="workload_zookeeper_path">
  ## workload_zookeeper_path
</div>

`CREATE WORKLOAD` および `CREATE RESOURCE` クエリをすべて保存するために使用される ZooKeeper node のパスです。一貫性を保つため、すべての SQL 定義はこの単一の znode の値として保存されます。デフォルトでは ZooKeeper は使用されず、定義は[ディスク](/ja/reference/settings/server-settings/settings/workload#workload_path)に保存されます。

**例**

```xml
<workload_zookeeper_path>/clickhouse/workload/definitions.sql</workload_zookeeper_path>
```
