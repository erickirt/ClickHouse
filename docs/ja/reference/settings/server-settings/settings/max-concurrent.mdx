これらの設定はClickHouse サーバーを設定するもので、ClickHouseのソースコードから自動生成されています。

<div id="max_concurrent_insert_queries">
  ## max_concurrent_insert_queries
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

同時実行される `insert` クエリの総数の上限。

<Note>
  値が `0` (デフォルト) の場合は、無制限を意味します。

  この設定は実行時に変更でき、変更は即座に反映されます。すでに実行中のクエリには影響しません。
</Note>

<div id="max_concurrent_queries">
  ## max_concurrent_queries
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

同時実行されるクエリの総数の上限です。`INSERT` クエリおよび `SELECT` クエリの上限、ならびに全ユーザーに対するクエリの最大数の上限も考慮する必要がある点に注意してください。

関連項目:

* [`max_concurrent_insert_queries`](/ja/reference/settings/server-settings/settings/max-concurrent#max_concurrent_insert_queries)
* [`max_concurrent_select_queries`](/ja/reference/settings/server-settings/settings/max-concurrent#max_concurrent_select_queries)
* [`max_concurrent_queries_for_all_users`](/ja/reference/settings/session-settings#max_concurrent_queries_for_all_users)

<Note>
  `0` (デフォルト) は無制限を意味します。

  この設定は実行時に変更でき、変更は即座に反映されます。すでに実行中のクエリには影響しません。
</Note>

  ## max_concurrent_select_queries
