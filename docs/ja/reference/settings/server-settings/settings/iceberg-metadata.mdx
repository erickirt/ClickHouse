これらの設定はClickHouseサーバーを構成するもので、ClickHouseのソースコードから自動生成されています。

<div id="iceberg_metadata_files_cache_max_entries">
  ## iceberg_metadata_files_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />Iceberg metadata files cache の最大エントリ数。0 は無効です。

<div id="iceberg_metadata_files_cache_policy">
  ## iceberg_metadata_files_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Iceberg メタデータキャッシュのポリシー名。

<div id="iceberg_metadata_files_cache_size">
  ## iceberg_metadata_files_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="134217728" changeable_without_restart="Yes" />Icebergのメタデータキャッシュの最大サイズ (バイト単位) 。0 は無効を意味します。

<div id="iceberg_metadata_files_cache_size_ratio">
  ## iceberg_metadata_files_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />Iceberg メタデータキャッシュにおける protected queue (SLRU ポリシーの場合) のサイズが、キャッシュ全体のサイズに占める比率。
