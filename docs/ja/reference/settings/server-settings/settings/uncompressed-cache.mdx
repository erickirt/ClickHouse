これらの設定は ClickHouse サーバーを構成するもので、ClickHouse のソースコードから自動生成されています。

<div id="uncompressed_cache_policy">
  ## uncompressed_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />非圧縮 cache のポリシー名。

<div id="uncompressed_cache_size">
  ## uncompressed_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

MergeTree family のテーブルエンジンで使用される非圧縮データの最大サイズ (バイト単位) 。

server には共有の cache が 1 つあります。メモリは必要に応じて割り当てられます。cache は、`use_uncompressed_cache` オプションが有効な場合に使用されます。

非圧縮 cache は、特に非常に短いクエリでは効果的な場合があります。

<Note>
  `0` は無効を意味します。

  この設定は runtime 中に変更でき、すぐに反映されます。
</Note>

<div id="uncompressed_cache_size_ratio">
  ## uncompressed_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />非圧縮 cache における保護キュー (SLRUポリシーの場合) のサイズの、cache 全体のサイズに対する比率です。
