これらの設定はClickHouse serverを設定するためのもので、ClickHouseのソースコードから自動生成されています。

<div id="vector_similarity_index_cache_max_entries">
  ## vector_similarity_index_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000" changeable_without_restart="No" />ベクトル類似度索引のキャッシュサイズ (エントリ数) 。0 は無効を意味します。

<div id="vector_similarity_index_cache_policy">
  ## vector_similarity_index_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />ベクトル類似度索引キャッシュのポリシー名。

<div id="vector_similarity_index_cache_size">
  ## vector_similarity_index_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="5368709120" changeable_without_restart="Yes" />ベクトル類似度索引キャッシュのサイズです。0 は無効を意味します。

<Note>
  この設定は実行時に変更でき、変更は直ちに反映されます。
</Note>

<div id="vector_similarity_index_cache_size_ratio">
  ## vector_similarity_index_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />ベクトル類似度索引キャッシュ内の保護キュー (SLRUポリシーの場合) のサイズが、キャッシュ全体のサイズに占める割合。
