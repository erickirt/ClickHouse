これらは ClickHouse server を設定する項目で、ClickHouse のソースコードから自動生成されています。

<div id="custom_cached_disks_base_directory">
  ## custom_cached_disks_base_directory
</div>

この設定は、カスタム (SQL で作成された) キャッシュ付きディスクのキャッシュパスを指定します。
`custom_cached_disks_base_directory` は、カスタムディスクについては `filesystem_caches_path` (`filesystem_caches_path.xml` に記載) よりも優先され、
前者が存在しない場合は後者が使用されます。
ファイルシステムキャッシュの設定パスはそのディレクトリ内になければならず、
そうでない場合は、ディスクの作成を防ぐために例外がスローされます。

<Note>
  これは、サーバーのアップグレード前の古いバージョンで作成されたディスクには影響しません。
  この場合は、サーバーが正常に起動できるよう、例外はスローされません。
</Note>

例:

```xml
<custom_cached_disks_base_directory>/var/lib/clickhouse/caches/</custom_cached_disks_base_directory>
```

<div id="custom_settings_prefixes">
  ## custom_settings_prefixes
</div>

[カスタム設定](/ja/concepts/features/configuration/settings/settings-query-level#custom_settings) で使用するプレフィックスの一覧です。
複数のプレフィックスを指定する場合は、カンマで区切ります。

**例**

```xml
<custom_settings_prefixes>SQL_</custom_settings_prefixes>
```
