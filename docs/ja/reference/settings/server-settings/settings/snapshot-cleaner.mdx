これらの設定は ClickHouse server を構成するものであり、ClickHouse のソースコードから自動生成されています。

<div id="snapshot_cleaner_period">
  ## snapshot_cleaner_period
</div>

<SettingsInfoBlock type="UInt64" default_value="120" changeable_without_restart="No" />

SharedMergeTree のスナップショットパーツを完全に削除する間隔です。ClickHouse Cloud でのみ利用できます

<div id="snapshot_cleaner_pool_size">
  ## snapshot_cleaner_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="128" changeable_without_restart="No" />

shared merge tree の snapshot cleaner のクリーンアップ処理に使用するスレッド数。ClickHouse Cloud でのみ利用可能
