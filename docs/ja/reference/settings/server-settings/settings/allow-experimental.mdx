これらの設定は ClickHouse server の構成に使用され、ClickHouse のソースから自動生成されています。

<div id="allow_experimental_executable_udf_drivers">
  ## allow_experimental_executable_udf_drivers
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />`user_defined_executable_function_drivers_config` で宣言された、実行可能なユーザー定義関数用ドライバーの実験的サポートを有効にします。ドライバーは、`CREATE FUNCTION ... ENGINE = DriverName(...) AS '...'` で指定されたユーザーのコードスニペットを、実行可能な UDF に変換します。

<div id="allow_experimental_webassembly_udf">
  ## allow_experimental_webassembly_udf
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />WebAssembly UDF の実験的なサポートを有効にします

<div id="allow_experimental_webterminal">
  ## allow_experimental_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />`enable_webterminal` の旧称 (実験的な名称) です。`enable_webterminal` が設定されていない場合は、後方互換性のため引き続き有効として扱われます。この設定は非推奨です。
