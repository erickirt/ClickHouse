これらの設定はClickHouseサーバーを設定するためのもので、ClickHouseのソースコードから自動生成されています。

<div id="shutdown_wait_backups_and_restores">
  ## shutdown_wait_backups_and_restores
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />true に設定すると、ClickHouse はシャットダウンする前に、実行中のバックアップと復元が完了するまで待機します。

<div id="shutdown_wait_unfinished">
  ## shutdown_wait_unfinished
</div>

<SettingsInfoBlock type="UInt64" default_value="5" changeable_without_restart="No" />未完了のクエリの完了を待機する遅延時間 (秒)

<div id="shutdown_wait_unfinished_queries">
  ## shutdown_wait_unfinished_queries
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />true に設定すると、ClickHouse はシャットダウン前に、実行中のクエリが終了するまで待機します。
