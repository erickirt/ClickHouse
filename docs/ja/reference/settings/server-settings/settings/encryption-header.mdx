これらの設定はClickHouseサーバーの設定項目であり、ClickHouseのソースコードから自動生成されています。

<div id="encryption_header_cache_policy">
  ## encryption_header_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />暗号化ヘッダーキャッシュポリシー名。

<div id="encryption_header_cache_size">
  ## encryption_header_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="52428800" changeable_without_restart="Yes" />

暗号化ファイルから読み込まれた暗号化ヘッダーの暗号化ヘッダーキャッシュの最大サイズです。実験的な ReaderExecutor の読み取りパスでのみ使用されます。

<Note>
  この設定は実行時に変更でき、変更内容は即座に反映されます。
</Note>

<div id="encryption_header_cache_size_ratio">
  ## encryption_header_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />暗号化ヘッダーキャッシュにおける、保護キュー (SLRU policy の場合) のサイズが、cache 全体のサイズに対して占める割合です。
