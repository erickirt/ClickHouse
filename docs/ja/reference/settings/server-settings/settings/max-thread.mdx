これらの設定はClickHouse serverを設定するもので、ClickHouseのソースコードから自動生成されています。

<div id="max_thread_pool_free_size">
  ## max_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />

グローバルスレッドプール内の**アイドル状態の**スレッド数が[`max_thread_pool_free_size`](/ja/reference/settings/server-settings/settings/max-thread#max_thread_pool_free_size)を超えると、ClickHouseは一部のスレッドが使用しているリソースを解放し、プールサイズを縮小します。必要に応じて、スレッドは再度作成されます。

**例**

```xml
<max_thread_pool_free_size>1200</max_thread_pool_free_size>
```

<div id="max_thread_pool_size">
  ## max_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />

ClickHouse は、クエリの処理にグローバルスレッドプールのスレッドを使用します。クエリを処理するアイドル状態のスレッドがない場合は、プール内に新しいスレッドが作成されます。`max_thread_pool_size` は、プール内のスレッド数の最大値を制限します。

**例**

```xml
<max_thread_pool_size>12000</max_thread_pool_size>
```
