これらの設定はClickHouse serverを設定するためのもので、ClickHouseのソースから自動生成されています。

<div id="tcp_close_connection_after_queries_num">
  ## tcp_close_connection_after_queries_num
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />TCP 接続が閉じられるまでに、その TCP 接続で許可されるクエリの最大数です。クエリ数を無制限にするには、0 に設定します。

<div id="tcp_close_connection_after_queries_seconds">
  ## tcp_close_connection_after_queries_seconds
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />TCP 接続 が閉じられるまでの最大存続時間 (秒) 。接続の存続時間を無制限にするには、0 に設定します。
