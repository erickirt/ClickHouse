これらの設定は ClickHouse サーバー を構成するためのもので、ClickHouse のソースコードから自動生成されています。

<div id="os_collect_psi_metrics">
  ## os_collect_psi_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />/proc/pressure/ ファイルの PSI メトリクスの収集を有効にします。

<div id="os_cpu_busy_time_threshold">
  ## os_cpu_busy_time_threshold
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000" changeable_without_restart="No" />CPU が実際に処理を行っていると見なすための、マイクロ秒単位の OS CPU ビジー時間 (`OSCPUVirtualTimeMicroseconds` メトリクス) のしきい値です。ビジー時間がこの値を下回る場合、CPU は過負荷とは見なされません。
