これらの設定は ClickHouse サーバー の設定に使用されるもので、ClickHouse のソースコードから自動生成されています。

<div id="database_replicated_allow_detach_permanently">
  ## database_replicated_allow_detach_permanently
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Replicated データベースでテーブルを永続的にデタッチすることを許可します

<div id="database_replicated_drop_broken_tables">
  ## database_replicated_drop_broken_tables
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Replicated データベースでは、予期しないテーブルを別のローカルデータベースに移動する代わりに削除します
