これらは ClickHouse server を設定する項目で、ClickHouse のソースコードから自動生成されています。

<div id="max_replicated_fetches_network_bandwidth_for_server">
  ## max_replicated_fetches_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />レプリケーションフェッチにおける、ネットワーク経由のデータ転送速度の上限を 1 秒あたりのバイト数で指定します。0 は無制限を意味します。

<div id="max_replicated_sends_network_bandwidth_for_server">
  ## max_replicated_sends_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />レプリケート送信における、ネットワーク経由のデータ転送速度の最大値をバイト/秒で指定します。0 は無制限を意味します。

<div id="max_replicated_table_num_to_throw">
  ## max_replicated_table_num_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

レプリケートテーブルの数がこの値を超えると、サーバーは例外をスローします。

次のデータベースエンジンのテーブルのみをカウントします。

* Atomic
* Ordinary
* Replicated
* Lazy

<Note>
  値 `0` は、制限なしを意味します。
</Note>

**例**

```xml
<max_replicated_table_num_to_throw>400</max_replicated_table_num_to_throw>
```
