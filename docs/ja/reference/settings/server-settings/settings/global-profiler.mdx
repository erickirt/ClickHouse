これらの設定はClickHouse サーバーの構成に使用され、ClickHouseのソースコードから自動生成されています。

<div id="global_profiler_cpu_time_period_ns">
  ## global_profiler_cpu_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />グローバルプロファイラの CPU クロックタイマーの周期 (ナノ秒単位) 。CPU クロックのグローバルプロファイラを無効にするには、値を 0 に設定します。推奨値は、単一クエリでは少なくとも 10000000 (1 秒あたり 100 回) 、クラスター全体のプロファイリングでは 1000000000 (1 秒あたり 1 回) です。

  ## global_profiler_real_time_period_ns
</div>
