これらの設定はClickHouseサーバーを構成するもので、ClickHouseのソースコードから自動生成されています。

<div id="postgresql_port">
  ## postgresql_port
</div>

PostgreSQLプロトコルでクライアントと通信するためのポートです。

<Note>
  * 正の整数を指定すると、待ち受けるポート番号になります
  * PostgreSQLプロトコルでのクライアントとの通信を無効にするには、値を空にします
</Note>

**例**

```xml
<postgresql_port>9005</postgresql_port>
```

<div id="postgresql_require_secure_transport">
  ## postgresql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />true に設定すると、[postgresql&#95;port](/ja/reference/settings/server-settings/settings/postgresql#postgresql_port) で接続するクライアントとの通信にセキュアな接続が必須になります。`<sslmode=disable>` オプションを指定した接続は拒否されます。[OpenSSL](/ja/reference/settings/server-settings/settings/other#openssl) の設定とあわせて使用してください。
