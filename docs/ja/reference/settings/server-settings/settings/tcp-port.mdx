これらの設定はClickHouseサーバーを構成するもので、ClickHouseのソースコードから自動生成されています。

<div id="tcp_port">
  ## tcp_port
</div>

TCPプロトコル経由でクライアントと通信するためのポート。

**例**

```xml
<tcp_port>9000</tcp_port>
```

<div id="tcp_port_secure">
  ## tcp_port_secure
</div>

クライアントとのセキュアな通信に使用する TCP ポートです。[OpenSSL](/ja/reference/settings/server-settings/settings/other#openssl) の設定と併せて使用します。

**デフォルト値**

```xml
<tcp_port_secure>9440</tcp_port_secure>
```
