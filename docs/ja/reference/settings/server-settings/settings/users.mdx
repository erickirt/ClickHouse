これらの設定はClickHouse サーバーの設定を行うもので、ClickHouseのソースコードから自動生成されています。

<div id="users_config">
  ## users_config
</div>

次の内容を含むファイルへのパス：

* ユーザー設定。
* アクセス権。
* 設定プロファイル。
* クォータ設定。

**例**

```xml
<users_config>users.xml</users_config>
```

<div id="users_to_ignore_early_memory_limit_check">
  ## users_to_ignore_early_memory_limit_check
</div>

早期のメモリ制限チェックの対象外とするユーザーのカンマ区切りリストです。ユーザーがこのリストに含まれていない場合、合計のメモリ使用量が制限を超えると、クエリは拒否されます。
