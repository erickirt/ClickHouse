これらの設定はClickHouseサーバーの設定であり、ClickHouseのソースコードから自動生成されています。

<div id="s3queue_disable_streaming">
  ## s3queue_disable_streaming
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Yes" />テーブルが作成されており、マテリアライズドビューがアタッチされている場合でも、S3Queue のストリーミングを無効にします

<div id="s3queue_log">
  ## s3queue_log
</div>

`s3queue_log` システムテーブルの設定です。

<SystemLogParameters />

デフォルト設定は次のとおりです。

```xml
<s3queue_log>
    <database>system</database>
    <table>s3queue_log</table>
    <partition_by>toYYYYMM(event_date)</partition_by>
    <flush_interval_milliseconds>7500</flush_interval_milliseconds>
</s3queue_log>
```
