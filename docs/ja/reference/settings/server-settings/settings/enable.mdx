これらの設定はClickHouseサーバーの構成に使用され、ClickHouseのソースコードから自動生成されています。

<div id="enable_azure_sdk_logging">
  ## enable_azure_sdk_logging
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Azure SDK のログを有効にします

<div id="enable_webterminal">
  ## enable_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />`/webterminal` HTTPエンドポイントでウェブターミナルのインターフェイスを有効にします。WebSocket 経由で、ブラウザ内で対話的な `clickhouse-client` セッションを提供します。`false` の場合、`/webterminal` へのリクエストには HTTPステータス `403 Forbidden` が返されます。
