これらの設定は ClickHouse server を設定するためのもので、ClickHouse のソースから自動生成されています。

<div id="max_io_thread_pool_free_size">
  ## max_io_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

I/O スレッドプール内の**アイドル状態**のスレッド数が `max_io_thread_pool_free_size` を超えると、ClickHouse は待機中のスレッドが占有しているリソースを解放し、プールサイズを縮小します。必要に応じて、スレッドは再度作成されます。

<div id="max_io_thread_pool_size">
  ## max_io_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="100" changeable_without_restart="Yes" />

ClickHouse は、一部の I/O 操作 (たとえば S3 とのやり取り) に I/O スレッドプールのスレッドを使用します。`max_io_thread_pool_size` は、プール内のスレッドの最大数を制限します。
