これらの設定はClickHouseサーバーを設定するもので、ClickHouseのソースから自動生成されています。

<div id="max_snapshot_commit_thread_pool_free_size">
  ## max_snapshot_commit_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />スナップショット commit thread pool 内のアイドルスレッド数が `max_snapshot_commit_thread_pool_free_size` を超えると、ClickHouse はアイドル状態のスレッドが占有しているリソースを解放し、プールのサイズを縮小します。必要に応じて、スレッドは再作成されます。

<div id="max_snapshot_commit_thread_pool_size">
  ## max_snapshot_commit_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="64" changeable_without_restart="はい" />スナップショットのコミットに使用するスレッド数。
