これらの設定はClickHouse serverを設定するもので、ClickHouseのソースコードから自動生成されています。

<div id="dictionaries_config">
  ## dictionaries_config
</div>

Dictionaryの設定ファイルへのパスです。

パス:

* 絶対パス、またはserver設定ファイルを基準とした相対パスを指定します。
* パスにはワイルドカード `*` と `?` を含めることができます。

関連項目:

* &quot;[Dictionaries](/ja/reference/statements/create/dictionary)&quot;。

**例**

```xml
<dictionaries_config>*_dictionary.xml</dictionaries_config>
```

<div id="dictionaries_lazy_load">
  ## dictionaries_lazy_load
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />

Dictionary の遅延読み込み。

* `true` の場合、各 Dictionary は初回の使用時に読み込まれます。読み込みに失敗すると、その Dictionary を使用している関数は例外をスローします。
* `false` の場合、サーバーは起動時にすべての Dictionary を読み込みます。

<Note>
  サーバーは、すべての Dictionary の読み込みが完了するまで起動時に待機し、その後で接続を受け付けます
  (例外: [`wait_dictionaries_load_at_startup`](/ja/reference/settings/server-settings/settings/other#wait_dictionaries_load_at_startup) が `false` に設定されている場合) 。
</Note>

**例**

```xml
<dictionaries_lazy_load>true</dictionaries_lazy_load>
```

<div id="dictionaries_lib_path">
  ## dictionaries_lib_path
</div>

<SettingsInfoBlock type="String" default_value="/var/lib/clickhouse/dictionaries_lib/" changeable_without_restart="No" />

Dictionary lib が格納されているディレクトリです。

**例**

```xml
<dictionaries_lib_path>/var/lib/clickhouse/dictionaries_lib/</dictionaries_lib_path>
```
