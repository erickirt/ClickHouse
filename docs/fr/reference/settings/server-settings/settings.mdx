<a id="server-settings" />

Cette section contient la description des paramètres du serveur. Il s’agit de paramètres qui
ne peuvent pas être modifiés au niveau de la session ou de la requête.

Pour plus d’informations sur les fichiers de configuration dans ClickHouse, consultez [&quot;&quot;Fichiers de configuration&quot;&quot;](/fr/concepts/features/configuration/server-config/configuration-files).

Les autres paramètres sont décrits dans la section &quot;&quot;[Paramètres](/fr/concepts/features/configuration/settings/overview)&quot;&quot;.
Avant d’étudier les paramètres, nous vous recommandons de lire la section [Fichiers de configuration](/fr/concepts/features/configuration/server-config/configuration-files)
et de noter l’utilisation des substitutions (les attributs `incl` et `optional`).
