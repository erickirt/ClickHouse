Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir des sources de ClickHouse.

<div id="dictionaries_config">
  ## dictionaries_config
</div>

Le chemin du fichier de configuration des dictionnaires.

Chemin :

* Indiquez le chemin absolu ou le chemin relatif au fichier de configuration du serveur.
* Le chemin peut contenir les caractères génériques * et ?.

Voir aussi :

* &quot;[Dictionaries](/fr/reference/statements/create/dictionary)&quot;.

**Exemple**

```xml
<dictionaries_config>*_dictionary.xml</dictionaries_config>
```

<div id="dictionaries_lazy_load">
  ## dictionaries_lazy_load
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />

Chargement différé des dictionnaires.

* Si `true`, chaque dictionnaire est chargé lors de sa première utilisation. Si le chargement échoue, la fonction qui utilise le dictionnaire lève une exception.
* Si `false`, le serveur charge tous les dictionnaires au démarrage.

<Note>
  Au démarrage, le serveur attend que tous les dictionnaires aient fini de se charger avant d&#39;accepter des connexions
  (exception : si [`wait_dictionaries_load_at_startup`](/fr/reference/settings/server-settings/settings/other#wait_dictionaries_load_at_startup) est défini sur `false`).
</Note>

**Exemple**

```xml
<dictionaries_lazy_load>true</dictionaries_lazy_load>
```

<div id="dictionaries_lib_path">
  ## dictionaries_lib_path
</div>

<SettingsInfoBlock type="String" default_value="/var/lib/clickhouse/dictionaries_lib/" changeable_without_restart="No" />

Le répertoire contenant la bibliothèque de dictionnaires.

**Exemple**

```xml
<dictionaries_lib_path>/var/lib/clickhouse/dictionaries_lib/</dictionaries_lib_path>
```
