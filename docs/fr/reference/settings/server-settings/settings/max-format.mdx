Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="max_format_parsing_thread_pool_free_size">
  ## max_format_parsing_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Oui" />

Nombre maximal de threads de secours inactifs à conserver dans le pool de threads pour l’analyse syntaxique des données d’entrée.

<div id="max_format_parsing_thread_pool_size">
  ## max_format_parsing_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="100" changeable_without_restart="Oui" />

Nombre maximal total de threads à utiliser pour l’analyse syntaxique des données d’entrée.
