Ces paramètres configurent le ClickHouse server et sont générés automatiquement à partir du code source de ClickHouse.

<div id="primary_index_cache_policy">
  ## primary_index_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Nom de la politique de cache de l’index primaire.

<div id="primary_index_cache_prewarm_ratio">
  ## primary_index_cache_prewarm_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.95" changeable_without_restart="No" />La proportion de la taille totale du cache de marques à remplir pendant le préchauffage.

<div id="primary_index_cache_size">
  ## primary_index_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="5368709120" changeable_without_restart="Oui" />Taille maximale du cache de l’index primaire (index de la famille de tables MergeTree).

<div id="primary_index_cache_size_ratio">
  ## primary_index_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="Non" />La taille de la file d’attente protégée (dans le cas de la politique SLRU) dans le cache de l’index primaire, par rapport à la taille totale du cache.
