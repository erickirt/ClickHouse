Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir des sources de ClickHouse.

<div id="index_uncompressed_cache_policy">
  ## index_uncompressed_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Nom de la politique du cache non compressé de l’index secondaire.

<div id="index_uncompressed_cache_size">
  ## index_uncompressed_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Taille maximale du cache pour les blocs non compressés des indices `MergeTree`.

<Note>
  Une valeur de `0` signifie que le paramètre est désactivé.

  Ce paramètre peut être modifié en cours d’exécution et prendra effet immédiatement.
</Note>

<div id="index_uncompressed_cache_size_ratio">
  ## index_uncompressed_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />Taille de la file protégée (dans le cas de la politique SLRU) dans le cache non compressé de l’index secondaire, par rapport à la taille totale du cache.
