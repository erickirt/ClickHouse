Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="users_config">
  ## users_config
</div>

Chemin vers le fichier qui contient :

* Les configurations des utilisateurs.
* Les droits d&#39;accès.
* Les profils de paramètres.
* Les paramètres de quotas.

**Exemple**

```xml
<users_config>users.xml</users_config>
```

<div id="users_to_ignore_early_memory_limit_check">
  ## users_to_ignore_early_memory_limit_check
</div>

Liste d’utilisateurs, séparés par des virgules, pour lesquels la vérification anticipée de la limite de mémoire est ignorée. Si l’utilisateur ne figure pas dans cette liste, une requête sera rejetée si l’utilisation totale de la mémoire dépasse la limite.
