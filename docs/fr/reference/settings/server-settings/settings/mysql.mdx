Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="mysql_port">
  ## mysql_port
</div>

Port utilisé pour communiquer avec les clients via le protocole MySQL.

<Note>
  * Les entiers positifs indiquent le numéro de port sur lequel écouter
  * Des valeurs vides permettent de désactiver la communication avec les clients via le protocole MySQL.
</Note>

**Exemple**

```xml
<mysql_port>9004</mysql_port>
```

<div id="mysql_require_secure_transport">
  ## mysql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Si cette valeur est définie sur true, une communication sécurisée avec les clients via [mysql&#95;port](/fr/reference/settings/server-settings/settings/mysql#mysql_port) est requise. Les connexions avec l&#39;option `<--ssl-mode=none>` seront refusées. Utilisez ce paramètre avec les paramètres [OpenSSL](/fr/reference/settings/server-settings/settings/other#openssl).
