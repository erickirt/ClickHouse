Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="parquet_metadata_cache_max_entries">
  ## parquet_metadata_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="5000" changeable_without_restart="Non" />Taille maximale du cache des fichiers de métadonnées Parquet, en nombre d’entrées. Zéro signifie que le cache est désactivé.

<div id="parquet_metadata_cache_policy">
  ## parquet_metadata_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Nom de la politique de cache de métadonnées Parquet.

<div id="parquet_metadata_cache_size">
  ## parquet_metadata_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="536870912" changeable_without_restart="Oui" />Taille maximale du cache de métadonnées Parquet, en octets. Zéro signifie que le cache est désactivé.

<div id="parquet_metadata_cache_size_ratio">
  ## parquet_metadata_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="Non" />La taille de la file d’attente protégée (dans le cas d’une politique SLRU) du cache de métadonnées Parquet, par rapport à la taille totale du cache.
