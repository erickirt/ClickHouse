Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="interserver_https_host">
  ## interserver_https_host
</div>

Semblable à `<interserver_http_host>`, sauf qu’ici, ce nom d’hôte peut être utilisé par d’autres serveurs pour accéder à ce serveur via `<HTTPS>`.

**Exemple**

```xml
<interserver_https_host>example.clickhouse.com</interserver_https_host>
```

<div id="interserver_https_port">
  ## interserver_https_port
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />

Port utilisé pour l’échange de données entre les serveurs ClickHouse via `<HTTPS>`.

**Exemple**

```xml
<interserver_https_port>9010</interserver_https_port>
```
