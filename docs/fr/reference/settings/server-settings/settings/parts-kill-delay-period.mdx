Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="parts_kill_delay_period">
  ## parts_kill_delay_period
</div>

<SettingsInfoBlock type="UInt64" default_value="30" changeable_without_restart="No" />

Période avant la suppression complète des parts pour SharedMergeTree. Disponible uniquement dans ClickHouse Cloud

<div id="parts_kill_delay_period_random_add">
  ## parts_kill_delay_period_random_add
</div>

<SettingsInfoBlock type="UInt64" default_value="10" changeable_without_restart="No" />

Ajoute à `kill_delay_period` une valeur répartie uniformément entre 0 et x secondes afin d’éviter un effet de troupeau ainsi qu’un DoS ultérieur de ZooKeeper en cas de très grand nombre de tables. Only available in ClickHouse Cloud
