<div id="backups">
  ## sauvegardes
</div>

Paramètres de sauvegarde, utilisés lors de l’exécution des instructions [`BACKUP` et `RESTORE`](/fr/concepts/features/backup-restore/overview).

<div id="backups_io_thread_pool_queue_size">
  ## backups_io_thread_pool_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Nombre maximal de jobs pouvant être planifiés dans le pool de threads d’E/S des sauvegardes. Il est recommandé de laisser cette file d’attente illimitée en raison de la logique actuelle des sauvegardes sur S3.

<Note>
  Une valeur de `0` (par défaut) signifie que la file d’attente est illimitée.
</Note>
