<div id="os_threads_nice_value_distributed_cache_tcp_handler">
  ## os_threads_nice_value_distributed_cache_tcp_handler
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="No" />

Valeur nice Linux pour les threads du handler TCP du cache distribué. Des valeurs plus faibles correspondent à une priorité CPU plus élevée.

Nécessite la capacité CAP&#95;SYS&#95;NICE, sinon est sans effet.

Valeurs possibles : -20 à 19.

<div id="os_threads_nice_value_merge_mutate">
  ## os_threads_nice_value_merge_mutate
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="No" />

Valeur nice Linux pour les threads de fusion et de mutation. Des valeurs plus faibles correspondent à une priorité CPU plus élevée.

Nécessite la capacité CAP&#95;SYS&#95;NICE, sinon est sans effet.

Valeurs possibles : -20 à 19.

<div id="os_threads_nice_value_zookeeper_client_send_receive">
  ## os_threads_nice_value_zookeeper_client_send_receive
</div>

<SettingsInfoBlock type="Int32" default_value="0" changeable_without_restart="No" />

Valeur nice Linux des threads d’envoi et de réception du client ZooKeeper. Plus la valeur est faible, plus la priorité CPU est élevée.

Nécessite la capacité CAP&#95;SYS&#95;NICE, sinon cette option est sans effet.

Valeurs possibles : -20 à 19.
