<div id="jemalloc_profiler_sampling_rate">
  ## jemalloc_profiler_sampling_rate
</div>

<SettingsInfoBlock type="UInt64" default_value="19" changeable_without_restart="No" />

Contrôle `lg_prof_sample` de jemalloc — le logarithme en base 2 de l’intervalle moyen (en octets) entre les échantillons d’allocation.
La valeur par défaut, 19, correspond à 512 Kio. Une valeur plus faible augmente la fréquence d’échantillonnage (plus de surcharge, plus de détails), tandis qu’une valeur plus élevée la réduit.
Modifier cette valeur appelle `prof.reset`, ce qui réinitialise toutes les statistiques de profilage accumulées. Le profilage doit être activé (`MALLOC_CONF=prof:true`).
