Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="query_condition_cache_policy">
  ## query_condition_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Nom de la politique du cache de conditions de requête.

<div id="query_condition_cache_size">
  ## query_condition_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="104857600" changeable_without_restart="Yes" />

Taille maximale du cache de conditions de requête.

<Note>
  Ce paramètre peut être modifié à l’exécution et prend effet immédiatement.
</Note>

<div id="query_condition_cache_size_ratio">
  ## query_condition_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />La taille de la file protégée (dans le cas de la politique SLRU) du cache de conditions de requête, par rapport à la taille totale du cache.
