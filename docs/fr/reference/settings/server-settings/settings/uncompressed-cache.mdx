Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir des sources de ClickHouse.

<div id="uncompressed_cache_policy">
  ## uncompressed_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Nom de la politique du cache non compressé.

<div id="uncompressed_cache_size">
  ## uncompressed_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Taille maximale (en octets) des données non compressées utilisées par les moteurs de table de la famille MergeTree.

Le serveur dispose d’un cache partagé. La mémoire est allouée à la demande. Le cache est utilisé si l’option `use_uncompressed_cache` est activée.

Le cache non compressé peut être avantageux pour des requêtes très courtes dans certains cas.

<Note>
  Une valeur de `0` signifie que ce paramètre est désactivé.

  Ce paramètre peut être modifié en cours d’exécution et prend effet immédiatement.
</Note>

<div id="uncompressed_cache_size_ratio">
  ## uncompressed_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="No" />La taille de la file d’attente protégée (dans le cas de la politique SLRU) du cache non compressé, par rapport à la taille totale du cache.
