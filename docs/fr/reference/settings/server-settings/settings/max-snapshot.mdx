Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir des sources de ClickHouse.

<div id="max_snapshot_commit_thread_pool_free_size">
  ## max_snapshot_commit_thread_pool_free_size
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Oui" />Si le nombre de threads inactifs dans le pool de threads de commit des instantanés dépasse `max_snapshot_commit_thread_pool_free_size`, ClickHouse libérera les ressources occupées par ces threads inactifs et réduira la taille du pool. Les threads pourront être recréés si nécessaire.

<div id="max_snapshot_commit_thread_pool_size">
  ## max_snapshot_commit_thread_pool_size
</div>

<SettingsInfoBlock type="UInt64" default_value="64" changeable_without_restart="Yes" />Le nombre de threads utilisés pour valider les instantanés.
