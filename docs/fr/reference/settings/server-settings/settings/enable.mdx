<div id="enable_webterminal">
  ## enable_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Active l’interface du terminal web au endpoint HTTP `/webterminal`. Fournit une session `clickhouse-client` interactive dans le navigateur via WebSocket. Lorsque `false`, les requêtes vers `/webterminal` renvoient le code HTTP `403 Forbidden`.
