Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="keeper_hosts">
  ## keeper_hosts
</div>

Paramètre dynamique. Contient un ensemble d’hôtes [Zoo]Keeper auxquels ClickHouse peut potentiellement se connecter. N’expose aucune information issue de `<auxiliary_zookeepers>`

<div id="keeper_multiread_batch_size">
  ## keeper_multiread_batch_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="Non" />

Taille maximale du lot pour une requête MultiRead vers [Zoo]Keeper prenant en charge le traitement par lots. Si cette valeur est définie sur 0, le traitement par lots est désactivé. Disponible uniquement dans ClickHouse Cloud.
