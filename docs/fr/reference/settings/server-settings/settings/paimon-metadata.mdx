Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir des sources de ClickHouse.

<div id="paimon_metadata_files_cache_max_entries">
  ## paimon_metadata_files_cache_max_entries
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="Oui" />Nombre maximal d&#39;entrées dans le cache des fichiers de métadonnées paimon. Zéro signifie qu&#39;il n&#39;y a aucune limite sur le nombre d&#39;entrées.

<div id="paimon_metadata_files_cache_policy">
  ## paimon_metadata_files_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="Non" />Nom de la politique de cache des métadonnées de Paimon.

<div id="paimon_metadata_files_cache_size">
  ## paimon_metadata_files_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" changeable_without_restart="Oui" />Taille maximale du cache des métadonnées de Paimon, en octets. Une valeur de zéro signifie qu’il est désactivé.

<div id="paimon_metadata_files_cache_size_ratio">
  ## paimon_metadata_files_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="Non" />La taille de la file d’attente protégée (dans le cas d’une stratégie SLRU) du cache des métadonnées de Paimon, par rapport à la taille totale du cache.
