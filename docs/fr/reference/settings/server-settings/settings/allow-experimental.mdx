Ces paramètres configurent le ClickHouse server et sont générés automatiquement à partir du code source de ClickHouse.

<div id="allow_experimental_executable_udf_drivers">
  ## allow_experimental_executable_udf_drivers
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Non" />Active la prise en charge expérimentale des drivers pour les fonctions utilisateur exécutables, déclarées via `user_defined_executable_function_drivers_config`. Un driver convertit un extrait de code utilisateur fourni dans `CREATE FUNCTION ... ENGINE = DriverName(...) AS '...'` en une UDF exécutable.

<div id="allow_experimental_webassembly_udf">
  ## allow_experimental_webassembly_udf
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Non" />Active la prise en charge expérimentale des UDF WebAssembly

<div id="allow_experimental_webterminal">
  ## allow_experimental_webterminal
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="Non" />Ancien nom (expérimental) de `enable_webterminal`. Toujours pris en charge pour la rétrocompatibilité lorsque `enable_webterminal` n’est pas défini. Ce paramètre est déprécié.
