Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="global_profiler_cpu_time_period_ns">
  ## global_profiler_cpu_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />Période du temporisateur d’horloge CPU du profileur global (en nanosecondes). Définissez cette valeur sur 0 pour désactiver le profileur global d’horloge CPU. La valeur recommandée est d’au moins 10000000 (100 fois par seconde) pour des requêtes individuelles, ou 1000000000 (une fois par seconde) pour le profilage à l’échelle du cluster.

<div id="global_profiler_real_time_period_ns">
  ## global_profiler_real_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="10000000000" changeable_without_restart="No" />Période du temporisateur d’horloge réelle du profileur global (en nanosecondes). Définissez cette valeur sur 0 pour désactiver le profileur global d’horloge réelle. La valeur recommandée est d’au moins 10000000 (100 fois par seconde) pour les requêtes individuelles, ou 1000000000 (une fois par seconde) pour le profilage à l’échelle du cluster.
