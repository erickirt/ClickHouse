Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="postgresql_port">
  ## postgresql_port
</div>

Port utilisé pour communiquer avec les clients via le protocole PostgreSQL.

<Note>
  * Les entiers positifs indiquent le numéro de port à écouter
  * Des valeurs vides permettent de désactiver la communication avec les clients via le protocole PostgreSQL.
</Note>

**Exemple**

```xml
<postgresql_port>9005</postgresql_port>
```

<div id="postgresql_require_secure_transport">
  ## postgresql_require_secure_transport
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Non" />Si cette option est définie sur true, une communication sécurisée avec les clients via [postgresql&#95;port](/fr/reference/settings/server-settings/settings/postgresql#postgresql_port) est requise. Les connexions avec l’option `<sslmode=disable>` seront refusées. Utilisez ce paramètre avec les paramètres [OpenSSL](/fr/reference/settings/server-settings/settings/other#openssl).
