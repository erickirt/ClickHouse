Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="os_collect_psi_metrics">
  ## os_collect_psi_metrics
</div>

<SettingsInfoBlock type="Bool" default_value="1" changeable_without_restart="No" />Active la collecte des métriques PSI à partir des fichiers /proc/pressure/.

<div id="os_cpu_busy_time_threshold">
  ## os_cpu_busy_time_threshold
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000" changeable_without_restart="No" />Seuil du temps d’activité du CPU de l’OS, en microsecondes (métrique OSCPUVirtualTimeMicroseconds), à partir duquel le CPU est considéré comme effectuant un travail utile ; aucune surcharge CPU n’est prise en compte si ce temps d’activité est inférieur à cette valeur.
