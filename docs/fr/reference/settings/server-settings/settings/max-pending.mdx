Ces paramètres servent à configurer le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="max_pending_mutations_execution_time_to_warn">
  ## max_pending_mutations_execution_time_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="86400" changeable_without_restart="Oui" />

Si l’une des mutations en attente dépasse la valeur spécifiée en secondes, le serveur ClickHouse ajoutera des messages d’avertissement à la table `system.warnings`.

**Exemple**

```xml
<max_pending_mutations_execution_time_to_warn>10000</max_pending_mutations_execution_time_to_warn>
```

<div id="max_pending_mutations_to_warn">
  ## max_pending_mutations_to_warn
</div>

<SettingsInfoBlock type="UInt64" default_value="500" changeable_without_restart="Oui" />

Si le nombre de mutations en attente dépasse la valeur spécifiée, le serveur ClickHouse ajoutera des messages d’avertissement à la table `system.warnings`.

**Exemple**

```xml
<max_pending_mutations_to_warn>400</max_pending_mutations_to_warn>
```
