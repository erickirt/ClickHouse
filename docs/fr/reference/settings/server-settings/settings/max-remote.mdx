Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="max_remote_read_connections">
  ## max_remote_read_connections
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" changeable_without_restart="No" />Nombre maximal de connexions de lecture à distance ouvertes maintenues actives par `ReaderExecutor` afin d’optimiser la lecture séquentielle. 0 désactive la réutilisation des connexions.

<div id="max_remote_read_network_bandwidth_for_server">
  ## max_remote_read_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Le débit maximal de transfert de données sur le réseau, en octets par seconde, pour la lecture.

<Note>
  Une valeur de `0` (par défaut) signifie qu’il n’y a pas de limite.
</Note>

<div id="max_remote_write_network_bandwidth_for_server">
  ## max_remote_write_network_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Le débit maximal des échanges de données sur le réseau, en octets par seconde, en écriture.

<Note>
  Une valeur de `0` (par défaut) signifie illimité.
</Note>
