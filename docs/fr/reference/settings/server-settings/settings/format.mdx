Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="format_parsing_thread_pool_queue_size">
  ## format_parsing_thread_pool_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="Yes" />

Le nombre maximal de tâches pouvant être planifiées dans le pool de threads pour l’analyse syntaxique des données d’entrée.

<Note>
  Une valeur de `0` signifie sans limite.
</Note>

<div id="format_schema_path">
  ## format_schema_path
</div>

<SettingsInfoBlock type="String" default_value="/var/lib/clickhouse/format_schemas/" changeable_without_restart="No" />

Chemin vers le répertoire contenant les schémas des données d’entrée, par exemple les schémas du format [CapnProto](/fr/reference/formats/CapnProto).

**Exemple**

```xml
<!-- Directory containing schema files for various input formats. -->
<format_schema_path>/var/lib/clickhouse/format_schemas/</format_schema_path>
```
