Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="merge_tree">
  ## merge_tree
</div>

Ajustement fin des tables dans [MergeTree](/fr/reference/engines/table-engines/mergetree-family/mergetree).

Pour plus d’informations, consultez le fichier d’en-tête MergeTreeSettings.h.

**Exemple**

```xml
<merge_tree>
    <max_suspicious_broken_parts>5</max_suspicious_broken_parts>
</merge_tree>
```

<div id="merge_workload">
  ## merge_workload
</div>

<SettingsInfoBlock type="String" default_value="default" changeable_without_restart="Yes" />

Permet de réguler l’utilisation et le partage des ressources entre les fusions et les autres charges de travail. La valeur spécifiée est utilisée comme valeur du paramètre `workload` pour toutes les fusions en arrière-plan. Peut être surchargée par un paramètre MergeTree.
