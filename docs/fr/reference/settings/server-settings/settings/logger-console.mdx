Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="logger.console">
  ## logger.console
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Non" />Active la journalisation dans la console. Définissez la valeur sur `<1>` ou `<true>` pour l’activer. La valeur par défaut est `<1>` si ClickHouse ne s’exécute pas en mode daemon, sinon `<0>`.

<div id="logger.console_log_level">
  ## logger.console_log_level
</div>

<SettingsInfoBlock type="String" default_value="trace" changeable_without_restart="No" />Niveau de journalisation pour la sortie sur la console. La valeur par défaut est `<level>`.
