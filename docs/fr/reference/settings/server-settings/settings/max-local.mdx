Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir des sources de ClickHouse.

<div id="max_local_read_bandwidth_for_server">
  ## max_local_read_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Yes" />

Le débit maximal des lectures locales en octets par seconde.

<Note>
  Une valeur de `0` signifie qu’il n’y a pas de limite.
</Note>

<div id="max_local_write_bandwidth_for_server">
  ## max_local_write_bandwidth_for_server
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="Oui" />

Le débit maximal des écritures locales, en octets par seconde.

<Note>
  Une valeur de `0` signifie qu’il n’y a pas de limite.
</Note>
