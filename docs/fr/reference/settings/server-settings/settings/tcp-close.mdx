Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir des sources de ClickHouse.

<div id="tcp_close_connection_after_queries_num">
  ## tcp_close_connection_after_queries_num
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />Nombre maximal de requêtes autorisées par connexion TCP avant que la connexion ne soit fermée. Définissez cette valeur sur 0 pour autoriser un nombre illimité de requêtes.

<div id="tcp_close_connection_after_queries_seconds">
  ## tcp_close_connection_after_queries_seconds
</div>

<SettingsInfoBlock type="UInt64" default_value="0" changeable_without_restart="No" />Durée de vie maximale d’une connexion TCP, en secondes, avant sa fermeture. Définissez `0` pour une durée de vie de connexion illimitée.
