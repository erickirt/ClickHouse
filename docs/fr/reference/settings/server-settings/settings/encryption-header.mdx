Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="encryption_header_cache_policy">
  ## encryption_header_cache_policy
</div>

<SettingsInfoBlock type="String" default_value="SLRU" changeable_without_restart="No" />Nom de la politique du cache des en-têtes de chiffrement.

<div id="encryption_header_cache_size">
  ## encryption_header_cache_size
</div>

<SettingsInfoBlock type="UInt64" default_value="52428800" changeable_without_restart="Oui" />

Taille maximale du cache des en-têtes de chiffrement lus depuis des fichiers chiffrés. Utilisé uniquement par le chemin de lecture ReaderExecutor expérimental.

<Note>
  Ce paramètre peut être modifié à l’exécution et prendra effet immédiatement.
</Note>

<div id="encryption_header_cache_size_ratio">
  ## encryption_header_cache_size_ratio
</div>

<SettingsInfoBlock type="Double" default_value="0.5" changeable_without_restart="Non" />La taille de la file protégée (dans le cas d’une politique SLRU) du cache des en-têtes de chiffrement, par rapport à la taille totale du cache.
