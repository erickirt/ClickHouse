Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="threadpool_local_fs_reader_pool_size">
  ## threadpool_local_fs_reader_pool_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="100" changeable_without_restart="No" />Le nombre de threads dans le pool de threads utilisé pour la lecture depuis le système de fichiers local lorsque `local_filesystem_read_method = 'pread_threadpool'`.

<div id="threadpool_local_fs_reader_queue_size">
  ## threadpool_local_fs_reader_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />Nombre maximal de tâches pouvant être planifiées dans le pool de threads pour la lecture depuis le système de fichiers local.
