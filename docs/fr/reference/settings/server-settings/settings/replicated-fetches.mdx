Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="replicated_fetches_http_connection_timeout">
  ## replicated_fetches_http_connection_timeout
</div>

<SettingsInfoBlock type="Secondes" default_value="0" changeable_without_restart="Non" />Délai d’expiration de la connexion HTTP pour les requêtes de récupération de parts. Hérité du profil par défaut `http_connection_timeout` s’il n’est pas défini explicitement.

<div id="replicated_fetches_http_receive_timeout">
  ## replicated_fetches_http_receive_timeout
</div>

<SettingsInfoBlock type="Secondes" default_value="0" changeable_without_restart="Non" />Délai maximal de réception HTTP pour les requêtes de récupération de parts. Hérité du profil par défaut `http_receive_timeout` s’il n’est pas défini explicitement.

<div id="replicated_fetches_http_send_timeout">
  ## replicated_fetches_http_send_timeout
</div>

<SettingsInfoBlock type="Secondes" default_value="0" changeable_without_restart="No" />Délai d’expiration de l’envoi HTTP pour les requêtes de récupération de parts. Hérité du profil par défaut `http_send_timeout` s’il n’est pas explicitement défini.
