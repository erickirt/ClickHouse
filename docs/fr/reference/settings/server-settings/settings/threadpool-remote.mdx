Ces paramètres servent à configurer le serveur ClickHouse et sont générés automatiquement à partir des sources de ClickHouse.

<div id="threadpool_remote_fs_reader_pool_size">
  ## threadpool_remote_fs_reader_pool_size
</div>

<SettingsInfoBlock type="NonZeroUInt64" default_value="250" changeable_without_restart="No" />Nombre de threads dans le pool de threads utilisé pour lire depuis le système de fichiers distant lorsque `remote_filesystem_read_method = 'threadpool'`.

<div id="threadpool_remote_fs_reader_queue_size">
  ## threadpool_remote_fs_reader_queue_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10000" changeable_without_restart="No" />Le nombre maximal de jobs pouvant être planifiés dans le pool de threads pour la lecture à partir d’un système de fichiers distant.
