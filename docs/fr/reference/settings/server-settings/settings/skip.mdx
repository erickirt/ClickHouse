Ces paramètres configurent le serveur ClickHouse et sont générés automatiquement à partir du code source de ClickHouse.

<div id="skip_binary_checksum_checks">
  ## skip_binary_checksum_checks
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="No" />Ignore les vérifications d’intégrité de la somme de contrôle du binaire de ClickHouse

<div id="skip_check_for_incorrect_settings">
  ## skip_check_for_incorrect_settings
</div>

<SettingsInfoBlock type="Bool" default_value="0" changeable_without_restart="Non" />

S’il est défini sur true, la validité des paramètres du serveur ne sera pas vérifiée.

**Exemple**

```xml
<skip_check_for_incorrect_settings>1</skip_check_for_incorrect_settings>
```
