<div id="detach_not_byte_identical_parts">
  ## detach_not_byte_identical_parts
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Active ou désactive le détachement d&#39;une part de données sur une réplique après une fusion ou une
mutation, si elle n&#39;est pas identique octet pour octet aux parts de données des autres répliques. Si
ce paramètre est désactivé, la part de données est supprimée. Activez ce paramètre si vous souhaitez
analyser ces parts ultérieurement.

Ce paramètre s&#39;applique aux tables `MergeTree` avec
la [réplication des données](/fr/reference/engines/table-engines/mergetree-family/replacingmergetree) activée.

Valeurs possibles :

* `0` — Les parts sont supprimées.
* `1` — Les parts sont détachées.

<div id="detach_old_local_parts_when_cloning_replica">
  ## detach_old_local_parts_when_cloning_replica
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Ne détache pas les anciennes parts locales lors de la réparation d’une réplique perdue.

Valeurs possibles :

* `true`
* `false`
