<div id="fsync_after_insert">
  ## fsync_after_insert
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Effectue un fsync pour chaque part insérée. Réduit considérablement les performances des
insertions ; son utilisation avec des wide parts n’est pas recommandée.

<div id="fsync_part_directory">
  ## fsync_part_directory
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Effectue un fsync sur le répertoire de la part une fois toutes les opérations sur la part terminées (écritures, renommages, etc.).
