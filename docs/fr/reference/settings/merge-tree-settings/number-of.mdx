<div id="number_of_free_entries_in_pool_to_execute_mutation">
  ## number_of_free_entries_in_pool_to_execute_mutation
</div>

<SettingsInfoBlock type="UInt64" default_value="20" />

Lorsque le nombre d&#39;entrées libres dans le pool est inférieur à la valeur spécifiée, les mutations de parts ne sont pas exécutées. Cela permet de laisser des threads disponibles pour les fusions régulières et d&#39;éviter les erreurs &quot;Too many parts&quot;.

Valeurs possibles :

* Tout entier positif.

**Utilisation**

La valeur du paramètre `number_of_free_entries_in_pool_to_execute_mutation`
doit être inférieure à la valeur de [background&#95;pool&#95;size](/fr/reference/settings/server-settings/settings#background_pool_size)

* [background&#95;merges&#95;mutations&#95;concurrency&#95;ratio](/fr/reference/settings/server-settings/settings#background_merges_mutations_concurrency_ratio).
  Sinon, ClickHouse génère une exception.

<div id="number_of_free_entries_in_pool_to_execute_optimize_entire_partition">
  ## number_of_free_entries_in_pool_to_execute_optimize_entire_partition
</div>

<SettingsInfoBlock type="UInt64" default_value="25" />

Lorsque le nombre d&#39;entrées libres dans le pool est inférieur au nombre spécifié, ne pas
exécuter l&#39;optimisation de la partition entière en arrière-plan (cette tâche est générée
lorsque `min_age_to_force_merge_seconds` est défini et que
`min_age_to_force_merge_on_partition_only` est activé). Cela permet de laisser des threads libres
pour les fusions habituelles et d&#39;éviter « Too many parts ».

Valeurs possibles :

* Entier positif.

La valeur du paramètre `number_of_free_entries_in_pool_to_execute_optimize_entire_partition`
doit être inférieure à la valeur de
[background&#95;pool&#95;size](/fr/reference/settings/server-settings/settings#background_pool_size)

* [background&#95;merges&#95;mutations&#95;concurrency&#95;ratio](/fr/reference/settings/server-settings/settings#background_merges_mutations_concurrency_ratio).
  Sinon, ClickHouse lève une exception.

<div id="number_of_free_entries_in_pool_to_lower_max_size_of_merge">
  ## number_of_free_entries_in_pool_to_lower_max_size_of_merge
</div>

<SettingsInfoBlock type="UInt64" default_value="8" />

Lorsque le nombre d&#39;entrées libres dans le pool
(ou la file d&#39;attente répliquée) devient inférieur au nombre spécifié, commence à réduire la taille maximale de fusion à traiter
(ou à placer dans la file d&#39;attente).
Cela permet de traiter les petites fusions, sans saturer le pool avec des
fusions de longue durée.

Valeurs possibles :

* Tout entier positif.

<div id="number_of_mutations_to_delay">
  ## number_of_mutations_to_delay
</div>

<SettingsInfoBlock type="UInt64" default_value="500" />

Si la table comporte au moins
ce nombre de mutations non terminées, ralentissez artificiellement les mutations de la table.
Désactivé si la valeur est 0

<div id="number_of_mutations_to_throw">
  ## number_of_mutations_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="1000" />

Si la table comporte au moins ce nombre de mutations inachevées, l&#39;exception &#39;Too many mutations&#39; est générée.
Désactivé si défini sur 0

  ## number_of_partitions_to_consider_for_merge
