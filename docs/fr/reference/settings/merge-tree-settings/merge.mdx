<div id="merge_workload">
  ## merge_workload
</div>

Utilisé pour réguler l’utilisation et le partage des ressources entre les fusions et les autres charges de travail. La valeur spécifiée est utilisée comme valeur du paramètre `workload` pour les fusions en arrière-plan de cette table. Si elle n’est pas spécifiée (chaîne vide), le paramètre du serveur `merge_workload` est utilisé à la place.
