Lorsqu’il est activé, les INSERT créent de nouvelles parts avec des projections.
Sinon, celles-ci peuvent être créées explicitement avec [MATERIALIZE PROJECTION](/fr/reference/statements/alter/projection#materialize-projection)

<div id="materialize_projections_on_merge">
  ## materialize_projections_on_merge
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.6"},{"label": "0"},{"label": "Nouveau paramètre"}]}]} />

Lorsqu&#39;il est activé, une fusion reconstruit une projection absente de l&#39;ensemble de ses parts source (par exemple parce qu&#39;elles ont été
insérées avec `materialize_projections_on_insert = 0`), afin que la part fusionnée dispose de la projection.

Les fusions ne combinent toujours que des parts qui partagent le même ensemble de projections. Pour ajouter rétroactivement une projection à toutes les parts existantes,
utilisez un [MATERIALIZE PROJECTION](/fr/reference/statements/alter/projection#materialize-projection) explicite. Les projections
sont également créées lors des INSERT avec [materialize&#95;projections&#95;on&#95;insert](/fr/reference/settings/merge-tree-settings/materialize-projections#materialize_projections_on_insert).
