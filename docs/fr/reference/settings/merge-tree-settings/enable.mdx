<div id="enable_mixed_granularity_parts">
  ## enable_mixed_granularity_parts
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Active ou désactive l&#39;utilisation du paramètre `index_granularity_bytes` pour contrôler la taille des
granules. Avant la version 19.11, seul le paramètre
`index_granularity` permettait de limiter la taille des granules. Le paramètre
`index_granularity_bytes` améliore les performances de ClickHouse lors de la
lecture de données à partir de tables contenant de grandes lignes (de dizaines ou de centaines de mégaoctets).
Si vous avez des tables avec de grandes lignes, vous pouvez activer ce paramètre pour ces tables
afin d&#39;améliorer l&#39;efficacité des requêtes `SELECT`.
