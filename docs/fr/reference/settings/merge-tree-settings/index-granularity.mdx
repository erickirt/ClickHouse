<div id="index_granularity">
  ## index_granularity
</div>

<SettingsInfoBlock type="UInt64" default_value="8192" />

Nombre maximal de lignes de données entre les marques d’un index. Autrement dit, le nombre de lignes
correspondant à une valeur de clé primaire.

<div id="index_granularity_bytes">
  ## index_granularity_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="10485760" />

Taille maximale des granules de données, en octets.

Pour limiter la taille du granule uniquement en fonction du nombre de lignes, définissez cette valeur sur `0` (non recommandé).
