<div id="remote_fs_execute_merges_on_single_replica_time_threshold">
  ## remote_fs_execute_merges_on_single_replica_time_threshold
</div>

<SettingsInfoBlock type="Secondes" default_value="10800" />

Lorsque ce paramètre est supérieur à zéro, une seule réplique lance
immédiatement la fusion si la partie fusionnée se trouve sur un stockage partagé.

<Note>
  La réplication zero-copy n’est pas prête pour la production
  La réplication zero-copy est désactivée par défaut dans ClickHouse version 22.8 et
  ultérieures.

  Cette fonctionnalité n’est pas recommandée pour une utilisation en production.
</Note>

Valeurs possibles :

* Tout entier positif.
