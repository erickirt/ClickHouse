  ## add_minmax_index_for_block_number_column
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.5"},{"label": "0"},{"label": "Nouveau paramètre."}]}]} />

Lorsqu’il est activé, un index minmax de saut de données implicite est ajouté pour la colonne virtuelle persistante `_block_number`.
Nécessite `enable_block_number_column = 1` pour être effectif. L’index est construit uniquement lors des fusions,
et non lors des insertions : au moment de l’insertion, le numéro de bloc est provisoire et l’index porterait sur une constante.
