Disponible uniquement dans ClickHouse Cloud. Taille minimale non compressée, en octets, pour

<div id="min_bytes_for_wide_part">
  ## min_bytes_for_wide_part
</div>

<SettingsInfoBlock type="UInt64" default_value="10485760" />

Nombre minimal d’octets/lignes dans une part de données pouvant être stockée au format `Wide`. Vous pouvez définir l’un de ces paramètres, les deux, ou aucun.

<div id="min_bytes_to_rebalance_partition_over_jbod">
  ## min_bytes_to_rebalance_partition_over_jbod
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Définit la quantité minimale d’octets permettant d’activer l’équilibrage lors de la répartition de nouvelles grandes
parts sur les disques du volume [JBOD](https://en.wikipedia.org/wiki/Non-RAID_drive_architectures).

Valeurs possibles :

* Entier positif.
* `0` — L’équilibrage est désactivé.

**Utilisation**

La valeur du paramètre `min_bytes_to_rebalance_partition_over_jbod` ne doit
pas être inférieure à la valeur de
[max&#95;bytes&#95;to&#95;merge&#95;at&#95;max&#95;space&#95;in&#95;pool](/fr/reference/settings/merge-tree-settings/max-bytes#max_bytes_to_merge_at_max_space_in_pool)
/ 1024. Sinon, ClickHouse lève une exception.
