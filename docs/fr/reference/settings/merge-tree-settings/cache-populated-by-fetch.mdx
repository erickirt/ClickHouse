<div id="cache_populated_by_fetch">
  ## cache_populated_by_fetch
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<Note>
  Ce paramètre s’applique uniquement à ClickHouse Cloud.
</Note>

Lorsque `cache_populated_by_fetch` est désactivé (paramètre par défaut), les nouvelles parties de données
ne sont chargées dans le cache du système de fichiers que lorsqu’une requête nécessitant
ces parties est exécutée.

S’il est activé, `cache_populated_by_fetch` fera au contraire en sorte que tous les nœuds chargent
les nouvelles parties de données depuis le stockage dans leur cache du système de fichiers, sans qu’une requête
soit nécessaire pour déclencher cette action.
