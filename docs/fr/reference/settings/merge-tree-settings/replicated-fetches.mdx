<div id="replicated_fetches_min_part_level">
  ## replicated_fetches_min_part_level
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "0"},{"label": "Nouveau paramètre"}]}]} />

Niveau minimal des parts à récupérer depuis d&#39;autres répliques. Les parts dont le niveau est inférieur à ce seuil sont reportées
(conservées dans la file d&#39;attente de réplication et réévaluées à chaque cycle de planification, et non ignorées définitivement).
Utilisez 1 pour reporter la récupération des parts de niveau 0 (non fusionnées), afin de réduire le surcoût de réplication lors d&#39;une ingestion importante.
Par défaut : 0 (récupère toutes les parts, quel que soit leur niveau).
