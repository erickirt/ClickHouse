<div id="use_minimalistic_checksums_in_zookeeper">
  ## use_minimalistic_checksums_in_zookeeper
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Utilise un format réduit (quelques dizaines d’octets) pour les sommes de contrôle des parties de données dans ZooKeeper au lieu
du format habituel (quelques dizaines de Ko). Avant de l’activer, vérifiez que toutes les répliques prennent en charge
le nouveau format.

<div id="use_minimalistic_part_header_in_zookeeper">
  ## use_minimalistic_part_header_in_zookeeper
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Méthode de stockage des en-têtes des parties de données dans ZooKeeper. Si ce paramètre est activé, ZooKeeper
stocke moins de données. Pour plus de détails, voir [ici](/fr/reference/settings/server-settings/settings#use_minimalistic_part_header_in_zookeeper).
