<div id="max_suspicious_broken_parts">
  ## max_suspicious_broken_parts
</div>

<SettingsInfoBlock type="UInt64" default_value="100" />

Si le nombre de parts corrompues dans une partition donnée dépasse la valeur
`max_suspicious_broken_parts`, la suppression automatique est interdite.

Valeurs possibles :

* Tout entier positif.

<div id="max_suspicious_broken_parts_bytes">
  ## max_suspicious_broken_parts_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="1073741824" />

Taille maximale de l’ensemble des parts corrompues ; au-delà, la suppression automatique est refusée.

Valeurs possibles :

* Tout entier positif.
