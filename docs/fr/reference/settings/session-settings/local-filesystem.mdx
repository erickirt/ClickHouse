<div id="local_filesystem_read_method">
  ## local_filesystem_read_method
</div>

<SettingsInfoBlock type="String" default_value="pread_threadpool" />

Méthode de lecture des données depuis le système de fichiers local, parmi les suivantes : read, pread, mmap, io&#95;uring, pread&#95;threadpool.

La méthode &#39;io&#95;uring&#39; est expérimentale et ne fonctionne pas avec Log, TinyLog, StripeLog, File, Set et Join, ni avec les autres tables utilisant des fichiers auxquels on peut ajouter des données en présence de lectures et d&#39;écritures concurrentes.
Si vous lisez divers articles sur &#39;io&#95;uring&#39; sur Internet, ne vous y fiez pas aveuglément. Ce n&#39;est pas une meilleure méthode de lecture de fichiers, sauf dans le cas d&#39;un grand nombre de petites requêtes d&#39;IO, ce qui n&#39;est pas le cas dans ClickHouse. Il n&#39;y a aucune raison d&#39;activer &#39;io&#95;uring&#39;.

<div id="local_filesystem_read_prefetch">
  ## local_filesystem_read_prefetch
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Utilise la prélecture lors de la lecture de données depuis le système de fichiers local.
