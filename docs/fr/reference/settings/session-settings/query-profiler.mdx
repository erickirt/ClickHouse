<div id="query_profiler_cpu_time_period_ns">
  ## query_profiler_cpu_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000000" />

Définit la période d’un minuteur d’horloge CPU pour le [profiler de requêtes](/fr/concepts/features/performance/troubleshoot/sampling-query-profiler). Ce minuteur ne compte que le temps CPU.

Valeurs possibles :

* Un entier positif en nanosecondes.

  Valeurs recommandées :

  * 10000000 (100 fois par seconde) nanosecondes et plus pour des requêtes individuelles.
  * 1000000000 (une fois par seconde) pour le profilage à l’échelle du cluster.

* 0 pour désactiver le minuteur.

Voir aussi :

* table système [trace&#95;log](/fr/reference/system-tables/trace_log)

<div id="query_profiler_real_time_period_ns">
  ## query_profiler_real_time_period_ns
</div>

<SettingsInfoBlock type="UInt64" default_value="1000000000" />

Définit la période d&#39;un minuteur en temps réel pour le [profiler de requêtes](/fr/concepts/features/performance/troubleshoot/sampling-query-profiler). Le minuteur en temps réel mesure le temps réel (wall-clock time).

Valeurs possibles :

* Entier positif, en nanosecondes.

  Valeurs recommandées :

  * 10000000 (100 fois par seconde) nanosecondes ou moins pour des requêtes individuelles.
  * 1000000000 (une fois par seconde) pour le profilage à l&#39;échelle du cluster.

* 0 pour désactiver le minuteur.

Voir aussi :

* Table système [trace&#95;log](/fr/reference/system-tables/trace_log)

Valeur par défaut dans Cloud : `3000000000`.
