  ## materialize_skip_indexes_on_insert
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "24.6"},{"label": "1"},{"label": "Nouveau paramètre permettant de désactiver la matérialisation des skip indexes à l'insertion"}]}]} />

Indique si les opérations `INSERT` construisent et stockent les skip indexes. Si cette option est désactivée, les skip indexes seront uniquement construits et stockés [lors des fusions](/fr/reference/settings/merge-tree-settings#materialize_skip_indexes_on_merge) ou via un [MATERIALIZE INDEX](/fr/reference/statements/alter/skipping-index#materialize-index) explicite.
