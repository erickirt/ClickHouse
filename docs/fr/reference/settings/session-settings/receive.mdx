<div id="receive_data_timeout_ms">
  ## receive_data_timeout_ms
</div>

<SettingsInfoBlock type="Millisecondes" default_value="2000" />

Délai d’attente de connexion pour la réception du premier paquet de données ou d’un paquet indiquant une progression positive depuis une réplique

<div id="receive_timeout">
  ## receive_timeout
</div>

<SettingsInfoBlock type="Seconds" default_value="300" />

Délai d’attente de réception des données depuis le réseau, en secondes. Si aucun octet n’est reçu pendant cet intervalle, une exception est levée. Si vous définissez ce paramètre côté client, le &#39;send&#95;timeout&#39; du socket sera également défini sur l’extrémité correspondante de la connexion, côté serveur.
