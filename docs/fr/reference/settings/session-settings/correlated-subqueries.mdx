  ## correlated_subqueries_default_join_kind
</div>

<SettingsInfoBlock type="DecorrelationJoinKind" default_value="right" />

Contrôle le type de jointure utilisé dans le plan de requête décorrelé. La valeur par défaut est `right`, ce qui signifie que le plan décorrelé contiendra des RIGHT JOIN avec l&#39;entrée de la sous-requête sur le côté droit.

Valeurs possibles :

* `left` - Le processus de décorrélation produira des LEFT JOIN et la table d&#39;entrée apparaîtra sur le côté gauche.
* `right` - Le processus de décorrélation produira des RIGHT JOIN et la table d&#39;entrée apparaîtra sur le côté droit.
