  ## max_download_buffer_size
</div>

<SettingsInfoBlock type="UInt64" default_value="10485760" />

La taille maximale du tampon de téléchargement parallèle pour chaque thread (par exemple, pour le moteur URL).

<div id="max_download_threads">
  ## max_download_threads
</div>

<SettingsInfoBlock type="MaxThreads" default_value="4" />

Nombre maximal de threads utilisés pour télécharger des données (par ex. avec le moteur URL).
