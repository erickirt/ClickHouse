<SettingsInfoBlock type="OverflowMode" default_value="throw" />

Définit le comportement lorsque le volume de données lu dépasse l&#39;une des limites d&#39;un nœud terminal.

Options possibles :

* `throw` : lever une exception (par défaut).
* `break` : arrêter l&#39;exécution de la requête et renvoyer un résultat partiel.
