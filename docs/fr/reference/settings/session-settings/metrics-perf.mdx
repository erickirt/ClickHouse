  ## metrics_perf_events_list
</div>

Liste de métriques perf séparées par des virgules, qui seront mesurées pendant l’exécution des requêtes. Une valeur vide signifie tous les événements. Voir PerfEventInfo dans les sources pour la liste des événements disponibles.
