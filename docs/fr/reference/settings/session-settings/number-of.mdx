  ## number_of_mutations_to_delay
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Si la table mutée contient au moins ce nombre de mutations non terminées, les mutations de la table sont artificiellement ralenties. 0 : désactivé

  ## number_of_mutations_to_throw
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />
