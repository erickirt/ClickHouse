  ## use_statistics_for_part_pruning
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.4"},{"label": "1"},{"label": "Nouveau paramètre permettant d’utiliser les statistiques pour l’élagage des parts lors de l’exécution des requêtes."}]}]} />

Utilise les statistiques pour filtrer les parts lors de l’exécution des requêtes.

Lorsqu’il est activé, l’élagage dans les requêtes SELECT s’appuie sur les statistiques de colonnes (par ex. les statistiques MinMax) pour éliminer les parts qui ne peuvent pas contenir de données correspondantes avant de lire la moindre donnée.

Valeurs possibles :

* 0 — Désactivé.
* 1 — Activé.
