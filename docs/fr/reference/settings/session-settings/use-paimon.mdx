  ## use_paimon_metadata_files_cache
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "0"},{"label": "Nouveau paramètre permettant d’activer la mise en cache en mémoire des fichiers de métadonnées Paimon analysés (listes de manifestes et manifestes). Pour les moteurs de table Paimon persistants, il doit être activé avant l’initialisation des métadonnées ; les fonctions de table l’évaluent pour chaque requête. Cela évite les téléchargements répétés et la désérialisation des fichiers de métadonnées depuis le stockage objet lors des requêtes suivantes."}]}]} />

S’il est activé, la fonction de table paimon et le stockage paimon peuvent utiliser le cache des fichiers de métadonnées paimon.

Les fonctions de table Paimon évaluent ce paramètre pour chaque requête, tandis que les moteurs de table Paimon persistants le figent lors de l’initialisation des métadonnées : pour modifier cette décision pour une table déjà créée, celle-ci doit être supprimée puis recréée.

Valeurs possibles :

* 0 - Désactivé
* 1 - Activé
