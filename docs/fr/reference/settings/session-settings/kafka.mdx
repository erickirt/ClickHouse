<div id="kafka_disable_num_consumers_limit">
  ## kafka_disable_num_consumers_limit
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Désactive la limite de `kafka_num_consumers`, qui dépend du nombre de cœurs de CPU disponibles.

<div id="kafka_max_wait_ms">
  ## kafka_max_wait_ms
</div>

<SettingsInfoBlock type="Millisecondes" default_value="5000" />

Le délai d’attente, en millisecondes, pour lire les messages depuis [Kafka](/fr/reference/engines/table-engines/integrations/kafka) avant une nouvelle tentative.

Valeurs possibles :

* Entier positif.
* 0 — délai d’expiration infini.

Voir aussi :

* [Apache Kafka](https://kafka.apache.org/)
