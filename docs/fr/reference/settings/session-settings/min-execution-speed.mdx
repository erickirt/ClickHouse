Ces paramètres sont disponibles dans [system.settings](/fr/reference/system-tables/settings) et sont générés automatiquement à partir du [source](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp).

<div id="min_execution_speed">
  ## min_execution_speed
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Vitesse d&#39;exécution minimale en lignes par seconde. Vérifiée pour chaque bloc de données lorsque
[`timeout_before_checking_execution_speed`](/fr/reference/settings/session-settings/other#timeout_before_checking_execution_speed)
expire. Si la vitesse d&#39;exécution est inférieure, une exception est levée.

<div id="min_execution_speed_bytes">
  ## min_execution_speed_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Le nombre minimal d’octets exécutés par seconde. Vérifié sur chaque bloc de données lorsque
[`timeout_before_checking_execution_speed`](/fr/reference/settings/session-settings/other#timeout_before_checking_execution_speed)
expire. Si la vitesse d’exécution est inférieure, une exception est levée.
