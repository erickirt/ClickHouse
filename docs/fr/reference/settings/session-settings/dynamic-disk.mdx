  ## dynamic_disk_allow_from_env

Autorise l&#39;utilisation de substitutions `from_env` dans la configuration dynamique des disques (c.-à-d. dans les arguments de la fonction `disk()`).
Désactivé par défaut pour empêcher les utilisateurs de lire des variables d&#39;environnement arbitraires lors de la définition du stockage d&#39;une table.
