<div id="allow_suspicious_low_cardinality_types">
  ## allow_suspicious_low_cardinality_types
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

Autorise ou restreint l’utilisation de [LowCardinality](/fr/reference/data-types/lowcardinality) avec des types de données de taille fixe de 8 octets ou moins : les types de données numériques et `FixedString(8_bytes_or_less)`.

Pour les petites valeurs fixes, l’utilisation de `LowCardinality` est généralement inefficace, car ClickHouse stocke un index numérique pour chaque ligne. Par conséquent :

* L’espace disque utilisé peut augmenter.
* La consommation de RAM peut être plus élevée, selon la taille du dictionnaire.
* Certaines fonctions peuvent être plus lentes en raison d’opérations d’encodage supplémentaires.

Les temps de fusion dans les tables utilisant le moteur [MergeTree](/fr/reference/engines/table-engines/mergetree-family/mergetree) peuvent augmenter pour toutes les raisons décrites ci-dessus.

Valeurs possibles :

* 1 — L’utilisation de `LowCardinality` n’est pas restreinte.
* 0 — L’utilisation de `LowCardinality` est restreinte.
