<div id="use_page_cache_for_local_disks">
  ## use_page_cache_for_local_disks
</div>

<SettingsInfoBlock type="Bool" default_value="0" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.2"},{"label": "0"},{"label": "Nouveau paramètre pour utiliser le cache de pages en espace utilisateur pour les disques locaux"}]}]} />

Utilise le cache de pages en espace utilisateur lors de la lecture depuis des disques locaux. Utilisé pour les tests, il est peu probable qu&#39;il améliore les performances en pratique. Nécessite `local_filesystem_read_method = 'pread'` ou `'read'`. Cela ne désactive pas l&#39;OS page cache ; `min_bytes_to_use_direct_io` peut être utilisé à cette fin. N&#39;affecte que les tables ordinaires, pas la fonction de table `file()` ni le moteur de table `File()`.
