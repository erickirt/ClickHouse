<div id="partial_merge_join_left_table_buffer_bytes">
  ## partial_merge_join_left_table_buffer_bytes
</div>

<SettingsInfoBlock type="UInt64" default_value="0" />

Si la valeur n’est pas égale à 0, regroupe les blocs de la table de gauche en blocs plus volumineux pour le côté gauche d’une jointure par fusion partielle. Utilise jusqu’à 2x la mémoire spécifiée par thread de jointure.

<div id="partial_merge_join_rows_in_right_blocks">
  ## partial_merge_join_rows_in_right_blocks
</div>

<SettingsInfoBlock type="UInt64" default_value="65536" />

Limite la taille des blocs de données de jointure du côté droit dans l’algorithme de jointure par fusion partielle pour les requêtes [JOIN](/fr/reference/statements/select/join).

Serveur ClickHouse :

1. Divise les données de jointure du côté droit en blocs contenant jusqu’au nombre de lignes spécifié.
2. Crée un index pour chaque bloc à partir de ses valeurs minimale et maximale.
3. Décharge les blocs préparés sur le disque lorsque c’est possible.

Valeurs possibles :

* Tout entier positif. Plage de valeurs recommandée : [1000, 100000].
