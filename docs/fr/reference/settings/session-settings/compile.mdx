<div id="compile_aggregate_expressions">
  ## compile_aggregate_expressions
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

Active ou désactive la compilation JIT des fonctions d&#39;agrégation en code natif. L&#39;activation de ce réglage peut améliorer les performances.

Valeurs possibles :

* 0 — L&#39;agrégation est effectuée sans compilation JIT.
* 1 — L&#39;agrégation est effectuée à l&#39;aide de la compilation JIT.

<div id="compile_regular_expressions">
  ## compile_regular_expressions
</div>

<SettingsInfoBlock type="Bool" default_value="1" />

<VersionHistory rows={[{"id": "row-1","items": [{"label": "26.7"},{"label": "1"},{"label": "Nouveau paramètre permettant d’activer la compilation JIT des expressions régulières simples dans des fonctions comme `match` et `extract`."}]}]} />

Compiler les expressions régulières simples utilisées dans des fonctions comme `match` et `extract` en code natif. Les motifs ne faisant pas partie du sous-ensemble pris en charge basculent automatiquement vers le moteur général.
