Ces paramètres sont disponibles dans [system.settings](/fr/reference/system-tables/settings) et sont générés automatiquement à partir du [source](https://github.com/ClickHouse/ClickHouse/blob/master/src/Core/Settings.cpp).

<div id="timeout_overflow_mode">
  ## timeout_overflow_mode
</div>

<SettingsInfoBlock type="OverflowMode" default_value="throw" />

Définit le comportement à adopter si la requête s&#39;exécute plus longtemps que `max_execution_time` ou si le
temps d&#39;exécution estimé dépasse `max_estimated_execution_time`.

Valeurs possibles :

* `throw` : lever une exception (par défaut).
* `break` : arrêter l&#39;exécution de la requête et renvoyer un résultat partiel, comme si les
  données source étaient épuisées.

<div id="timeout_overflow_mode_leaf">
  ## timeout_overflow_mode_leaf
</div>

<SettingsInfoBlock type="OverflowMode" default_value="throw" />

Définit ce qui se passe lorsque la requête exécutée sur le nœud feuille dépasse `max_execution_time_leaf`.

Valeurs possibles :

* `throw` : lever une exception (par défaut).
* `break` : arrêter l&#39;exécution de la requête et renvoyer un résultat partiel, comme si les
  données source étaient épuisées.
