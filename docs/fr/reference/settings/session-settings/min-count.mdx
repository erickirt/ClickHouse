<div id="min_count_to_compile_aggregate_expression">
  ## min_count_to_compile_aggregate_expression
</div>

<SettingsInfoBlock type="UInt64" default_value="3" />

Le nombre minimal d&#39;expressions d&#39;agrégation identiques à partir duquel la compilation JIT démarre. Fonctionne uniquement si le paramètre [compile&#95;aggregate&#95;expressions](/fr/reference/settings/session-settings/compile#compile_aggregate_expressions) est activé.

Valeurs possibles :

* Entier positif.
* 0 — Les expressions d&#39;agrégation identiques sont toujours compilées en JIT.
