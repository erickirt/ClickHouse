<div id="syntax">
  ## Syntaxe
</div>

```sql
SELECT ... FROM loop(database, table);
SELECT ... FROM loop(database.table);
SELECT ... FROM loop(table);
SELECT ... FROM loop(other_table_function(...));
```

<div id="arguments">
  ## Arguments
</div>

| Argument                    | Description                                                                                                                            |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| `database`                  | nom de la base de données.                                                                                                             |
| `table`                     | nom de la table.                                                                                                                       |
| `other_table_function(...)` | une autre fonction de table. Exemple : `SELECT * FROM loop(numbers(10));` ici, `other_table_function(...)` correspond à `numbers(10)`. |

<div id="returned_values">
  ## Valeurs renvoyées
</div>

Boucle infinie renvoyant le résultat de la requête.

<div id="examples">
  ## Exemples
</div>

Sélection de données dans ClickHouse :

```sql
SELECT * FROM loop(test_database, test_table);
SELECT * FROM loop(test_database.test_table);
SELECT * FROM loop(test_table);
```

Ou avec d’autres fonctions de table :

```sql
SELECT * FROM loop(numbers(3)) LIMIT 7;
```
