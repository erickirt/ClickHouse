<div id="quantilesBFloat16Weighted">
  ## quantilesBFloat16Weighted
</div>

Introduit dans : v21.10.0

Comme [`quantilesBFloat16`](/fr/reference/functions/aggregate-functions/quantilesBFloat16), mais tient compte du poids de chaque valeur.

Calcule simultanément plusieurs [quantiles](https://en.wikipedia.org/wiki/Quantile) approximatifs d&#39;un échantillon composé de nombres [bfloat16](https://en.wikipedia.org/wiki/Bfloat16_floating-point_format) à différents niveaux, en une seule passe.

**Syntaxe**

```sql
quantilesBFloat16Weighted(level1, level2, ...)(expr, weight)
```

**Paramètres**

* `level` — Niveaux de quantiles. Un ou plusieurs nombres constants à virgule flottante compris entre 0 et 1. Nous recommandons d&#39;utiliser des valeurs de `level` dans la plage `[0.01, 0.99]`. [`Float*`](/fr/reference/data-types/float)

**Arguments**

* `expr` — Expression appliquée aux valeurs de la colonne, renvoyant un type de données numérique, `Date` ou `DateTime`. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Date`](/fr/reference/data-types/date) ou [`DateTime`](/fr/reference/data-types/datetime)
* `weight` — Colonne contenant les poids des éléments de la séquence. Le poids correspond au nombre d&#39;occurrences d&#39;une valeur. [`UInt*`](/fr/reference/data-types/int-uint)

**Valeur renvoyée**

Tableau de quantiles approximatifs pour les niveaux spécifiés, dans le même ordre que celui dans lequel les niveaux ont été indiqués. Pour les entrées `Date` et `DateTime`, le format de sortie correspond au format d&#39;entrée. [`Array(Float64)`](/fr/reference/data-types/array) ou [`Array(Date)`](/fr/reference/data-types/array) ou [`Array(DateTime)`](/fr/reference/data-types/array)

**Exemples**

**Calcul de plusieurs quantiles bfloat16 pondérés**

```sql title=Query
SELECT quantilesBFloat16Weighted(0.25, 0.5, 0.75)(number, 1) FROM numbers(10)
```

```response title=Response
┌─quantilesBFloat16Weighted(0.25, 0.5, 0.75)(number, 1)─┐
│ [2,4,7]                                               │
└───────────────────────────────────────────────────────┘
```
