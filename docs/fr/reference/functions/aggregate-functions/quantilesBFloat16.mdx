<div id="quantilesBFloat16">
  ## quantilesBFloat16
</div>

Introduite dans : v21.7.0

Calcule simultanément plusieurs [quantiles](https://en.wikipedia.org/wiki/Quantile) approximatifs d’un échantillon composé de nombres [bfloat16](https://en.wikipedia.org/wiki/Bfloat16_floating-point_format), à différents niveaux.

Cette fonction est équivalente à [`quantileBFloat16`](/fr/reference/functions/aggregate-functions/quantileBFloat16), mais elle permet de calculer plusieurs niveaux de quantile en un seul passage, ce qui est plus efficace que d’appeler des fonctions de quantile séparées.

**Syntaxe**

```sql
quantilesBFloat16(level1, level2, ...)(expr)
```

**Paramètres**

* `level` — Niveaux de quantiles. Un ou plusieurs nombres constants à virgule flottante compris entre 0 et 1. Nous recommandons d&#39;utiliser des valeurs `level` dans la plage `[0.01, 0.99]`. [`Float*`](/fr/reference/data-types/float)

**Arguments**

* `expr` — Expression appliquée aux valeurs de la colonne, produisant des types de données numériques, `Date` ou `DateTime`. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Date`](/fr/reference/data-types/date) ou [`DateTime`](/fr/reference/data-types/datetime)

**Valeur de retour**

Tableau de quantiles approximatifs pour les niveaux spécifiés, dans le même ordre que celui des niveaux indiqués. Pour les entrées `Date` et `DateTime`, le format de sortie correspond au format d&#39;entrée. [`Array(Float64)`](/fr/reference/data-types/array) ou [`Array(Date)`](/fr/reference/data-types/array) ou [`Array(DateTime)`](/fr/reference/data-types/array)

**Exemples**

**Calcul de plusieurs quantiles bfloat16**

```sql title=Query
SELECT quantilesBFloat16(0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesBFloat16(0.25, 0.5, 0.75)(number)─┐
│ [2,4,7]                                    │
└────────────────────────────────────────────┘
```
