<div id="quantilesExactLow">
  ## quantilesExactLow
</div>

Introduit dans : v20.8.0

Calcule exactement plusieurs [quantiles](https://en.wikipedia.org/wiki/Quantile) d’une séquence de données numériques à différents niveaux en une seule fois. Pour un nombre pair d’éléments au niveau `0.5`, elle renvoie la plus petite des deux valeurs centrales.

Cette fonction est équivalente à [`quantileExactLow`](/fr/reference/functions/aggregate-functions/quantileExactLow), mais permet de calculer plusieurs niveaux de quantile en un seul passage, ce qui est plus efficace que d’appeler des fonctions de quantile séparées.

**Syntaxe**

```sql
quantilesExactLow(level1, level2, ...)(expr)
```

**Paramètres**

* `level` — Niveaux des quantiles. Un ou plusieurs nombres constants à virgule flottante de 0 à 1. Nous recommandons d&#39;utiliser des valeurs de `level` dans la plage `[0.01, 0.99]`. [`Float*`](/fr/reference/data-types/float)

**Arguments**

* `expr` — Expression sur les valeurs de la colonne produisant des types de données numériques, `Date`, `DateTime` ou `DateTime64`. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal*`](/fr/reference/data-types/decimal) ou [`Date`](/fr/reference/data-types/date) ou [`DateTime`](/fr/reference/data-types/datetime) ou [`DateTime64`](/fr/reference/data-types/datetime64)

**Valeur renvoyée**

Tableau de quantiles des niveaux spécifiés, dans le même ordre que celui des niveaux indiqués. Pour les types de données numériques, le format de sortie correspond au format d&#39;entrée. [`Array((U)Int*)`](/fr/reference/data-types/array) ou [`Array(Float*)`](/fr/reference/data-types/array) ou [`Array(Decimal*)`](/fr/reference/data-types/array) ou [`Array(Date)`](/fr/reference/data-types/array) ou [`Array(DateTime)`](/fr/reference/data-types/array) ou [`Array(DateTime64)`](/fr/reference/data-types/array)

**Exemples**

**Calcul de plusieurs quantiles exacts inférieurs**

```sql title=Query
SELECT quantilesExactLow(0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesExactLow(0.25, 0.5, 0.75)(number)─┐
│ [2,4,7]                                    │
└────────────────────────────────────────────┘
```
