<div id="quantilesExactHigh">
  ## quantilesExactHigh
</div>

Introduit dans : v20.8.0

Calcule exactement plusieurs [quantiles](https://en.wikipedia.org/wiki/Quantile) d’une séquence de données numériques à différents niveaux simultanément. Pour un nombre pair d’éléments au niveau `0.5`, elle renvoie la plus élevée des deux valeurs centrales.

Cette fonction est équivalente à [`quantileExactHigh`](/fr/reference/functions/aggregate-functions/quantileExactHigh), mais permet de calculer plusieurs niveaux de quantile en un seul passage, ce qui est plus efficace que d’appeler des fonctions de quantile individuelles.

**Syntaxe**

```sql
quantilesExactHigh(level1, level2, ...)(expr)
```

**Paramètres**

* `level` — Niveaux de quantile. Un ou plusieurs nombres constants à virgule flottante compris entre 0 et 1. Nous recommandons d&#39;utiliser des valeurs de `level` dans la plage `[0.01, 0.99]`. [`Float*`](/fr/reference/data-types/float)

**Arguments**

* `expr` — Expression sur les valeurs de la colonne, renvoyant des types de données numériques, `Date`, `DateTime` ou `DateTime64`. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal*`](/fr/reference/data-types/decimal) ou [`Date`](/fr/reference/data-types/date) ou [`DateTime`](/fr/reference/data-types/datetime) ou [`DateTime64`](/fr/reference/data-types/datetime64)

**Valeur renvoyée**

Tableau de quantiles des niveaux spécifiés, dans le même ordre que les niveaux indiqués. Pour les types de données numériques, le format de sortie correspond au format d&#39;entrée. [`Array((U)Int*)`](/fr/reference/data-types/array) ou [`Array(Float*)`](/fr/reference/data-types/array) ou [`Array(Decimal*)`](/fr/reference/data-types/array) ou [`Array(Date)`](/fr/reference/data-types/array) ou [`Array(DateTime)`](/fr/reference/data-types/array) ou [`Array(DateTime64)`](/fr/reference/data-types/array)

**Exemples**

**Calcul de plusieurs quantiles exacts de niveau élevé**

```sql title=Query
SELECT quantilesExactHigh(0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesExactHigh(0.25, 0.5, 0.75)(number)─┐
│ [2,5,7]                                     │
└─────────────────────────────────────────────┘
```
