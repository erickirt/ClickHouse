<div id="quantilesExact">
  ## quantilesExact
</div>

Introduit dans : v1.1.0

Calcule exactement plusieurs [quantiles](https://en.wikipedia.org/wiki/Quantile) d’une séquence de données numériques à différents niveaux, simultanément.

Cette fonction est équivalente à [`quantileExact`](/fr/reference/functions/aggregate-functions/quantileExact), mais elle permet de calculer plusieurs niveaux de quantile en un seul passage, ce qui est plus efficace que d’appeler séparément des fonctions de quantile.

Pour obtenir des valeurs exactes, toutes les valeurs transmises sont regroupées dans un tableau, qui est ensuite partiellement trié. Par conséquent, la fonction utilise une mémoire de `O(n)`, où `n` est le nombre de valeurs transmises.

**Syntaxe**

```sql
quantilesExact(level1, level2, ...)(expr)
```

**Paramètres**

* `level` — Niveaux de quantiles. Un ou plusieurs nombres à virgule flottante constants de 0 à 1. Nous recommandons d&#39;utiliser des valeurs `level` dans la plage `[0.01, 0.99]`. [`Float*`](/fr/reference/data-types/float)

**Arguments**

* `expr` — Expression appliquée aux valeurs de la colonne, produisant des types de données numériques, `Date`, `DateTime` ou `DateTime64`. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Int128`](/fr/reference/data-types/int-uint) ou [`UInt128`](/fr/reference/data-types/int-uint) ou [`Int256`](/fr/reference/data-types/int-uint) ou [`UInt256`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Decimal*`](/fr/reference/data-types/decimal) ou [`Date`](/fr/reference/data-types/date) ou [`DateTime`](/fr/reference/data-types/datetime) ou [`DateTime64`](/fr/reference/data-types/datetime64)

**Valeur renvoyée**

Tableau de quantiles pour les niveaux spécifiés, dans le même ordre que celui des niveaux indiqués. Pour les types de données numériques, le format de sortie correspond au format d&#39;entrée. [`Array((U)Int*)`](/fr/reference/data-types/array) ou [`Array(Int128)`](/fr/reference/data-types/array) ou [`Array(UInt128)`](/fr/reference/data-types/array) ou [`Array(Int256)`](/fr/reference/data-types/array) ou [`Array(UInt256)`](/fr/reference/data-types/array) ou [`Array(Float*)`](/fr/reference/data-types/array) ou [`Array(Decimal*)`](/fr/reference/data-types/array) ou [`Array(Date)`](/fr/reference/data-types/array) ou [`Array(DateTime)`](/fr/reference/data-types/array) ou [`Array(DateTime64)`](/fr/reference/data-types/array)

**Exemples**

**Calcul de plusieurs quantiles exacts**

```sql title=Query
SELECT quantilesExact(0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesExact(0.25, 0.5, 0.75)(number)─┐
│ [2,5,7]                                 │
└─────────────────────────────────────────┘
```
