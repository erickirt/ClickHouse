<div id="quantilesDeterministic">
  ## quantilesDeterministic
</div>

Introduit dans : v1.1.0

Calcule simultanément plusieurs [quantiles](https://en.wikipedia.org/wiki/Quantile) approximatifs d’une séquence de données numériques à différents niveaux, à l’aide d’un échantillonnage en réservoir déterministe.

Cette fonction est équivalente à [`quantileDeterministic`](/fr/reference/functions/aggregate-functions/quantileDeterministic), mais elle permet de calculer plusieurs niveaux de quantile en une seule passe, ce qui est plus efficace que d’appeler des fonctions de quantile individuellement.

**Syntaxe**

```sql
quantilesDeterministic(level1, level2, ...)(expr, determinator)
```

**Paramètres**

* `level` — Niveaux de quantiles. Un ou plusieurs nombres constants à virgule flottante compris entre 0 et 1. Nous recommandons d&#39;utiliser des valeurs `level` dans la plage `[0.01, 0.99]`. [`Float*`](/fr/reference/data-types/float)

**Arguments**

* `expr` — Expression sur les valeurs de la colonne produisant des types de données numériques, `Date` ou `DateTime`. [`(U)Int*`](/fr/reference/data-types/int-uint) ou [`Int128`](/fr/reference/data-types/int-uint) ou [`UInt128`](/fr/reference/data-types/int-uint) ou [`Int256`](/fr/reference/data-types/int-uint) ou [`UInt256`](/fr/reference/data-types/int-uint) ou [`Float*`](/fr/reference/data-types/float) ou [`Date`](/fr/reference/data-types/date) ou [`DateTime`](/fr/reference/data-types/datetime)
* `determinator` — Nombre dont le hachage est utilisé à la place d&#39;un générateur de nombres aléatoires dans l&#39;algorithme d&#39;échantillonnage en réservoir afin de rendre le résultat de l&#39;échantillonnage déterministe. [`UInt*`](/fr/reference/data-types/int-uint)

**Valeur de retour**

Tableau de quantiles approximatifs pour les niveaux spécifiés, dans le même ordre que celui où les niveaux ont été indiqués. [`Array(Float64)`](/fr/reference/data-types/array) ou [`Array(Date)`](/fr/reference/data-types/array) ou [`Array(DateTime)`](/fr/reference/data-types/array)

**Exemples**

**Calcul de plusieurs quantiles déterministes**

```sql title=Query
SELECT quantilesDeterministic(0.25, 0.5, 0.75)(number, number) FROM numbers(10)
```

```response title=Response
┌─quantilesDeterministic(0.25, 0.5, 0.75)(number, number)─┐
│ [2.25,4.5,6.75]                                         │
└─────────────────────────────────────────────────────────┘
```
