<div id="quantilesDD">
  ## quantilesDD
</div>

Introduit dans : v24.1.0

Calcule simultanément plusieurs [quantiles](https://en.wikipedia.org/wiki/Quantile) approximatifs d’un échantillon, à différents niveaux, avec des garanties d’erreur relative.

Cette fonction est équivalente à [`quantileDD`](/fr/reference/functions/aggregate-functions/quantileddsketch), mais elle permet de calculer plusieurs niveaux de quantile en un seul passage, ce qui est plus efficace que d’appeler séparément des fonctions de quantile individuelles.

**Syntaxe**

```sql
quantilesDD(relative_accuracy, level1, level2, ...)(expr)
```

**Paramètres**

* `relative_accuracy` — Précision relative des quantiles. Les valeurs possibles sont comprises entre 0 et 1. La valeur recommandée est de 0.001 ou plus. [`Float*`](/fr/reference/data-types/float)
* `level` — Niveaux des quantiles. Un ou plusieurs nombres à virgule flottante constants compris entre 0 et 1. Nous recommandons d&#39;utiliser des valeurs de `level` comprises dans la plage `[0.01, 0.99]`. [`Float*`](/fr/reference/data-types/float)

**Arguments**

* `expr` — Expression appliquée aux valeurs de la colonne, produisant des types de données numériques, `Date` ou `DateTime`. [`(U)Int*`](/fr/reference/data-types/int-uint) or [`Float*`](/fr/reference/data-types/float) or [`Date`](/fr/reference/data-types/date) or [`DateTime`](/fr/reference/data-types/datetime)

**Valeur renvoyée**

Tableau de quantiles approximatifs pour les niveaux spécifiés, dans le même ordre que celui dans lequel les niveaux ont été indiqués. Pour les entrées `Date` et `DateTime`, le format de sortie correspond au format d&#39;entrée. [`Array(Float64)`](/fr/reference/data-types/array) or [`Array(Date)`](/fr/reference/data-types/array) or [`Array(DateTime)`](/fr/reference/data-types/array)

**Exemples**

**Calcul de plusieurs quantiles avec DDSketch**

```sql title=Query
SELECT quantilesDD(0.01, 0.25, 0.5, 0.75)(number) FROM numbers(10)
```

```response title=Response
┌─quantilesDD(0.01, 0.25, 0.5, 0.75)(number)────────────────┐
│ [1.9936617014173448,4.0148353330285875,5.989510371172621] │
└───────────────────────────────────────────────────────────┘
```
