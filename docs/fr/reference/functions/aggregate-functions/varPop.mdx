## varPop {#varPop}

Introduced in: v1.1.0

Calculates the variance de population.

The variance de population is calculated using the formula:

$$
\frac{\Sigma{(x - \bar{x})^2}}{n}
$$

<br/>

Where:
- $x$ is each value in the population
- $\bar{x}$ is the population mean
- $n$ is the population size

<Note>
Cette fonction utilise un algorithme numériquement instable. Si vous avez besoin de [stabilité numérique](https://en.wikipedia.org/wiki/Numerical_stability) dans vos calculs, utilisez la fonction [`varPopStable`](/reference/functions/aggregate-functions/varPopStable). Elle est plus lente, mais produit une erreur de calcul plus faible.
</Note>
    

**Syntax**

```sql
varPop(x)
```

**Aliases**: `VAR_POP`

**Arguments**

- `x` — Population of values to find the variance de population of. [`(U)Int*`](/reference/data-types/int-uint) or [`Float*`](/reference/data-types/float) or [`Decimal*`](/reference/data-types/decimal)

**Returned value**

Returns the variance de population of `x`. [`Float64`](/reference/data-types/float)

**Examples**

**Computing variance de population**

```sql title=Query
DROP TABLE IF EXISTS test_data;
CREATE TABLE test_data
(
    x UInt8,
)
ENGINE = Memory;

INSERT INTO test_data VALUES (3), (3), (3), (4), (4), (5), (5), (7), (11), (15);

SELECT
    varPop(x) AS var_pop
FROM test_data;
```

```response title=Response
┌─var_pop─┐
│    14.4 │
└─────────┘
```
