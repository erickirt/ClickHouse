<div id="description">
  ## Description
</div>

Affiche tous les stemmers disponibles.
Ils peuvent être utilisés dans la fonction [stem](/fr/reference/functions/regular-functions/nlp-functions).

<Info>
  **Disponibilité**

  `system.stemmers` est présent uniquement dans les versions de ClickHouse compilées avec la dépendance `libstemmer` (`USE_LIBSTEMMER`). Dans les versions qui n’en disposent pas, la table n’existe pas et les requêtes qui l’interrogent échouent avec `UNKNOWN_TABLE`. Vous pouvez vérifier si cette option est activée dans votre build avec :

  ```sql
  SELECT value FROM system.build_options WHERE name = 'USE_LIBSTEMMER';
  ```
</Info>

Colonnes :

{/*AUTOGENERATED_START*/}

* `name` ([String](/fr/reference/data-types/index)) — Identifiant du stemmer Snowball (langue/algorithme)

{/*AUTOGENERATED_END*/}
