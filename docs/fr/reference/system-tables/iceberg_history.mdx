<div id="description">
  ## Description
</div>

Cette table système contient l&#39;historique des instantanés des tables Iceberg présentes dans ClickHouse. Elle sera vide si vous n&#39;avez pas de tables Iceberg dans ClickHouse.

<div id="columns">
  ## Colonnes
</div>

{/*AUTOGENERATED_START*/}

* `database` ([String](/fr/reference/data-types/index)) — Nom de la base de données.
* `table` ([String](/fr/reference/data-types/index)) — Nom de la table.
* `made_current_at` ([Nullable(DateTime64(3))](/fr/reference/data-types/index)) — Date et heure auxquelles cet instantané est devenu l’instantané courant.
* `snapshot_id` ([UInt64](/fr/reference/data-types/index)) — ID de l’instantané utilisé pour identifier un instantané.
* `parent_id` ([UInt64](/fr/reference/data-types/index)) — ID du parent de cet instantané.
* `is_current_ancestor` ([UInt8](/fr/reference/data-types/index)) — Indicateur indiquant si cet instantané est un ancêtre de l’instantané courant.
* `operation` ([Enum8(&#39;UNKNOWN&#39; = -1, &#39;APPEND&#39; = 0, &#39;OVERWRITE&#39; = 1, &#39;DELETE&#39; = 2, &#39;REPLACE&#39; = 3)](/fr/reference/data-types/index)) — Opération de l’instantané (APPEND, OVERWRITE, DELETE, REPLACE et UNKNOWN). Le statut UNKNOWN signifie soit que nous n’avons pas pu lire le résumé, soit qu’il est vide (il est facultatif pour v1). Le champ &#39;operation&#39; correct est obligatoire
* `summary` ([Map(String, String)](/fr/reference/data-types/index)) — Champs de résumé de l’instantané
