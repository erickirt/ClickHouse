  ### Gerar uma chave de acesso

  ### Criar uma chave HMAC e um segredo para uma conta de serviço

  Abra **Cloud Storage &gt; Settings &gt; Interoperability** e escolha uma **chave de acesso** existente ou **CREATE A KEY FOR A SERVICE ACCOUNT**. Este guia aborda o processo de criar uma nova chave para uma nova conta de serviço.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-a-service-account-key.webp" alt="Gerando uma chave HMAC de conta de serviço no GCS" />
    </Frame>
  </div>

  ### Adicionar uma nova conta de serviço

  Se este for um projeto sem nenhuma conta de serviço existente, clique em **CREATE NEW ACCOUNT**.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-0.webp" alt="Adicionando uma nova conta de serviço no GCS" />
    </Frame>
  </div>

  Há três etapas para criar a conta de serviço. Na primeira, dê à conta um nome, ID e descrição significativos.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-a.webp" alt="Definindo um novo nome e ID de conta de serviço no GCS" />
    </Frame>
  </div>

  Na caixa de diálogo de configurações de Interoperability, recomenda-se a IAM role **Storage Object Admin**; selecione essa função na segunda etapa.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-2.webp" alt="Selecionando a IAM role Storage Object Admin no GCS" />
    </Frame>
  </div>

  A terceira etapa é opcional e não é usada neste guia. Você pode permitir que usuários tenham esses privilégios com base nas suas políticas.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-3.webp" alt="Configurando opções adicionais para a nova conta de serviço no GCS" />
    </Frame>
  </div>

  A chave HMAC da conta de serviço será exibida. Salve essas informações, pois elas serão usadas na configuração do ClickHouse.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-guide-key.webp" alt="Recuperando a chave HMAC gerada para o GCS" />
    </Frame>
  </div>
