  ### Generar una clave de acceso

  ### Crear una clave HMAC y un secreto para una cuenta de servicio

  Abra **Cloud Storage &gt; Settings &gt; Interoperability** y elija una **Access key** existente o **CREATE A KEY FOR A SERVICE ACCOUNT**. Esta guía explica cómo crear una nueva clave para una nueva cuenta de servicio.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-a-service-account-key.webp" alt="Generación de una clave HMAC de cuenta de servicio en GCS" />
    </Frame>
  </div>

  ### Agregar una nueva cuenta de servicio

  Si se trata de un proyecto sin ninguna cuenta de servicio existente, haga clic en **CREATE NEW ACCOUNT**.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-0.webp" alt="Adición de una nueva cuenta de servicio en GCS" />
    </Frame>
  </div>

  Hay tres pasos para crear la cuenta de servicio; en el primero, asigne a la cuenta un nombre, un ID y una descripción significativos.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-a.webp" alt="Definición de un nuevo nombre e ID de cuenta de servicio en GCS" />
    </Frame>
  </div>

  En el cuadro de diálogo de configuración de Interoperability, se recomienda el rol de IAM **Storage Object Admin**; seleccione ese rol en el segundo paso.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-2.webp" alt="Selección del rol de IAM Storage Object Admin en GCS" />
    </Frame>
  </div>

  El tercer paso es opcional y no se utiliza en esta guía. Puede permitir que los usuarios tengan estos privilegios según sus políticas.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-3.webp" alt="Configuración de ajustes adicionales para la nueva cuenta de servicio en GCS" />
    </Frame>
  </div>

  Se mostrará la clave HMAC de la cuenta de servicio. Guarde esta información, ya que se utilizará en la configuración de ClickHouse.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-guide-key.webp" alt="Recuperación de la clave HMAC generada para GCS" />
    </Frame>
  </div>
