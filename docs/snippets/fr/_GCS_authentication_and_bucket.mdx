  ### Générer une clé d’accès

  ### Créer une clé HMAC et un secret pour un compte de service

  Ouvrez **Cloud Storage &gt; Settings &gt; Interoperability** et choisissez soit une **Access key** existante, soit **CREATE A KEY FOR A SERVICE ACCOUNT**. Ce guide décrit la procédure de création d’une nouvelle clé pour un nouveau compte de service.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-a-service-account-key.webp" alt="Génération d’une clé HMAC de compte de service dans GCS" />
    </Frame>
  </div>

  ### Ajouter un nouveau compte de service

  S’il s’agit d’un projet sans compte de service existant, cliquez sur **CREATE NEW ACCOUNT**.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-0.webp" alt="Ajout d’un nouveau compte de service dans GCS" />
    </Frame>
  </div>

  La création du compte de service comporte trois étapes ; lors de la première, attribuez au compte un nom, un ID et une description explicites.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-a.webp" alt="Définition du nom et de l’ID d’un nouveau compte de service dans GCS" />
    </Frame>
  </div>

  Dans la boîte de dialogue des paramètres Interoperability, le rôle IAM **Storage Object Admin** est recommandé ; sélectionnez-le à la deuxième étape.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-2.webp" alt="Sélection du rôle IAM Storage Object Admin dans GCS" />
    </Frame>
  </div>

  La troisième étape est facultative et n’est pas utilisée dans ce guide. Vous pouvez accorder ces privilèges à des utilisateurs selon vos politiques.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-create-service-account-3.webp" alt="Configuration de paramètres supplémentaires pour le nouveau compte de service dans GCS" />
    </Frame>
  </div>

  La clé HMAC du compte de service s’affichera. Enregistrez ces informations, car elles seront utilisées dans la configuration de ClickHouse.

  <div className="ch-image-md">
    <Frame>
      <img src="/images/integrations/data-ingestion/s3/GCS-guide-key.webp" alt="Récupération de la clé HMAC générée pour GCS" />
    </Frame>
  </div>
